NAME function::print_ubacktrace_brief - Print stack back trace for current task. EXPERIMENTAL!

SYNOPSIS print_ubacktrace_brief()

ARGUMENTS None

DESCRIPTION Equivalent to print_ubacktrace, but output for each symbol is shorter (just name and offset), and the function address is printed if it can´t be mapped to a name.
