NAME struct_hrtimer_sleeper - simple sleeper structure

SYNOPSIS struct hrtimer_sleeper { struct hrtimer timer; struct task_struct * task; };

MEMBERS timer embedded timer structure

task task to wake up

DESCRIPTION task is set to NULL, when the timer expires.
