NAME auparse_get_type - get records type

SYNOPSIS #include <auparse.h>

int auparse_get_type(auparse_state_t *au);

DESCRIPTION auparse_get_type will return the integer value for the current record of the current event.

RETURN VALUE Returns 0 if an error occurs; otherwise, the records type.

SEE ALSO auparse_next_record(3).
