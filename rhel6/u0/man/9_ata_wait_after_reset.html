NAME ata_wait_after_reset - wait for link to become ready after reset

SYNOPSIS int ata_wait_after_reset(struct ata_link * link, unsigned long deadline, int (*check_ready) (struct ata_link *link));

ARGUMENTS link link to be waited on

deadline deadline jiffies for the operation

check_ready callback to check link readiness

DESCRIPTION Wait for link to become ready after reset.

LOCKING EH context.

RETURNS 0 if linke is ready before deadline; otherwise, -errno.
