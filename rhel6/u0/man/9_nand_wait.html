NAND_WAIT(9) Internal Functions Provided NAND_WAIT(9)

NAME nand_wait - [DEFAULT] wait until the command is done

SYNOPSIS int nand_wait(struct mtd_info * mtd, struct nand_chip * chip);

ARGUMENTS mtd MTD device structure

chip NAND chip structure

DESCRIPTION Wait for command done. This applies to erase and program only Erase can take up to 400ms and program up to 20ms according to general NAND and SmartMedia specs
