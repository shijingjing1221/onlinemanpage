NAME perf-buildid-cache - Manage build-id cache.

SYNOPSIS perf buildid-list <options>

DESCRIPTION This command manages the build-id cache. It can add and remove files to the cache. In the future it should as well purge older entries, set upper limits for the space used by the cache, etc.

OPTIONS -a, --add= Add specified file to the cache.

-r, --remove= Remove specified file to the cache.

-v, --verbose Be more verbose.

SEE ALSO perf-record(1), perf-report(1)
