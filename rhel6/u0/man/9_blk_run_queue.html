BLK_RUN_QUEUE(9) Block Devices BLK_RUN_QUEUE(9)

NAME blk_run_queue - run a single device queue

SYNOPSIS void blk_run_queue(struct request_queue * q);

ARGUMENTS q The queue to run

DESCRIPTION Invoke request handling on this queue, if it has pending work to do. May be used to restart queueing when a request has completed.
