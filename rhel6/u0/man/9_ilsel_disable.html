NAME ilsel_disable - Disable an ILSEL set

SYNOPSIS void ilsel_disable(unsigned int irq);

ARGUMENTS irq Bit position for ILSEL set value (retval from enable routines)

DESCRIPTION Disable a previously enabled ILSEL set.
