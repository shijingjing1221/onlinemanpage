NAME rio_local_read_config_16 - Read 16 bits from local configuration space

SYNOPSIS int rio_local_read_config_16(struct rio_mport * port, u32 offset, u16 * data);

ARGUMENTS port Master port

offset Offset into local configuration space

data Pointer to read data into

DESCRIPTION Reads 16 bits of data from the specified offset within the local device´s configuration space.
