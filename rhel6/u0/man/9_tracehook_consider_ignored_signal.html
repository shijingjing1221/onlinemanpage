NAME tracehook_consider_ignored_signal - suppress short-circuit of ignored signal

SYNOPSIS int tracehook_consider_ignored_signal(struct task_struct * task, int sig);

ARGUMENTS task task receiving the signal

sig signal number being sent

DESCRIPTION Return zero iff tracing doesn´t care to examine this ignored signal, so it can short-circuit normal delivery and never even get queued.

Called with task->sighand->siglock held.
