NAME function::user_string_n - Retrieves string of given length from user space.

SYNOPSIS user_string_n:string(addr:long,n:long)

ARGUMENTS addr The user space address to retrieve the string from.

n The maximum length of the string (if not null terminated).

DESCRIPTION Returns the C string of a maximum given length from a given user space address. Returns <unknown> on the rare cases when userspace data is not accessible at the given address.
