NAME lss16toppm Convert an LSS-16 image to PPM

SYNOPSIS lss16toppm [-map] [< file.lss] [> file.ppm]

DESCRIPTION This manual page documents briefly the lss16toppm command.

The lss16toppm utility converts an LSS-16 image to a PPM image.

OPTIONS A summary of options is included below.

-map Output the color map to standard error.

SEE ALSO ppmtolss16(1)
