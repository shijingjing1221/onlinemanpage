NAME FcFreeTypeQuery - compute pattern from font file (and index)

SYNOPSIS #include <fontconfig.h> #include <fcfreetype.h>

FcPattern * FcFreeTypeQuery(const FcChar8 *file); (int id); (FcBlanks *blanks); (int *count); .fi

DESCRIPTION Constructs a pattern representing the idth font in file. The number of fonts in file is returned in count.

VERSION Fontconfig version 2.8.0
