IPC_INIT_IDS(9) Kernel IPC facilities IPC_INIT_IDS(9)

NAME ipc_init_ids - initialise IPC identifiers

SYNOPSIS void ipc_init_ids(struct ipc_ids * ids);

ARGUMENTS ids Identifier set

DESCRIPTION Set up the sequence range to use for the ipc identifier range (limited below IPCMNI) then initialise the ids idr.
