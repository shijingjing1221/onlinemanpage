NAME asn1_length_der - API function

SYNOPSIS #include <libtasn1.h>

void asn1_length_der(unsigned long int len, unsigned char * ans, int * ans_len);

ARGUMENTS unsigned long int len value to convert.

unsigned char * ans string returned.

int * ans_len number of meaningful bytes of ANS (ans[0]..ans[ans_len-1]).

DESCRIPTION Creates the DER coding for the LEN parameter (only the length). The ans buffer is pre-allocated and must have room for the output.
