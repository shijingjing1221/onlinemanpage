I2O_PCI_IRQ_DISABLE(9) Message-based devices I2O_PCI_IRQ_DISABLE(9)

NAME i2o_pci_irq_disable - Free interrupt for I2O controller

SYNOPSIS void i2o_pci_irq_disable(struct i2o_controller * c);

ARGUMENTS c I2O controller

DESCRIPTION Disable interrupts in I2O controller and then free interrupt.
