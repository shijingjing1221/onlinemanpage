NAME gnutls_protocol_get_version - Returns the version of the currently used protocol

SYNOPSIS #include <gnutls/gnutls.h>

gnutls_protocol_t gnutls_protocol_get_version(gnutls_session_t ses- sion);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

DESCRIPTION Get TLS version, a gnutls_protocol_t value.

RETURNS the version of the currently used protocol.
