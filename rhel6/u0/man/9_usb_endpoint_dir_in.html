NAME usb_endpoint_dir_in - check if the endpoint has IN direction

SYNOPSIS int usb_endpoint_dir_in(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns true if the endpoint is of type IN, otherwise it returns false.
