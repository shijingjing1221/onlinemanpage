NAME atomic_inc_and_test - increment and test

SYNOPSIS int atomic_inc_and_test(atomic_t * v);

ARGUMENTS v pointer of type atomic_t

DESCRIPTION Atomically increments v by 1 and returns true if the result is zero, or false for all other cases.
