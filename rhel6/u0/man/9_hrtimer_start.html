NAME hrtimer_start - (re)start an hrtimer on the current CPU

SYNOPSIS int hrtimer_start(struct hrtimer * timer, ktime_t tim, const enum hrtimer_mode mode);

ARGUMENTS timer the timer to be added

tim expiry time

mode expiry mode: absolute (HRTIMER_ABS) or relative (HRTIMER_REL)

RETURNS 0 on success 1 when the timer was active
