NAME stringprep_utf8_to_unichar - API function

SYNOPSIS #include <stringprep.h>

uint32_t stringprep_utf8_to_unichar(const char * p);

ARGUMENTS const char * p a pointer to Unicode character encoded as UTF-8

DESCRIPTION Converts a sequence of bytes encoded as UTF-8 to a Unicode character. If p does not point to a valid UTF-8 encoded character, results are undefined.

RETURN VALUE the resulting character.
