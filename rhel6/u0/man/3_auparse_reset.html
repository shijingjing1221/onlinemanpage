NAME auparse_reset - reset audit parser instance

SYNOPSIS #include <auparse.h>

int auparse_reset(auparse_state_t *au);

DESCRIPTION auparse_reset resets all internal cursors to the beginning. It closes files and descriptors.

RETURN VALUE Returns -1 if an error occurs; otherwise, 0 for success.

SEE ALSO auparse_init(3), auparse_destroy(3).
