NAME gnutls_x509_crl_get_crt_count - get number of revoked certificates in a CRL

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crl_get_crt_count(gnutls_x509_crl_t crl);

ARGUMENTS gnutls_x509_crl_t crl should contain a gnutls_x509_crl_t structure

DESCRIPTION This function will return the number of revoked certificates in the given CRL.

RETURNS number of certificates, a negative value on failure.
