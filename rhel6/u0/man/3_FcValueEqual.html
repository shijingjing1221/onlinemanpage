NAME FcValueEqual - Test two values for equality

SYNOPSIS #include <fontconfig.h>

FcBool FcValueEqual(FcValue v_a); (FcValue v_b); .fi

DESCRIPTION Compares two values. Integers and Doubles are compared as numbers; oth- erwise the two values have to be the same type to be considered equal. Strings are compared ignoring case.

VERSION Fontconfig version 2.8.0
