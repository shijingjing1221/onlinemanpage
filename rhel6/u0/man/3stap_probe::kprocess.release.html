NAME probe::kprocess.release - Process released

SYNOPSIS kprocess.release

VALUES pid PID of the process being released

task A task handle to the process being released

CONTEXT The context of the parent, if it wanted notification of this process´ termination, else the context of the process itself.

DESCRIPTION Fires when a process is released from the kernel. This always follows a kprocess.exit, though it may be delayed somewhat if the process waits in a zombie state.
