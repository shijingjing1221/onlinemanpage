VMTRUNCATE(9) Memory Management in Linux VMTRUNCATE(9)

NAME vmtruncate - unmap mappings freed by truncate syscall

SYNOPSIS int vmtruncate(struct inode * inode, loff_t offset);

ARGUMENTS inode inode of the file used

offset file offset to start truncating

DESCRIPTION NOTE! We have to be ready to update the memory sharing between the file and the memory map for a potential last incomplete page. Ugly, but necessary.
