XtAppSetExitFlag(3) XT FUNCTIONS XtAppSetExitFlag(3)

NAME XtAppSetExitFlag, XtAppGetExitFlag - thread support functions

SYNTAX void XtAppSetExitFlag(XtAppContext app_context);

Boolean XtAppGetExitFlag(XtAppContext app_context);

ARGUMENTS app_context Specifies the application context.

DESCRIPTION To indicate that an application context should exit, use XtAppSetExit- Flag.

To test the exit status of an application context, use XtAppGetExit- Flag.
