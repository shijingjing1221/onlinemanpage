NAME enum_set_key_cmd - key command

SYNOPSIS enum set_key_cmd { SET_KEY, DISABLE_KEY };

CONSTANTS SET_KEY a key is set

DISABLE_KEY a key must be disabled

DESCRIPTION Used with the set_key callback in struct ieee80211_ops, this indicates whether a key is being removed or added.
