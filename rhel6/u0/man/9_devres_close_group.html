DEVRES_CLOSE_GROUP(9) Driver Basics DEVRES_CLOSE_GROUP(9)

NAME devres_close_group - Close a devres group

SYNOPSIS void devres_close_group(struct device * dev, void * id);

ARGUMENTS dev Device to close devres group for

id ID of target group, can be NULL

DESCRIPTION Close the group identified by id. If id is NULL, the latest open group is selected.
