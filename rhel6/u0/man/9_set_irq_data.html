NAME set_irq_data - set irq type data for an irq

SYNOPSIS int set_irq_data(unsigned int irq, void * data);

ARGUMENTS irq Interrupt number

data Pointer to interrupt specific data

DESCRIPTION Set the hardware irq controller data for an irq
