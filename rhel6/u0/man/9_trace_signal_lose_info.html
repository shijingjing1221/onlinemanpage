NAME trace_signal_lose_info - called when siginfo is lost

SYNOPSIS void trace_signal_lose_info(int sig, int group, struct siginfo * info);

ARGUMENTS sig signal number

group signal to process group or not (bool)

info pointer to struct siginfo

DESCRIPTION Kernel generates ´sig´ signal but loses ´info´ siginfo, because siginfo queue is overflow. ´group´ is not 0 if the signal will be sent to a process group. ´sig´ is always one of non-RT signals.
