NAME hpiinv - An openhpi sample application that shows resources inventory records.

SYNOPSIS hpiinv [-x] [-a asset_tag]

DESCRIPTION hpiinv walks the RPT (Resource Present Table) looking for resources that have Inventory Capability. It displays all inventory records found.

OPTIONS -a Sets the asset tag -x Display debug messages -z Display extra debug messages

ENVIRONMENT VARIABLES For generic OpenHPI, there is *no* additional environment variable required. For some devices; sahalee, mBMC; BMCONLY flag needs to be set for additional device characterization based on rdr.IdString.Data.
