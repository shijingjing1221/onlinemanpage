NAME struct_regulator_consumer_supply - supply -> device mapping

SYNOPSIS struct regulator_consumer_supply { struct device * dev; const char * dev_name; const char * supply; };

MEMBERS dev Device structure for the consumer.

dev_name Result of dev_name for the consumer.

supply Name for the supply.

DESCRIPTION This maps a supply name to a device. Only one of dev or dev_name can be specified. Use of dev_name allows support for buses which make struct device available late such as I2C and is the preferred form.
