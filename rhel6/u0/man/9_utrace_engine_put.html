NAME utrace_engine_put - release a reference on a struct utrace_engine

SYNOPSIS void utrace_engine_put(struct utrace_engine * engine);

ARGUMENTS engine

struct utrace_engine pointer

DESCRIPTION You must hold a reference on engine, and you lose that reference. If it was the last one, engine becomes an invalid pointer.
