NAME selinux_policy_root - return the path of the SELinux policy files for this machine.

SYNOPSIS #include <selinux/selinux.h>

char *selinux_policy_root();

DESCRIPTION selinux_policy_root Reads the contents of the /etc/selinux/config file to determine which policy files should be used for this machine.

RETURN VALUE On success, returns a directory path containing the SELinux policy files. On failure, NULL is returned.
