NAME infokey - compile customizations for Info

SYNOPSIS infokey [OPTION]... [INPUT-FILE]

DESCRIPTION Compile infokey source file to infokey file. Reads INPUT-FILE (default $HOME/.infokey) and writes compiled key file to (by default) $HOME/.info.

OPTIONS --output FILE output to FILE instead of $HOME/.info

--help display this help and exit.

--version display version information and exit.
