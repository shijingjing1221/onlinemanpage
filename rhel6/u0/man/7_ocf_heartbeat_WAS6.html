NAME ocf_heartbeat_WAS6 - Manages a WebSphere Application Server 6 instance

SYNOPSIS WAS6 [start | stop | status | monitor | validate-all | meta-data | methods]

DESCRIPTION Resource script for WAS6. It manages a Websphere Application Server (WAS6) as an HA resource.

SUPPORTED PARAMETERS profile The WAS profile name. (optional, string, default default)

SUPPORTED ACTIONS This resource agent supports the following actions (operations):

start Starts the resource. Suggested minimum timeout: 300.

stop Stops the resource. Suggested minimum timeout: 300.

status Performs a status check. Suggested minimum timeout: 30. Suggested interval: 10.

monitor Performs a detailed status check. Suggested minimum timeout: 30. Suggested interval: 10.

validate-all Performs a validation of the resource configuration. Suggested minimum timeout: 5.

meta-data Retrieves resource agent metadata (internal use only). Suggested minimum timeout: 5.

methods Suggested minimum timeout: 5.

EXAMPLE The following is an example configuration for a WAS6 resource using the crm(8) shell:

primitive example_WAS6 ocf:heartbeat:WAS6 op monitor depth="0" timeout="30" interval="10"
