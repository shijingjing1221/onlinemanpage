FUNCTION::BACKTRACE(3stap) Context Functions FUNCTION::BACKTRACE(3stap)

NAME function::backtrace - Hex backtrace of current stack

SYNOPSIS backtrace:string()

ARGUMENTS None

DESCRIPTION Return a string of hex addresses that are a backtrace of the stack. Output may be truncated as per maximum string length.
