NAME rio_device_probe - Tell if a RIO device structure has a matching RIO device id structure

SYNOPSIS int rio_device_probe(struct device * dev);

ARGUMENTS dev the RIO device structure to match against

DESCRIPTION return 0 and set rio_dev->driver when drv claims rio_dev, else error
