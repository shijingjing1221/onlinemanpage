NAME ata_host_suspend - suspend host

SYNOPSIS int ata_host_suspend(struct ata_host * host, pm_message_t mesg);

ARGUMENTS host host to suspend

mesg PM message

DESCRIPTION Suspend host. Actual operation is performed by EH. This function requests EH to perform PM operations and waits for EH to finish.

LOCKING Kernel thread context (may sleep).

RETURNS 0 on success, -errno on failure.
