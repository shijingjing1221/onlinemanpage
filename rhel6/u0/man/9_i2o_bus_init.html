I2O_BUS_INIT(9) Message-based devices I2O_BUS_INIT(9)

NAME i2o_bus_init - Bus Adapter OSM initialization function

SYNOPSIS int i2o_bus_init(void);

ARGUMENTS void no arguments

DESCRIPTION Only register the Bus Adapter OSM in the I2O core.

Returns 0 on success or negative error code on failure.
