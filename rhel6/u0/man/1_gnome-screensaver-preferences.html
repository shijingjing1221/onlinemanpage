NAME gnome-screensaver-preferences - configure GNOME Screensaver

SYNOPSIS gnome-screensaver-preferences [OPTIONS...]

DESCRIPTION Manages preferences for the GNOME screen saver.

OPTIONS This program only accepts the standard GTK options.
