NAME filp_open - open file and return file pointer

SYNOPSIS struct file * filp_open(const char * filename, int flags, int mode);

ARGUMENTS filename path to open

flags open flags as per the open(2) second argument

mode mode for the new file if O_CREAT is set, else ignored

DESCRIPTION This is the helper to open a file from kernelspace if you really have to. But in generally you should not do this, so please move along, nothing to see here..
