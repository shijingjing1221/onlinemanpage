NAME remove_all_active_ranges - Remove all currently registered regions

SYNOPSIS void remove_all_active_ranges(void);

ARGUMENTS void no arguments

DESCRIPTION During discovery, it may be found that a table like SRAT is invalid and an alternative discovery method must be used. This function removes all currently registered regions.
