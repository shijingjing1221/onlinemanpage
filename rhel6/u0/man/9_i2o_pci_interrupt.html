I2O_PCI_INTERRUPT(9) Message-based devices I2O_PCI_INTERRUPT(9)

NAME i2o_pci_interrupt - Interrupt handler for I2O controller

SYNOPSIS irqreturn_t i2o_pci_interrupt(int irq, void * dev_id);

ARGUMENTS irq interrupt line

dev_id pointer to the I2O controller

DESCRIPTION Handle an interrupt from a PCI based I2O controller. This turns out to be rather simple. We keep the controller pointer in the cookie.
