NAME gnutls_x509_crt_verify_hash - verify the given signed digest

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_verify_hash(gnutls_x509_crt_t crt, unsigned int flags, const gnutls_datum_t * hash, const gnutls_datum_t * signature);

ARGUMENTS gnutls_x509_crt_t crt Holds the certificate

unsigned int flags should be 0 for now

const gnutls_datum_t * hash holds the hash digest to be verified

const gnutls_datum_t * signature contains the signature

DESCRIPTION This function will verify the given signed digest, using the parameters from the certificate.

RETURNS In case of a verification failure 0 is returned, and 1 on success.
