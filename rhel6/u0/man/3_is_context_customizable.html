NAME is_context_customizable - check whether SELinux context type is cus- tomizable by the administrator.

SYNOPSIS #include <selinux/selinux.h>

int is_context_customizable(security_context_t scon);

DESCRIPTION is_context_customizable

This function checks whether the type of scon is in the /etc/selinux/SELINUXTYPE/context/customizable_types file. A customiz- able type is a file context type that administrators set on files, usu- ally to allow certain domains to share the file content. restorecon and setfiles, by default, leave these context in place.

RETURN VALUE returns 1 if security context is customizable or 0 if it is not. returns -1 on error

FILE /etc/selinux/SELINUXTYPE/context/customizable_types
