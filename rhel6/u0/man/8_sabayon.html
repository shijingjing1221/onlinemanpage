NAME sabayon - A tool to graphically manage GNOME desktop profiles.

SYNOPSIS sabayon

DESCRIPTION Sabayon is a tool to graphically manage GNOME desktop profiles. The sabayon program is the administrative tool that allows you to create, modify, and save profiles. It must be run with root priviledges.

OPTIONS None.

FILES /etc/sabayon/users.xml XML file that maps userids to profile mappings

/etc/sabayon/groups.xml XML file that maps userids to profile mappings

/etc/sabayon/sabayon-debug-log.conf Config file for logging options.

/etc/sabayon/profiles/ Directory where profiles are stored.

RESOURCES http://live.gnome.org/Sabayon
