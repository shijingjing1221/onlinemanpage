NAME rio_update_route_tables - Updates route tables in switches

SYNOPSIS void rio_update_route_tables(struct rio_mport * port);

ARGUMENTS port Master port associated with the RIO network

DESCRIPTION For each enumerated device, ensure that each switch in a system has correct routing entries. Add routes for devices that where unknown dirung the first enumeration pass through the switch.
