NAME relay_alloc_buf - allocate a channel buffer

SYNOPSIS void * relay_alloc_buf(struct rchan_buf * buf, size_t * size);

ARGUMENTS buf the buffer struct

size total size of the buffer

DESCRIPTION Returns a pointer to the resulting buffer, NULL if unsuccessful. The passed in size will get page aligned, if it isn´t already.
