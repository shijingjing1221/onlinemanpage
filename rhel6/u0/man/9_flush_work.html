NAME flush_work - block until a work_struct´s callback has terminated

SYNOPSIS int flush_work(struct work_struct * work);

ARGUMENTS work the work which is to be flushed

DESCRIPTION Returns false if work has already terminated.

It is expected that, prior to calling flush_work, the caller has arranged for the work to not be requeued, otherwise it doesn´t make sense to use this function.
