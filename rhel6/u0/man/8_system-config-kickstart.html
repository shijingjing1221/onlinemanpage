SYSTEM-CONFIG-KICKSTART(8) Kickstart Configurator SYSTEM-CONFIG-KICKSTART(8)

NAME system-config-kickstart - graphical interface for creating kickstart files

SYNOPSIS system-config-kickstart

DESCRIPTION system-config-kickstart provides a simple method of creating a kick- start file that can be used to automate the installation process on Red Hat Linux.

OPTIONS None

FILES /usr/bin/system-config-kickstart /usr/share/system-config-kickstart/*

EXAMPLES To run the program type:

system-config-kickstart
