SND_BUG(9) Miscellaneous Functions SND_BUG(9)

NAME snd_BUG - give a BUG warning message and stack trace

SYNOPSIS snd_BUG(void);

ARGUMENTS None

DESCRIPTION Calls WARN if CONFIG_SND_DEBUG is set. Ignored when CONFIG_SND_DEBUG is not set.
