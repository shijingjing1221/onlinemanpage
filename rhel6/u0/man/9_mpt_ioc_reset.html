NAME mpt_ioc_reset - Base cleanup for hard reset

SYNOPSIS int mpt_ioc_reset(MPT_ADAPTER * ioc, int reset_phase);

ARGUMENTS ioc Pointer to the adapter structure

reset_phase Indicates pre- or post-reset functionality

REMARK Frees resources with internally generated commands.
