NAME inc_nlink - directly increment an inode´s link count

SYNOPSIS void inc_nlink(struct inode * inode);

ARGUMENTS inode inode

DESCRIPTION This is a low-level filesystem helper to replace any direct filesystem manipulation of i_nlink. Currently, it is only here for parity with dec_nlink.
