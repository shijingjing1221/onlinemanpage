NAME atomic_dec_and_test - decrement and test

SYNOPSIS int atomic_dec_and_test(atomic_t * v);

ARGUMENTS v pointer of type atomic_t

DESCRIPTION Atomically decrements v by 1 and returns true if the result is 0, or false for all other cases.
