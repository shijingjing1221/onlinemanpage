NAME ipcperms - check IPC permissions

SYNOPSIS int ipcperms(struct kern_ipc_perm * ipcp, short flag);

ARGUMENTS ipcp IPC permission set

flag desired permission set.

DESCRIPTION Check user, group, other permissions for access to ipc resources. return 0 if allowed
