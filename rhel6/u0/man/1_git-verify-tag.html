NAME git-verify-tag - Check the GPG signature of tags

SYNOPSIS git verify-tag <tag>...

DESCRIPTION Validates the gpg signature created by git tag.

OPTIONS <tag>... SHA1 identifiers of git tag objects.

AUTHOR Written by Jan Harkes <jaharkes@cs.cmu.edu[1]> and Eric W. Biederman <ebiederm@xmission.com[2]>
