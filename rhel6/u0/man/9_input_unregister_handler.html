INPUT_UNREGISTER_HAN(9) Input Subsystem INPUT_UNREGISTER_HAN(9)

NAME input_unregister_handler - unregisters an input handler

SYNOPSIS void input_unregister_handler(struct input_handler * handler);

ARGUMENTS handler handler to be unregistered

DESCRIPTION This function disconnects a handler from its input devices and removes it from lists of known handlers.
