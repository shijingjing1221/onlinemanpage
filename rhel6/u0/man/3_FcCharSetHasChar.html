NAME FcCharSetHasChar - Check a charset for a char

SYNOPSIS #include <fontconfig.h>

FcBool FcCharSetHasChar(const FcCharSet *fcs); (FcChar32 ucs4); .fi

DESCRIPTION Returns whether fcs contains the char ucs4.
