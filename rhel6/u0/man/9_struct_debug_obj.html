NAME struct_debug_obj - representaion of an tracked object

SYNOPSIS struct debug_obj { struct hlist_node node; enum debug_obj_state state; void * object; struct debug_obj_descr * descr; };

MEMBERS node hlist node to link the object into the tracker list

state tracked object state

object pointer to the real object

descr pointer to an object type specific debug description structure
