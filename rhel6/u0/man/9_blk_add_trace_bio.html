NAME blk_add_trace_bio - Add a trace for a bio oriented action

SYNOPSIS void blk_add_trace_bio(struct request_queue * q, struct bio * bio, u32 what);

ARGUMENTS q queue the io is for

bio the source bio

what the action

DESCRIPTION Records an action against a bio. Will log the bio offset + size.
