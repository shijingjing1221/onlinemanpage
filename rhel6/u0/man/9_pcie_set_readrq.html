PCIE_SET_READRQ(9) Hardware Interfaces PCIE_SET_READRQ(9)

NAME pcie_set_readrq - set PCI Express maximum memory read request

SYNOPSIS int pcie_set_readrq(struct pci_dev * dev, int rq);

ARGUMENTS dev PCI device to query

rq maximum memory read count in bytes valid values are 128, 256, 512, 1024, 2048, 4096

DESCRIPTION If possible sets maximum read byte count
