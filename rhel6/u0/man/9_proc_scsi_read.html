PROC_SCSI_READ(9) SCSI mid layer PROC_SCSI_READ(9)

NAME proc_scsi_read - handle read from /proc by calling host´s proc_info command

SYNOPSIS int proc_scsi_read(char * buffer, char ** start, off_t offset, int length, int * eof, void * data);

ARGUMENTS buffer passed to proc_info

start passed to proc_info

offset passed to proc_info

length passed to proc_info

eof returns whether length read was less than requested

data pointer to a struct Scsi_Host
