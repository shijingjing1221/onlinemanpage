NAME cdev_init - initialize a cdev structure

SYNOPSIS void cdev_init(struct cdev * cdev, const struct file_operations * fops);

ARGUMENTS cdev the structure to initialize

fops the file_operations for this device

DESCRIPTION Initializes cdev, remembering fops, making it ready to add to the system with cdev_add.
