NAME ata_host_resume - resume host

SYNOPSIS void ata_host_resume(struct ata_host * host);

ARGUMENTS host host to resume

DESCRIPTION Resume host. Actual operation is performed by EH. This function requests EH to perform PM operations and returns. Note that all resume operations are performed parallely.

LOCKING Kernel thread context (may sleep).
