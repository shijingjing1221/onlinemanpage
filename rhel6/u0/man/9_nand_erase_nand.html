NAME nand_erase_nand - [Internal] erase block(s)

SYNOPSIS int nand_erase_nand(struct mtd_info * mtd, struct erase_info * instr, int allowbbt);

ARGUMENTS mtd MTD device structure

instr erase instruction

allowbbt allow erasing the bbt area

DESCRIPTION Erase one ore more blocks
