NAME function::user_long - Retrieves a long value stored in user space.

SYNOPSIS user_long:long(addr:long)

ARGUMENTS addr The user space address to retrieve the long from.

DESCRIPTION Returns the long value from a given user space address. Returns zero when user space data is not accessible. Note that the size of the long depends on the architecture of the current user space task (for those architectures that support both 64/32 bit compat tasks).
