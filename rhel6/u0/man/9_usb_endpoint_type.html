USB_ENDPOINT_TYPE(9) Kernel Mode Gadget API USB_ENDPOINT_TYPE(9)

NAME usb_endpoint_type - get the endpoint´s transfer type

SYNOPSIS int usb_endpoint_type(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns one of USB_ENDPOINT_XFER_{CONTROL, ISOC, BULK, INT} according to epd´s transfer type.
