NAME sfcbuuid - Script to create a universally unique identifier for the installed Small-Footprint CIM Broker (sfcb)

SYNOPSIS sfcbuuid [options]

DESCRIPTION sfcbuuid can be used to create a universally unique identifier (uuid) for sfcb. The uuid will be stored in a file (usually /var/lib/sfcb/uuid). Whenever sfcb needs to return its uuid to a requestor it will read the uuid file from this file.

OPTIONS -u uuidfile Full path name of the sfcb uuid file. Default is /var/lib/sfcb/uuid

-f Remove an existing uuid file without asking for permission.

-h Display usage information and exit.
