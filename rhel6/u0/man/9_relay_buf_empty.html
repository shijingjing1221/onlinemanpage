NAME relay_buf_empty - boolean, is the channel buffer empty?

SYNOPSIS int relay_buf_empty(struct rchan_buf * buf);

ARGUMENTS buf channel buffer

DESCRIPTION Returns 1 if the buffer is empty, 0 otherwise.
