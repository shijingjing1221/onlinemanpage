NAME utrace_signal_action - enum utrace_signal_action from callback action

SYNOPSIS enum utrace_signal_action utrace_signal_action(u32 action);

ARGUMENTS action

report_signal callback action argument or return value

DESCRIPTION This extracts the enum utrace_signal_action from action, which is the action argument to a report_signal callback or the return value from one.
