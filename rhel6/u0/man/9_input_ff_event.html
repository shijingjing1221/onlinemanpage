NAME input_ff_event - generic handler for force-feedback events

SYNOPSIS int input_ff_event(struct input_dev * dev, unsigned int type, unsigned int code, int value);

ARGUMENTS dev input device to send the effect to

type event type (anything but EV_FF is ignored)

code event code

value event value
