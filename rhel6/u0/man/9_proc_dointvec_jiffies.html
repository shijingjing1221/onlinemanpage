NAME proc_dointvec_jiffies - read a vector of integers as seconds

SYNOPSIS int proc_dointvec_jiffies(struct ctl_table * table, int write, void __user * buffer, size_t * lenp, loff_t * ppos);

ARGUMENTS table the sysctl table

write

TRUE if this is a write to the sysctl file

buffer the user buffer

lenp the size of the user buffer

ppos file position

DESCRIPTION Reads/writes up to table->maxlen/sizeof(unsigned int) integer values from/to the user buffer, treated as an ASCII string. The values read are assumed to be in seconds, and are converted into jiffies.

Returns 0 on success.
