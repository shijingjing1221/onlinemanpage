NAME probe::socket.writev.return - Conclusion of message sent via socket_writev

SYNOPSIS socket.writev.return

VALUES success Was send successful? (1 = yes, 0 = no)

protocol Protocol value

flags Socket flags value

name Name of this probe

state Socket state value

size Size of message sent (in bytes) or error code if success = 0

type Socket type value

family Protocol family value

CONTEXT The message receiver.

DESCRIPTION Fires at the conclusion of sending a message on a socket via the sock_writev function
