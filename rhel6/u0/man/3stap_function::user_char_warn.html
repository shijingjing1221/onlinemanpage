NAME function::user_char_warn - Retrieves a char value stored in user space.

SYNOPSIS user_char_warn:long(addr:long)

ARGUMENTS addr The user space address to retrieve the char from.

DESCRIPTION Returns the char value from a given user space address. Returns zero when user space and warns (but does not abort) about the failure.
