NAME round_jiffies_up - function to round jiffies up to a full second

SYNOPSIS unsigned long round_jiffies_up(unsigned long j);

ARGUMENTS j the time in (absolute) jiffies that should be rounded

DESCRIPTION This is the same as round_jiffies except that it will never round down. This is useful for timeouts for which the exact time of firing does not matter too much, as long as they don´t fire too early.
