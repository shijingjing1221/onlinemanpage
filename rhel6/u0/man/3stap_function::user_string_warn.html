NAME function::user_string_warn - Retrieves string from user space.

SYNOPSIS user_string_warn:string(addr:long)

ARGUMENTS addr The user space address to retrieve the string from.

DESCRIPTION Returns the null terminated C string from a given user space memory address. Reports <unknown> on the rare cases when userspace data is not accessible and warns (but does not abort) about the failure.
