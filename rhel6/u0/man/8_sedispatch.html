NAME sedispatch - setroubleshoot audit dispatcher

SYNOPSIS sedispatch [-b] [- [-h] [-s] [-S] [-l id] [-a file] [-v] [-V] [-u] [-p]

DESCRIPTION This manual page describes the sedispatch program.

sedispatch is audit dispatcher. It scans audit messages for SELinux AVC messages, formats them into a dbus message and sends it to setrou- bleshootd
