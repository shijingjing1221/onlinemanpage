NAME tracehook_force_sigpending - let tracing force signal_pending(current) on

SYNOPSIS int tracehook_force_sigpending(void);

ARGUMENTS void no arguments

DESCRIPTION Called when recomputing our signal_pending flag. Return nonzero to force the signal_pending flag on, so that tracehook_get_signal will be called before the next return to user mode.

Called with current->sighand->siglock held.
