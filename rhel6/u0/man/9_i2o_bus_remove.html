I2O_BUS_REMOVE(9) Message-based devices I2O_BUS_REMOVE(9)

NAME i2o_bus_remove - remove the I2O Bus Adapter device from the system again

SYNOPSIS int i2o_bus_remove(struct device * dev);

ARGUMENTS dev I2O Bus Adapter device which should be removed

DESCRIPTION Always returns 0.
