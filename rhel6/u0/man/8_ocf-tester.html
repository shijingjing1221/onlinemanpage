NAME ocf-tester - Part of the Linux-HA project

SYNOPSIS ocf-tester [-Lh] -n resource_name [-o name=value]* /full/path/to/resource/agent

DESCRIPTION Tool for testing if a cluster resource is OCF compliant

OPTIONS -h,--help This text

-n name Name of the resource

-o name=value Name and value of any parameters required by the agent

-L Use lrmadmin/lrmd for tests
