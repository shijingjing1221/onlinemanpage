NAME mrftopbm - convert an MRF image to PBM format

SYNOPSIS mrftopbm [ -a ] [ input.mrf ]

DESCRIPTION This program is part of Netpbm(1).

mrftopbm converts an MRF image to PBM format.

mrftopbm takes the MRF image from the file named by the input.mrf argu- ment, or Standard Input if you dont specify input.mrf. The output goes to Standard Output.

For more information about mrf, see theMRF specification (1).

OPTIONS -a causes mrftopbm to include the edges, if any, in the output PBM. This may help when debugging a compressor s edge optimization.
