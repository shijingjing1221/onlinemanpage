NAME i2c_smbus_write_byte - SMBus send byte protocol

SYNOPSIS s32 i2c_smbus_write_byte(struct i2c_client * client, u8 value);

ARGUMENTS client Handle to slave device

value Byte to be sent

DESCRIPTION This executes the SMBus send byte protocol, returning negative errno else zero on success.
