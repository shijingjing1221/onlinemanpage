SQ_FLUSH_RANGE(9) Memory Management SQ_FLUSH_RANGE(9)

NAME sq_flush_range - Flush (prefetch) a specific SQ range

SYNOPSIS void sq_flush_range(unsigned long start, unsigned int len);

ARGUMENTS start the store queue address to start flushing from

len the length to flush

DESCRIPTION Flushes the store queue cache from start to start + len in a linear fashion.
