NAME _toupper - transliterate lowercase characters to uppercase

SYNOPSIS #include <ctype.h>

int _toupper(int c);

DESCRIPTION The _toupper() macro shall be equivalent to toupper() except that the application shall ensure that the argument c is a lowercase letter.

RETURN VALUE Upon successful completion, _toupper() shall return the uppercase let- ter corresponding to the argument passed.

ERRORS No errors are defined.
