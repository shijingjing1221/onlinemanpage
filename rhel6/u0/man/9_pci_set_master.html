PCI_SET_MASTER(9) Hardware Interfaces PCI_SET_MASTER(9)

NAME pci_set_master - enables bus-mastering for device dev

SYNOPSIS void pci_set_master(struct pci_dev * dev);

ARGUMENTS dev the PCI device to enable

DESCRIPTION Enables bus-mastering on the device and calls pcibios_set_master to do the needed arch specific settings.
