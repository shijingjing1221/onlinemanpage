NAME gnutls_x509_crl_set_this_update - This function will set the CRLs issuing time

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crl_set_this_update(gnutls_x509_crl_t crl, time_t act_time);

ARGUMENTS gnutls_x509_crl_t crl should contain a gnutls_x509_crl_t structure

time_t act_time The actual time

DESCRIPTION This function will set the time this CRL was issued.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
