NAME gnutls_x509_crt_set_proxy - Set the proxyCertInfo extension

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_set_proxy(gnutls_x509_crt_t crt, int pathLenCon- straint, const char * policyLanguage, const char * policy, size_t sizeof_policy);

ARGUMENTS gnutls_x509_crt_t crt a certificate of type gnutls_x509_crt_t

int pathLenConstraint non-negative values indicate maximum length of path, and negative values indicate that the pathLenConstraints field should not be present.

const char * policyLanguage OID describing the language of policy.

const char * policy opaque byte array with policy language, can be NULL

size_t sizeof_policy size of policy.

DESCRIPTION This function will set the proxyCertInfo extension.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
