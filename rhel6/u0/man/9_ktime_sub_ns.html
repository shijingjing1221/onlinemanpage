NAME ktime_sub_ns - Subtract a scalar nanoseconds value from a ktime_t variable

SYNOPSIS ktime_t ktime_sub_ns(const ktime_t kt, u64 nsec);

ARGUMENTS kt minuend

nsec the scalar nsec value to subtract

DESCRIPTION Returns the subtraction of nsec from kt in ktime_t format
