Detailed Description template<typename _IntType = int, typename _RealType = double> class std::poisson_distribution< _IntType, _RealType > A discrete Poisson random number distribution.

The formula for the Poisson probability mass function is $ p(i) = ac{mean^i}{i!} e^{-mean} $ where $ mean $ is the parameter of the distribution.
