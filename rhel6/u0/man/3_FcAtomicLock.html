NAME FcAtomicLock - lock a file

SYNOPSIS #include <fontconfig.h>

FcBool FcAtomicLock(FcAtomic *atomic); .fi

DESCRIPTION Attempts to lock the file referenced by atomic. Returns FcFalse if the file is already locked, else returns FcTrue and leaves the file locked.

VERSION Fontconfig version 2.8.0
