NAME clk_set_rate - set the clock rate for a clock source

SYNOPSIS int clk_set_rate(struct clk * clk, unsigned long rate);

ARGUMENTS clk clock source

rate desired clock rate in Hz

DESCRIPTION Returns success (0) or negative errno.
