ENABLE_CMF(9) The ccw bus ENABLE_CMF(9)

NAME enable_cmf - switch on the channel measurement for a specific device

SYNOPSIS int enable_cmf(struct ccw_device * cdev);

ARGUMENTS cdev The ccw device to be enabled

DESCRIPTION Returns 0 for success or a negative error value.

CONTEXT non-atomic
