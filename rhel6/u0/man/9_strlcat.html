STRLCAT(9) Basic C Library Functions STRLCAT(9)

NAME strlcat - Append a length-limited, NUL-terminated string to another

SYNOPSIS size_t strlcat(char * dest, const char * src, size_t count);

ARGUMENTS dest The string to be appended to

src The string to append to it

count The size of the destination buffer.
