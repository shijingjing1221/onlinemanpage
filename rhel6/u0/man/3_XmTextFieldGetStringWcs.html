NAME XmTextFieldGetStringWcs A TextField function that retrieves a copy of the wide character string value of a TextField widget

SYNOPSIS #include <Xm/TextF.h> wchar_t * XmTextFieldGetStringWcs( Widget widget);

DESCRIPTION XmTextFieldGetStringWcs retrieves a copy of the wide character string value of the TextField widget. The application is responsible for free- ing the storage associated with the string by calling XtFree.

widget Specifies the TextField widget ID

For a complete definition of TextField and its associated resources, see XmTextField(3).

RETURN Returns the wide character string value of the TextField widget. The function returns an empty string if the length of the TextField wid- gets string is 0 (zero).

RELATED XmTextField(3) and XmTextFieldGetString(3).
