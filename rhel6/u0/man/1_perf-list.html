NAME perf-list - List all symbolic event types

SYNOPSIS perf list

DESCRIPTION This command displays the symbolic event types which can be selected in the various perf commands with the -e option.

OPTIONS None

SEE ALSO perf-stat(1), perf-top(1), perf-record(1)
