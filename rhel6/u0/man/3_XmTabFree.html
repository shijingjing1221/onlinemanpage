NAME XmTabFree A convenience function that frees a tab

SYNOPSIS #include <Xm/Xm.h> void XmTabFree( XmTab tab);

DESCRIPTION XmTabFree frees the memory associated with the specified tab.

tab Specifies the tab to be freed.
