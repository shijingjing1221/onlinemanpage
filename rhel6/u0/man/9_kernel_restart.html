NAME kernel_restart - reboot the system

SYNOPSIS void kernel_restart(char * cmd);

ARGUMENTS cmd pointer to buffer containing command to execute for restart or NULL

DESCRIPTION Shutdown everything and perform a clean reboot. This is not safe to call in interrupt context.
