NAME modifyrepo - Modify a repomd (xml-rpm-metadata) repository

SYNOPSIS modifyrepo <input metadata> <output repodata>

DESCRIPTION modifyrepo is a program that allows you to insert arbitrary metadata into a repomd (xml-based rpm metadata) repository.

EXAMPLES $ modifyrepo metadata.xml /repository/repodata Wrote: /repository/repodata/metadata.xml.gz type = metadata location = repodata/metadata.xml.gz checksum = 1d7ee93db2964e7f85e07ec19b3204591da1050c timestamp = 1196716296.0 open-checksum = 824d936dc7dfff029379797b311af0cc66af4115 Wrote: /repository/repodata/repomd.xml

SEE ALSO createrepo (8)
