NAME remove_irq - free an interrupt

SYNOPSIS void remove_irq(unsigned int irq, struct irqaction * act);

ARGUMENTS irq Interrupt line to free

act irqaction for the interrupt

DESCRIPTION Used to remove interrupts statically setup by the early boot process.
