NAME nand_write_page_syndrome - [REPLACABLE] hardware ecc syndrom based page write

SYNOPSIS void nand_write_page_syndrome(struct mtd_info * mtd, struct nand_chip * chip, const uint8_t * buf);

ARGUMENTS mtd mtd info structure

chip nand chip info structure

buf data buffer

DESCRIPTION The hw generator calculates the error syndrome automatically. Therefor we need a special oob layout and handling.
