NAME function::user_string2 - Retrieves string from user space with alternative error string..

SYNOPSIS user_string2:string(addr:long,err_msg:string)

ARGUMENTS addr The user space address to retrieve the string from.

err_msg The error message to return when data isn´t available.

DESCRIPTION Returns the null terminated C string from a given user space memory address. Reports the given error message on the rare cases when userspace data is not accessible.
