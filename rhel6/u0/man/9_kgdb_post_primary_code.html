KGDB_POST_PRIMARY_CO(9) KGDB Internals KGDB_POST_PRIMARY_CO(9)

NAME kgdb_post_primary_code - (optional) Save error vector/code numbers.

SYNOPSIS void kgdb_post_primary_code(struct pt_regs * regs, int e_vector, int err_code);

ARGUMENTS regs Original pt_regs.

e_vector Original error vector.

err_code Original error code.

DESCRIPTION This is usually needed on architectures which support SMP and KGDB. This function is called after all the secondary cpus have been put to a know spin state and the primary CPU has control over KGDB.
