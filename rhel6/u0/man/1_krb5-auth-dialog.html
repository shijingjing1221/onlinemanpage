NAME krb5-auth-dialog - Kerberos 5 credential checking

SYNOPSIS krb5-auth-dialog [--always|--auto]

DESCRIPTION The krb5-auth-dialog application will periodically check (every 30 sec- onds) if the user has Kerberos credentials, and if so, if they will expire soon (in less than 30 minutes). If it determines that this is the case, krb5-auth-dialog will attempt to obtain fresh credentials, prompting the user for whatever information is necessary.

ARGUMENTS --always By default, krb5-auth-dialog will exit if it finds that the user has no Kerberos credentials. If this option is specified, then it will instead attempt to get credentials.

--auto Overrides the --always option, restoring the default behavior.
