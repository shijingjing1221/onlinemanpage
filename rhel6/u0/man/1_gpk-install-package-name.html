NAME gpk-install-package-name - GNOME PackageKit Package Name Helper

SYNOPSIS gpk-install-package-name [ --verbose ] [ package_name ]

DESCRIPTION This manual page documents briefly the gpk-install-package-name com- mand.

gpk-install-package-name allows you to install a a package from the package name.
