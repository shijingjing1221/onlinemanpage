NAME FcConfigAppFontClear - Remove all app fonts from font database

SYNOPSIS #include <fontconfig.h>

void FcConfigAppFontClear(FcConfig *config); .fi

DESCRIPTION Clears the set of application-specific fonts. If config is NULL, the current configuration is used.

VERSION Fontconfig version 2.8.0
