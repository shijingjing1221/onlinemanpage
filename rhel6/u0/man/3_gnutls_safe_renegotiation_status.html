NAME gnutls_safe_renegotiation_status - API function

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_safe_renegotiation_status(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

DESCRIPTION Can be used to check whether safe renegotiation is being used in the current session.

RETURNS 0 when safe renegotiation is not used and non zero when safe renegotia- tion is used.

SINCE 2.10.0
