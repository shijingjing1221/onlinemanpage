NAME mpt_adapter_dispose - Free all resources associated with an MPT adapter

SYNOPSIS void mpt_adapter_dispose(MPT_ADAPTER * ioc);

ARGUMENTS ioc Pointer to MPT adapter structure

DESCRIPTION This routine unregisters h/w resources and frees all alloc´d memory associated with a MPT adapter structure.
