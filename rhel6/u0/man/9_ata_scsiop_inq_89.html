NAME ata_scsiop_inq_89 - Simulate INQUIRY VPD page 89, ATA info

SYNOPSIS unsigned int ata_scsiop_inq_89(struct ata_scsi_args * args, u8 * rbuf);

ARGUMENTS args device IDENTIFY data / SCSI command of interest.

rbuf Response buffer, to which simulated SCSI cmd output is sent.

DESCRIPTION Yields SAT-specified ATA VPD page.

LOCKING spin_lock_irqsave(host lock)
