NAME probe::tcp.recvmsg - Receiving TCP message

SYNOPSIS tcp.recvmsg

VALUES saddr A string representing the source IP address

daddr A string representing the destination IP address

name Name of this probe

sport TCP source port

dport TCP destination port

size Number of bytes to be received

sock Network socket

CONTEXT The process which receives a tcp message
