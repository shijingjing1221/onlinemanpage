BDGET_DISK(9) Block Devices BDGET_DISK(9)

NAME bdget_disk - do bdget by gendisk and partition number

SYNOPSIS struct block_device * bdget_disk(struct gendisk * disk, int partno);

ARGUMENTS disk gendisk of interest

partno partition number

DESCRIPTION Find partition partno from disk, do bdget on it.

CONTEXT Don´t care.

RETURNS Resulting block_device on success, NULL on failure.
