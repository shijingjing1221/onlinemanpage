NAME mkofm - front end to mktextfm(1)

SYNOPSIS mkofm ARGS

DESCRIPTION mkofm takes all ARGS and passes them to mktextfm(1). For more informa- tion about appropriate arguments, see mktextfm(1).

SEE ALSO mktextfm(1)

BUGS None known.
