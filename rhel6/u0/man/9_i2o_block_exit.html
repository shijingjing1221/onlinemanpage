I2O_BLOCK_EXIT(9) Message-based devices I2O_BLOCK_EXIT(9)

NAME i2o_block_exit - Block OSM exit function

SYNOPSIS void __exit i2o_block_exit(void);

ARGUMENTS void no arguments

DESCRIPTION Unregisters Block OSM from I2O core, unregisters i2o_block block device and frees the mempool and slab.
