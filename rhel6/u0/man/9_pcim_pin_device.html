PCIM_PIN_DEVICE(9) Hardware Interfaces PCIM_PIN_DEVICE(9)

NAME pcim_pin_device - Pin managed PCI device

SYNOPSIS void pcim_pin_device(struct pci_dev * pdev);

ARGUMENTS pdev PCI device to pin

DESCRIPTION Pin managed PCI device pdev. Pinned device won´t be disabled on driver detach. pdev must have been enabled with pcim_enable_device.
