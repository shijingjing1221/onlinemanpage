I2O_DRIVER_INIT(9) Message-based devices I2O_DRIVER_INIT(9)

NAME i2o_driver_init - initialize I2O drivers (OSMs)

SYNOPSIS int i2o_driver_init(void);

ARGUMENTS void no arguments

DESCRIPTION Registers the I2O bus and allocate memory for the array of OSMs.

Returns 0 on success or negative error code on failure.
