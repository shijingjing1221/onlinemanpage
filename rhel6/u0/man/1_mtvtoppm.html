NAME mtvtoppm - convert output from an MTV or PRT ray tracer into a PPM

SYNOPSIS mtvtoppm [mtvfile]

DESCRIPTION This program is part of Netpbm(1).

mtvtoppm reads an input file from Mark VanDeWetterings MTV ray tracer and produces a PPM image as output.

The PRT raytracer also produces this format.

SEE ALSO ppm(1)
