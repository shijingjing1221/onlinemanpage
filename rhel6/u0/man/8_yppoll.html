NAME yppoll - return version and master server of a NIS map

SYNOPSYS yppoll [ -h host ] [ -d domain ] mapname

DESCRIPTION yppoll

OPTIONS -h host Ask the NIS server process running on host for information about mapname. If host is not specified, the server polled is the default server returned by ypwhich(1).

-d domain Specify a domain other than the default domain as returned by domainname(1).

SEE ALSO domainname(8), ypbind(8), ypcat(8), ypmatch(1), ypserv(8), yset(8), ypwhich(1)
