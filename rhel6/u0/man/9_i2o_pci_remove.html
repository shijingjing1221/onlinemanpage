I2O_PCI_REMOVE(9) Message-based devices I2O_PCI_REMOVE(9)

NAME i2o_pci_remove - Removes a I2O controller from the system

SYNOPSIS void __devexit i2o_pci_remove(struct pci_dev * pdev);

ARGUMENTS pdev I2O controller which should be removed

DESCRIPTION Reset the I2O controller, disable interrupts and remove all allocated resources.
