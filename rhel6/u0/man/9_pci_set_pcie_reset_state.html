PCI_SET_PCIE_RESET_S(9) Hardware Interfaces PCI_SET_PCIE_RESET_S(9)

NAME pci_set_pcie_reset_state - set reset state for device dev

SYNOPSIS int pci_set_pcie_reset_state(struct pci_dev * dev, enum pcie_reset_state state);

ARGUMENTS dev the PCI-E device reset

state Reset state to enter into

DESCRIPTION Sets the PCI reset state for the device.
