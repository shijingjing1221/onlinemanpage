NAME atrun - run jobs queued for later execution

SYNOPSIS atrun [-l load_avg] [-d]

DESCRIPTION atrun runs jobs queued by at(1). It is a shell script invoking /usr/sbin/atd with the -s option, and is provided for backward compati- bility with older installations.

SEE ALSO at(1), atd(8).
