NAME GetPortFacts - Send PortFacts request to MPT adapter.

SYNOPSIS int GetPortFacts(MPT_ADAPTER * ioc, int portnum, int sleepFlag);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

portnum Port number

sleepFlag Specifies whether the process can sleep

DESCRIPTION Returns 0 for success, non-zero for failure.
