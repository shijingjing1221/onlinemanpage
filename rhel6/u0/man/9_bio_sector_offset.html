NAME bio_sector_offset - Find hardware sector offset in bio

SYNOPSIS sector_t bio_sector_offset(struct bio * bio, unsigned short index, unsigned int offset);

ARGUMENTS bio bio to inspect

index bio_vec index

offset offset in bv_page

DESCRIPTION Return the number of hardware sectors between beginning of bio and an end point indicated by a bio_vec index and an offset within that vector´s page.
