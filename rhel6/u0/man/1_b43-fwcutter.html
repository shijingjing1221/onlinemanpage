NAME b43-fwcutter - manual page for b43-fwcutter

SYNOPSIS b43-fwcutter [OPTION] [DRIVER]

DESCRIPTION b43-fwcutter can extract the firmware for your Broadcom 43xx hardware from different closed source drivers. The b43 driver depends on this firmware files and cant work without them.

Currently b43-fwcutter supports Apple MacOS X, Microsoft Windows 98/ME/2000/XP and Linux drivers, but keep in mind that b43-fwcutter doesnt support all driver versions.

Example:

b43-fwcutter bcmwl5.sys

to cut the firmware out of bcmwl5.sys

OPTIONS -l|--list List supported driver versions

-i|--identify Only identify the driver file (dont extract)

-w|--target-dir DIR Extract and write firmware to DIR

-a|--alt-iv Extract alternative initvals (only 3.10.x.x)

-p|--postfix ".FOO" Postfix for firmware filenames (.FOO.fw)

-v|--version Print b43-fwcutter version

-h|--help Print help information
