ADD_TIMER_ON(9) Driver Basics ADD_TIMER_ON(9)

NAME add_timer_on - start a timer on a particular CPU

SYNOPSIS void add_timer_on(struct timer_list * timer, int cpu);

ARGUMENTS timer the timer to be added

cpu the CPU to start it on

DESCRIPTION This is not very scalable on SMP. Double adds are not possible.
