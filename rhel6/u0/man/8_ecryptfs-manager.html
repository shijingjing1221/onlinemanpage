NAME ecryptfs-manager - eCryptfs key manager.

DESCRIPTION ecryptfs-manager is an application that manages eCryptfs objects such as keys.

You can use ecryptfs-manager to ask key modules to generate new keys for you, for instance.

SEE ALSO ecryptfs(7), ecryptfsd(8), mount.ecryptfs(8)
