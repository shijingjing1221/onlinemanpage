NAME XmCreateDropDownList The Drop-down list ComboBox widget creation function

SYNOPSIS #include <Xm/ComboBox.h> Widget XmCreateDropDownList( Widget parent, String name, ArgList arglist, Cardinal arg_count);

DESCRIPTION XmCreateDropDownList creates an instance of a ComboBox widget of XmN- comboBoxType XmDROP_DOWN_LIST and returns the associated widget ID.

parent Specifies the parent widget ID.

name Specifies the name of the created widget.

arglist Specifies the argument list.

arg_count Specifies the number of attribute/value pairs in the argument list (arglist).

For a complete definition of ComboBox and its associated resources, see XmComboBox(3).

RETURN Returns the ComboBox widget ID.

RELATED XmComboBox(3).
