NAME hrtimer_cancel - cancel a timer and wait for the handler to finish.

SYNOPSIS int hrtimer_cancel(struct hrtimer * timer);

ARGUMENTS timer the timer to be cancelled

RETURNS 0 when the timer was not active 1 when the timer was active
