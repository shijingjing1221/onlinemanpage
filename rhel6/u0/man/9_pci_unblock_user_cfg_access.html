NAME pci_unblock_user_cfg_access - Unblock userspace PCI config reads/writes

SYNOPSIS void pci_unblock_user_cfg_access(struct pci_dev * dev);

ARGUMENTS dev pci device struct

DESCRIPTION This function allows userspace PCI config accesses to resume.
