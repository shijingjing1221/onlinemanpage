NAME ata_port_probe - Mark port as enabled

SYNOPSIS void ata_port_probe(struct ata_port * ap);

ARGUMENTS ap Port for which we indicate enablement

DESCRIPTION Modify ap data structure such that the system thinks that the entire port is enabled.

LOCKING host lock, or some other form of serialization.
