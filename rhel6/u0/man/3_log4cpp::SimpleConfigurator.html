Detailed Description This class implements a simple Configurator for log4cpp.

It is a temporary hack with an undocumented configuration format.

Deprecated As of version 0.3.2 log4cpp includes a log4j format compatible PropertyConfigurator, removing the need for SimpleConfigurator. This class will be removed in 0.4.0.
