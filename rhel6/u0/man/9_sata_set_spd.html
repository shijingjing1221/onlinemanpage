NAME sata_set_spd - set SATA spd according to spd limit

SYNOPSIS int sata_set_spd(struct ata_link * link);

ARGUMENTS link Link to set SATA spd for

DESCRIPTION Set SATA spd of link according to sata_spd_limit.

LOCKING Inherited from caller.

RETURNS 0 if spd doesn´t need to be changed, 1 if spd has been changed. Negative errno if SCR registers are inaccessible.
