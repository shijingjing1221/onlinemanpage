DEV_UNICAST_ADD(9) Network device support DEV_UNICAST_ADD(9)

NAME dev_unicast_add - add a secondary unicast address

SYNOPSIS int dev_unicast_add(struct net_device * dev, void * addr);

ARGUMENTS dev device

addr address to add

DESCRIPTION Add a secondary unicast address to the device or increase the reference count if it already exists.

The caller must hold the rtnl_mutex.
