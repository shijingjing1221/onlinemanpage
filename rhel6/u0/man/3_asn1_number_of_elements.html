NAME asn1_number_of_elements - Counts the number of elements of a structure.

SYNOPSIS #include <libtasn1.h>

asn1_retCode asn1_number_of_elements(ASN1_TYPE element, const char * name, int * num);

ARGUMENTS ASN1_TYPE element pointer to the root of an ASN1 structure.

const char * name the name of a sub-structure of ROOT.

int * num pointer to an integer where the result will be stored

DESCRIPTION Counts the number of elements of a sub-structure called NAME with names equal to "?1","?2", ...

RETURNS ASN1_SUCCESS Creation OK.

ASN1_ELEMENT_NOT_FOUND NAME isnt known.

ASN1_GENERIC_ERROR Pointer num equal to NULL.
