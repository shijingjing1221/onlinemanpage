FUNCTION::WARN(3stap) Logging Tapset FUNCTION::WARN(3stap)

NAME function::warn - Send a line to the warning stream.

SYNOPSIS warn(msg:string)

ARGUMENTS msg The formatted message string.

DESCRIPTION An implicit end-of-line is added. staprun prepends the string WARNING:.
