NAME FcCharSetIntersect - Intersect charsets

SYNOPSIS #include <fontconfig.h>

FcCharSet * FcCharSetIntersect(const FcCharSet *a); (const FcCharSet *b); .fi

DESCRIPTION Returns a set including only those chars found in both a and b.

VERSION Fontconfig version 2.8.0
