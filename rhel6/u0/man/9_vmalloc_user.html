VMALLOC_USER(9) Memory Management in Linux VMALLOC_USER(9)

NAME vmalloc_user - allocate zeroed virtually contiguous memory for userspace

SYNOPSIS void * vmalloc_user(unsigned long size);

ARGUMENTS size allocation size

DESCRIPTION The resulting memory area is zeroed so it can be mapped to userspace without leaking data.
