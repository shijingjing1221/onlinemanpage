NAME pci_block_user_cfg_access - Block userspace PCI config reads/writes

SYNOPSIS void pci_block_user_cfg_access(struct pci_dev * dev);

ARGUMENTS dev pci device struct

DESCRIPTION When user access is blocked, any reads or writes to config space will sleep until access is unblocked again. We don´t allow nesting of block/unblock calls.
