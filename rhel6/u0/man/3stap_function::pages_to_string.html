NAME function::pages_to_string - Turns pages into a human readable string

SYNOPSIS pages_to_string:string(pages:long)

ARGUMENTS pages Number of pages to translate.

DESCRIPTION Multiplies pages by page_size to get the number of bytes and returns the result of bytes_to_string.
