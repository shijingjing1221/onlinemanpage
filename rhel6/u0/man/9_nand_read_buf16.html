NAME nand_read_buf16 - [DEFAULT] read chip data into buffer

SYNOPSIS void nand_read_buf16(struct mtd_info * mtd, uint8_t * buf, int len);

ARGUMENTS mtd MTD device structure

buf buffer to store date

len number of bytes to read

DESCRIPTION Default read function for 16bit buswith
