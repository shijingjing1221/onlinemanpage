I2O_MSG_POST(9) Message-based devices I2O_MSG_POST(9)

NAME i2o_msg_post - Post I2O message to I2O controller

SYNOPSIS void i2o_msg_post(struct i2o_controller * c, struct i2o_message * msg);

ARGUMENTS c I2O controller to which the message should be send

msg message returned by i2o_msg_get

DESCRIPTION Post the message to the I2O controller and return immediately.
