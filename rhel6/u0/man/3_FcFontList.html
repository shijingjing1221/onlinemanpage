NAME FcFontList - List fonts

SYNOPSIS #include <fontconfig.h>

FcFontSet * FcFontList(FcConfig *config); (FcPattern *p); (FcObjectSet *os); .fi

DESCRIPTION Selects fonts matching p, creates patterns from those fonts containing only the objects in os and returns the set of unique such patterns. If config is NULL, the default configuration is checked to be up to date, and used.

VERSION Fontconfig version 2.8.0
