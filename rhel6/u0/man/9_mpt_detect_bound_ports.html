NAME mpt_detect_bound_ports - Search for matching PCI bus/dev_function

SYNOPSIS void mpt_detect_bound_ports(MPT_ADAPTER * ioc, struct pci_dev * pdev);

ARGUMENTS ioc Pointer to MPT adapter structure

pdev Pointer to (struct pci_dev) structure

DESCRIPTION Search for PCI bus/dev_function which matches PCI bus/dev_function (+/-1) for newly discovered 929, 929X, 1030 or 1035.

If match on PCI dev_function +/-1 is found, bind the two MPT adapters using alt_ioc pointer fields in their MPT_ADAPTER structures.
