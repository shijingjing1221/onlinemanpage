NAME utrace_set_events_pid - choose which event reports a tracing engine gets

SYNOPSIS __must_check int utrace_set_events_pid(struct pid * pid, struct utrace_engine * engine, unsigned long eventmask);

ARGUMENTS pid thread to affect

engine attached engine to affect

eventmask new event mask

DESCRIPTION This is the same as utrace_set_events, but takes a struct pid pointer rather than a struct task_struct pointer. The caller must hold a ref on pid, but does not need to worry about the task staying valid. If it´s been reaped so that pid points nowhere, then this call returns -ESRCH.
