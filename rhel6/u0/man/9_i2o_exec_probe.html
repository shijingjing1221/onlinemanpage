I2O_EXEC_PROBE(9) Message-based devices I2O_EXEC_PROBE(9)

NAME i2o_exec_probe - Called if a new I2O device (executive class) appears

SYNOPSIS int i2o_exec_probe(struct device * dev);

ARGUMENTS dev I2O device which should be probed

DESCRIPTION Registers event notification for every event from Executive device. The return is always 0, because we want all devices of class Executive.

Returns 0 on success.
