JOURNAL_CLEAR_ERR(9) The Linux Journalling API JOURNAL_CLEAR_ERR(9)

NAME journal_clear_err - clears the journal´s error state

SYNOPSIS int journal_clear_err(journal_t * journal);

ARGUMENTS journal journal to act on.

DESCRIPTION An error must be cleared or Acked to take a FS out of readonly mode.
