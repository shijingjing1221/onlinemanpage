NAME SDL_PollEvent - Polls for currently pending events.

SYNOPSIS #include "SDL.h"

int SDL_PollEvent(SDL_Event *event);

DESCRIPTION Polls for currently pending events, and returns 1 if there are any pending events, or 0 if there are none available.

If event is not NULL, the next event is removed from the queue and stored in that area.

EXAMPLES SDL_Event event; /* Event structure */
