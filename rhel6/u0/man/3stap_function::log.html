FUNCTION::LOG(3stap) Logging Tapset FUNCTION::LOG(3stap)

NAME function::log - Send a line to the common trace buffer.

SYNOPSIS log(msg:string)

ARGUMENTS msg The formatted message string.

DESCRIPTION An implicit end-of-line is added. Same as println with a single string argument.
