NAME gnutls_openpgp_crt_check_hostname - compare hostname with the keys hostname

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_crt_check_hostname(gnutls_openpgp_crt_t key, const char * hostname);

ARGUMENTS gnutls_openpgp_crt_t key should contain a gnutls_openpgp_crt_t structure

const char * hostname A null terminated string that contains a DNS name

DESCRIPTION This function will check if the given key s owner matches the given hostname. This is a basic implementation of the matching described in RFC2818 (HTTPS), which takes into account wildcards.

RETURNS GNUTLS_E_SUCCESS on success, or an error code.
