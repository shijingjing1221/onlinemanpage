NAME tapset::pagefault - systemtap pagefault probe points

DESCRIPTION This family of probe points is used to probe page fault events. It contains the following probe points:

vm.pagefault Fires when there is a pagefault

Arguments:

address The address caused this page fault.

write_access 1 means this is a write access and 0 means this is a read access

SEE ALSO stap(1), stapprobes(3stap)
