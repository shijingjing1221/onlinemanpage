USB_ENDPOINT_NUM(9) Kernel Mode Gadget API USB_ENDPOINT_NUM(9)

NAME usb_endpoint_num - get the endpoint´s number

SYNOPSIS int usb_endpoint_num(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns epd´s number: 0 to 15.
