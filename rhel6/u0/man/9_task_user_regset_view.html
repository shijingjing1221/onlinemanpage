TASK_USER_REGSET_VIE(9) Machine State TASK_USER_REGSET_VIE(9)

NAME task_user_regset_view - Return the process´s native regset view.

SYNOPSIS const struct user_regset_view * task_user_regset_view(struct task_struct * tsk);

ARGUMENTS tsk a thread of the process in question

DESCRIPTION Return the struct user_regset_view that is native for the given process. For example, what it would access when it called ptrace. Throughout the life of the process, this only changes at exec.
