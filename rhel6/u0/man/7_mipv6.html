NAME mipv6 - Linux Mobile IPv6 protocol extensions

SYNOPSIS #include <netinet/in.h> #include <netinet/ip6.h> #include <netinet/icmp6.h> #include <netinet/ip6mh.h>

DESCRIPTION Linux 2.6 optionally implements extensions for Mobile IPv6 protocol in the IPv6 stack. This man page describes extensions to the IPv6 API, proposed in draft-chakrabarti-mobileip-mipext-advapi-02.
