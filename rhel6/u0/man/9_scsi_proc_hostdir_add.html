NAME scsi_proc_hostdir_add - Create directory in /proc for a scsi host

SYNOPSIS void scsi_proc_hostdir_add(struct scsi_host_template * sht);

ARGUMENTS sht owner of this directory

DESCRIPTION Sets sht->proc_dir to the new directory.
