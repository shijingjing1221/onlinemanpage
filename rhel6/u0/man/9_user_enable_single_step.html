NAME user_enable_single_step - single-step in user-mode task

SYNOPSIS void user_enable_single_step(struct task_struct * task);

ARGUMENTS task either current or a task stopped in TASK_TRACED

DESCRIPTION This can only be called when arch_has_single_step has returned nonzero. Set task so that when it returns to user mode, it will trap after the next single instruction executes. If arch_has_block_step is defined, this must clear the effects of user_enable_block_step too.
