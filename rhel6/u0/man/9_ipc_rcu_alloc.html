IPC_RCU_ALLOC(9) Kernel IPC facilities IPC_RCU_ALLOC(9)

NAME ipc_rcu_alloc - allocate ipc and rcu space

SYNOPSIS void* ipc_rcu_alloc(int size);

ARGUMENTS size size desired

DESCRIPTION Allocate memory for the rcu header structure + the object. Returns the pointer to the object. NULL is returned if the allocation fails.
