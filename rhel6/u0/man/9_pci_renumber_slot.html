NAME pci_renumber_slot - update struct pci_slot -> number

SYNOPSIS void pci_renumber_slot(struct pci_slot * slot, int slot_nr);

ARGUMENTS slot

struct pci_slot to update

slot_nr new number for slot

DESCRIPTION The primary purpose of this interface is to allow callers who earlier created a placeholder slot in pci_create_slot by passing a -1 as slot_nr, to update their struct pci_slot with the correct slot_nr.
