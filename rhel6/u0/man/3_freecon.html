freecon(3) SELinux API documentation freecon(3)

NAME freecon, freeconary - free memory associated with SELinux security con- texts.

SYNOPSIS #include <selinux/selinux.h>

void freecon(security_context_t con);

void freeconary(security_context_t *con);

DESCRIPTION freecon frees the memory allocated for a security context.

freeconary frees the memory allocated for a context array.

If con is NULL, no operation is performed.
