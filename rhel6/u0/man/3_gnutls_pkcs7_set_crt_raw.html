NAME gnutls_pkcs7_set_crt_raw - add a certificate in a PKCS7 certificate set

SYNOPSIS #include <gnutls/x509.h>

int gnutls_pkcs7_set_crt_raw(gnutls_pkcs7_t pkcs7, const gnutls_datum_t * crt);

ARGUMENTS gnutls_pkcs7_t pkcs7 should contain a gnutls_pkcs7_t structure

const gnutls_datum_t * crt the DER encoded certificate to be added

DESCRIPTION This function will add a certificate to the PKCS7 or RFC2630 certifi- cate set.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
