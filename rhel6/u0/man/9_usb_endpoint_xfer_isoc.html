NAME usb_endpoint_xfer_isoc - check if the endpoint has isochronous transfer type

SYNOPSIS int usb_endpoint_xfer_isoc(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns true if the endpoint is of type isochronous, otherwise it returns false.
