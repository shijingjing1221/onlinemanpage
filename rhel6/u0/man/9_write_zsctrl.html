NAME write_zsctrl - Write to a Z8530 control register

SYNOPSIS void write_zsctrl(struct z8530_channel * c, u8 val);

ARGUMENTS c The Z8530 channel

val Value to write

DESCRIPTION Write directly to the control register on the Z8530
