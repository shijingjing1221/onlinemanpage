NAME sss_groupdel - create a new group

SYNOPSIS sss_groupdel [options] GROUP

DESCRIPTION sss_groupdel deletes a group identified by its name GROUP from the system.

OPTIONS -h,--help Display help message and exit.

SEE ALSO sss_groupadd(8), sss_groupmod(8), sss_groupshow(8), sss_useradd(8), sss_userdel(8), sss_usermod(8).
