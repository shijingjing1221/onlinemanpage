NAME gnutls_compression_list - Get a list of supported compression methods

SYNOPSIS #include <gnutls/gnutls.h>

const gnutls_compression_method_t * gnutls_compression_list( void);

ARGUMENTS void

DESCRIPTION Get a list of compression methods. Note that to be able to use LZO compression, you must link to libgnutls-extra and call gnutls_global_init_extra().

RETURNS a zero-terminated list of gnutls_compression_method_t integers indicat- ing the available compression methods.
