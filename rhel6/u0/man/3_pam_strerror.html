NAME pam_strerror - return string describing PAM error code

SYNOPSIS #include <security/pam_appl.h>

const char *pam_strerror(pam_handle_t *pamh, int errnum);

DESCRIPTION The pam_strerror function returns a pointer to a string describing the error code passed in the argument errnum, possibly using the LC_MESSAGES part of the current locale to select the appropriate language. This string must not be modified by the application. No library function will modify this string.

RETURN VALUES This function returns always a pointer to a string.
