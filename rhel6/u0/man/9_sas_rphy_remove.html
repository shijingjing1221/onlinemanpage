NAME sas_rphy_remove - remove SAS remote PHY

SYNOPSIS void sas_rphy_remove(struct sas_rphy * rphy);

ARGUMENTS rphy SAS remote phy to remove

DESCRIPTION Removes the specified SAS remote PHY.
