NAME pci_vpd_truncate - Set available Vital Product Data size

SYNOPSIS int pci_vpd_truncate(struct pci_dev * dev, size_t size);

ARGUMENTS dev pci device struct

size available memory in bytes

DESCRIPTION Adjust size of available VPD area.
