LIST_REPLACE(9) Data Types LIST_REPLACE(9)

NAME list_replace - replace old entry by new one

SYNOPSIS void list_replace(struct list_head * old, struct list_head * new);

ARGUMENTS old the element to be replaced

new the new element to insert

DESCRIPTION If old was empty, it will be overwritten.
