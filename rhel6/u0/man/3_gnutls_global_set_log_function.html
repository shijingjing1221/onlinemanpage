NAME gnutls_global_set_log_function - set the logging function

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_global_set_log_function(gnutls_log_func log_func);

ARGUMENTS gnutls_log_func log_func its a log function

DESCRIPTION This is the function where you set the logging function gnutls is going to use. This function only accepts a character array. Normally you may not use this function since it is only used for debugging purposes.

gnutls_log_func is of the form, void (*gnutls_log_func)( int level, const char*);
