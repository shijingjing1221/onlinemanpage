NAME XmDataFieldGetString A DataField function that accesses the string value

SYNOPSIS #include <Xm/DataF.h> char * XmDataFieldGetString( Widget widget);

DESCRIPTION XmDataFieldGetString accesses the string value of the DataField widget. The application is responsible for freeing the storage associated with the string by calling XtFree.

widget Specifies the DataField widget ID

For a complete definition of DataField and its associated resources, see XmDataField(3).

RETURN Returns a character pointer to the string value of the DataField wid- get. This returned value is a copy of the value of the XmNvalue resource. Returns an empty string if the length of the DataField wid- gets string is 0 (zero).

RELATED XmDataField(3) and XmDataFieldGetStringWcs(3).
