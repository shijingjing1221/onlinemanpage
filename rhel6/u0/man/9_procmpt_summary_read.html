PROCMPT_SUMMARY_READ(9) Message-based devices PROCMPT_SUMMARY_READ(9)

NAME procmpt_summary_read - Handle read request of a summary file

SYNOPSIS int procmpt_summary_read(char * buf, char ** start, off_t offset, int request, int * eof, void * data);

ARGUMENTS buf Pointer to area to write information

start Pointer to start pointer

offset Offset to start writing

request Amount of read data requested

eof Pointer to EOF integer

data Pointer

DESCRIPTION Handles read request from /proc/mpt/summary or /proc/mpt/iocN/summary. Returns number of characters written to process performing the read.
