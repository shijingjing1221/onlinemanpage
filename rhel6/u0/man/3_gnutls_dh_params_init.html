NAME gnutls_dh_params_init - initialize the DH parameters

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_dh_params_init(gnutls_dh_params_t * dh_params);

ARGUMENTS gnutls_dh_params_t * dh_params Is a structure that will hold the prime numbers

DESCRIPTION This function will initialize the DH parameters structure.

RETURNS On success, GNUTLS_E_SUCCESS (zero) is returned, otherwise an error code is returned.
