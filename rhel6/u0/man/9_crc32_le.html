CRC32_LE(9) Basic Kernel Library Functions CRC32_LE(9)

NAME crc32_le - Calculate bitwise little-endian Ethernet AUTODIN II CRC32

SYNOPSIS u32 __pure crc32_le(u32 crc, unsigned char const * p, size_t len);

ARGUMENTS crc seed value for computation. ~0 for Ethernet, sometimes 0 for other uses, or the previous crc32 value if computing incrementally.

p pointer to buffer over which CRC is run

len length of buffer p
