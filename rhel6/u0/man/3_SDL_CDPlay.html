NAME SDL_CDPlay - Play a CD

SYNOPSIS #include "SDL.h"

int SDL_CDPlay(SDL_CD *cdrom, int start, int length);

DESCRIPTION Plays the given cdrom, starting a frame start for length frames.

RETURN VALUES Returns 0 on success, or -1 on an error.

SEE ALSO SDL_CDPlayTracks, SDL_CDStop
