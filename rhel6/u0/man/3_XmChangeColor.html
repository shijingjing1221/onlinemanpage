NAME XmChangeColor Recalculates all associated colors of a widget

SYNOPSIS #include <Xm/Xm.h> void XmChangeColor( Widget widget, Pixel background);

DESCRIPTION XmChangeColor handles all color modifications for the specified widget when a new background pixel value is specified. This function recalcu- lates the foreground, select, and shadow colors based on the new back- ground color and sets the corresponding resources for the widget. If a color calculation procedure has been set by a call to XmSetColorCalcu- lation, XmChangeColor uses that procedure to calculate the new colors. Otherwise, the routine uses a default procedure.

widget Specifies the widget ID whose colors will be updated

background Specifies the background color pixel value

RELATED XmGetColorCalculation(3), XmGetColors(3), and XmSetColorCalculation(3).
