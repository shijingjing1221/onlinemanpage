NAME hpiwdt - This sample openhpi application reads and enables the watchdog timer.

SYNOPSIS hpiwdt [-derx -t sec]

DESCRIPTION hpiwdt reads and enables the watchdog timer.

OPTIONS -e enables the watchdog timer -d disables the watchdog timer -r resets the watchdog timer -t N sets timeout to N seconds -x show eXtra debug messages
