NAME wcslen - determine the length of a wide-character string

SYNOPSIS #include <wchar.h>

size_t wcslen(const wchar_t *s);

DESCRIPTION The wcslen() function is the wide-character equivalent of the strlen(3) function. It determines the length of the wide-character string pointed to by s, not including the terminating L’ ’ character.

RETURN VALUE The wcslen() function returns the number of wide characters in s.

CONFORMING TO C99.

SEE ALSO strlen(3)
