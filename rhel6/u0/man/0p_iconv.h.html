NAME iconv.h - codeset conversion facility

SYNOPSIS #include <iconv.h>

DESCRIPTION The <iconv.h> header shall define the following type:

iconv_t Identifies the conversion from one codeset to another.

The following shall be declared as functions and may also be defined as macros. Function prototypes shall be provided.

iconv_t iconv_open(const char *, const char *); size_t iconv(iconv_t, char **restrict, size_t *restrict, char **restrict, size_t *restrict); int iconv_close(iconv_t);

The following sections are informative.

APPLICATION USAGE None.

RATIONALE None.

FUTURE DIRECTIONS None.
