NAME bh_uptodate_or_lock - Test whether the buffer is uptodate

SYNOPSIS int bh_uptodate_or_lock(struct buffer_head * bh);

ARGUMENTS bh struct buffer_head

DESCRIPTION Return true if the buffer is up-to-date and false, with the buffer locked, if not.
