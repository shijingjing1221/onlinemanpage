PCI_RESCAN_BUS(9) Hardware Interfaces PCI_RESCAN_BUS(9)

NAME pci_rescan_bus - scan a PCI bus for devices.

SYNOPSIS unsigned int __ref pci_rescan_bus(struct pci_bus * bus);

ARGUMENTS bus PCI bus to scan

DESCRIPTION Scan a PCI bus and child buses for new devices, adds them, and enables them.

Returns the max number of subordinate bus discovered.
