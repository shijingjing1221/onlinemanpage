NAME ppmtopi1 - convert a PPM image into an Atari Degas .pi1 file

SYNOPSIS ppmtopi1

[ppmfile]

DESCRIPTION This program is part of Netpbm(1).

ppmtopi1 reads a PPM image as input and produces an Atari Degas .pi1 file as output.

SEE ALSO pi1toppm(1), ppm(1), pbmtopi3(1), pi3topbm(1)
