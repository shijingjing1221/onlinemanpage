SCSI_QUEUE_WORK(9) SCSI mid layer SCSI_QUEUE_WORK(9)

NAME scsi_queue_work - Queue work to the Scsi_Host workqueue.

SYNOPSIS int scsi_queue_work(struct Scsi_Host * shost, struct work_struct * work);

ARGUMENTS shost Pointer to Scsi_Host.

work Work to queue for execution.

RETURN VALUE 1 - work queued for execution 0 - work is already queued -EINVAL - work queue doesn´t exist
