NAME register_sysctl_table - register a sysctl table hierarchy

SYNOPSIS struct ctl_table_header * register_sysctl_table(struct ctl_table * table);

ARGUMENTS table the top-level table structure

DESCRIPTION Register a sysctl table hierarchy. table should be a filled in ctl_table array. A completely 0 filled entry terminates the table.

See register_sysctl_paths for more details.
