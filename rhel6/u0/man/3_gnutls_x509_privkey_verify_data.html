NAME gnutls_x509_privkey_verify_data - Verify the signed data using private key

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_privkey_verify_data(gnutls_x509_privkey_t key, unsigned int flags, const gnutls_datum_t * data, const gnutls_datum_t * signa- ture);

ARGUMENTS gnutls_x509_privkey_t key Holds the key

unsigned int flags should be 0 for now

const gnutls_datum_t * data holds the data to be signed

const gnutls_datum_t * signature contains the signature

DESCRIPTION This function will verify the given signed data, using the parameters in the private key.

RETURNS In case of a verification failure 0 is returned, and 1 on success.
