NAME asn1_copy_node - API function

SYNOPSIS #include <libtasn1.h>

asn1_retCode asn1_copy_node(ASN1_TYPE dst, const char * dst_name, ASN1_TYPE src, const char * src_name);

ARGUMENTS ASN1_TYPE dst Destination ASN1_TYPE node.

const char * dst_name Field name in destination node.

ASN1_TYPE src Source ASN1_TYPE node.

const char * src_name Field name in source node.

DESCRIPTION Create a deep copy of a ASN1_TYPE variable.

RETURN VALUE Return ASN1_SUCCESS on success.
