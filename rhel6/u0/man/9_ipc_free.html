IPC_FREE(9) Kernel IPC facilities IPC_FREE(9)

NAME ipc_free - free ipc space

SYNOPSIS void ipc_free(void * ptr, int size);

ARGUMENTS ptr pointer returned by ipc_alloc

size size of block

DESCRIPTION Free a block created with ipc_alloc. The caller must know the size used in the allocation call.
