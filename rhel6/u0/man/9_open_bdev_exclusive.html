NAME open_bdev_exclusive - open a block device by name and set it up for use

SYNOPSIS struct block_device * open_bdev_exclusive(const char * path, fmode_t mode, void * holder);

ARGUMENTS path special file representing the block device

mode FMODE_... combination to pass be used

holder owner for exclusion

DESCRIPTION Open the blockdevice described by the special file at path, claim it for the holder.
