NAME PAPI_num_components - return the number of components currently installed

SYNOPSIS C Interface #include <papi.h> int PAPI_num_components();

DESCRIPTION PAPI_num_components() returns the number of counting components installed in the library.

ARGUMENTS This function takes no arguments.

RETURN VALUES This function returns a value greater than zero.

EXAMPLES /* Query the library for a component count. */ printf("%d components installed.0, PAPI_num_components());

BUGS None.
