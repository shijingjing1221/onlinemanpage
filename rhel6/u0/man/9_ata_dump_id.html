NAME ata_dump_id - IDENTIFY DEVICE info debugging output

SYNOPSIS void ata_dump_id(const u16 * id);

ARGUMENTS id IDENTIFY DEVICE page to dump

DESCRIPTION Dump selected 16-bit words from the given IDENTIFY DEVICE page.

LOCKING caller.
