NAME vfs_dirsort - Sort directory contents

SYNOPSIS vfs objects = dirsort

DESCRIPTION This VFS module is part of the samba(7) suite.

The vfs_dirsort module sorts directory entries alphabetically before sending them to the client.

Please be aware that adding this module might have negative performance implications for large directories.

EXAMPLES Sort directories for all shares:

[global] vfs objects = dirsort

VERSION This man page is correct for version 3.3 of the Samba suite.
