NAME ieee80211_free_hw - free hardware descriptor

SYNOPSIS void ieee80211_free_hw(struct ieee80211_hw * hw);

ARGUMENTS hw the hardware to free

DESCRIPTION This function frees everything that was allocated, including the private data for the driver. You must call ieee80211_unregister_hw before calling this function.
