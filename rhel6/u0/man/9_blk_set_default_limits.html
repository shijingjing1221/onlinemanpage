NAME blk_set_default_limits - reset limits to default values

SYNOPSIS void blk_set_default_limits(struct queue_limits * lim);

ARGUMENTS lim the queue_limits structure to reset

DESCRIPTION Returns a queue_limit struct to its default state. Can be used by stacking drivers like DM that stage table swaps and reuse an existing device queue.
