NAME d_add - add dentry to hash queues

SYNOPSIS void d_add(struct dentry * entry, struct inode * inode);

ARGUMENTS entry dentry to add

inode The inode to attach to this dentry

DESCRIPTION This adds the entry to the hash queues and initializes inode. The entry was actually filled in earlier during d_alloc.
