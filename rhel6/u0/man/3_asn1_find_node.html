NAME asn1_find_node - API function

SYNOPSIS #include <libtasn1.h>

ASN1_TYPE asn1_find_node(ASN1_TYPE pointer, const char * name);

ARGUMENTS ASN1_TYPE pointer NODE_ASN element pointer.

const char * name null terminated string with the elements name to find.

DESCRIPTION Searches for an element called NAME starting from POINTER. The name is composed by differents identifiers separated by dots. When *POINTER has a name, the first identifier must be the name of *POINTER, other- wise it must be the name of one child of *POINTER.

RETURN VALUE the searching result. NULL if not found.
