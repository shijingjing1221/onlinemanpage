NAME sata_scr_valid - test whether SCRs are accessible

SYNOPSIS int sata_scr_valid(struct ata_link * link);

ARGUMENTS link ATA link to test SCR accessibility for

DESCRIPTION Test whether SCRs are accessible for link.

LOCKING None.

RETURNS 1 if SCRs are accessible, 0 otherwise.
