PCI_PME_CAPABLE(9) Hardware Interfaces PCI_PME_CAPABLE(9)

NAME pci_pme_capable - check the capability of PCI device to generate PME#

SYNOPSIS bool pci_pme_capable(struct pci_dev * dev, pci_power_t state);

ARGUMENTS dev PCI device to handle.

state PCI state from which device will issue PME#.
