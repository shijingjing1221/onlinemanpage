NAME selinux_lsetfilecon_default - set the file context to the system defaults.

SYNOPSIS #include <selinux/selinux.h>

int selinux_lsetfilecon_default(const char *path);

DESCRIPTION selinux_lsetfilecon_default sets the file context to the system defaults.

RETURN VALUE Returns zero on success or -1 otherwise.
