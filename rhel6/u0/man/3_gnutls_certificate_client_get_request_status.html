NAME gnutls_certificate_client_get_request_status - return the certificate request status

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_certificate_client_get_request_status(gnutls_session_t ses- sion);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION Get whether client certificate is requested or not.

RETURNS 0 if the peer (server) did not request client authentication or 1 oth- erwise, or a negative value in case of error.
