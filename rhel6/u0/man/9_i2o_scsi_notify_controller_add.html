NAME i2o_scsi_notify_controller_add - Retrieve notifications of added controllers

SYNOPSIS void i2o_scsi_notify_controller_add(struct i2o_controller * c);

ARGUMENTS c the controller which was added

DESCRIPTION If a I2O controller is added, we catch the notification to add a corresponding Scsi_Host.
