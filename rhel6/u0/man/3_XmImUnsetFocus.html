NAME XmImUnsetFocus An input manager function that notifies an input method that a widget has lost input focus

SYNOPSIS #include <Xm/XmIm.h> void XmImUnsetFocus( Widget widget);

DESCRIPTION XmImUnsetFocus unsets a specified widgets focus, then notifies the input manager that the specified widget has lost its input focus.

Note that the Text, TextField, and List widgets already call the XmImUnsetFocus internally. Therefore, further calls to the XmImUnsetFo- cus function for those widgets are unnecessary.

widget Specifies the ID of the widget registered with the input man- ager

RELATED XmImSetFocusValues(3) and XmImVaSetFocusValues(3).
