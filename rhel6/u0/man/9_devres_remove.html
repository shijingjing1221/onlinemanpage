NAME devres_remove - Find a device resource and remove it

SYNOPSIS void * devres_remove(struct device * dev, dr_release_t release, dr_match_t match, void * match_data);

ARGUMENTS dev Device to find resource from

release Look for resources associated with this release function

match Match function (optional)

match_data Data for the match function

DESCRIPTION Find the latest devres of dev associated with release and for which match returns 1. If match is NULL, it´s considered to match all. If found, the resource is removed atomically and returned.

RETURNS Pointer to removed devres on success, NULL if not found.
