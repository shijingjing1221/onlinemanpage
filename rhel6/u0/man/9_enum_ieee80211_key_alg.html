NAME enum_ieee80211_key_alg - key algorithm

SYNOPSIS enum ieee80211_key_alg { ALG_WEP, ALG_TKIP, ALG_CCMP, ALG_AES_CMAC };

CONSTANTS ALG_WEP WEP40 or WEP104

ALG_TKIP TKIP

ALG_CCMP CCMP (AES)

ALG_AES_CMAC AES-128-CMAC
