PCI_INTX(9) Hardware Interfaces PCI_INTX(9)

NAME pci_intx - enables/disables PCI INTx for device dev

SYNOPSIS void pci_intx(struct pci_dev * pdev, int enable);

ARGUMENTS pdev the PCI device to operate on

enable boolean: whether to enable or disable PCI INTx

DESCRIPTION Enables/disables PCI INTx for device dev
