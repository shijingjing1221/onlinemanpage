NAME SDL_WM_ToggleFullScreen - Toggles fullscreen mode

SYNOPSIS #include "SDL.h"

int SDL_WM_ToggleFullScreen(SDL_Surface *surface);

DESCRIPTION Toggles the application between windowed and fullscreen mode, if sup- ported. (X11 is the only target currently supported, BeOS support is experimental).

RETURN VALUE Returns 0 on failure or 1 on success.
