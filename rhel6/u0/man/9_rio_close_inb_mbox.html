NAME rio_close_inb_mbox - Shut down MPC85xx inbound mailbox

SYNOPSIS void rio_close_inb_mbox(struct rio_mport * mport, int mbox);

ARGUMENTS mport Master port implementing the inbound message unit

mbox Mailbox to close

DESCRIPTION Disables the inbound message unit, free all buffers, and frees the inbound message interrupt.
