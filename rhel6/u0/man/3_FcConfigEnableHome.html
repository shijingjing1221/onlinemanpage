NAME FcConfigEnableHome - controls use of the home directory.

SYNOPSIS #include <fontconfig.h>

FcBol FcConfigEnableHome(FcBool enable); .fi

DESCRIPTION If enable is FcTrue, then Fontconfig will use various files which are specified relative to the users home directory (using the ~ notation in the configuration). When enable is FcFalse, then all use of the home directory in these contexts will be disabled. The previous setting of the value is returned.

VERSION Fontconfig version 2.8.0
