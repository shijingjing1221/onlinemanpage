USB_ANCHOR_URB(9) USB Core APIs USB_ANCHOR_URB(9)

NAME usb_anchor_urb - anchors an URB while it is processed

SYNOPSIS void usb_anchor_urb(struct urb * urb, struct usb_anchor * anchor);

ARGUMENTS urb pointer to the urb to anchor

anchor pointer to the anchor

DESCRIPTION This can be called to have access to URBs which are to be executed without bothering to track them
