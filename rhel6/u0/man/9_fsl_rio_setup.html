NAME fsl_rio_setup - Setup Freescale PowerPC RapidIO interface

SYNOPSIS int fsl_rio_setup(struct of_device * dev);

ARGUMENTS dev of_device pointer

DESCRIPTION Initializes MPC85xx RapidIO hardware interface, configures master port with system-specific info, and registers the master port with the RapidIO subsystem.
