NAME cas - Tool to configure core analysis environment.

DESCRIPTION cas is a tool used to automatically configure an environment for viewing coredumps.

OPTIONS -i identifier Specify an id to associate with current cas job, normally a sup- port ticket of some sort.

-f file Define a core file to be processed.

-e email Provide an email address for result notification

-m Extract current loaded modules from debug kernels.

RESOURCES A configuration file is maintained in /etc/cas.conf.
