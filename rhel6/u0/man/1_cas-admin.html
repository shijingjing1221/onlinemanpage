NAME cas-admin - Administrator tools for building rpm/server database.

DESCRIPTION cas-admin is a tool used to build necessary data for use with cas.

OPTIONS -s Populate server database for automatic processing of core files.

-b Populate a database with debug kernel information.

-p Purge data based on set days (see purge options)

PURGE OPTIONS -d Define how many days back you wish to purge data, default is 90 days.

RESOURCES A configuration file is maintained in /etc/cas.conf.
