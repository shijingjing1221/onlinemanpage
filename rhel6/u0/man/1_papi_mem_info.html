NAME papi_mem_info - provides information on the memory architecture of the current processor.

SYNOPSIS papi_mem_info

DESCRIPTION papi_mem_info is a PAPI utility program that reports information about the cache memory architecture of the current processor, including num- ber, types, sizes and associativities of instruction and data caches and Translation Lookaside Buffers.

OPTIONS This utility has no command line options.

BUGS There are no known bugs in this utility.

If you find a bug, it should be reported to the PAPI Mailing List at <ptools-perfapi@ptools.org>.
