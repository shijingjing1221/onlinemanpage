NAME git-stripspace - Filter out empty lines

SYNOPSIS git stripspace [-s | --strip-comments] < <stream>

DESCRIPTION Remove multiple empty lines, and empty lines at beginning and end.

OPTIONS -s, --strip-comments In addition to empty lines, also strip lines starting with #.

<stream> Byte stream to act on.
