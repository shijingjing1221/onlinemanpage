CLK_ENABLE(9) Clock Framework CLK_ENABLE(9)

NAME clk_enable - inform the system when the clock source should be running.

SYNOPSIS int clk_enable(struct clk * clk);

ARGUMENTS clk clock source

DESCRIPTION If the clock can not be enabled/disabled, this should return success.

Returns success (0) or negative errno.
