NAME FcConfigGetFonts - Get config font set

SYNOPSIS #include <fontconfig.h>

FcFontSet * FcConfigGetFonts(FcConfig *config); (FcSetName set); .fi

DESCRIPTION Returns one of the two sets of fonts from the configuration as speci- fied by set. This font set is owned by the library and must not be mod- ified or freed. If config is NULL, the current configuration is used.

VERSION Fontconfig version 2.8.0
