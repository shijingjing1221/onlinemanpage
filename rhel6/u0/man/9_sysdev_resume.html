SYSDEV_RESUME(9) Device drivers infrastructure SYSDEV_RESUME(9)

NAME sysdev_resume - Bring system devices back to life.

SYNOPSIS int sysdev_resume(void);

ARGUMENTS void no arguments

DESCRIPTION Similar to sysdev_suspend, but we iterate the list forwards to guarantee that parent devices are resumed before their children.

NOTE Interrupts are disabled when called.
