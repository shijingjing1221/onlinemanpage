AUDIT_SOCKADDR(9) Audit Interfaces AUDIT_SOCKADDR(9)

NAME audit_sockaddr - record audit data for sys_bind, sys_connect, sys_sendto

SYNOPSIS int audit_sockaddr(int len, void * a);

ARGUMENTS len data length in user space

a data address in kernel space

DESCRIPTION Returns 0 for success or NULL context or < 0 on error.
