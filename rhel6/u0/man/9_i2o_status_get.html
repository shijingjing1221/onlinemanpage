I2O_STATUS_GET(9) Message-based devices I2O_STATUS_GET(9)

NAME i2o_status_get - Get the status block from the I2O controller

SYNOPSIS int i2o_status_get(struct i2o_controller * c);

ARGUMENTS c I2O controller

DESCRIPTION Issue a status query on the controller. This updates the attached status block. The status block could then be accessed through c->status_block.

Returns 0 on success or negative error code on failure.
