NAME i2o_iop_quiesce_all - Quiesce all I2O controllers on the system

SYNOPSIS void i2o_iop_quiesce_all(void);

ARGUMENTS void no arguments

DESCRIPTION Quiesce all I2O controllers which are connected to the system.
