D_MOVE(9) The Linux VFS D_MOVE(9)

NAME d_move - move a dentry

SYNOPSIS void d_move(struct dentry * dentry, struct dentry * target);

ARGUMENTS dentry entry to move

target new dentry

DESCRIPTION Update the dcache to reflect the move of a file name. Negative dcache entries should not be moved in this way.
