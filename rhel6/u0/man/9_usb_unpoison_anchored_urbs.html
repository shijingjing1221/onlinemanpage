NAME usb_unpoison_anchored_urbs - let an anchor be used successfully again

SYNOPSIS void usb_unpoison_anchored_urbs(struct usb_anchor * anchor);

ARGUMENTS anchor anchor the requests are bound to

DESCRIPTION Reverses the effect of usb_poison_anchored_urbs the anchor can be used normally after it returns
