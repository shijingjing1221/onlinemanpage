NAME mpt_set_taskmgmt_in_progress_flag - set flags associated with task management

SYNOPSIS int mpt_set_taskmgmt_in_progress_flag(MPT_ADAPTER * ioc);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

DESCRIPTION Returns 0 for SUCCESS or -1 if FAILED.

If -1 is return, then it was not possible to set the flags
