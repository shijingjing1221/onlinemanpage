NAME gnutls_sign_algorithm_get_name - Returns a string with the name of the specified sign algorithm

SYNOPSIS #include <gnutls/gnutls.h>

const char * gnutls_sign_algorithm_get_name(gnutls_sign_algorithm_t sign);

ARGUMENTS gnutls_sign_algorithm_t sign is a sign algorithm

DESCRIPTION Convert a gnutls_sign_algorithm_t value to a string.

RETURNS a string that contains the name of the specified sign algorithm, or NULL.
