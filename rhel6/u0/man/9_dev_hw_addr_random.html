DEV_HW_ADDR_RANDOM(9) Network device support DEV_HW_ADDR_RANDOM(9)

NAME dev_hw_addr_random - Create random MAC and set device flag

SYNOPSIS void dev_hw_addr_random(struct net_device * dev, u8 * hwaddr);

ARGUMENTS dev pointer to net_device structure

hwaddr -- undescribed --

DESCRIPTION Generate random MAC to be used by a device and set addr_assign_type so the state can be read by sysfs and be used by udev.
