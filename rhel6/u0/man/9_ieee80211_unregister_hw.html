NAME ieee80211_unregister_hw - Unregister a hardware device

SYNOPSIS void ieee80211_unregister_hw(struct ieee80211_hw * hw);

ARGUMENTS hw the hardware to unregister

DESCRIPTION This function instructs mac80211 to free allocated resources and unregister netdevices from the networking subsystem.
