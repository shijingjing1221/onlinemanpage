NAME tracehook_report_exit - task has begun to exit

SYNOPSIS void tracehook_report_exit(long * exit_code);

ARGUMENTS exit_code pointer to value destined for current->exit_code

DESCRIPTION exit_code points to the value passed to do_exit, which tracing might change here. This is almost the first thing in do_exit, before freeing any resources or setting the PF_EXITING flag.

Called with no locks held.
