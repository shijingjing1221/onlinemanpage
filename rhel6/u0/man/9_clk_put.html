CLK_PUT(9) Clock Framework CLK_PUT(9)

NAME clk_put - "free" the clock source

SYNOPSIS void clk_put(struct clk * clk);

ARGUMENTS clk clock source

NOTE drivers must ensure that all clk_enable calls made on this clock source are balanced by clk_disable calls prior to calling this function.

clk_put should not be called from within interrupt context.
