NAME function::stringat - Returns the char at a given position in the string.

SYNOPSIS stringat:long(str:string,pos:long)

ARGUMENTS str The string to fetch the character from.

pos The position to get the character from. 0 = start of the string.

DESCRIPTION Returns the character at a given position in the string or zero if the string doesn´t have as many characters.
