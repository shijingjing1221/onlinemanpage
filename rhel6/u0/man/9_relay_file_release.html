NAME relay_file_release - release file op for relay files

SYNOPSIS int relay_file_release(struct inode * inode, struct file * filp);

ARGUMENTS inode the inode

filp the file

DESCRIPTION Decrements the channel refcount, as the filesystem is no longer using it.
