NAME seq_write - write arbitrary data to buffer

SYNOPSIS int seq_write(struct seq_file * seq, const void * data, size_t len);

ARGUMENTS seq seq_file identifying the buffer to which data should be written

data data address

len number of bytes

DESCRIPTION Return 0 on success, non-zero otherwise.
