SAS_REMOVE_HOST(9) SCSI mid layer SAS_REMOVE_HOST(9)

NAME sas_remove_host - tear down a Scsi_Host´s SAS data structures

SYNOPSIS void sas_remove_host(struct Scsi_Host * shost);

ARGUMENTS shost Scsi Host that is torn down

DESCRIPTION Removes all SAS PHYs and remote PHYs for a given Scsi_Host. Must be called just before scsi_remove_host for SAS HBAs.
