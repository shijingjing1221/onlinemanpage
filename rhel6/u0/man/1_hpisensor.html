NAME hpisensor - This sample openhpi application displays sensor info for all resources with Sensor Capability

SYNOPSIS hpisensor [-t -r -x -e]

DESCRIPTION hpisensor displays sensor info for all resources with Sensor Capability .

OPTIONS -t = show Thresholds also -r = show Range values also -s = show EventState also -e entity path = limit to a single entity -x = show eXtra debug messages
