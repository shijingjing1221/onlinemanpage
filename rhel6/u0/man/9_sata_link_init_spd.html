NAME sata_link_init_spd - Initialize link->sata_spd_limit

SYNOPSIS int sata_link_init_spd(struct ata_link * link);

ARGUMENTS link Link to configure sata_spd_limit for

DESCRIPTION Initialize link->[hw_]sata_spd_limit to the currently configured value.

LOCKING Kernel thread context (may sleep).

RETURNS 0 on success, -errno on failure.
