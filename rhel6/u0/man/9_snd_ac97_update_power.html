SND_AC97_UPDATE_POWE(9) Control/Mixer API SND_AC97_UPDATE_POWE(9)

NAME snd_ac97_update_power - update the powerdown register

SYNOPSIS int snd_ac97_update_power(struct snd_ac97 * ac97, int reg, int powerup);

ARGUMENTS ac97 the codec instance

reg the rate register, e.g. AC97_PCM_FRONT_DAC_RATE

powerup non-zero when power up the part

DESCRIPTION Update the AC97 powerdown register bits of the given part.
