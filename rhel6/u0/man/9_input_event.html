INPUT_EVENT(9) Input Subsystem INPUT_EVENT(9)

NAME input_event - report new input event

SYNOPSIS void input_event(struct input_dev * dev, unsigned int type, unsigned int code, int value);

ARGUMENTS dev device that generated the event

type type of the event

code event code

value value of the event

DESCRIPTION This function should be used by drivers implementing various input devices. See also input_inject_event.
