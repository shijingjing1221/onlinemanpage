NAME function::proc_mem_string - Human readable string of current proc memory usage

SYNOPSIS proc_mem_string:string()

ARGUMENTS None

DESCRIPTION Returns a human readable string showing the size, rss, shr, txt and data of the memory used by the current process. For example size: 301m, rss: 11m, shr: 8m, txt: 52k, data: 2248k.
