NAME schedule_delayed_work - put work task in global workqueue after delay

SYNOPSIS int schedule_delayed_work(struct delayed_work * dwork, unsigned long delay);

ARGUMENTS dwork job to be done

delay number of jiffies to wait or 0 for immediate execution

DESCRIPTION After waiting for a given time this puts a job in the kernel-global workqueue.
