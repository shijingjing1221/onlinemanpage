NAME getgid - get the real group ID

SYNOPSIS #include <unistd.h>

gid_t getgid(void);

DESCRIPTION The getgid() function shall return the real group ID of the calling process.

RETURN VALUE The getgid() function shall always be successful and no return value is reserved to indicate an error.

ERRORS No errors are defined.
