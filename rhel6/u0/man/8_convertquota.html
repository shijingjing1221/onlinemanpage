NAME convertquota - convert quota from old file format to new one

SYNOPSIS convertquota [ -ug ] [ -e | -f ] filesystem

DESCRIPTION convertquota converts old quota files quota.user and quota.group to files aquota.user and aquota.group in new format currently used by 2.4.0-ac? and newer or by SuSE or Red Hat Linux 2.4 kernels on filesys- tem.

New file format allows using quotas for 32-bit uids / gids, setting quotas for root, accounting used space in bytes (and so allowing use of quotas in ReiserFS) and it is also architecture independent. This for- mat introduces Radix Tree (a simple form of tree structure) to quota file.

OPTIONS -u, --user convert user quota file. This is the default.

-g, --group convert group quota file.

-f, --convert-format convert from old file format to new one. This is the default.

-e, --convert-endian convert new file format from big endian to little endian.

-V, --version print version information.

FILES aquota.user new user quota file

aquota.group new group quota file
