NAME function::caller - Return name and address of calling function

SYNOPSIS caller:string()

ARGUMENTS None

DESCRIPTION Return the address and name of the calling function.

THIS IS EQUIVALENT TO CALLING sprintf( s 0xx , symname(caller_addr, caller_addr)) Works only for return probes at this time.
