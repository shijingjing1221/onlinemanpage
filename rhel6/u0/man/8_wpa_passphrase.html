NAME wpa_passphrase - Generate a WPA PSK from an ASCII passphrase for a SSID

SYNOPSIS wpa_passphrase [ ssid ] [ passphrase ]

OVERVIEW wpa_passphrase pre-computes PSK entries for network configuration blocks of a wpa_supplicant.conf file. An ASCII passphrase and SSID are used to generate a 256-bit PSK.

OPTIONS ssid The SSID whose passphrase should be derived.

passphrase The passphrase to use. If not included on the command line, passphrase will be read from standard input.
