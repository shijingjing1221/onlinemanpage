NAME mcstransd - MCS (Multiple Category System) daemon. Translates SELinux MCS/MLS labels to human readable form.

SYNOPSIS mcstransd

DESCRIPTION This manual page describes the mcstransd program.

This daemon reads /etc/selinux/{SELINUXTYPE}/setrans.conf configuration file, and communicates with libselinux via a socket in /var/run/setrans.
