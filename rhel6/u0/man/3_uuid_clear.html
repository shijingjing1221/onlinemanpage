UUID_CLEAR(3) Libuuid API UUID_CLEAR(3)

NAME uuid_clear - reset value of UUID variable to the NULL value

SYNOPSIS #include <uuid/uuid.h>

void uuid_clear(uuid_t uu);

DESCRIPTION The uuid_clear function sets the value of the supplied uuid variable uu to the NULL value.

AUTHOR Theodore Y. Tso

AVAILABILITY libuuid is part of the util-linux-ng package since version 2.15.1 and is available from ftp://ftp.kernel.org/pub/linux/utils/util-linux-ng/.

SEE ALSO uuid(3), uuid_compare(3), uuid_copy(3), uuid_generate(3), uuid_is_null(3), uuid_parse(3), uuid_unparse(3)
