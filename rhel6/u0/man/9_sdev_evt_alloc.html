SDEV_EVT_ALLOC(9) SCSI mid layer SDEV_EVT_ALLOC(9)

NAME sdev_evt_alloc - allocate a new scsi event

SYNOPSIS struct scsi_event * sdev_evt_alloc(enum scsi_device_event evt_type, gfp_t gfpflags);

ARGUMENTS evt_type type of event to allocate

gfpflags GFP flags for allocation

DESCRIPTION Allocates and returns a new scsi_event.
