NAME lssubsys - list hierarchies containing given subsystem

SYNOPSIS lssubsys [-m|--mount-points] [-a|--all] [controller] [...] lssubsys [-h|--help]

DESCRIPTION The command list hierarchies containing all subsystems or only given subsystem(s).

controller defines subsystem which hierarchy will be shown. If this param- eter is not used then the command list all subsystems which are present.

-m, --mount-points Display mount points.

-a, --all Display not mounted subsystems.

-h, --help Display help and exit.
