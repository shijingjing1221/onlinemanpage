NAME pci_bus_add_devices - insert newly discovered PCI devices

SYNOPSIS void pci_bus_add_devices(const struct pci_bus * bus);

ARGUMENTS bus bus to check for new devices

DESCRIPTION Add newly discovered PCI devices (which are on the bus->devices list) to the global PCI device list, add the sysfs and procfs entries. Where a bridge is found, add the discovered bus to the parents list of child buses, and recurse (breadth-first to be compatible with 2.4)

Call hotplug for each new devices.
