NAME piix_set_dmamode - Initialize host controller PATA DMA timings

SYNOPSIS void piix_set_dmamode(struct ata_port * ap, struct ata_device * adev);

ARGUMENTS ap Port whose timings we are configuring

adev um

DESCRIPTION Set MW/UDMA mode for device, in host controller PCI config space.

LOCKING None (inherited from caller).
