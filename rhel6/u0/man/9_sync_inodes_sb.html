NAME sync_inodes_sb - sync sb inode pages

SYNOPSIS void sync_inodes_sb(struct super_block * sb);

ARGUMENTS sb the superblock

DESCRIPTION This function writes and waits on any dirty inode belonging to this super_block. The number of pages synced is returned.
