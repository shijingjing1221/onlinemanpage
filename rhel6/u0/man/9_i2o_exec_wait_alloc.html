I2O_EXEC_WAIT_ALLOC(9) Message-based devices I2O_EXEC_WAIT_ALLOC(9)

NAME i2o_exec_wait_alloc - Allocate a i2o_exec_wait struct an initialize it

SYNOPSIS struct i2o_exec_wait * i2o_exec_wait_alloc(void);

ARGUMENTS void no arguments

DESCRIPTION Allocate the i2o_exec_wait struct and initialize the wait.

Returns i2o_exec_wait pointer on success or negative error code on failure.
