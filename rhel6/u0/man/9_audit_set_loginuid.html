NAME audit_set_loginuid - set a task´s audit_context loginuid

SYNOPSIS int audit_set_loginuid(struct task_struct * task, uid_t loginuid);

ARGUMENTS task task whose audit context is being modified

loginuid loginuid value

DESCRIPTION Returns 0.

Called (set) from fs/proc/base.c::proc_loginuid_write.
