NAME piix_check_450nx_errata - Check for problem 450NX setup

SYNOPSIS int piix_check_450nx_errata(struct pci_dev * ata_dev);

ARGUMENTS ata_dev the PCI device to check

DESCRIPTION Check for the present of 450NX errata #19 and errata #25. If they are found return an error code so we can turn off DMA
