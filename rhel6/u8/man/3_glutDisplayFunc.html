NAME glutDisplayFunc - Sets the Display callback for the current window.

LIBRARY OpenGLUT - windowcallback

SYNOPSIS #include <openglut.h>

void glutDisplayFunc(void( *callback )( void ));

PARAMETERS callback Client function for normal redisplay event.

DESCRIPTION Sets the display callback for the current window . All windows, including subwindows, must have a display callback registered. OpenG- LUT will call the callback function whenever it thinks that the window may require updating.

This callback is bound to the current window .

CAVEATS Unlike most callbacks, it is illegal to try to disable the display call- back by setting it to NULL .

Multiple redisplays may be coalesced into a single event for invoking the callback only once.

SEE ALSO glutPostRedisplay(3) glutOverlayDisplayFunc(3)
