IPC_RMID(9) Kernel IPC facilities IPC_RMID(9)

NAME ipc_rmid - remove an IPC identifier

SYNOPSIS void ipc_rmid(struct ipc_ids * ids, struct kern_ipc_perm * ipcp);

ARGUMENTS ids IPC identifier set

ipcp ipc perm structure containing the identifier to remove

DESCRIPTION ipc_ids.rw_mutex (as a writer) and the spinlock for this ID are held before this function is called, and remain locked on the exit.
