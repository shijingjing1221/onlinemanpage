NAME bio_kmalloc - allocate a bio for I/O using kmalloc

SYNOPSIS struct bio * bio_kmalloc(gfp_t gfp_mask, int nr_iovecs);

ARGUMENTS gfp_mask the GFP_ mask given to the slab allocator

nr_iovecs number of iovecs to pre-allocate

DESCRIPTION Allocate a new bio with nr_iovecs bvecs. If gfp_mask contains __GFP_WAIT, the allocation is guaranteed to succeed.
