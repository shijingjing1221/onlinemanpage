NAME acm_cdc_notify - issue CDC notification to host

SYNOPSIS int acm_cdc_notify(struct f_acm * acm, u8 type, u16 value, void * data, unsigned length);

ARGUMENTS acm wraps host to be notified

type notification type

value Refer to cdc specs, wValue field.

data data to be sent

length size of data

CONTEXT irqs blocked, acm->lock held, acm_notify_req non-null

DESCRIPTION Returns zero on sucess or a negative errno.

See section 6.3.5 of the CDC 1.1 specification for information

ABOUT THE ONLY NOTIFICATION WE ISSUE SerialState change.
