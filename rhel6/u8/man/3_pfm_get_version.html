NAME pfm_get_version - get performance monitoring library version

SYNOPSIS #include <perfmon/pfmlib.h>

int pfm_get_version(unsigned int *version);

DESCRIPTION This function can be called at any time to get the revision level of the library. The version is encoded into an unsigned integer and returned in the version argument. A revision number is composed of two fields: a major number and a minor number. Both can be extracted from the returned argument using macros provided in the header file:

PFMLIB_MAJ_VERSION(v) returns the major number encoded in v.

PFMLIB_MIN_VERSION(v) returns the minor number encoded in v.

RETURN The function returns whether or not it was successful. A return value of PFMLIB_SUCCESS indicates success, otherwise the value is the error code.

ERRORS PFMLIB_ERR_INVAL the argument is invalid, most likely a NULL pointer.
