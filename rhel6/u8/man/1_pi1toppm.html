NAME pi1toppm - convert an Atari Degas .pi1 into a PPM image

SYNOPSIS pi1toppm

[pi1file]

DESCRIPTION This program is part of Netpbm(1).

pi1toppm reads an Atari Degas .pi1 file as input and produces a PPM image as output.

SEE ALSO ppmtopi1(1), pc1toppm(1), ppm(1), pi3topbm(1), pbmtopi3(1)
