NAME function::proc_mem_shr - Program shared pages (from shared mappings)

SYNOPSIS proc_mem_shr:long()

ARGUMENTS None

DESCRIPTION Returns the shared pages (from shared mappings) of the current process, or zero when there is no current process or the number of pages couldn´t be retrieved.
