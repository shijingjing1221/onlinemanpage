ETHER_ADDR_EQUAL(9) Network device support ETHER_ADDR_EQUAL(9)

NAME ether_addr_equal - Compare two Ethernet addresses

SYNOPSIS bool ether_addr_equal(const u8 * addr1, const u8 * addr2);

ARGUMENTS addr1 Pointer to a six-byte array containing the Ethernet address

addr2 Pointer other six-byte array containing the Ethernet address

DESCRIPTION Compare two ethernet addresses, returns true if equal
