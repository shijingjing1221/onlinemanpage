NAME scsi_schedule_eh - schedule EH for SCSI host

SYNOPSIS void scsi_schedule_eh(struct Scsi_Host * shost);

ARGUMENTS shost SCSI host to invoke error handling on.

DESCRIPTION Schedule SCSI EH without scmd.
