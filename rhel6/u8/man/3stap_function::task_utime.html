FUNCTION::TASK_UTIME(3stap) Task Time Tapset FUNCTION::TASK_UTIME(3stap)

NAME function::task_utime - User time of the current task

SYNOPSIS task_utime:long()

ARGUMENTS None

DESCRIPTION Returns the user time of the current task in cputime. Does not include any time used by other tasks in this process, nor does it include any time of the children of this task.
