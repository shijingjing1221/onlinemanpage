I2O_IOP_ENABLE(9) Message-based devices I2O_IOP_ENABLE(9)

NAME i2o_iop_enable - move controller from ready to OPERATIONAL

SYNOPSIS int i2o_iop_enable(struct i2o_controller * c);

ARGUMENTS c I2O controller

DESCRIPTION Enable IOP. This allows the IOP to resume external operations and reverses the effect of a quiesce. Returns zero or an error code if an error occurs.
