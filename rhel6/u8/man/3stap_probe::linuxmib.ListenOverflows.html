NAME probe::linuxmib.ListenOverflows - Count of times a listen queue overflowed

SYNOPSIS linuxmib.ListenOverflows

VALUES sk Pointer to the struct sock being acted on

op Value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function linuxmib_filter_key. If the packet passes the filter is is counted in the global ListenOverflows (equivalent to SNMP´s MIB LINUX_MIB_LISTENOVERFLOWS)
