NAME user_enable_block_step - step until branch in user-mode task

SYNOPSIS void user_enable_block_step(struct task_struct * task);

ARGUMENTS task either current or a task stopped in TASK_TRACED

DESCRIPTION This can only be called when arch_has_block_step has returned nonzero, and will never be called when single-instruction stepping is being used. Set task so that when it returns to user mode, it will trap after the next branch or trap taken.
