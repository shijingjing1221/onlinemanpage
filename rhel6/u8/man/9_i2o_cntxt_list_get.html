I2O_CNTXT_LIST_GET(9) Message-based devices I2O_CNTXT_LIST_GET(9)

NAME i2o_cntxt_list_get - Get a pointer from the context list and remove it

SYNOPSIS void * i2o_cntxt_list_get(struct i2o_controller * c, u32 context);

ARGUMENTS c controller to which the context list belong

context context id to which the pointer belong

DESCRIPTION Returns pointer to the matching context id on success or NULL on failure.
