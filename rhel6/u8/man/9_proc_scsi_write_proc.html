PROC_SCSI_WRITE_PROC(9) SCSI mid layer PROC_SCSI_WRITE_PROC(9)

NAME proc_scsi_write_proc - Handle write to /proc by calling host´s proc_info

SYNOPSIS int proc_scsi_write_proc(struct file * file, const char __user * buf, unsigned long count, void * data);

ARGUMENTS file not used

buf source of data to write.

count number of bytes (at most PROC_BLOCK_SIZE) to write.

data pointer to struct Scsi_Host
