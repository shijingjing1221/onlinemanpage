NAME kgdb_disable_hw_debug - (optional) Disable hardware debugging hook

SYNOPSIS void kgdb_disable_hw_debug(struct pt_regs * regs);

ARGUMENTS regs Current struct pt_regs.

DESCRIPTION This function will be called if the particular architecture must disable hardware debugging while it is processing gdb packets or handling exception.
