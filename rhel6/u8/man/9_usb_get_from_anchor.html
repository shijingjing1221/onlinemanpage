NAME usb_get_from_anchor - get an anchor´s oldest urb

SYNOPSIS struct urb * usb_get_from_anchor(struct usb_anchor * anchor);

ARGUMENTS anchor the anchor whose urb you want

DESCRIPTION this will take the oldest urb from an anchor, unanchor and return it
