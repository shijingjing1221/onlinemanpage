NAME gnutls_openpgp_crt_get_key_id - Gets the keyID

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_crt_get_key_id(gnutls_openpgp_crt_t key, gnutls_openpgp_keyid_t keyid);

ARGUMENTS gnutls_openpgp_crt_t key the structure that contains the OpenPGP public key.

gnutls_openpgp_keyid_t keyid the buffer to save the keyid.

DESCRIPTION Get key id string.

RETURNS the 64-bit keyID of the OpenPGP key.

SINCE 2.4.0
