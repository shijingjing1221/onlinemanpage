NAME groups - print the groups a user is in

SYNOPSIS groups [OPTION]... [USERNAME]...

DESCRIPTION Print group memberships for each USERNAME or, if no USERNAME is speci- fied, for the current process (which may differ if the groups database has changed).

--help display this help and exit

--version output version information and exit
