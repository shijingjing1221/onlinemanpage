NAME neotoppm - convert an Atari Neochrome .neo into a PPM image

SYNOPSIS neotoppm [neofile]

DESCRIPTION This program is part of Netpbm(1).

neotoppm reads an Atari Neochrome .neo file as input and produces a PPM image as output.

SEE ALSO ppmtoneo(1), ppm(1)
