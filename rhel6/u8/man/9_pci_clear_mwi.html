PCI_CLEAR_MWI(9) Hardware Interfaces PCI_CLEAR_MWI(9)

NAME pci_clear_mwi - disables Memory-Write-Invalidate for device dev

SYNOPSIS void pci_clear_mwi(struct pci_dev * dev);

ARGUMENTS dev the PCI device to disable

DESCRIPTION Disables PCI Memory-Write-Invalidate transaction on the device
