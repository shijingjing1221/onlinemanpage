MEMZERO_EXPLICIT(9) Basic C Library Functions MEMZERO_EXPLICIT(9)

NAME memzero_explicit - Fill a region of memory (e.g. sensitive keying data) with 0s.

SYNOPSIS void memzero_explicit(void * s, size_t count);

ARGUMENTS s Pointer to the start of the area.

count The size of the area.

DESCRIPTION memzero_explicit doesn´t need an arch-specific version as it just invokes the one of memset implicitly.
