NAND_WRITE(9) Internal Functions Provided NAND_WRITE(9)

NAME nand_write - [MTD Interface] NAND write with ECC

SYNOPSIS int nand_write(struct mtd_info * mtd, loff_t to, size_t len, size_t * retlen, const uint8_t * buf);

ARGUMENTS mtd MTD device structure

to offset to write to

len number of bytes to write

retlen pointer to variable to store the number of written bytes

buf the data to write

DESCRIPTION NAND write with ECC
