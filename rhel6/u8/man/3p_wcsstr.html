NAME wcsstr - find a wide-character substring

SYNOPSIS #include <wchar.h>

wchar_t *wcsstr(const wchar_t *restrict ws1, const wchar_t *restrict ws2);

DESCRIPTION The wcsstr() function shall locate the first occurrence in the wide- character string pointed to by ws1 of the sequence of wide characters (excluding the terminating null wide character) in the wide-character string pointed to by ws2.

RETURN VALUE Upon successful completion, wcsstr() shall return a pointer to the located wide-character string, or a null pointer if the wide-character string is not found.

If ws2 points to a wide-character string with zero length, the function shall return ws1.

ERRORS No errors are defined.
