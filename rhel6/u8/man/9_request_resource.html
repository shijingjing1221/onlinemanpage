REQUEST_RESOURCE(9) Hardware Interfaces REQUEST_RESOURCE(9)

NAME request_resource - request and reserve an I/O or memory resource

SYNOPSIS int request_resource(struct resource * root, struct resource * new);

ARGUMENTS root root resource descriptor

new resource descriptor desired by caller

DESCRIPTION Returns 0 for success, negative error code on error.
