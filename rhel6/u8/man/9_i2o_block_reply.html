I2O_BLOCK_REPLY(9) Message-based devices I2O_BLOCK_REPLY(9)

NAME i2o_block_reply - Block OSM reply handler.

SYNOPSIS int i2o_block_reply(struct i2o_controller * c, u32 m, struct i2o_message * msg);

ARGUMENTS c I2O controller from which the message arrives

m message id of reply

msg the actual I2O message reply

DESCRIPTION This function gets all the message replies.
