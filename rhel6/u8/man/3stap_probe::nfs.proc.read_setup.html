NAME probe::nfs.proc.read_setup - NFS client setting up a read RPC task

SYNOPSIS nfs.proc.read_setup

VALUES count read bytes in this execution

version NFS version

prot transfer protocol

size read bytes in this execution

offset the file offset

server_ip IP address of server

DESCRIPTION The read_setup function is used to setup a read RPC task. It is not doing the actual read operation.
