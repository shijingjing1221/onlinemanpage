NAME probe::ioscheduler_trace.unplug_io - Fires when a request queue is unplugged;

SYNOPSIS ioscheduler_trace.unplug_io

VALUES name Name of the probe point

rq_queue request queue

DESCRIPTION Either, when number of pending requests in the queue exceeds threshold or, upon expiration of timer that was activated when queue was plugged.
