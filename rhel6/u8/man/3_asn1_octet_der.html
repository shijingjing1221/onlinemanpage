NAME asn1_octet_der - API function

SYNOPSIS #include <libtasn1.h>

void asn1_octet_der(const unsigned char * str, int str_len, unsigned char * der, int * der_len);

ARGUMENTS const unsigned char * str OCTET string.

int str_len STR length (str[0]..str[str_len-1]).

unsigned char * der string returned.

int * der_len number of meaningful bytes of DER (der[0]..der[ans_len-1]).

DESCRIPTION Creates the DER coding for an OCTET type (length included).
