NAME pwdhash - Generator of password hashes

SYNOPSIS pwdhash [-D config-dir] [-H] [-s scheme | -c comparepwd] password

DESCRIPTION Generates password hashes which can also be used in LDIF password fields. This uses the Directory Server password generator.

OPTIONS A summary of options is included below:

-H Show summary of options.

-s scheme Password scheme to be used (e.g. MD5, SHA1, SHA256, SHA512, SSHA, SSHA256, SSHA512)

-c comparepassword Password to be compared against

-D configdir Takes the password schema directly from the ns-slapd configura- tion
