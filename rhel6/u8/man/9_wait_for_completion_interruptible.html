NAME wait_for_completion_interruptible - waits for completion of a task (w/intr)

SYNOPSIS int __sched wait_for_completion_interruptible(struct completion * x);

ARGUMENTS x holds the state of this particular completion

DESCRIPTION This waits for completion of a specific task to be signaled. It is interruptible.
