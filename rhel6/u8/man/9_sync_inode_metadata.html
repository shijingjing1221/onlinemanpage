NAME sync_inode_metadata - write an inode to disk

SYNOPSIS int sync_inode_metadata(struct inode * inode, int wait);

ARGUMENTS inode the inode to sync

wait wait for I/O to complete.

DESCRIPTION Write an inode to disk and adjust it´s dirty state after completion.

NOTE only writes the actual inode, no associated data or other metadata.
