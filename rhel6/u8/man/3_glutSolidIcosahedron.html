NAME glutSolidIcosahedron - Draw a solid icosahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidIcosahedron(void);

DESCRIPTION This function draws a regular, solid 20-sided polyhedron centered at the origin. The distance from the origin to the vertices is 1.
