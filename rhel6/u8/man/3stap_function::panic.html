NAME function::panic - trigger a panic

SYNOPSIS panic(msg:string)

ARGUMENTS msg message to pass to kernel´s panic function

DESCRIPTION This function triggers an immediate panic of the running kernel with a user-specified panic message. It requires guru mode.
