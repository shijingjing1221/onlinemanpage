NAME gnutls_psk_set_params_function - set the DH or RSA parameters callback

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_psk_set_params_function(gnutls_psk_server_credentials_t res, gnutls_params_function * func);

ARGUMENTS gnutls_psk_server_credentials_t res is a gnutls_psk_server_credentials_t structure

gnutls_params_function * func is the function to be called

DESCRIPTION This function will set a callback in order for the server to get the Diffie-Hellman or RSA parameters for psk authentication. The callback should return zero on success.
