NAME corosync-pload - Start payload test.

SYNOPSIS corosync-pload

DESCRIPTION corosync-pload starts the pload service. This is used as a test tool to generate traffic. All nodes will begin transmitting messages and they will stop when 1500000 messages have been received per node.

To view the progress of this execute:

$ corosync-objctl runtime.services.pload. runtime.services.pload.service_id=13 runtime.services.pload.0.tx=3 runtime.services.pload.0.rx=2 runtime.services.pload.1.tx=2478788 runtime.services.pload.1.rx=4467870

At the end of the run a log message will be written showing the number of messages written and the time that this took.

corosync[4931]: [PLOAD ] 1500000 Writes 300 bytes per write 30.838 seconds runtime, 48640.968 TP/S, 13.916 MB/S.
