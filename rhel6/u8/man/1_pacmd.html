NAME pacmd - Reconfigure a PulseAudio sound server during runtime

SYNOPSIS pacmd

DESCRIPTION This tool can be used to introspect or reconfigure a running PulseAudio sound server during runtime. It connects to the sound server and offers a simple live shell that can be used to enter the commands also under- stood in the default.pa configuration scripts.

This program takes no command line options.
