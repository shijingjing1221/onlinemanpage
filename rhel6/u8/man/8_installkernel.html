NAME installkernel - tool to script kernel installation

SYNOPSIS installkernel <kernel-version> <bootimage> <mapfile>

DESCRIPTION installkernel installs a new kernel image onto the system from the Linux source tree. It is called by the Linux kernel makefiles when make install is invoked there.

The new kernel is installed into {directory}/vmlinuz-{version}. If a symbolic link {directory}/vmlinuz already exists, it is refreshed by making a link from {directory}/vmlinuz to the new kernel, and the pre- viously installed kernel is available as {directory}/vmlinuz.old.
