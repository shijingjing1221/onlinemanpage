NAME user_regset_set_fn - type of set function in struct user_regset

SYNOPSIS typedef int user_regset_set_fn(struct task_struct * target, const struct user_regset * regset, unsigned int pos, unsigned int count, const void * kbuf, const void __user * ubuf);

ARGUMENTS target thread being examined

regset regset being examined

pos offset into the regset data to access, in bytes

count amount of data to copy, in bytes

kbuf if not NULL, a kernel-space pointer to copy from

ubuf if kbuf is NULL, a user-space pointer to copy from

DESCRIPTION Store register values. Return 0 on success; -EIO or -ENODEV are usual failure returns. The pos and count values are in bytes, but must be properly aligned. If kbuf is non-null, that buffer is used and ubuf is ignored. If kbuf is NULL, then ubuf gives a userland pointer to access directly, and an -EFAULT return value is possible.
