NAME snd_pcm_format_linear - Check the PCM format is linear

SYNOPSIS int snd_pcm_format_linear(snd_pcm_format_t format);

ARGUMENTS format the format to check

DESCRIPTION Returns 1 if the given PCM format is linear, 0 if not.
