NAME bluetooth-properties - GTK dialog for managing properties of the Linux Bluetooth stack

SYNOPSIS bluetooth-properties

DESCRIPTION bluetooth-properties will display a dialog for changing Bluetooth pref- erences. bluetooth-properties is part of bluez-gnome, see also http://www.bluez.org

OPTIONS bluetooth-properties takes no options
