NAME error::pass3 - systemtap pass-3 errors

DESCRIPTION Errors during pass 3 (translation) occur only rarely.

unsupported code generation Some script language constructs are not available in every probe point. For example, the @perf() counter-reading function may only be used in process.* probes.

GATHERING MORE INFORMATION Increasing the verbosity of pass-3 with an option such as --vp 002 may help pinpoint the problem.
