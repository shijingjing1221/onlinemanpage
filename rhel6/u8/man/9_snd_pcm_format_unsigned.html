SND_PCM_FORMAT_UNSIG(9) PCM API SND_PCM_FORMAT_UNSIG(9)

NAME snd_pcm_format_unsigned - Check the PCM format is unsigned linear

SYNOPSIS int snd_pcm_format_unsigned(snd_pcm_format_t format);

ARGUMENTS format the format to check

DESCRIPTION Returns 1 if the given PCM format is unsigned linear, 0 if signed linear, and a negative error code for non-linear formats.
