NAME hpifan - An openhpi sample application that shows "Fan Control" management instruments

SYNOPSIS hpifan [-h] [-s fan_speed_level]

DESCRIPTION hpifan walks the RPT (Resource Present Table) for resouces that have "Fan Control" management instruments (SAHPI_CTRL_FAN_SPEED).

OPTIONS -h help -s <speed> set fan speed for ALL fans
