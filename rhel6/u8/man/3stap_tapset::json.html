NAME tapset::json - systemtap json tapset

DESCRIPTION The JSON tapset provides probes, functions, and macros to generate a JSON metadata and data file. The JSON metadata file is located in /proc/systemtap/MODULE/metadata.json. The JSON data file is located in /proc/systemtap/MODULE/data.json. The JSON data file is updated with current data every time the file is read.

See function::json_set_prefix(3stap) for details.
