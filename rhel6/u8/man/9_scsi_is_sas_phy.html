NAME scsi_is_sas_phy - check if a struct device represents a SAS PHY

SYNOPSIS int scsi_is_sas_phy(const struct device * dev);

ARGUMENTS dev device to check

RETURNS 1 if the device represents a SAS PHY, 0 else
