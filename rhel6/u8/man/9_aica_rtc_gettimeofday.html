NAME aica_rtc_gettimeofday - Get the time from the AICA RTC

SYNOPSIS void aica_rtc_gettimeofday(struct timespec * ts);

ARGUMENTS ts pointer to resulting timespec

DESCRIPTION Grabs the current RTC seconds counter and adjusts it to the Unix Epoch.
