NAME blk_cleanup_queue - shutdown a request queue

SYNOPSIS void blk_cleanup_queue(struct request_queue * q);

ARGUMENTS q request queue to shutdown

DESCRIPTION Mark q DEAD, drain all pending requests, destroy and put it. All future requests will be failed immediately with -ENODEV.
