NAME pbmtocmuwm - convert a PBM image into a CMU window manager bitmap

SYNOPSIS pbmtocmuwm [pbmfile]

DESCRIPTION This program is part of Netpbm(1).

pbmtocmuwm reads a portable bitmap as input and produces a CMU window manager bitmap as output.

SEE ALSO cmuwmtopbm(1), pbm(1)
