NAME i2o_block_request_free - Frees a I2O block request

SYNOPSIS void i2o_block_request_free(struct i2o_block_request * ireq);

ARGUMENTS ireq I2O block request which should be freed

DESCRIPTION Frees the allocated memory (give it back to the request mempool).
