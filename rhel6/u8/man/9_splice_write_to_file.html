NAME splice_write_to_file - splice data from a pipe to a file

SYNOPSIS ssize_t splice_write_to_file(struct pipe_inode_info * pipe, struct file * out, loff_t * ppos, size_t len, unsigned int flags, splice_write_actor actor);

ARGUMENTS pipe pipe info

out file to write to

ppos position in out

len number of bytes to splice

flags splice modifier flags

actor worker that does the splicing from the pipe to the file

DESCRIPTION Will either move or copy pages (determined by flags options) from the given pipe inode to the given file.
