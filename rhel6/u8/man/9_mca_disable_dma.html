MCA_DISABLE_DMA(9) Hardware Interfaces MCA_DISABLE_DMA(9)

NAME mca_disable_dma - channel to disable DMA on

SYNOPSIS void mca_disable_dma(unsigned int dmanr);

ARGUMENTS dmanr DMA channel

DESCRIPTION Enable the MCA bus DMA on a channel. This can be called from IRQ context.
