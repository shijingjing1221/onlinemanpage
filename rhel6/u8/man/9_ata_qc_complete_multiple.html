NAME ata_qc_complete_multiple - Complete multiple qcs successfully

SYNOPSIS int ata_qc_complete_multiple(struct ata_port * ap, u32 qc_active);

ARGUMENTS ap port in question

qc_active new qc_active mask

DESCRIPTION Complete in-flight commands. This functions is meant to be called from low-level driver´s interrupt routine to complete requests normally. ap->qc_active and qc_active is compared and commands are completed accordingly.

LOCKING spin_lock_irqsave(host lock)

RETURNS Number of completed commands on success, -errno otherwise.
