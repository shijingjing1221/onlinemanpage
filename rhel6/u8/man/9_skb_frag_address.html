NAME skb_frag_address - gets the address of the data contained in a paged fragment

SYNOPSIS void * skb_frag_address(const skb_frag_t * frag);

ARGUMENTS frag the paged fragment buffer

DESCRIPTION Returns the address of the data within frag. The page must already be mapped.
