NAME gnutls_x509_crt_set_crq_extensions - Associate the Certificate with a request

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_set_crq_extensions(gnutls_x509_crt_t crt, gnutls_x509_crq_t crq);

ARGUMENTS gnutls_x509_crt_t crt a certificate of type gnutls_x509_crt_t

gnutls_x509_crq_t crq holds a certificate request

DESCRIPTION This function will set extensions from the given request to the cer- tificate.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.

SINCE 2.8.0
