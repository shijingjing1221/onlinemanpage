NAME gnutls_openpgp_privkey_sign_hash - sign the given data using the pri- vate key params

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_privkey_sign_hash(gnutls_openpgp_privkey_t key, const gnutls_datum_t * hash, gnutls_datum_t * signature);

ARGUMENTS gnutls_openpgp_privkey_t key Holds the key

const gnutls_datum_t * hash holds the data to be signed

gnutls_datum_t * signature will contain newly allocated signature

DESCRIPTION This function will sign the given hash using the private key. You should use gnutls_openpgp_privkey_set_subkey() before calling this function to set the subkey to use.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
