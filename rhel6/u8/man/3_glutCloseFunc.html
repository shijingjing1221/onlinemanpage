NAME glutCloseFunc - Window destruction callback.

LIBRARY OpenGLUT - windowcallback

SYNOPSIS #include <openglut.h>

void glutCloseFunc(void( *callback )( void ));

PARAMETERS callback Client window destruction hook.

DESCRIPTION When a window is destroyed by user-action in traditional GLUT, the appli- cation terminates. In freeglut and OpenGLUT, the application can choose to persist and treat the window close event as a normal event. This callback is how that event is transmitted to the application.

This callback is bound to the current window .

CAVEATS This function is exactly the same as glutWMCloseFunc(), which has been deprecated. This function should be used instead.
