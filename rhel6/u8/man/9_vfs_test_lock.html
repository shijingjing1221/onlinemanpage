NAME vfs_test_lock - test file byte range lock

SYNOPSIS int vfs_test_lock(struct file * filp, struct file_lock * fl);

ARGUMENTS filp The file to test lock for

fl The lock to test; also used to hold result

DESCRIPTION Returns -ERRNO on failure. Indicates presence of conflicting lock by setting conf->fl_type to something other than F_UNLCK.
