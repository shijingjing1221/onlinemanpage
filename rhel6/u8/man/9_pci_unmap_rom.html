PCI_UNMAP_ROM(9) Hardware Interfaces PCI_UNMAP_ROM(9)

NAME pci_unmap_rom - unmap the ROM from kernel space

SYNOPSIS void pci_unmap_rom(struct pci_dev * pdev, void __iomem * rom);

ARGUMENTS pdev pointer to pci device struct

rom virtual address of the previous mapping

DESCRIPTION Remove a mapping of a previously mapped ROM
