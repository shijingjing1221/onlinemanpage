NAME crc_ccitt - recompute the CRC for the data buffer

SYNOPSIS u16 crc_ccitt(u16 crc, u8 const * buffer, size_t len);

ARGUMENTS crc previous CRC value

buffer data pointer

len number of bytes in the buffer
