NAME hipstopgm - convert a HIPS file into a PGM image

SYNOPSIS hipstopgm [hipsfile]

DESCRIPTION This program is part of Netpbm(1).

hipstopgm reads a HIPS file as input and produces a PGM image as out- put.

If the HIPS file contains more than one frame in sequence, hipstopgm will concatenate all the frames vertically.

HIPS is a format developed at the Human Information Processing Labora- tory, NYU.
