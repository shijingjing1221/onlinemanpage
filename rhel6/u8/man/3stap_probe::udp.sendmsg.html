NAME probe::udp.sendmsg - Fires whenever a process sends a UDP message

SYNOPSIS udp.sendmsg

VALUES name The name of this probe

dport UDP destination port

size Number of bytes sent by the process

saddr A string representing the source IP address

daddr A string representing the destination IP address

sport UDP source port

sock Network socket used by the process

family IP address family

CONTEXT The process which sent a UDP message
