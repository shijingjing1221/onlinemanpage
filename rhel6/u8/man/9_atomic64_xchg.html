NAME atomic64_xchg - xchg atomic64 variable

SYNOPSIS u64 atomic64_xchg(atomic64_t * ptr, u64 new_val);

ARGUMENTS ptr pointer to type atomic64_t

new_val value to assign

DESCRIPTION Atomically xchgs the value of ptr to new_val and returns the old value.
