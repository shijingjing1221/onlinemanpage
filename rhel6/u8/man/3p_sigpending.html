NAME sigpending - examine pending signals

SYNOPSIS #include <signal.h>

int sigpending(sigset_t *set);

DESCRIPTION The sigpending() function shall store, in the location referenced by the set argument, the set of signals that are blocked from delivery to the calling thread and that are pending on the process or the calling thread.

RETURN VALUE Upon successful completion, sigpending() shall return 0; otherwise, -1 shall be returned and errno set to indicate the error.

ERRORS No errors are defined.
