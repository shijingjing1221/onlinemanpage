I2O_CONFIG_INIT(9) Message-based devices I2O_CONFIG_INIT(9)

NAME i2o_config_init - Configuration OSM initialization function

SYNOPSIS int i2o_config_init(void);

ARGUMENTS void no arguments

DESCRIPTION Registers Configuration OSM in the I2O core and if old ioctl´s are compiled in initialize them.

Returns 0 on success or negative error code on failure.
