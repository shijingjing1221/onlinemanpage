NAME ata_dev_enable_pm - enable SATA interface power management

SYNOPSIS void ata_dev_enable_pm(struct ata_device * dev, enum link_pm policy);

ARGUMENTS dev device to enable power management

policy the link power management policy

DESCRIPTION Enable SATA Interface power management. This will enable Device Interface Power Management (DIPM) for min_power policy, and then call driver specific callbacks for enabling Host Initiated Power management.

LOCKING Caller.

RETURNS -EINVAL if IPM is not supported, 0 otherwise.
