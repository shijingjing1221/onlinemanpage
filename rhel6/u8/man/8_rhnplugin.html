NAME rhnplugin - Red Hat Network support for yum(8).

DESCRIPTION rhnplugin provides Red Hat Network support for yum(8) via its plugin architecture. A system registered with Red Hat Network will be able to update and install packages from Red Hat Network.

FILES /etc/sysconfig/rhn/up2date The common configuration file used by RHN client programs.

/etc/yum/pluginconf.d/rhnplugin.conf A file containing plugin specific configuration options.

SEE ALSO yum(8), rhn_check(8), rhn_register(8), rhnplugin.conf(5), up2date(5).
