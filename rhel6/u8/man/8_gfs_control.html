NAME gfs_control - a utility for the gfs_controld daemon

SYNOPSIS gfs_control [COMMAND] [OPTIONS] [ name ]

DESCRIPTION ls Display internal gfs_controld state about mount groups.

dump Dump gfs_controld debug buffer.

plocks name Dump posix locks from gfs_controld for the mount group. (For compat mode, otherwise see dlm_tool(8).)

OPTIONS -n Show all node information in ls.

-h Print a help message describing available options, then exit.

-V Print program version information, then exit.

SEE ALSO gfs_controld(8), dlm_tool(8)
