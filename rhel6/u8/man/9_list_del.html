LIST_DEL(9) Data Types LIST_DEL(9)

NAME list_del - deletes entry from list.

SYNOPSIS void list_del(struct list_head * entry);

ARGUMENTS entry the element to delete from the list.

NOTE list_empty on entry does not return true after this, the entry is in an undefined state.
