NAME snd_pcm_playback_empty - check whether the playback buffer is empty

SYNOPSIS int snd_pcm_playback_empty(struct snd_pcm_substream * substream);

ARGUMENTS substream the pcm substream instance

DESCRIPTION Checks whether the playback buffer is empty.

Returns non-zero if empty, or zero if not.
