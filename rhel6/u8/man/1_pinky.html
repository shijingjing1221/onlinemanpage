NAME pinky - lightweight finger

SYNOPSIS pinky [OPTION]... [USER]...

DESCRIPTION -l produce long format output for the specified USERs

-b omit the users home directory and shell in long format

-h omit the users project file in long format

-p omit the users plan file in long format

-s do short format output, this is the default

-f omit the line of column headings in short format

-w omit the users full name in short format

-i omit the users full name and remote host in short format

-q omit the users full name, remote host and idle time in short format

--help display this help and exit

--version output version information and exit

A lightweight finger program; print user information. The utmp file will be /var/run/utmp.
