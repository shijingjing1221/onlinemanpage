NAME ctl_deliver - perform operations on the duplicate delivery database

SYNOPSIS ctl_deliver [ -C config-file ] -d [ -f filename ]

DESCRIPTION Ctl_deliver is used to perform various administrative operations on the duplicate delivery database.

Ctl_deliver reads its configuration options out of the imapd.conf(5) file unless specified otherwise by -C.

OPTIONS -C config-file Read configuration options from config-file.

-d Dump the contents of the database to standard output in a portable flat-text format.

-f filename Use the database specified by filename instead of the default (configdirectory/deliver.db).

FILES /etc/imapd.conf

SEE ALSO imapd.conf(5), cyrus-master(8)
