NAME i2c_smbus_process_call - SMBus process call protocol

SYNOPSIS s32 i2c_smbus_process_call(struct i2c_client * client, u8 command, u16 value);

ARGUMENTS client Handle to slave device

command Byte interpreted by slave

value 16-bit word being written

DESCRIPTION This executes the SMBus process call protocol, returning negative errno else a 16-bit unsigned word received from the device.
