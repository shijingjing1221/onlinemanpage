FUNCTION::NS_UID(3stap) Context Functions FUNCTION::NS_UID(3stap)

NAME function::ns_uid - Returns the user ID of a target process as seen in a user namespace

SYNOPSIS ns_uid:long()

ARGUMENTS None

DESCRIPTION This function returns the user ID of the target process as seen in the target user namespace if provided, or the stap process namespace.
