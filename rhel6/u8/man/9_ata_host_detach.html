NAME ata_host_detach - Detach all ports of an ATA host

SYNOPSIS void ata_host_detach(struct ata_host * host);

ARGUMENTS host Host to detach

DESCRIPTION Detach all ports of host.

LOCKING Kernel thread context (may sleep).
