NAME function::task_ns_tid - The thread identifier of the task as seen in a namespace

SYNOPSIS task_ns_tid:long(task:long)

ARGUMENTS task task_struct pointer

DESCRIPTION This function returns the thread id of the given task as seen in the pid namespace.
