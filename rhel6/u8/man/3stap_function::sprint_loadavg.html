NAME function::sprint_loadavg - Report a pretty-printed load average

SYNOPSIS sprint_loadavg:string()

ARGUMENTS None

DESCRIPTION Returns the a string with three decimal numbers in the usual format for 1-, 5- and 15-minute load averages.
