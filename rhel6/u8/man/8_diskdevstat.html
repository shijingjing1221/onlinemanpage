NAME diskdevstat - tool for recording harddisk activity

SYNOPSIS diskdevstat [update interval] [total duration] [display histogram]

DESCRIPTION diskdevstat is a simple systemtap script to record harddisk activity of processes and display statistics for read/write operations.

update interval Sets sampling interval in seconds.

total duration Sets total measurement time in seconds.

display histogram If this parameter is present, the histogram will be shown at the end of the measurement.

SEE ALSO tuned(8) netdevstat(8) varnetload(8) scomes(8) stap(1)
