NAME glutMenuDestroyFunc - Destruction callback for menus.

LIBRARY OpenGLUT - menucallback

SYNOPSIS #include <openglut.h>

void glutMenuDestroyFunc(void( *callback )( void ));

PARAMETERS callback Client menu destruction hook.

DESCRIPTION When a menu is destroyed, OpenGLUT will call this hook, if defined on that menu.

This callback is not bound to any window.

This callback is bound to a specific menu.

CAVEATS For emphasis, we repeat: This callback does not bind to any window.
