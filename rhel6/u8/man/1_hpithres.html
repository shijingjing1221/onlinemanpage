NAME hpithres - This sample openhpi application interactively displays sensors and sensor info.

SYNOPSIS hpithres [-x]

DESCRIPTION hpithres interactively displays sensor info for resources with Sensor Capability. Resources, sensors,and sensor info can be individually selected.

OPTIONS -x Display debug messages
