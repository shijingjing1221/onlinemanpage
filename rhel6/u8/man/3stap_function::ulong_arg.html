NAME function::ulong_arg - Return function argument as unsigned long

SYNOPSIS ulong_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the value of argument n as an unsigned long. On architectures where a long is 32 bits, the value is zero-extended to 64 bits.
