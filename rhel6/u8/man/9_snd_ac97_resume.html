NAME snd_ac97_resume - General resume function for AC97 codec

SYNOPSIS void snd_ac97_resume(struct snd_ac97 * ac97);

ARGUMENTS ac97 the ac97 instance

DESCRIPTION Do the standard resume procedure, power up and restoring the old register values.
