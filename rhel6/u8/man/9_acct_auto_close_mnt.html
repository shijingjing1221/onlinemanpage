ACCT_AUTO_CLOSE_MNT(9) Accounting Framework ACCT_AUTO_CLOSE_MNT(9)

NAME acct_auto_close_mnt - turn off a filesystem´s accounting if it is on

SYNOPSIS void acct_auto_close_mnt(struct vfsmount * m);

ARGUMENTS m vfsmount being shut down

DESCRIPTION If the accounting is turned on for a file in the subtree pointed to to by m, turn accounting off. Done when m is about to die.
