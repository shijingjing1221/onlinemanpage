NAME seekdir - set the position of the next readdir() call in the directory stream.

SYNOPSIS #include <dirent.h>

void seekdir(DIR *dirp, long offset);

Feature Test Macro Requirements for glibc (see feature_test_macros(7)):

seekdir(): _BSD_SOURCE || _SVID_SOURCE || _XOPEN_SOURCE

DESCRIPTION The seekdir() function sets the location in the directory stream from which the next readdir(2) call will start. seekdir() should be used with an offset returned by telldir(3).

RETURN VALUE The seekdir() function returns no value.

CONFORMING TO 4.3BSD, POSIX.1-2001.

NOTES In glibc up to version 2.1.1, the type of the offset argument was off_t. POSIX.1-2001 specifies long, and this is the type used since glibc 2.1.2.
