MEMCHR_INV(9) Basic C Library Functions MEMCHR_INV(9)

NAME memchr_inv - Find an unmatching character in an area of memory.

SYNOPSIS void * memchr_inv(const void * start, int c, size_t bytes);

ARGUMENTS start The memory area

c Find a character other than c

bytes The size of the area.

DESCRIPTION returns the address of the first character other than c, or NULL if the whole buffer contains just c.
