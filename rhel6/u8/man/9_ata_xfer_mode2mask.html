NAME ata_xfer_mode2mask - Find matching xfer_mask for XFER_*

SYNOPSIS unsigned long ata_xfer_mode2mask(u8 xfer_mode);

ARGUMENTS xfer_mode XFER_* of interest

DESCRIPTION Return matching xfer_mask for xfer_mode.

LOCKING None.

RETURNS Matching xfer_mask, 0 if no match found.
