NAME regulator_list_voltage - enumerate supported voltages

SYNOPSIS int regulator_list_voltage(struct regulator * regulator, unsigned selector);

ARGUMENTS regulator regulator source

selector identify voltage to list

CONTEXT can sleep

DESCRIPTION Returns a voltage that can be passed to regulator_set_voltage(), zero if this selector code can´t be used on this sytem, or a negative errno.
