NAME function::user_int32 - Retrieves a 32-bit integer value stored in user space

SYNOPSIS user_int32:long(addr:long)

ARGUMENTS addr the user space address to retrieve the 32-bit integer from

DESCRIPTION Returns the 32-bit integer value from a given user space address. Returns zero when user space data is not accessible.
