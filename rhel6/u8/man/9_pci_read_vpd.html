NAME pci_read_vpd - Read one entry from Vital Product Data

SYNOPSIS ssize_t pci_read_vpd(struct pci_dev * dev, loff_t pos, size_t count, void * buf);

ARGUMENTS dev pci device struct

pos offset in vpd space

count number of bytes to read

buf pointer to where to store result
