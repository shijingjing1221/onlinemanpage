NAME autopoint - copies standard gettext infrastructure

SYNOPSIS autopoint [OPTION]...

DESCRIPTION Copies standard gettext infrastructure files into a source package.

OPTIONS --help print this help and exit

--version print version information and exit

-f, --force force overwriting of files that already exist

-n, --dry-run print modifications but dont perform them
