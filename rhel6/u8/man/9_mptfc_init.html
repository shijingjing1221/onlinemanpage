NAME mptfc_init - Register MPT adapter(s) as SCSI host(s) with SCSI mid-layer.

SYNOPSIS int mptfc_init(void);

ARGUMENTS void no arguments

DESCRIPTION Returns 0 for success, non-zero for failure.
