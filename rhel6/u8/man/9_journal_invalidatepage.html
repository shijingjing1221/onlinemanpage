JOURNAL_INVALIDATEPA(9) The Linux Journalling API JOURNAL_INVALIDATEPA(9)

NAME journal_invalidatepage - invalidate a journal page

SYNOPSIS void journal_invalidatepage(journal_t * journal, struct page * page, unsigned long offset);

ARGUMENTS journal journal to use for flush

page page to flush

offset length of page to invalidate.

DESCRIPTION Reap page buffers containing data after offset in page.
