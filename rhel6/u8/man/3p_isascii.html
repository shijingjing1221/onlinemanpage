NAME isascii - test for a 7-bit US-ASCII character

SYNOPSIS #include <ctype.h>

int isascii(int c);

DESCRIPTION The isascii() function shall test whether c is a 7-bit US-ASCII charac- ter code.

The isascii() function is defined on all integer values.

RETURN VALUE The isascii() function shall return non-zero if c is a 7-bit US-ASCII character code between 0 and octal 0177 inclusive; otherwise, it shall return 0.

ERRORS No errors are defined.
