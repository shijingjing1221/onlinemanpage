FUNCTION::HZ(3stap) Timestamp Functions FUNCTION::HZ(3stap)

NAME function::HZ - Kernel HZ

SYNOPSIS HZ:long()

ARGUMENTS None

DESCRIPTION This function returns the value of the kernel HZ macro, which corresponds to the rate of increase of the jiffies value.
