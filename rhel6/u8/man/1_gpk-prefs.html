NAME gpk-prefs - GNOME PackageKit Update Preferences

SYNOPSIS gpk-prefs [ --verbose ]

DESCRIPTION This manual page documents briefly the gpk-prefs command.

gpk-prefs allows you to change preferences for keeping your system up to date.
