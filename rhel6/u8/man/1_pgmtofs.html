NAME pgmtofs - convert PGM image to Usenix FaceSaver(tm) format

SYNOPSIS pgmtofs

[pgmfile]

DESCRIPTION This program is part of Netpbm(1).

pgmtofs reads a PGM image as input and produces Usenix FaceSaver(tm) format as output.

FaceSaver is a registered trademark of Metron Computerware Ltd. of Oak- land, CA.
