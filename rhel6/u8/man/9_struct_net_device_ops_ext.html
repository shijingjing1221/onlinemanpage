NAME struct_net_device_ops_ext - structure for extending net_device_ops in RHEL

SYNOPSIS struct net_device_ops_ext { size_t size; };

MEMBERS size This field should be initialized to the size of the structure by the drivers.

DESCRIPTION New methods can be added at the end. Do not change existing ones.
