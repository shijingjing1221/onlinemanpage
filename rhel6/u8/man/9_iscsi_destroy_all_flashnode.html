NAME iscsi_destroy_all_flashnode - destory all flashnode session entries

SYNOPSIS void iscsi_destroy_all_flashnode(struct Scsi_Host * shost);

ARGUMENTS shost pointer to host data

DESCRIPTION Destroys all the flashnode session entries and all corresponding children flashnode connection entries from sysfs
