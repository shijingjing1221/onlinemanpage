NAME function::proc_mem_size - Total program virtual memory size in pages

SYNOPSIS proc_mem_size:long()

ARGUMENTS None

DESCRIPTION Returns the total virtual memory size in pages of the current process, or zero when there is no current process or the number of pages couldn´t be retrieved.
