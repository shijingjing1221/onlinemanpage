SETUP_PER_ZONE_WMARK(9) Memory Management in Linux SETUP_PER_ZONE_WMARK(9)

NAME setup_per_zone_wmarks - called when min_free_kbytes changes or when memory is hot-{added|removed}

SYNOPSIS void setup_per_zone_wmarks(void);

ARGUMENTS void no arguments

DESCRIPTION Ensures that the watermark[min,low,high] values for each zone are set correctly with respect to min_free_kbytes.
