NAME ecryptfs-rewrap-passphrase - unwrap an eCryptfs wrapped passphrase, rewrap it with a new passphrase, and write it back to file.

SYNOPSIS ecryptfs-rewrap-passphrase [file]

printf "%s1s" "old wrapping passphrase" "new wrapping passphrase" | ecryptfs-rewrap-passphrase [file] -

DESCRIPTION ecryptfs-rewrap-passphrase is a utility to change the wrapping passphrase on a wrapped passphrase file.

SEE ALSO ecryptfs(7), ecryptfs-unwrap-passphrase(1), ecryptfs-wrap-passphrase(1)
