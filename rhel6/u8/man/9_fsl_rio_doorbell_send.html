NAME fsl_rio_doorbell_send - Send a MPC85xx doorbell message

SYNOPSIS int fsl_rio_doorbell_send(struct rio_mport * mport, int index, u16 destid, u16 data);

ARGUMENTS mport RapidIO master port info

index ID of RapidIO interface

destid Destination ID of target device

data 16-bit info field of RapidIO doorbell message

DESCRIPTION Sends a MPC85xx doorbell message. Returns 0 on success or -EINVAL on failure.
