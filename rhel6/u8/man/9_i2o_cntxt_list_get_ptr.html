NAME i2o_cntxt_list_get_ptr - Get a context id from the context list

SYNOPSIS u32 i2o_cntxt_list_get_ptr(struct i2o_controller * c, void * ptr);

ARGUMENTS c controller to which the context list belong

ptr pointer to which the context id should be fetched

DESCRIPTION Returns context id which matches to the pointer on succes or 0 on failure.
