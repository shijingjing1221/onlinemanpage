CRC16(9) Basic Kernel Library Functions CRC16(9)

NAME crc16 - compute the CRC-16 for the data buffer

SYNOPSIS u16 crc16(u16 crc, u8 const * buffer, size_t len);

ARGUMENTS crc previous CRC value

buffer data pointer

len number of bytes in the buffer

DESCRIPTION Returns the updated CRC value.
