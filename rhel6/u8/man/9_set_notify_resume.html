SET_NOTIFY_RESUME(9) Kernel Internals SET_NOTIFY_RESUME(9)

NAME set_notify_resume - cause tracehook_notify_resume to be called

SYNOPSIS void set_notify_resume(struct task_struct * task);

ARGUMENTS task task that will call tracehook_notify_resume

DESCRIPTION Calling this arranges that task will call tracehook_notify_resume before returning to user mode. If it´s already running in user mode, it will enter the kernel and call tracehook_notify_resume soon. If it´s blocked, it will not be woken.
