NAME tld_get_table - API function

SYNOPSIS #include <tld.h>

const Tld_table * tld_get_table(const char * tld, const Tld_table ** tables);

ARGUMENTS const char * tld TLD name (e.g. "com") as zero terminated ASCII byte string.

const Tld_table ** tables Zero terminated array of Tld_table info-structures for TLDs.

DESCRIPTION Get the TLD table for a named TLD by searching through the given TLD table array.

RETURN VALUE Return structure corresponding to TLD tld by going thru tables, or return NULL if no such structure is found.
