NAME ipsec_secrets - re-read the ipsec.secrets file

SYNOPSIS ipsec secrets

DESCRIPTION secrets is an alias for ipsec auto --rereadsecrets and rereads all the secrets in the ipsec.secrets file.

SEE ALSO ipsec.secrets(5)

HISTORY Older versions of libreswan (and openswan) would allow the secrets files to contain filenames to encrypted RSA private keys. These are no longer supported and must reside in the NSS database. The NSS database itself can be protected with a passphrase.

BUGS None
