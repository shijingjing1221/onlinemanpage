I2O_SCSI_INIT(9) Message-based devices I2O_SCSI_INIT(9)

NAME i2o_scsi_init - SCSI OSM initialization function

SYNOPSIS int i2o_scsi_init(void);

ARGUMENTS void no arguments

DESCRIPTION Register SCSI OSM into I2O core.

Returns 0 on success or negative error code on failure.
