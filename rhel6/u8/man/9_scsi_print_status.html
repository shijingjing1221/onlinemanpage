SCSI_PRINT_STATUS(9) SCSI mid layer SCSI_PRINT_STATUS(9)

NAME scsi_print_status - print scsi status description

SYNOPSIS void scsi_print_status(unsigned char scsi_status);

ARGUMENTS scsi_status scsi status value

DESCRIPTION If the status is recognized, the description is printed. Otherwise Unknown status is output. No trailing space. If CONFIG_SCSI_CONSTANTS is not set, then print status in hex (e.g. 0x2 for Check Condition).
