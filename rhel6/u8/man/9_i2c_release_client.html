I2C_RELEASE_CLIENT(9) I2C and SMBus Subsystem I2C_RELEASE_CLIENT(9)

NAME i2c_release_client - release a use of the i2c client structure

SYNOPSIS void i2c_release_client(struct i2c_client * client);

ARGUMENTS client the client being no longer referenced

DESCRIPTION Must be called when a user of a client is finished with it.
