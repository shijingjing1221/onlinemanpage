NAME ata_msense_caching - Simulate MODE SENSE caching info page

SYNOPSIS unsigned int ata_msense_caching(u16 * id, u8 * buf);

ARGUMENTS id device IDENTIFY data

buf output buffer

DESCRIPTION Generate a caching info page, which conditionally indicates write caching to the SCSI layer, depending on device capabilities.

LOCKING None.
