PCI_MSI_OFF(9) Hardware Interfaces PCI_MSI_OFF(9)

NAME pci_msi_off - disables any msi or msix capabilities

SYNOPSIS void pci_msi_off(struct pci_dev * dev);

ARGUMENTS dev the PCI device to operate on

DESCRIPTION If you want to use msi see pci_enable_msi and friends. This is a lower level primitive that allows us to disable msi operation at the device level.
