NAME i2o_scsi_notify_device_remove - Retrieve notifications of removed devices

SYNOPSIS void i2o_scsi_notify_device_remove(struct i2o_device * i2o_dev);

ARGUMENTS i2o_dev the I2O device which was removed

DESCRIPTION If a I2O device is removed, we catch the notification to remove the corresponding SCSI device.
