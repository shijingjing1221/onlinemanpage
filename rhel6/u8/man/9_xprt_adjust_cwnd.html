NAME xprt_adjust_cwnd - adjust transport congestion window

SYNOPSIS void xprt_adjust_cwnd(struct rpc_task * task, int result);

ARGUMENTS task recently completed RPC request used to adjust window

result result code of completed RPC request

DESCRIPTION The transport code maintains an estimate on the maximum number of out- standing RPC requests, using a smoothed version of the congestion avoidance implemented in 44BSD. This is basically the Van Jacobson

CONGESTION ALGORITHM If a retransmit occurs, the congestion window is halved; otherwise, it is incremented by 1/cwnd when

- a reply is received and - a full number of requests are outstanding and - the congestion window hasn´t been updated recently.
