NAME ata_qc_new - Request an available ATA command, for queueing

SYNOPSIS struct ata_queued_cmd * ata_qc_new(struct ata_port * ap);

ARGUMENTS ap target port

DESCRIPTION Some ATA host controllers may implement a queue depth which is less than ATA_MAX_QUEUE. So we shouldn´t allocate a tag which is beyond the hardware limitation.

LOCKING None.
