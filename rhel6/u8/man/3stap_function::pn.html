FUNCTION::PN(3stap) Context Functions FUNCTION::PN(3stap)

NAME function::pn - Returns the active probe name

SYNOPSIS pn:string()

ARGUMENTS None

DESCRIPTION This function returns the script-level probe point associated with a currently running probe handler, including wild-card expansion effects. Context: The current probe point.
