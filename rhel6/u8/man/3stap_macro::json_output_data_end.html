NAME macro::json_output_data_end - End the json output.

SYNOPSIS @json_output_data_end()

ARGUMENTS None

DESCRIPTION The json_output_data_end macro is designed to be called from the ´json_data´ probe from the user´s script. It marks the end of the JSON output.
