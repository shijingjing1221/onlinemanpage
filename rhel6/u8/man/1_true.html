TRUE(1) User Commands TRUE(1)

NAME true - do nothing, successfully

SYNOPSIS true [ignored command line arguments] true OPTION

DESCRIPTION Exit with a status code indicating success.

--help display this help and exit

--version output version information and exit

NOTE: your shell may have its own version of true, which usually super- sedes the version described here. Please refer to your shells docu- mentation for details about the options it supports.
