NAME inode_inc_iversion - increments i_version

SYNOPSIS void inode_inc_iversion(struct inode * inode);

ARGUMENTS inode inode that need to be updated

DESCRIPTION Every time the inode is modified, the i_version field will be incremented. The filesystem has to be mounted with i_version flag
