NAME ht_destroy_irq - destroy an irq created with ht_create_irq

SYNOPSIS void ht_destroy_irq(unsigned int irq);

ARGUMENTS irq irq to be destroyed

DESCRIPTION This reverses ht_create_irq removing the specified irq from existence. The irq should be free before this happens.
