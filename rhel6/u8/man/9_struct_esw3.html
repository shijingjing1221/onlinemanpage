NAME struct_esw3 - Format 3 Extended Status Word (ESW)

SYNOPSIS struct esw3 { __u8 zero0; __u8 lpum; __u16 res; struct erw erw; __u32 zeros[3]; };

MEMBERS zero0 reserved zeros

lpum last path used mask

res reserved

erw extended report word

zeros[3] three fullwords of zeros
