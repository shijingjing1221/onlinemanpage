NAME ps_mem - Memory profiling tool

SYNOPSIS ps_mem [-h|--help] [-s|--show-cmdline] [-p PID] [-w N]

DESCRIPTION The ps_mem tool can determine how much RAM is used per program (not per process).

In detail it reports:

sum(private RAM for program processes) + sum(Shared RAM for pro- gram processes).

The shared RAM is problematic to calculate, and the tool automatically selects the most accurate method available for the running kernel.

OPTIONS -h --help Show help message

-s --show-cmdline Show complete program path with options

-p PID Show memory consumption of process with specified PID

-w N Report memory consumption every N seconds
