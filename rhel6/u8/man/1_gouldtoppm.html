NAME gouldtoppm - convert Gould scanner file into a PPM image

SYNOPSIS gouldtoppm [gouldfile]

DESCRIPTION This program is part of Netpbm(1).

gouldtoppm reads a file produced by the Gould scanner as input and pro- duces a PPM image as output.
