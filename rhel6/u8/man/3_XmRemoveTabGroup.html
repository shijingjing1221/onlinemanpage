NAME XmRemoveTabGroup A function that removes a tab group

SYNOPSIS #include <Xm/Xm.h> void XmRemoveTabGroup( Widget tab_group);

DESCRIPTION This function is obsolete and its behavior is replaced by setting XmN- navigationType to XmNONE. XmRemoveTabGroup removes a widget from the list of tab groups associated with a particular widget hierarchy and sets the widgets XmNnavigationType to XmNONE.

tab_group Specifies the widget ID

RELATED XmAddTabGroup(3), XmManager(3), and XmPrimitive(3).
