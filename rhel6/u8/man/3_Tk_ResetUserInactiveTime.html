NAME Tk_GetUserInactiveTime, Tk_ResetUserInactiveTime - discover user inac- tivity time

SYNOPSIS #include <tk.h>

long Tk_GetUserInactiveTime(display)

Tk_GetUserInactiveTime(display)

ARGUMENTS Display *display (in) The display on which the user inactivity timer is to be queried or reset. _________________________________________________________________

DESCRIPTION Tk_GetUserInactiveTime returns the number of milliseconds that have passed since the last user interaction (usually via keyboard or mouse) with the respective display. On systems and displays that do not sup- port querying the user inactiviy time, -1 is returned. Tk_GetUserInac- tiveTime resets the user inactivity timer of the given display to zero. On windowing systems that do not support multiple displays display can be passed as NULL.

KEYWORDS idle, inactive
