NAME auparse_get_time - get events time

SYNOPSIS #include <auparse.h>

time_t auparse_get_time(auparse_state_t *au);

DESCRIPTION auparse_get_time will access just the time portion of the timestamp data structure for the current event.

RETURN VALUE Returns 0 if an error occurs; otherwise, the valid time value in time_t format.

SEE ALSO time(3), auparse_get_timestamp(3), auparse_get_milli(3). auparse_get_serial(3). auparse_get_node(3).
