NAME git-count-objects - Count unpacked number of objects and their disk consumption

SYNOPSIS git count-objects [-v]

DESCRIPTION This counts the number of unpacked object files and disk space consumed by them, to help you decide when it is a good time to repack.

OPTIONS -v, --verbose In addition to the number of loose objects and disk space consumed, it reports the number of in-pack objects, number of packs, disk space consumed by those packs, and number of objects that can be removed by running git prune-packed.
