NAME SDL_ExposeEvent - Quit requested event

STRUCTURE DEFINITION typedef struct{ Uint8 type } SDL_ExposeEvent;

STRUCTURE DATA type SDL_VIDEOEXPOSE

DESCRIPTION SDL_ExposeEvent is a member of the SDL_Event union and is used whan an event of type SDL_VIDEOEXPOSE is reported.

A VIDEOEXPOSE event is triggered when the screen has been modified out- side of the application, usually by the window manager and needs to be redrawn.
