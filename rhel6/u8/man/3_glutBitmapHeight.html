NAME glutBitmapHeight - Return the height of a given font, in pixels.

LIBRARY OpenGLUT - bitmapfont

SYNOPSIS #include <openglut.h>

int glutBitmapHeight(void *font);

PARAMETERS font A bitmapped font identifier.

DESCRIPTION Return the line-to-line vertical spacing (in pixels) between lines of a bitmapped font.

Returns 0 if font is invalid.

CAVEATS Does not report the height used by individual characters. This may limit its usefulness. (Compare with other font-metric queries.)
