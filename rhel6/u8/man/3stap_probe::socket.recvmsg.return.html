NAME probe::socket.recvmsg.return - Return from Message being received on socket

SYNOPSIS socket.recvmsg.return

VALUES success Was receive successful? (1 = yes, 0 = no)

protocol Protocol value

flags Socket flags value

name Name of this probe

state Socket state value

size Size of message received (in bytes) or error code if success = 0

type Socket type value

family Protocol family value

CONTEXT The message receiver.

DESCRIPTION Fires at the conclusion of receiving a message on a socket via the sock_recvmsg function.
