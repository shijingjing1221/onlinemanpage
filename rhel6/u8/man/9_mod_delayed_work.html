NAME mod_delayed_work - modify delay of or queue a delayed work

SYNOPSIS bool mod_delayed_work(struct workqueue_struct * wq, struct delayed_work * dwork, unsigned long delay);

ARGUMENTS wq workqueue to use

dwork work to queue

delay number of jiffies to wait before queueing

WARNING unlike upstream mod_delayed_work from newer kernel, this is not safe for atomic ctx! Be sure to review call sites.
