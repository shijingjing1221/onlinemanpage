NAME ldns-resolver - tries to create a resolver from a resolv.conf file.

SYNOPSIS ldns-resolver file

DESCRIPTION ldns-resolver tries to create a resolver from a resolv.conf file. This is only useful to test the library for robusteness with input data.

OPTIONS ldns-resolver takes a filename of the resolv.conf file as input. For example ldns-resolver /etc/resolv.conf will show if the file can be parsed successfully.

AUTHOR Written by the ldns team as an example for ldns usage.
