NAME glutWireCylinder - Draw a wireframe cylinder.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutWireCylinder(GLdouble radius, GLdouble height, GLint slices, GLint stacks);

PARAMETERS radius Radius of cylinder.

height Z height.

slices Number of divisions around the z axis.

stacks Number of divisions along the z axis.

DESCRIPTION glutWireCylinder() draws a wireframe of a cylinder, the center of whose base is at the origin, and whose axis parallels the z axis.

SEE ALSO glutSolidCylinder(3)
