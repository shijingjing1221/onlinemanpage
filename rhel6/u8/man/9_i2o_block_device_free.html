I2O_BLOCK_DEVICE_FRE(9) Message-based devices I2O_BLOCK_DEVICE_FRE(9)

NAME i2o_block_device_free - free the memory of the I2O Block device

SYNOPSIS void i2o_block_device_free(struct i2o_block_device * dev);

ARGUMENTS dev I2O Block device, which should be cleaned up

DESCRIPTION Frees the request queue, gendisk and the i2o_block_device structure.
