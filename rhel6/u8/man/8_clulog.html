NAME clulog - Log a message to the cluster and/or system logs

SYNOPSIS clulog -s severity [-m program_name ] message

DESCRIPTION The clulog command sends a message to syslogd(8).

OPTIONS -s <severity> Log at the specified severity (0-7; 0=ALERT, 7=DEBUG).

-l <priority_filter> Apply the specified filter (0-7) for the message. By default, clulog uses the rgmanagers assigned log level.

-m <program_name> Add the specified program name to the log message

SEE ALSO syslog(2), syslogd(8)
