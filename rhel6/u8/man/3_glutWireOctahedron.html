NAME glutWireOctahedron - Draw a wireframe octahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutWireOctahedron(void);

DESCRIPTION This function draws a regular wireframe 8-sided polyhedron centered at the origin. The vertices are at (+/-1, 0, 0), (0, +/-1, 0), (0, 0, +/-1).

CAVEATS We visit the same vertices the same number of times as for the solid octahedron, but the order is different.

Draws every edge twice.

The lines have normals, but the normals are from the facets, rather than upon the edge. If you squint too hard, the lighting on a wireframe octa- hedron does not look quite right.
