NAME init_copy_desc_masks - copy cpumasks for irq_desc

SYNOPSIS void init_copy_desc_masks(struct irq_desc * old_desc, struct irq_desc * new_desc);

ARGUMENTS old_desc pointer to old irq_desc struct

new_desc pointer to new irq_desc struct

DESCRIPTION Insures affinity and pending_masks are copied to new irq_desc. If !CONFIG_CPUMASKS_OFFSTACK the cpumasks are embedded in the irq_desc struct so the copy is redundant.
