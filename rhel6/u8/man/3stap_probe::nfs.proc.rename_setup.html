NAME probe::nfs.proc.rename_setup - NFS client setting up a rename RPC task

SYNOPSIS nfs.proc.rename_setup

VALUES version NFS version

prot transfer protocol

fh file handle of parent dir

server_ip IP address of server

DESCRIPTION The rename_setup function is used to setup a rename RPC task. Is is not doing the actual rename operation.
