NAME glutWireIcosahedron - Draw a wireframe icosahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutWireIcosahedron(void);

DESCRIPTION This function draws a regular, solid 20-sided polyhedron centered at the origin. The distance from the origin to the vertices is 1. No facet is normal to any of the x, y, or z axes.

SEE ALSO glutSolidIcosahedron(3)
