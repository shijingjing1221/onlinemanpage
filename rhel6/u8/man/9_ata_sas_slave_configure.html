NAME ata_sas_slave_configure - Default slave_config routine for libata devices

SYNOPSIS int ata_sas_slave_configure(struct scsi_device * sdev, struct ata_port * ap);

ARGUMENTS sdev SCSI device to configure

ap ATA port to which SCSI device is attached

RETURNS Zero.
