MPT_FREE_FW_MEMORY(9) Message-based devices MPT_FREE_FW_MEMORY(9)

NAME mpt_free_fw_memory - free firmware memory

SYNOPSIS void mpt_free_fw_memory(MPT_ADAPTER * ioc);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

DESCRIPTION If alt_img is NULL, delete from ioc structure. Else, delete a secondary image in same format.
