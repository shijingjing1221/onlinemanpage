NAME class_dev_iter_exit - finish iteration

SYNOPSIS void class_dev_iter_exit(struct class_dev_iter * iter);

ARGUMENTS iter class iterator to finish

DESCRIPTION Finish an iteration. Always call this function after iteration is complete whether the iteration ran till the end or not.
