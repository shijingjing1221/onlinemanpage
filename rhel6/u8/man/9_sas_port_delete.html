SAS_PORT_DELETE(9) SCSI mid layer SAS_PORT_DELETE(9)

NAME sas_port_delete - remove SAS PORT

SYNOPSIS void sas_port_delete(struct sas_port * port);

ARGUMENTS port SAS PORT to remove

DESCRIPTION Removes the specified SAS PORT. If the SAS PORT has an associated phys, unlink them from the port as well.
