BLK_ALLOC_DEVT(9) Block Devices BLK_ALLOC_DEVT(9)

NAME blk_alloc_devt - allocate a dev_t for a partition

SYNOPSIS int blk_alloc_devt(struct hd_struct * part, dev_t * devt);

ARGUMENTS part partition to allocate dev_t for

devt out parameter for resulting dev_t

DESCRIPTION Allocate a dev_t for block device.

RETURNS 0 on success, allocated dev_t is returned in *devt. -errno on failure.

CONTEXT Might sleep.
