NAME mpt_raid_phys_disk_get_num_paths - returns number paths associated to this phys_num

SYNOPSIS int mpt_raid_phys_disk_get_num_paths(MPT_ADAPTER * ioc, u8 phys_disk_num);

ARGUMENTS ioc Pointer to a Adapter Structure

phys_disk_num io unit unique phys disk num generated by the ioc

RETURN returns number paths
