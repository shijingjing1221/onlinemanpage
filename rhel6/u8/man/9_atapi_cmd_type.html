NAME atapi_cmd_type - Determine ATAPI command type from SCSI opcode

SYNOPSIS int atapi_cmd_type(u8 opcode);

ARGUMENTS opcode SCSI opcode

DESCRIPTION Determine ATAPI command type from opcode.

LOCKING None.

RETURNS ATAPI_{READ|WRITE|READ_CD|PASS_THRU|MISC}
