NAME abrt-oops.conf - Configuration file for ABRT´s Kernel Oops extractor

DESCRIPTION The configuration file consists of items in the format "Option = Value". The following items are recognized:

DropNotReportableOopses = yes / no If you want to see only reportable oopses, set to "yes". Default is no: do not drop them.

OnlyFatalMCE = yes / no If you want to see only fatal MCEs, set to "yes". Defaults is yes: detect only fatal ones.
