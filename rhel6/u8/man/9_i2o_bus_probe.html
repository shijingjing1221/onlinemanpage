NAME i2o_bus_probe - verify if dev is a I2O Bus Adapter device and install it

SYNOPSIS int i2o_bus_probe(struct device * dev);

ARGUMENTS dev device to verify if it is a I2O Bus Adapter device

DESCRIPTION Because we want all Bus Adapters always return 0. Except when we fail. Then we are sad.

Returns 0, except when we fail to excel.
