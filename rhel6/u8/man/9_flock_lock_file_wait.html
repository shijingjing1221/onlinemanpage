NAME flock_lock_file_wait - Apply a FLOCK-style lock to a file

SYNOPSIS int flock_lock_file_wait(struct file * filp, struct file_lock * fl);

ARGUMENTS filp The file to apply the lock to

fl The lock to be applied

DESCRIPTION Add a FLOCK style lock to a file.
