NAME pci_cfg_access_trylock - try to lock PCI config reads/writes

SYNOPSIS bool pci_cfg_access_trylock(struct pci_dev * dev);

ARGUMENTS dev pci device struct

DESCRIPTION Same as pci_cfg_access_lock, but will return 0 if access is already locked, 1 otherwise. This function can be used from atomic contexts.
