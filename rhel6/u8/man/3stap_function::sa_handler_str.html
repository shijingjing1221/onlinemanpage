NAME function::sa_handler_str - Returns the string representation of an sa_handler

SYNOPSIS sa_handler_str(handler:)

ARGUMENTS handler the sa_handler to convert to string.

DESCRIPTION Returns the string representation of an sa_handler. If it is not SIG_DFL, SIG_IGN or SIG_ERR, it will return the address of the handler.
