NAME ecryptfs-unwrap-passphrase - unwrap an eCryptfs mount passphrase from file.

SYNOPSIS ecryptfs-unwrap-passphrase [file]

printf "%s" "wrapping passphrase" | ecryptfs-unwrap-passphrase [file] -

DESCRIPTION ecryptfs-unwrap-passphrase is a utility to unwrap an eCryptfs mount passphrase from file, using a specified wrapping passphrase, and dis- play the decrypted result on standard out.

SEE ALSO ecryptfs(7), ecryptfs-rewrap-passphrase(1), ecryptfs-wrap-passphrase(1)
