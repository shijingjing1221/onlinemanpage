NAME gnutls_pem_base64_encode - convert raw data to Base64 encoded

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pem_base64_encode(const char * msg, const gnutls_datum_t * data, char * result, size_t * result_size);

ARGUMENTS const char * msg is a message to be put in the header

const gnutls_datum_t * data contain the raw data

char * result the place where base64 data will be copied

size_t * result_size holds the size of the result

DESCRIPTION This function will convert the given data to printable data, using the base64 encoding. This is the encoding used in PEM messages.

The output string will be null terminated, although the size will not include the terminating null.

RETURNS On success GNUTLS_E_SUCCESS (0) is returned, GNUTLS_E_SHORT_MEM- ORY_BUFFER is returned if the buffer given is not long enough, or 0 on success.
