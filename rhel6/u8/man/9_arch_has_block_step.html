ARCH_HAS_BLOCK_STEP(9) Kernel Internals ARCH_HAS_BLOCK_STEP(9)

NAME arch_has_block_step - does this CPU support user-mode block-step?

SYNOPSIS arch_has_block_step(void);

ARGUMENTS None

DESCRIPTION If this is defined, then there must be a function declaration or inline for user_enable_block_step, and arch_has_single_step must be defined too. arch_has_block_step should evaluate to nonzero iff the machine supports step-until-branch for user mode. It can be a constant or it can test a CPU feature bit.
