MPT_ADD_SGE(9) Message-based devices MPT_ADD_SGE(9)

NAME mpt_add_sge - Place a simple 32 bit SGE at address pAddr.

SYNOPSIS void mpt_add_sge(void * pAddr, u32 flagslength, dma_addr_t dma_addr);

ARGUMENTS pAddr virtual address for SGE

flagslength SGE flags and data transfer length

dma_addr Physical address

DESCRIPTION This routine places a MPT request frame back on the MPT adapter´s FreeQ.
