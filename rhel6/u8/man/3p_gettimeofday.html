NAME gettimeofday - get the date and time

SYNOPSIS #include <sys/time.h>

int gettimeofday(struct timeval *restrict tp, void *restrict tzp);

DESCRIPTION The gettimeofday() function shall obtain the current time, expressed as seconds and microseconds since the Epoch, and store it in the timeval structure pointed to by tp. The resolution of the system clock is unspecified.

If tzp is not a null pointer, the behavior is unspecified.

RETURN VALUE The gettimeofday() function shall return 0 and no value shall be reserved to indicate an error.

ERRORS No errors are defined.
