SND_PCM_SUSPEND_ALL(9) PCM API SND_PCM_SUSPEND_ALL(9)

NAME snd_pcm_suspend_all - trigger SUSPEND to all substreams in the given pcm

SYNOPSIS int snd_pcm_suspend_all(struct snd_pcm * pcm);

ARGUMENTS pcm the PCM instance

DESCRIPTION After this call, all streams are changed to SUSPENDED state.
