NAME sh64_teardown_tlb_slot - Teardown a translation.

SYNOPSIS void sh64_teardown_tlb_slot(unsigned long long config_addr);

ARGUMENTS config_addr Address of TLB slot.

DESCRIPTION Teardown any existing mapping in the TLB slot config_addr.
