CCW_DEVICE_CLEAR(9) The ccw bus CCW_DEVICE_CLEAR(9)

NAME ccw_device_clear - terminate I/O request processing

SYNOPSIS int ccw_device_clear(struct ccw_device * cdev, unsigned long intparm);

ARGUMENTS cdev target ccw device

intparm interruption parameter; value is only used if no I/O is outstanding, otherwise the intparm associated with the I/O request is returned

DESCRIPTION ccw_device_clear calls csch on cdev´s subchannel.

RETURNS 0 on success, -ENODEV on device not operational, -EINVAL on invalid device state.

CONTEXT Interrupts disabled, ccw device lock held
