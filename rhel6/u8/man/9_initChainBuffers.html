NAME initChainBuffers - Allocate memory for and initialize chain buffers

SYNOPSIS int initChainBuffers(MPT_ADAPTER * ioc);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

DESCRIPTION Allocates memory for and initializes chain buffers, chain buffer control arrays and spinlock.
