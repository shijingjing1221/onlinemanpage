NAME generic_pipe_buf_release - put a reference to a struct pipe_buffer

SYNOPSIS void generic_pipe_buf_release(struct pipe_inode_info * pipe, struct pipe_buffer * buf);

ARGUMENTS pipe the pipe that the buffer belongs to

buf the buffer to put a reference to

DESCRIPTION This function releases a reference to buf.
