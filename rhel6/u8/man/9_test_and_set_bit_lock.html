TEST_AND_SET_BIT_LOC(9) Basic C Library Functions TEST_AND_SET_BIT_LOC(9)

NAME test_and_set_bit_lock - Set a bit and return its old value for lock

SYNOPSIS int test_and_set_bit_lock(int nr, volatile unsigned long * addr);

ARGUMENTS nr Bit to set

addr Address to count from

DESCRIPTION This is the same as test_and_set_bit on x86.
