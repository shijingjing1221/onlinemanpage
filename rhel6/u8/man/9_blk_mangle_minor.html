NAME blk_mangle_minor - scatter minor numbers apart

SYNOPSIS int blk_mangle_minor(int minor);

ARGUMENTS minor minor number to mangle

DESCRIPTION Scatter consecutively allocated minor number apart if MANGLE_DEVT is enabled. Mangling twice gives the original value.

RETURNS Mangled value.

CONTEXT Don´t care.
