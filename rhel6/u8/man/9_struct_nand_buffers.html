NAME struct_nand_buffers - buffer structure for read/write

SYNOPSIS struct nand_buffers { uint8_t ecccalc[NAND_MAX_OOBSIZE]; uint8_t ecccode[NAND_MAX_OOBSIZE]; uint8_t databuf[NAND_MAX_PAGESIZE + NAND_MAX_OOBSIZE]; };

MEMBERS ecccalc[NAND_MAX_OOBSIZE] buffer for calculated ecc

ecccode[NAND_MAX_OOBSIZE] buffer for ecc read from flash

databuf[NAND_MAX_PAGESIZE + NAND_MAX_OOBSIZE] buffer for data - dynamically sized

DESCRIPTION Do not change the order of buffers. databuf and oobrbuf must be in consecutive order.
