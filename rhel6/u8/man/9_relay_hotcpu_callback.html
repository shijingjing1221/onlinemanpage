NAME relay_hotcpu_callback - CPU hotplug callback

SYNOPSIS int __cpuinit relay_hotcpu_callback(struct notifier_block * nb, unsigned long action, void * hcpu);

ARGUMENTS nb notifier block

action hotplug action to take

hcpu CPU number

DESCRIPTION Returns the success/failure of the operation. (NOTIFY_OK, NOTIFY_BAD)
