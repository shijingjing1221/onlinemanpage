NAME wait_for_completion - waits for completion of a task

SYNOPSIS void __sched wait_for_completion(struct completion * x);

ARGUMENTS x holds the state of this particular completion

DESCRIPTION This waits to be signaled for completion of a specific task. It is NOT interruptible and there is no timeout.

See also similar routines (i.e. wait_for_completion_timeout) with timeout and interrupt capability. Also see complete.
