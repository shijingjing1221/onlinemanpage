NAME checkquorum - Check Quorum Watchdog Script

SYNOPSIS checkquorum

DESCRIPTION The checkquorum watchdog script, when copied to the /etc/watchdog.d directory and after enabling/starting the watchdog daemon causes the node to reboot if quorum is lost and not regained within a user config- urable amount of time (default: 60 seconds).

OPTIONS The checkquorum script includes several options which can be set by editing the script with a text editor.

$wait_time Amount of time in seconds to wait after quorum is lost before trigger a reboot (Default: 60 seconds).

$hardreboot Instantly reboot the machine without cleanly shutting down the system. Useful when the machine may hang on reboot. Set to 1 to hard reboot the system, 0 to do a normal reboot.

NOTES checkquorum should never be called outside of watchdog except for debugging purposes.
