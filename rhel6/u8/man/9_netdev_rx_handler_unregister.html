NETDEV_RX_HANDLER_UN(9) Network device support NETDEV_RX_HANDLER_UN(9)

NAME netdev_rx_handler_unregister - unregister receive handler

SYNOPSIS void netdev_rx_handler_unregister(struct net_device * dev);

ARGUMENTS dev device to unregister a handler from

DESCRIPTION Unregister a receive hander from a device.

The caller must hold the rtnl_mutex.
