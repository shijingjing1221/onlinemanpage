NAME shrink_dcache_sb - shrink dcache for a superblock

SYNOPSIS void shrink_dcache_sb(struct super_block * sb);

ARGUMENTS sb superblock

DESCRIPTION Shrink the dcache for the specified super block. This is used to free the dcache before unmounting a file system
