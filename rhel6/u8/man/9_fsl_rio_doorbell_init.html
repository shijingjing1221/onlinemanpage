NAME fsl_rio_doorbell_init - MPC85xx doorbell interface init

SYNOPSIS int fsl_rio_doorbell_init(struct rio_mport * mport);

ARGUMENTS mport Master port implementing the inbound doorbell unit

DESCRIPTION Initializes doorbell unit hardware and inbound DMA buffer ring. Called from fsl_rio_setup. Returns 0 on success or -ENOMEM on failure.
