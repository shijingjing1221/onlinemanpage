NAME clamp - return a value clamped to a given range with strict typechecking

SYNOPSIS clamp(val, min, max);

ARGUMENTS val current value

min minimum allowable value

max maximum allowable value

DESCRIPTION This macro does strict typechecking of min/max to make sure they are of the same type as val. See the unnecessary pointer comparisons.
