SND_PCM_RATE_TO_RATE(9) PCM API SND_PCM_RATE_TO_RATE(9)

NAME snd_pcm_rate_to_rate_bit - converts sample rate to SNDRV_PCM_RATE_xxx bit

SYNOPSIS unsigned int snd_pcm_rate_to_rate_bit(unsigned int rate);

ARGUMENTS rate the sample rate to convert

DESCRIPTION Returns the SNDRV_PCM_RATE_xxx flag that corresponds to the given rate, or SNDRV_PCM_RATE_KNOT for an unknown rate.
