NAME ata_scsiop_inq_std - Simulate INQUIRY command

SYNOPSIS unsigned int ata_scsiop_inq_std(struct ata_scsi_args * args, u8 * rbuf);

ARGUMENTS args device IDENTIFY data / SCSI command of interest.

rbuf Response buffer, to which simulated SCSI cmd output is sent.

DESCRIPTION Returns standard device identification data associated with non-VPD INQUIRY command output.

LOCKING spin_lock_irqsave(host lock)
