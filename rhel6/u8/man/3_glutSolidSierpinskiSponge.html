NAME glutSolidSierpinskiSponge - Draw a solid Spierspinski s sponge.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidSierpinskiSponge(int num_levels, const GLdouble offset[ 3 ], GLdouble scale);

PARAMETERS num_levels Recursive depth.

offset Location vector.

scale Relative size.

DESCRIPTION This function recursively draws a few levels of a solid-shaded Sierpin- ski s Sponge. If num_levels is 0, draws 1 tetrahedron. The offset is a translation. The z axis is normal to the base. The sponge is centered at the origin.

CAVEATS Runtime is exponential in num_levels .
