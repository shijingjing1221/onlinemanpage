NAME kpsereadlink - print contents of symbolic link

SYNOPSIS kpsereadlink file

DESCRIPTION If file is a symbolic link, print its contents (what it links to), and exit successfully. Exit with a failure otherwise.

On systems that do not support symbolic links, kpsereadlink will always fail.

OPTIONS kpsereadlink accepts the following additional options:

--help Print help message and exit.

--version Print version information and exit.
