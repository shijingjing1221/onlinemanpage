BLK_END_REQUEST_ALL(9) Block Devices BLK_END_REQUEST_ALL(9)

NAME blk_end_request_all - Helper function for drives to finish the request.

SYNOPSIS void blk_end_request_all(struct request * rq, int error);

ARGUMENTS rq the request to finish

error

0 for success, < 0 for error

DESCRIPTION Completely finish rq.
