DMAM_POOL_CREATE(9) Memory Management in Linux DMAM_POOL_CREATE(9)

NAME dmam_pool_create - Managed dma_pool_create

SYNOPSIS struct dma_pool * dmam_pool_create(const char * name, struct device * dev, size_t size, size_t align, size_t allocation);

ARGUMENTS name name of pool, for diagnostics

dev device that will be doing the DMA

size size of the blocks in this pool.

align alignment requirement for blocks; must be a power of two

allocation returned blocks won´t cross this boundary (or zero)

DESCRIPTION Managed dma_pool_create. DMA pool created with this function is automatically destroyed on driver detach.
