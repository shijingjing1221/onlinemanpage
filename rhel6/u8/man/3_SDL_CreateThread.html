NAME SDL_CreateThread - Creates a new thread of execution that shares its parents properties.

SYNOPSIS #include "SDL.h" #include "SDL_thread.h"

SDL_Thread *SDL_CreateThread(int (*fn)(void *), void *data);

DESCRIPTION SDL_CreateThread creates a new thread of execution that shares all of its parent s global memory, signal handlers, file descriptors, etc, and runs the function fn passed the void pointer data The thread quits when this function returns.
