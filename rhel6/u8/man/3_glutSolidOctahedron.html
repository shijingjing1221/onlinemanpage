NAME glutSolidOctahedron - Draw a solid octahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidOctahedron(void);

DESCRIPTION This function draws a regular, solid 8-sided polyhedron centered at the origin. The vertices are at (+/-1, 0, 0), (0, +/-1, 0), (0, 0, +/-1).

CAVEATS We visit the same vertices the same number of times as in the wire octa- hedron, but the order is different.
