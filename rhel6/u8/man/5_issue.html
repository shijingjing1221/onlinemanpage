NAME issue - pre-login message and identification file

DESCRIPTION The file /etc/issue is a text file which contains a message or system identification to be printed before the login prompt. It may contain various @char andchar sequences, if supported by the getty-type pro- gram employed on the system.

FILES /etc/issue

SEE ALSO motd(5), agetty(8), mingetty(8)
