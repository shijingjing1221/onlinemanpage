NAME probe::nfs.aop.write_end - NFS client complete writing data

SYNOPSIS nfs.aop.write_end

VALUES i_size file length in bytes

dev device identifier

sb_flag super block flags

page_index offset within mapping, can used a page identifier and position identifier in the page frame

__page the address of page

size write bytes

i_flag file flags

to end address of this write operation

ino inode number

offset start address of this write operation

DESCRIPTION Fires when do a write operation on nfs, often after prepare_write

Update and possibly write a cached page of an NFS file.
