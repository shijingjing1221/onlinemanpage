NAME build-classpath - build a Java CLASSPATH with the named JARs

SYNOPSIS build-classpath JAR1 [JAR2... JARn]

DESCRIPTION Small utility to build a Java CLASSPATH with the named JARs. The CLASSPATH is returned to standard output.

EXAMPLES

build-classpath tyrex saxon wsdl4j will return a string to standard out that is a valid CLASSPATH containing tyrex, saxon, and wsdl4j JARs.
