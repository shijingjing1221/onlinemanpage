NAME devm_kfree - Resource-managed kfree

SYNOPSIS void devm_kfree(struct device * dev, void * p);

ARGUMENTS dev Device this memory belongs to

p Memory to free

DESCRIPTION Free memory allocated with dev_kzalloc.
