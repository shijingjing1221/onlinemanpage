SET_NETDEV_HW_FEATUR(9) Network device support SET_NETDEV_HW_FEATUR(9)

NAME set_netdev_hw_features - Set netdev´s hw_features

SYNOPSIS void set_netdev_hw_features(struct net_device * netdev, u32 hw_features);

ARGUMENTS netdev Network device

hw_features Bitmap of device´s changeable features

DESCRIPTION hw_features must not be changed after netdev initialization. Drivers that set hw_features must not have legacy ethtool ops.
