NAME pam_sm_close_session - PAM service function to terminate session management

SYNOPSIS #define PAM_SM_SESSION

#include <security/pam_modules.h>

PAM_EXTERN int pam_sm_close_session(pam_handle_t *pamh, int flags, int argc, const char **argv);

DESCRIPTION The pam_sm_close_session function is the service module´s implementation of the pam_close_session(3) interface.

This function is called to terminate a session. The only valid value for flags is zero or:

PAM_SILENT Do not emit any messages.

RETURN VALUES PAM_SESSION_ERR Cannot make/remove an entry for the specified session.

PAM_SUCCESS The session was successfully terminated.
