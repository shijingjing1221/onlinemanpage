PCI_NUM_VF(9) Hardware Interfaces PCI_NUM_VF(9)

NAME pci_num_vf - return number of VFs associated with a PF device_release_driver

SYNOPSIS int pci_num_vf(struct pci_dev * dev);

ARGUMENTS dev the PCI device

DESCRIPTION Returns number of VFs, or 0 if SR-IOV is not enabled.
