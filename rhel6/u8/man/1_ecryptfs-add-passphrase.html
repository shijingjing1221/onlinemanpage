NAME ecryptfs-add-passphrase - add an eCryptfs mount passphrase to the ker- nel keyring.

SYNOPSIS ecryptfs-add-passphrase [--fnek]

printf "%s" "passphrase" | ecryptfs-add-passphrase [--fnek] -

DESCRIPTION ecryptfs-add-passphrase is a utility to manually add a passphrase to the kernel keyring.

If the --fnek option is specified, the filename encryption key associ- ated with the input passphrase will also be added to the keyring.

SEE ALSO ecryptfs(7), keyctl(1)
