NAME writeback_inodes_sb_nr_if_idle - start writeback if none underway

SYNOPSIS int writeback_inodes_sb_nr_if_idle(struct super_block * sb, unsigned long nr);

ARGUMENTS sb the superblock

nr the number of pages to write

DESCRIPTION Invoke writeback_inodes_sb if no writeback is currently underway. Returns 1 if writeback was started, 0 if not.
