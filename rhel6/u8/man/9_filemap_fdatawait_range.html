FILEMAP_FDATAWAIT_RA(9) Memory Management in Linux FILEMAP_FDATAWAIT_RA(9)

NAME filemap_fdatawait_range - wait for all under-writeback pages to complete in a given range

SYNOPSIS int filemap_fdatawait_range(struct address_space * mapping, loff_t start, loff_t end);

ARGUMENTS mapping address space structure to wait for

start offset in bytes where the range starts

end offset in bytes where the range ends (inclusive)

DESCRIPTION Walk the list of under-writeback pages of the given address space in the given range and wait for all of them.

This is just a simple wrapper so that callers don´t have to convert offsets to page indexes themselves
