NAME function::proc_mem_data - Program data size (data + stack) in pages

SYNOPSIS proc_mem_data:long()

ARGUMENTS None

DESCRIPTION Returns the current process data size (data + stack) in pages, or zero when there is no current process or the number of pages couldn´t be retrieved.
