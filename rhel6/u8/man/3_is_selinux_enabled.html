is_selinux_enabled(3) SELinux API documentation is_selinux_enabled(3)

NAME is_selinux_enabled - check whether SELinux is enabled

NAME is_selinux_mls_enabled - check whether SELinux is enabled for (Multi Level Securty) MLS

SYNOPSIS #include <selinux/selinux.h>

int is_selinux_enabled();

int is_selinux_mls_enabled();

DESCRIPTION is_selinux_enabled returns 1 if SELinux is running or 0 if it is not. On error, -1 is returned.

is_selinux_mls_enabled returns 1 if SELinux is running in MLS mode or 0 if it is not.
