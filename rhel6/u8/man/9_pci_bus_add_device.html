PCI_BUS_ADD_DEVICE(9) Hardware Interfaces PCI_BUS_ADD_DEVICE(9)

NAME pci_bus_add_device - add a single device

SYNOPSIS int pci_bus_add_device(struct pci_dev * dev);

ARGUMENTS dev device to add

DESCRIPTION This adds a single pci device to the global device list and adds sysfs and procfs entries
