XtInitializeWidgetClass(3) XT FUNCTIONS XtInitializeWidgetClass(3)

NAME XtInitializeWidgetClass - initialize a widget class

SYNTAX void XtInitializeWidgetClass(WidgetClass object_class);

ARGUMENTS object_class Specifies the object class to initialize.

DESCRIPTION If the specified widget class is already initialized, XtInitialize- WidgetClass returns immediately.

If the class initialization procedure registers type converters, these type converters are not available until the first object of the class or subclass is created or XtInitializeWidgetClass is called.
