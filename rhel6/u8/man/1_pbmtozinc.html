NAME pbmtozinc - convert a PBM image into a Zinc bitmap

SYNOPSIS pbmtozinc

[pbmfile]

DESCRIPTION This program is part of Netpbm(1).

pbmtozinc reads a PBM image as input and produces a bitmap in the for- mat used by the Zinc Interface Library (ZIL) Version 1.0 as output.
