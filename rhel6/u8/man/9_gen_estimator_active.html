NAME gen_estimator_active - test if estimator is currently in use

SYNOPSIS bool gen_estimator_active(const struct gnet_stats_basic_packed * bstats, const struct gnet_stats_rate_est * rate_est);

ARGUMENTS bstats basic statistics

rate_est rate estimator statistics

DESCRIPTION Returns true if estimator is active, and false if not.
