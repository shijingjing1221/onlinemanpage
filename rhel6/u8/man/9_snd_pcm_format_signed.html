NAME snd_pcm_format_signed - Check the PCM format is signed linear

SYNOPSIS int snd_pcm_format_signed(snd_pcm_format_t format);

ARGUMENTS format the format to check

DESCRIPTION Returns 1 if the given PCM format is signed linear, 0 if unsigned linear, and a negative error code for non-linear formats.
