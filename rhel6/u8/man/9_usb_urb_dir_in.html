NAME usb_urb_dir_in - check if an URB describes an IN transfer

SYNOPSIS int usb_urb_dir_in(struct urb * urb);

ARGUMENTS urb URB to be checked

DESCRIPTION Returns 1 if urb describes an IN transfer (device-to-host), otherwise 0.
