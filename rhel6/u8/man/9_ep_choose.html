NAME ep_choose - select descriptor endpoint at current device speed

SYNOPSIS struct usb_endpoint_descriptor * ep_choose(struct usb_gadget * g, struct usb_endpoint_descriptor * hs, struct usb_endpoint_descriptor * fs);

ARGUMENTS g gadget, connected and running at some speed

hs descriptor to use for high speed operation

fs descriptor to use for full or low speed operation
