FUNCTION::NS_GID(3stap) Context Functions FUNCTION::NS_GID(3stap)

NAME function::ns_gid - Returns the group ID of a target process as seen in a user namespace

SYNOPSIS ns_gid:long()

ARGUMENTS None

DESCRIPTION This function returns the group ID of a target process as seen in the target user namespace if provided, or the stap process namespace.
