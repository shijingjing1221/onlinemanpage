PCIE_GET_MPS(9) Hardware Interfaces PCIE_GET_MPS(9)

NAME pcie_get_mps - get PCI Express maximum payload size

SYNOPSIS int pcie_get_mps(struct pci_dev * dev);

ARGUMENTS dev PCI device to query

DESCRIPTION Returns maximum payload size in bytes or appropriate error value.
