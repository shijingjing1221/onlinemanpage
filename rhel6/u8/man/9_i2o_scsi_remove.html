I2O_SCSI_REMOVE(9) Message-based devices I2O_SCSI_REMOVE(9)

NAME i2o_scsi_remove - Remove I2O device from SCSI core

SYNOPSIS int i2o_scsi_remove(struct device * dev);

ARGUMENTS dev device which should be removed

DESCRIPTION Removes the I2O device from the SCSI core again.

Returns 0 on success.
