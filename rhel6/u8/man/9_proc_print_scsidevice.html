PROC_PRINT_SCSIDEVIC(9) SCSI mid layer PROC_PRINT_SCSIDEVIC(9)

NAME proc_print_scsidevice - return data about this host

SYNOPSIS int proc_print_scsidevice(struct device * dev, void * data);

ARGUMENTS dev A scsi device

data

struct seq_file to output to.

DESCRIPTION prints Host, Channel, Id, Lun, Vendor, Model, Rev, Type, and revision.
