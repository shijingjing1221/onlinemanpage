NAME vbox - interact with recorded messages

SYNOPSIS vbox [-f|--force] [-o|--mono] [-s|--noledstatus] [-c|--playcmd player] [-m|--hostname server] [-p|--port port] [-r |--reload SECS] [-v|--ver- sion] [-h|--help]

DESCRIPTION This is the program with which you can listen to recorded messages.
