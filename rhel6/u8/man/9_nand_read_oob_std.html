NAME nand_read_oob_std - [REPLACABLE] the most common OOB data read function

SYNOPSIS int nand_read_oob_std(struct mtd_info * mtd, struct nand_chip * chip, int page, int sndcmd);

ARGUMENTS mtd mtd info structure

chip nand chip info structure

page page number to read

sndcmd flag whether to issue read command or not
