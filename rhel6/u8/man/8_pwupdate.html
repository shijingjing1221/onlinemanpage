NAME pwupdate - updates passwd and shadow NIS map

SYNOPSIS /usr/lib64/yp/pwupdate [ shadow ]

DESCRIPTION pwupdate is invoked by rpc.yppasswdd(8) after changing the password and it should update the passwd and shadow NIS map. Never run it by hand.

OPTIONS shadow Normally, only the passwd map will be updated. With this option, pwupdate will update the shadow map, too.

FILES /etc/passwd /etc/shadow

SEE ALSO rpc.yppasswdd(8), ypserv(8)
