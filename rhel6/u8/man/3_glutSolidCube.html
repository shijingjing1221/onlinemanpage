NAME glutSolidCube - Draw a solid cube centered at the origin.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidCube(GLdouble width);

PARAMETERS width The width, height and depth of the cube.

DESCRIPTION The glutSolidCube() function draws a solid-shaded cube with side-length given by width. The vertices of the cube are at (+/- width/2, +/- width/2, +/- width/2), so that the cube is centered at the origin.
