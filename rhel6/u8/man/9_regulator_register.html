REGULATOR_REGISTER(9) API reference REGULATOR_REGISTER(9)

NAME regulator_register - register regulator

SYNOPSIS struct regulator_dev * regulator_register(struct regulator_desc * regulator_desc, struct device * dev, struct regulator_init_data * init_data, void * driver_data);

ARGUMENTS regulator_desc regulator to register

dev struct device for the regulator

init_data platform provided init data, passed through by driver

driver_data private regulator data

DESCRIPTION Called by regulator drivers to register a regulator. Returns 0 on success.
