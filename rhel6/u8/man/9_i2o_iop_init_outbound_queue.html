I2O_IOP_INIT_OUTBOUN(9) Message-based devices I2O_IOP_INIT_OUTBOUN(9)

NAME i2o_iop_init_outbound_queue - setup the outbound message queue

SYNOPSIS int i2o_iop_init_outbound_queue(struct i2o_controller * c);

ARGUMENTS c I2O controller

DESCRIPTION Clear and (re)initialize IOP´s outbound queue and post the message frames to the IOP.

Returns 0 on success or negative error code on failure.
