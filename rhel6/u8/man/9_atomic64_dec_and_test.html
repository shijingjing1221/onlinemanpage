NAME atomic64_dec_and_test - decrement and test

SYNOPSIS int atomic64_dec_and_test(atomic64_t * ptr);

ARGUMENTS ptr pointer to type atomic64_t

DESCRIPTION Atomically decrements ptr by 1 and returns true if the result is 0, or false for all other cases.
