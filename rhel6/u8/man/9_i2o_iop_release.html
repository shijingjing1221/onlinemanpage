NAME i2o_iop_release - release the memory for a I2O controller

SYNOPSIS void i2o_iop_release(struct device * dev);

ARGUMENTS dev I2O controller which should be released

DESCRIPTION Release the allocated memory. This function is called if refcount of device reaches 0 automatically.
