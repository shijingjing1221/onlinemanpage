NAME libtasn1_strerror - Returns a string with a description of an error

SYNOPSIS #include <libtasn1.h>

const char * libtasn1_strerror(asn1_retCode error);

ARGUMENTS asn1_retCode error is an error returned by a libtasn1 function.

DESCRIPTION This function is similar to strerror. The only difference is that it accepts an error (number) returned by a libtasn1 function.

RETURNS Pointer to static zero-terminated string describing error code.

DEPRECATED Use asn1_strerror() instead.
