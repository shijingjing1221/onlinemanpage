NAME kgdb_arch_exit - Perform any architecture specific uninitalization.

SYNOPSIS void kgdb_arch_exit(void);

ARGUMENTS void no arguments

DESCRIPTION This function will handle the uninitalization of any architecture specific callbacks, for dynamic registration and unregistration.
