DESCRIPTION CMPIDateTime class implementation, representing the Encapsulated value objects.

This is the native CMPIDateTime implementation as used for remote providers. It reflects the well-defined interface of a regular CMPI- DateTime, however, it works independently from the management broker.

It is part of a native broker implementation that simulates CMPI data types rather than interacting with the entities in a full-grown CIMOM.
