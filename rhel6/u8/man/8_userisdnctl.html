NAME userisdnctl - Control and setup ISDN parameters

SYNOPSIS userisdnctl interface options

DESCRIPTION This tool fetches the interface configuration parameters for the ISDN and then calls the apropriate scripts to take the requested action, or report the interface status.

OPTIONS interface The interface to be used (will search interface parameters at /etc/sysconfig/ifcfg-INTERFACE)

dial Dial a connection calling the isdndial

hangup Finish an existing connection using the isdnhangup

status Show the status of an existing connection calling the isdnstatus

report Report the interface status by returning 0 if the interface sta- tus looks good and 1 otherwise

SEE ALSO isdnctrl(4), isdninfo(4)

SECURITY userisdnctl needs to be setuid root
