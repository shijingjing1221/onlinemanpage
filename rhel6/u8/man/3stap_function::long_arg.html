NAME function::long_arg - Return function argument as signed long

SYNOPSIS long_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the value of argument n as a signed long. On architectures where a long is 32 bits, the value is sign-extended to 64 bits.

SEE ALSO tapset::registers(3stap)
