NAME ppmtopuzz - convert a PPM image to an X11 puzzle file

SYNOPSIS ppmtopuzz

[ppmfile]

DESCRIPTION This program is part of Netpbm(1).

ppmtopuzz reads a PPM image as input and produces an X11 puzzle file as output. A puzzle file is for use with the puzzle program included with the X11 distribution. puzzles -picture option lets you specify an image file.

SEE ALSO ppm(1), puzzle
