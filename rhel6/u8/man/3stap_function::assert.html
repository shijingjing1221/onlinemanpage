NAME function::assert - evaluate assertion

SYNOPSIS assert(expression:,msg:)

ARGUMENTS expression The expression to evaluate

msg The formatted message string

DESCRIPTION This function checks the expression and aborts the current running probe if expression evaluates to zero. Uses error and may be caught by try{} catch{}.
