SYSCALL_GET_ERROR(9) Machine State SYSCALL_GET_ERROR(9)

NAME syscall_get_error - check result of traced system call

SYNOPSIS long syscall_get_error(struct task_struct * task, struct pt_regs * regs);

ARGUMENTS task task of interest, must be blocked

regs

task_pt_regs of task

DESCRIPTION Returns 0 if the system call succeeded, or -ERRORCODE if it failed.

It´s only valid to call this when task is stopped for tracing on exit from a system call, due to TIF_SYSCALL_TRACE or TIF_SYSCALL_AUDIT.
