SCSI_FINISH_COMMAND(9) SCSI mid layer SCSI_FINISH_COMMAND(9)

NAME scsi_finish_command - cleanup and pass command back to upper layer

SYNOPSIS void scsi_finish_command(struct scsi_cmnd * cmd);

ARGUMENTS cmd the command

DESCRIPTION Pass command off to upper layer for finishing of I/O request, waking processes that are waiting on results, etc.
