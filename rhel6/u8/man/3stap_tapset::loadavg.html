NAME tapset::loadavg - systemtap loadavg tapset

DESCRIPTION Functions in the loadavg tapset allow a probe handler to capture the load average.

See function::get_loadavg_index(3stap) for details.

sprint_loadavg Report a pretty-printed load average

See function::sprint_loadavg(3stap) for details.
