INPUT_CLOSE_DEVICE(9) Input Subsystem INPUT_CLOSE_DEVICE(9)

NAME input_close_device - close input device

SYNOPSIS void input_close_device(struct input_handle * handle);

ARGUMENTS handle handle through which device is being accessed

DESCRIPTION This function should be called by input handlers when they want to stop receive events from given input device.
