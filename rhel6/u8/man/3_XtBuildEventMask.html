NAME XtBuildEventMask - retrieve a widgets event mask

SYNTAX EventMask XtBuildEventMask(Widget w);

ARGUMENTS w Specifies the widget.

DESCRIPTION The XtBuildEventMask function returns the event mask representing the logical OR of all event masks for event handlers registered on the wid- get with XtAddEventHandler and all event translations, including accel- erators, installed on the widget. This is the same event mask stored into the XSetWindowAttributes structure by XtRealizeWidget and sent to the server when event handlers and translations are installed or removed on the realized widget.
