IPC_LOCK(9) Kernel IPC facilities IPC_LOCK(9)

NAME ipc_lock - Lock an ipc structure without rw_mutex held

SYNOPSIS struct kern_ipc_perm * ipc_lock(struct ipc_ids * ids, int id);

ARGUMENTS ids IPC identifier set

id ipc id to look for

DESCRIPTION Look for an id in the ipc ids idr and lock the associated ipc object.

The ipc object is locked on successful exit.
