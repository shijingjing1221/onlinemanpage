NAME encode_rs16 - Calculate the parity for data values (16bit data width)

SYNOPSIS int encode_rs16(struct rs_control * rs, uint16_t * data, int len, uint16_t * par, uint16_t invmsk);

ARGUMENTS rs the rs control structure

data data field of a given type

len data length

par parity data, must be initialized by caller (usually all 0)

invmsk invert data mask (will be xored on data, not on parity!)

DESCRIPTION Each field in the data array contains up to symbol size bits of valid data.
