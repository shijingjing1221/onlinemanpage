NAME probe::nfs.fop.read - NFS client read operation

SYNOPSIS nfs.fop.read

VALUES devname block device name

DESCRIPTION SystemTap uses the vfs.do_sync_read probe to implement this probe and as a result will get operations other than the NFS client read operations.
