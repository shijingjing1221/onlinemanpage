NAME amanda-scripts - Configuring and using the Script API

DESCRIPTION The Script API is a flexible system for invoking user-supplied scripts at various points in the execution of Amanda. This manual page describes the operation and configuration of the API. For help writing Script API scripts, see http://wiki.zmanda.com/index.php/Script_API.

SCRIPTS This section lists the scripts included with Amanda, see the individual man page for instructions on using them. For complete How-To information, consult the Amanda wiki at http://wiki.zmanda.com.

· amzfs-snapshot(8), - create/destroy zfs snapshot.

· script-email(8), - send email.
