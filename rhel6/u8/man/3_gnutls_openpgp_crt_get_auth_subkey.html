NAME gnutls_openpgp_crt_get_auth_subkey - Gets the keyID of an authentica- tion subkey

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_crt_get_auth_subkey(gnutls_openpgp_crt_t crt, gnutls_openpgp_keyid_t keyid, unsigned int flag);

ARGUMENTS gnutls_openpgp_crt_t crt the structure that contains the OpenPGP public key.

gnutls_openpgp_keyid_t keyid the struct to save the keyid.

unsigned int flag Non zero indicates that a valid subkey is always returned.

DESCRIPTION Returns the 64-bit keyID of the first valid OpenPGP subkey marked for authentication. If flag is non zero and no authentication subkey exists, then a valid subkey will be returned even if it is not marked for authentication. Returns the 64-bit keyID of the first valid OpenPGP subkey marked for authentication. If flag is non zero and no authentication subkey exists, then a valid subkey will be returned even if it is not marked for authentication.

RETURNS GNUTLS_E_SUCCESS on success, or an error code.
