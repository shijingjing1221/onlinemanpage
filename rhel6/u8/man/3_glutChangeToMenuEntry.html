NAME glutChangeToMenuEntry - Replace a menu entry with an item.

LIBRARY OpenGLUT - menus

SYNOPSIS #include <openglut.h>

void glutChangeToMenuEntry(int item, const char *label, int value);

PARAMETERS item Integer position down the list

label Menu item text

value Menu item callback value

DESCRIPTION Walks the list of the menu items and replaces the numbered item in the list with the given definition. Except that it replaces a pre-existing item, this function is much like glutAddMenuEntry().

SEE ALSO glutAddMenuEntry(3) glutAddSubMenu(3) glutChangeToSubMenu(3) glutRemoveMenuItem(3)
