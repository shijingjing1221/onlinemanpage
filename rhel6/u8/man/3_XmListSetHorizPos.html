NAME XmListSetHorizPos A List function that scrolls to the specified posi- tion in the list

SYNOPSIS #include <Xm/List.h> void XmListSetHorizPos( Widget widget, int position);

DESCRIPTION XmListSetHorizPos sets the XmNvalue resource of the horizontal Scroll- Bar to the specified position and updates the visible portion of the list with the new value if the List widget s XmNlistSizePolicy is set to XmCONSTANT or XmRESIZE_IF_POSSIBLE and the horizontal ScrollBar is currently visible. This is equivalent to moving the horizontal Scroll- Bar to the specified position.

widget Specifies the ID of the List widget

position Specifies the horizontal position

For a complete definition of List and its associated resources, see XmList(3).
