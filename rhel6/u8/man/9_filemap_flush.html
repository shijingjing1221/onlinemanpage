FILEMAP_FLUSH(9) Memory Management in Linux FILEMAP_FLUSH(9)

NAME filemap_flush - mostly a non-blocking flush

SYNOPSIS int filemap_flush(struct address_space * mapping);

ARGUMENTS mapping target address_space

DESCRIPTION This is a mostly non-blocking flush. Not suitable for data-integrity purposes - I/O may not be started against all dirty pages.
