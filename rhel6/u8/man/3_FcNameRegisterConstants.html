NAME FcNameRegisterConstants - Register symbolic constants

SYNOPSIS #include <fontconfig.h>

FcBool FcNameRegisterConstants(const FcConstant *consts); (int nconsts); .fi

DESCRIPTION Register nconsts new symbolic constants. Returns FcFalse if the con- stants cannot be registered (due to allocation failure). Otherwise returns FcTrue.

VERSION Fontconfig version 2.8.0
