NAME cexp2, cexp2f, cexp2l - base-2 exponent of a complex number

SYNOPSIS #include <complex.h>

double complex cexp2(double complex z); float complex cexp2f(float complex z); long double complex cexp2l(long double complex z);

Link with -lm.

DESCRIPTION The function returns 2 raised to the power of z.

CONFORMING TO These function names are reserved for future use in C99.

AVAILABILITY Not yet in glibc, as at version 2.8.

SEE ALSO cabs(3), cexp(3), clog10(3), complex(7)
