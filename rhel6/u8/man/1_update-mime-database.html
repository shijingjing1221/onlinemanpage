NAME update-mime-database - a program to build the Shared MIME-Info database cache

SYNOPSIS update-mime-database -h | -v | [ -V ] MIME-DIR

DESCRIPTION update-mime-database is a program which is responsible for updating the shared mime-info cache according to the system described in the Shared MIME-Info Database specification from the X Desktop Group.

OPTIONS -h Print out a command summary.

-v Print out the version information.

-V Be verbose.

ARGUMENTS MIME-DIR This is the directory for which update-mime-database should rebuild the cache. Mandatory if none of the options is provided.
