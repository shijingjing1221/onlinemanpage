NAME destroy_workqueue - safely terminate a workqueue

SYNOPSIS void destroy_workqueue(struct workqueue_struct * wq);

ARGUMENTS wq target workqueue

DESCRIPTION Safely destroy a workqueue. All work currently pending will be done first.
