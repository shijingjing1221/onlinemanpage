NAME cracklib-check - Check passwords using libcrack2

SYNOPSIS cracklib-check

DESCRIPTION cracklib-check takes a list of passwords from stdin and checks them via libcrack2s FascistCheck(3) sub routine.

RESULT cracklib-check prints each checked password and the corresponding result of FascistCheck(3) to stdout. The password and the result are separated by a colon.

SEE ALSO FascistCheck(3)
