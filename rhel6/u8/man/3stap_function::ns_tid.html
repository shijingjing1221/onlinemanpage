FUNCTION::NS_TID(3stap) Context Functions FUNCTION::NS_TID(3stap)

NAME function::ns_tid - Returns the thread ID of a target process as seen in a pid namespace

SYNOPSIS ns_tid:long()

ARGUMENTS None

DESCRIPTION This function returns the thread ID of a target process as seen in the target pid namespace if provided, or the stap process namespace.
