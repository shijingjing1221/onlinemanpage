NAME FcNameUnregisterConstants - Unregister symbolic constants

SYNOPSIS #include <fontconfig.h>

FcBool FcNameUnregisterConstants(const FcConstant *consts); (int nconsts); .fi

DESCRIPTION Unregister nconsts symbolic constants. Returns FcFalse if the specified constants were not registered. Otherwise returns FcTrue.

VERSION Fontconfig version 2.8.0
