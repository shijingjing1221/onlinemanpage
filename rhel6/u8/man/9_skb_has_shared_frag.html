NAME skb_has_shared_frag - can any frag be overwritten

SYNOPSIS bool skb_has_shared_frag(struct sk_buff * skb);

ARGUMENTS skb buffer to test

DESCRIPTION Return true if the skb has at least one frag that might be modified by an external entity (as in vmsplice/sendfile)
