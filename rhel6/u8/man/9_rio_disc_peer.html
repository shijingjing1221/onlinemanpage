NAME rio_disc_peer - Recursively discovers a RIO network through a master port

SYNOPSIS int rio_disc_peer(struct rio_net * net, struct rio_mport * port, u16 destid, u8 hopcount);

ARGUMENTS net RIO network being discovered

port Master port to send transactions

destid Current destination ID in network

hopcount Number of hops into the network

DESCRIPTION Recursively discovers a RIO network. Transactions are sent via the master port passed in port.
