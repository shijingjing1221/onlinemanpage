NAME rdev_get_drvdata - get rdev regulator driver data

SYNOPSIS void * rdev_get_drvdata(struct regulator_dev * rdev);

ARGUMENTS rdev regulator

DESCRIPTION Get rdev regulator driver private data. This call can be used in the regulator driver context.
