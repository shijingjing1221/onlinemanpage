CRC_ITU_T(9) Basic Kernel Library Functions CRC_ITU_T(9)

NAME crc_itu_t - Compute the CRC-ITU-T for the data buffer

SYNOPSIS u16 crc_itu_t(u16 crc, const u8 * buffer, size_t len);

ARGUMENTS crc previous CRC value

buffer data pointer

len number of bytes in the buffer

DESCRIPTION Returns the updated CRC value
