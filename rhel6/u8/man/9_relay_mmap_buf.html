NAME relay_mmap_buf - mmap channel buffer to process address space

SYNOPSIS int relay_mmap_buf(struct rchan_buf * buf, struct vm_area_struct * vma);

ARGUMENTS buf relay channel buffer

vma vm_area_struct describing memory to be mapped

DESCRIPTION Returns 0 if ok, negative on error

Caller should already have grabbed mmap_sem.
