NAME generic_file_aio_read - generic filesystem read routine

SYNOPSIS ssize_t generic_file_aio_read(struct kiocb * iocb, const struct iovec * iov, unsigned long nr_segs, loff_t pos);

ARGUMENTS iocb kernel I/O control block

iov io vector request

nr_segs number of segments in the iovec

pos current file position

DESCRIPTION This is the read routine for all filesystems that can use the page cache directly.
