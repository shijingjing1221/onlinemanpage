NAME tapset::guru-signal - systemtap guru-signal tapset

DESCRIPTION Functions in the guru-signal tapset allow a probe handler to queue a user-space signals. Such operations may only be safe from some kinds of probe points, therefore are guru-mode only.

raise raise a signal in the current thread

See function::raise(3stap) for details.

SEE ALSO function::raise(3stap), stap(1), stapprobes(3stap)
