NAME glutPositionWindow - Request to change the position of the current window

LIBRARY OpenGLUT - window

SYNOPSIS #include <openglut.h>

void glutPositionWindow(int x, int y);

PARAMETERS x Requested horizontal position of the current window

y Requested vertical position of the current window

DESCRIPTION The glutPositionWindow() function requests that the window system posi- tion a top-level or subwindow relative to the top-left corner. Subwin- dows are typically resized and repositioned in response to window resize events.

CAVEATS The position of top-level windows is ultimately determined by the window- ing system. Therefore, a position request by an OpenGLUT application may not necessarily succeed.

May not take immediate effect; wait for the callback.

Not applicable to offscreen windows.

SEE ALSO glutInit(3) glutInitWindowPosition(3) glutReshapeFunc(3) glutCreateSubWindow(3)
