NAME xcb_set_selection_owner - Sets the owner of a selection

SYNOPSIS #include <xcb/xproto.h>

Request function xcb_void_cookie_t xcb_set_selection_owner(xcb_connection_t *conn, xcb_window_t owner, xcb_atom_t selection, xcb_timestamp_t time);

REQUEST ARGUMENTS conn The XCB connection to X11.

owner The new owner of the selection.

The special value XCB_NONE means that the selection will have no owner.

selection The selection.

time Timestamp to avoid race conditions when running X over the network.

The selection will not be changed if time is earlier than the current last-change time of the selection or is later than the current X server time. Otherwise, the last-change time is set to the specified time.

The special value XCB_CURRENT_TIME will be replaced with the current server time.

DESCRIPTION Makes window the owner of the selection selection and updates the last- change time of the specified selection.
