NAME ata_pci_remove_one - PCI layer callback for device removal

SYNOPSIS void ata_pci_remove_one(struct pci_dev * pdev);

ARGUMENTS pdev PCI device that was removed

DESCRIPTION PCI layer indicates to libata via this hook that hot-unplug or module unload event has occurred. Detach all ports. Resource release is handled via devres.

LOCKING Inherited from PCI layer (may sleep).
