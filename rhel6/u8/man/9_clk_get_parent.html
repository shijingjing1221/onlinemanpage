NAME clk_get_parent - get the parent clock source for this clock

SYNOPSIS struct clk * clk_get_parent(struct clk * clk);

ARGUMENTS clk clock source

DESCRIPTION Returns struct clk corresponding to parent clock source, or valid IS_ERR condition containing errno.
