NAME eth_get_headlen - determine the the length of header for an ethernet frame

SYNOPSIS u32 eth_get_headlen(void * data, unsigned int len);

ARGUMENTS data pointer to start of frame

len total length of frame

DESCRIPTION Make a best effort attempt to pull the length for all of the headers for a given frame in a linear buffer.
