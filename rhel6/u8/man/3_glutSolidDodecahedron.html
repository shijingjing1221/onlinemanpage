NAME glutSolidDodecahedron - Draw a solid dodecahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidDodecahedron(void);

DESCRIPTION This function draws a regular, solid, 12-sided polyhedron centered at the origin. The distance from the origin to the vertices is sqrt(3). The facets are pentagons.
