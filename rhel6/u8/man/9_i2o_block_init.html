I2O_BLOCK_INIT(9) Message-based devices I2O_BLOCK_INIT(9)

NAME i2o_block_init - Block OSM initialization function

SYNOPSIS int i2o_block_init(void);

ARGUMENTS void no arguments

DESCRIPTION Allocate the slab and mempool for request structs, registers i2o_block block device and finally register the Block OSM in the I2O core.

Returns 0 on success or negative error code on failure.
