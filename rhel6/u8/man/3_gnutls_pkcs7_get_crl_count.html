NAME gnutls_pkcs7_get_crl_count - returns the number of CRLs in a PKCS7 crl set

SYNOPSIS #include <gnutls/x509.h>

int gnutls_pkcs7_get_crl_count(gnutls_pkcs7_t pkcs7);

ARGUMENTS gnutls_pkcs7_t pkcs7 should contain a gnutls_pkcs7_t structure

DESCRIPTION This function will return the number of certifcates in the PKCS7 or RFC2630 crl set.

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
