NAME libtasn1_perror - prints a string to stderr with a description of an error

SYNOPSIS #include <libtasn1.h>

void libtasn1_perror(asn1_retCode error);

ARGUMENTS asn1_retCode error is an error returned by a libtasn1 function.

DESCRIPTION This function is like perror(). The only difference is that it accepts an error returned by a libtasn1 function.

DEPRECATED Use asn1_perror() instead.
