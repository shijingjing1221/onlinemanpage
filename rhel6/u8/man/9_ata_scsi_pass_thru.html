NAME ata_scsi_pass_thru - convert ATA pass-thru CDB to taskfile

SYNOPSIS unsigned int ata_scsi_pass_thru(struct ata_queued_cmd * qc);

ARGUMENTS qc command structure to be initialized

DESCRIPTION Handles either 12 or 16-byte versions of the CDB.

RETURNS Zero on success, non-zero on failure.
