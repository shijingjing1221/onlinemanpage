NAME scsi_get_command - Allocate and setup a scsi command block

SYNOPSIS struct scsi_cmnd * scsi_get_command(struct scsi_device * dev, gfp_t gfp_mask);

ARGUMENTS dev parent scsi device

gfp_mask allocator flags

RETURNS The allocated scsi command structure.
