NAME usb_endpoint_is_isoc_in - check if the endpoint is isochronous IN

SYNOPSIS int usb_endpoint_is_isoc_in(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns true if the endpoint has isochronous transfer type and IN direction, otherwise it returns false.
