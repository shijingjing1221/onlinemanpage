FUNCTION::TZ_GMTOFF(3stap) Time utility functions FUNCTION::TZ_GMTOFF(3stap)

NAME function::tz_gmtoff - Return local time zone offset

SYNOPSIS tz_gmtoff()

ARGUMENTS None

DESCRIPTION Returns the local time zone offset (seconds west of UTC), as passed by staprun at script startup only.
