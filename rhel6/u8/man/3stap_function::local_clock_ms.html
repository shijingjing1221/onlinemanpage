NAME function::local_clock_ms - Number of milliseconds on the local cpu´s clock

SYNOPSIS local_clock_ms:long()

ARGUMENTS None

DESCRIPTION This function returns the number of milliseconds on the local cpu´s clock. This is always monotonic comparing on the same cpu, but may have some drift between cpus (within about a jiffy).
