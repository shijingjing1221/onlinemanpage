NAME gnutls_dh_params_deinit - deinitialize the DH parameters

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_dh_params_deinit(gnutls_dh_params_t dh_params);

ARGUMENTS gnutls_dh_params_t dh_params Is a structure that holds the prime numbers

DESCRIPTION This function will deinitialize the DH parameters structure.
