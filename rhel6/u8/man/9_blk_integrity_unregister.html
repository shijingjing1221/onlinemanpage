BLK_INTEGRITY_UNREGI(9) Block Devices BLK_INTEGRITY_UNREGI(9)

NAME blk_integrity_unregister - Remove block integrity profile

SYNOPSIS void blk_integrity_unregister(struct gendisk * disk);

ARGUMENTS disk disk whose integrity profile to deallocate

DESCRIPTION This function frees all memory used by the block integrity profile. To be called at device teardown.
