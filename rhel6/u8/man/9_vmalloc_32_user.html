VMALLOC_32_USER(9) Memory Management in Linux VMALLOC_32_USER(9)

NAME vmalloc_32_user - allocate zeroed virtually contiguous 32bit memory

SYNOPSIS void * vmalloc_32_user(unsigned long size);

ARGUMENTS size allocation size

DESCRIPTION The resulting memory area is 32bit addressable and zeroed so it can be mapped to userspace without leaking data.
