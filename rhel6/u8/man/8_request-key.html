NAME request-key - Handle key instantiation callback requests from the ker- nel

SYNOPSIS /sbin/request-key <op> <key> <uid> <gid> <threadring> <processring> <sessionring> [<info>]

DESCRIPTION This program is invoked by the kernel when the kernel is asked for a key that it doesnt have immediately available. The kernel creates a partially set up key and then calls out to this program to instantiate it. It is not intended to be called directly.

ERRORS All errors will be logged to the syslog.

FILES /etc/request-key.conf Instantiation handler configuration file.

/etc/request-key.d/<keytype>.conf Keytype specific configuration file.

SEE ALSO keyctl(1), request-key.conf(5)
