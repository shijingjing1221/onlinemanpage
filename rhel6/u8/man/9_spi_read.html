SPI_READ(9) Serial Peripheral Interface (S SPI_READ(9)

NAME spi_read - SPI synchronous read

SYNOPSIS int spi_read(struct spi_device * spi, u8 * buf, size_t len);

ARGUMENTS spi device from which data will be read

buf data buffer

len data buffer size

CONTEXT can sleep

DESCRIPTION This reads the buffer and returns zero or a negative error code. Callable only from contexts that can sleep.
