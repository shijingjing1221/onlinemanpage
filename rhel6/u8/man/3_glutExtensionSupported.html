NAME glutExtensionSupported - Determine if an OpenGL extension is available.

LIBRARY OpenGLUT - opengl

SYNOPSIS #include <openglut.h>

int glutExtensionSupported(const char *extension);

PARAMETERS extension A string-name of an extension.

DESCRIPTION Returns 0 if OpenGLUT cannot determine that the requested extension is definitely present. Only checks extensions from glGetString().
