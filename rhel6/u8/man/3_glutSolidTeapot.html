NAME glutSolidTeapot - Draw a solid teapot.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidTeapot(GLdouble size);

PARAMETERS size Scale factor.

DESCRIPTION Draws the standard Teapot, solid shaded, using OpenGL evaluators. This is the classic "Utah Teapot" of computer graphics. The base should lie in the xy-plane with "up" being along the positive z axis.

This is derived from SGI code. It should also be the same as the teapot modeled by Martin Newell in 1975.

SEE ALSO glutWireTeapot(3)
