NAME verify_blkparse - verifies an output file produced by blkparse

SYNOPSIS verify_blkparse <filename>

DESCRIPTION Verifies an output file from blkparse. All it does is check if the events in the file are correctly time ordered. If an entry is found that isnt ordered, its dumped to stdout.
