FUNCTION::TASK_STATE(3stap) Context Functions FUNCTION::TASK_STATE(3stap)

NAME function::task_state - The state of the task

SYNOPSIS task_state:long(task:long)

ARGUMENTS task task_struct pointer

DESCRIPTION Return the state of the given task, one of: TASK_RUNNING (0), TASK_INTERRUPTIBLE (1), TASK_UNINTERRUPTIBLE (2), TASK_STOPPED (4), TASK_TRACED (8), EXIT_ZOMBIE (16), or EXIT_DEAD (32).
