NAME ecryptfs-insert-wrapped-passphrase-into-keyring - unwrap a wrapped passphrase from file and insert into the kernel keyring.

SYNOPSIS ecryptfs-insert-wrapped-passphrase-into-keyring [file]

printf "%s" "wrapping passphrase" | ecryptfs-insert-wrapped-passphrase- into-keyring [file] -

DESCRIPTION ecryptfs-insert-wrapped-passphrase-into-keyring is a utility to manu- ally unwrap a passphrase from a file, and insert the unwrapped passphrase into the kernel keyring.

SEE ALSO ecryptfs(7), keyctl(1)
