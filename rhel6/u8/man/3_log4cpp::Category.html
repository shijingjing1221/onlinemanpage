Detailed Description This is the central class in the log4j package.

One of the distintive features of log4j (and hence log4cpp) are hierarchal categories and their evaluation.

Note: this function does not pass ownership of the categories in the vector to the caller, only the ownership of the vector. However vector<Category&>* is not legal C++, so we cant follow the default ownership conventions.
