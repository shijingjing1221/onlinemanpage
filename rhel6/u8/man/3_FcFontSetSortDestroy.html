NAME FcFontSetSortDestroy - DEPRECATED destroy a font set

SYNOPSIS #include <fontconfig.h>

FcFontSetSortDestroy(FcFontSet *set); .fi

DESCRIPTION This function is DEPRECATED. FcFontSetSortDestroy destroys set by call- ing FcFontSetDestroy. Applications should use FcFontSetDestroy directly instead.
