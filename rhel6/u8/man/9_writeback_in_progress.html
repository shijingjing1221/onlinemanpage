NAME writeback_in_progress - determine whether there is writeback in progress

SYNOPSIS int writeback_in_progress(struct backing_dev_info * bdi);

ARGUMENTS bdi the device´s backing_dev_info structure.

DESCRIPTION Determine whether there is writeback waiting to be handled against a backing device.
