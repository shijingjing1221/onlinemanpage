SND_PCM_SUSPEND(9) PCM API SND_PCM_SUSPEND(9)

NAME snd_pcm_suspend - trigger SUSPEND to all linked streams

SYNOPSIS int snd_pcm_suspend(struct snd_pcm_substream * substream);

ARGUMENTS substream the PCM substream

DESCRIPTION After this call, all streams are changed to SUSPENDED state.
