NAME rpmfile - list file modes and types in RPM packages

SYNOPSIS rpmfile [-h] [FILE...] [DIR...]

DESCRIPTION rpmfile produces three-column output: file name, file mode (octal), and file type, as determined by file(1). When processing a directory, an additional column is prepended to the output, which is typically RPM file basename.

OPTIONS -h Display this help and exit.
