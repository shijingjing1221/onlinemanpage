NAME pci_wait_for_pending_transaction - waits for pending transaction

SYNOPSIS int pci_wait_for_pending_transaction(struct pci_dev * dev);

ARGUMENTS dev the PCI device to operate on

DESCRIPTION Return 0 if transaction is pending 1 otherwise.
