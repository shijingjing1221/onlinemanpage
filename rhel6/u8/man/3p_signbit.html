NAME signbit - test sign

SYNOPSIS #include <math.h>

int signbit(real-floating x);

DESCRIPTION The signbit() macro shall determine whether the sign of its argument value is negative. NaNs, zeros, and infinities have a sign bit.

RETURN VALUE The signbit() macro shall return a non-zero value if and only if the sign of its argument value is negative.

ERRORS No errors are defined.
