NAND_SELECT_CHIP(9) Internal Functions Provided NAND_SELECT_CHIP(9)

NAME nand_select_chip - [DEFAULT] control CE line

SYNOPSIS void nand_select_chip(struct mtd_info * mtd, int chipnr);

ARGUMENTS mtd MTD device structure

chipnr chipnumber to select, -1 for deselect

DESCRIPTION Default select function for 1 chip devices.
