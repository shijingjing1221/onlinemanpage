NAME blk_free_devt - free a dev_t

SYNOPSIS void blk_free_devt(dev_t devt);

ARGUMENTS devt dev_t to free

DESCRIPTION Free devt which was allocated using blk_alloc_devt.

CONTEXT Might sleep.
