I2O_PROC_FS_DESTROY(9) Message-based devices I2O_PROC_FS_DESTROY(9)

NAME i2o_proc_fs_destroy - Cleanup the all i2o proc entries

SYNOPSIS int __exit i2o_proc_fs_destroy(void);

ARGUMENTS void no arguments

DESCRIPTION Iterate over each I2O controller and remove the entries for it.

Returns 0 on success or negative error code on failure.
