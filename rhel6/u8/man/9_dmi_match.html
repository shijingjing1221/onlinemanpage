NAME dmi_match - compare a string to the dmi field (if exists)

SYNOPSIS bool dmi_match(enum dmi_field f, const char * str);

ARGUMENTS f DMI field identifier

str string to compare the DMI field to

DESCRIPTION Returns true if the requested field equals to the str (including NULL).
