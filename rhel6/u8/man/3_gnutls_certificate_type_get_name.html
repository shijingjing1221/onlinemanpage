NAME gnutls_certificate_type_get_name - Returns a string with the name of the specified certificate type

SYNOPSIS #include <gnutls/gnutls.h>

const char * gnutls_certificate_type_get_name(gnutls_certificate_type_t type);

ARGUMENTS gnutls_certificate_type_t type is a certificate type

DESCRIPTION Convert a gnutls_certificate_type_t type to a string.

RETURNS a string that contains the name of the specified certificate type, or NULL in case of unknown types.
