NAME labs, llabs - return a long integer absolute value

SYNOPSIS #include <stdlib.h>

long labs(long i); long long llabs(long long i);

DESCRIPTION The labs() function shall compute the absolute value of the long inte- ger operand i. The llabs() function shall compute the absolute value of the long long integer operand i. If the result cannot be represented, the behavior is undefined.

RETURN VALUE The labs() function shall return the absolute value of the long integer operand. The labs() function shall return the absolute value of the long long integer operand.

ERRORS No errors are defined.
