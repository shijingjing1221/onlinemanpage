NAME pmdasnmp - Gateway from SNMP to PCP (PMDA)

DESCRIPTION pmdasnmp is a Performance Metrics Domain Agent (PMDA) which provides a generic gateway from PCP queries from a PCP client to SNMP queries to one or more SNMP agents.

INSTALLATION If you want access to the SNMP gateway performance metrics, do the following as root:

# cd $PCP_PMDAS_DIR/snmp # ./Install

If you want to undo the installation, do the following as root:

# cd $PCP_PMDAS_DIR/snmp # ./Remove

pmdasnmp is launched by pmcd(1) and should never be executed directly. The Install and Remove scripts notify pmcd(1) when the agent is installed or removed.
