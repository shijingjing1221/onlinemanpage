NAME function::uint_arg - Return function argument as unsigned int

SYNOPSIS uint_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the value of argument n as an unsigned int (i.e., a 32-bit integer zero-extended to 64 bits).
