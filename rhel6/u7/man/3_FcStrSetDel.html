NAME FcStrSetDel - delete from a string set

SYNOPSIS #include <fontconfig.h>

FcBool FcStrSetDel(FcStrSet *set); (const FcChar8 *s); .fi

DESCRIPTION Removes s from set, returning FcTrue if s was a member else FcFalse.
