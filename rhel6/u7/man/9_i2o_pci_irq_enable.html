I2O_PCI_IRQ_ENABLE(9) Message-based devices I2O_PCI_IRQ_ENABLE(9)

NAME i2o_pci_irq_enable - Allocate interrupt for I2O controller

SYNOPSIS int i2o_pci_irq_enable(struct i2o_controller * c);

ARGUMENTS c i2o_controller that the request is for

DESCRIPTION Allocate an interrupt for the I2O controller, and activate interrupts on the I2O controller.

Returns 0 on success or negative error code on failure.
