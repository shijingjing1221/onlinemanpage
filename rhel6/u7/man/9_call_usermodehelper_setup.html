NAME call_usermodehelper_setup - prepare to call a usermode helper

SYNOPSIS struct subprocess_info * call_usermodehelper_setup(char * path, char ** argv, char ** envp, gfp_t gfp_mask);

ARGUMENTS path path to usermode executable

argv arg vector for process

envp environment for process

gfp_mask gfp mask for memory allocation

DESCRIPTION Returns either NULL on allocation failure, or a subprocess_info structure. This should be passed to call_usermodehelper_exec to exec the process and free the structure.
