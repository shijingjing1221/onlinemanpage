PCI_LTR_SUPPORTED(9) Hardware Interfaces PCI_LTR_SUPPORTED(9)

NAME pci_ltr_supported - check whether a device supports LTR

SYNOPSIS bool pci_ltr_supported(struct pci_dev * dev);

ARGUMENTS dev PCI device

RETURNS True if dev supports latency tolerance reporting, false otherwise.
