I2O_PROC_INIT(9) Message-based devices I2O_PROC_INIT(9)

NAME i2o_proc_init - Init function for procfs

SYNOPSIS int i2o_proc_init(void);

ARGUMENTS void no arguments

DESCRIPTION Registers Proc OSM and creates procfs entries.

Returns 0 on success or negative error code on failure.
