FUNCTION::U_REGISTER(3stap) Context Functions FUNCTION::U_REGISTER(3stap)

NAME function::u_register - Return the unsigned value of the named CPU register

SYNOPSIS u_register:long(name:string)

ARGUMENTS name Name of the register to return

DESCRIPTION Same as register(name), except that if the register is 32 bits wide, it is zero-extended to 64 bits.
