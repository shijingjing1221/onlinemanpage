NAME wcscpy - copy a wide-character string

SYNOPSIS #include <wchar.h>

wchar_t *wcscpy(wchar_t *restrict ws1, const wchar_t *restrict ws2);

DESCRIPTION The wcscpy() function shall copy the wide-character string pointed to by ws2 (including the terminating null wide-character code) into the array pointed to by ws1. If copying takes place between objects that overlap, the behavior is undefined.

RETURN VALUE The wcscpy() function shall return ws1; no return value is reserved to indicate an error.

ERRORS No errors are defined.
