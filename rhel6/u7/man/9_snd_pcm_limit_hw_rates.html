NAME snd_pcm_limit_hw_rates - determine rate_min/rate_max fields

SYNOPSIS int snd_pcm_limit_hw_rates(struct snd_pcm_runtime * runtime);

ARGUMENTS runtime the runtime instance

DESCRIPTION Determines the rate_min and rate_max fields from the rates bits of the given runtime->hw.

Returns zero if successful.
