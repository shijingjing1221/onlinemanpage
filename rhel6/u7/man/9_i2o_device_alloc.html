I2O_DEVICE_ALLOC(9) Message-based devices I2O_DEVICE_ALLOC(9)

NAME i2o_device_alloc - Allocate a I2O device and initialize it

SYNOPSIS struct i2o_device * i2o_device_alloc(void);

ARGUMENTS void no arguments

DESCRIPTION Allocate the memory for a I2O device and initialize locks and lists

Returns the allocated I2O device or a negative error code if the device could not be allocated.
