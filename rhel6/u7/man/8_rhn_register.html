NAME rhn_register - Connect to RHN Satellite or Red Hat Network Classic.

SYNOPSIS rhn_register [options]

DESCRIPTION rhn_register is a client program that registers your system with Red Hat Network Classic (or a RHN Satellite). After registering, your sys- tem can recieve software updates, install new software, and remotely manage your system. It can run both in graphical and text modes.

OPTIONS -v, --verbose Increase the debugging output.

--proxy=PROXY Specify an http proxy to use.

--proxyUser=PROXYUSER Specify a username to use for an authenticated proxy. This option must be used in conjunction with proxyPassword.

--proxyPassword=PROXYPASSWORD Specify a password to use for an authenticated proxy. This option must be used in conjunction with proxyUser.

--nox Run in text mode.

--version Show the programs version number and exit.

-h, --help Show a help message and exit.

FILES /etc/sysconfig/rhn/systemid The digital server ID for this machine if the system has been registered onto RHN Satellite or Red Hat Network Classic. This file does not exist otherwise.

/etc/sysconfig/rhn/up2date The common configuration file used by RHN client programs.
