NAME glutSpecialUpFunc - Sets the special key release callback for the current window

LIBRARY OpenGLUT - input

SYNOPSIS #include <openglut.h>

void glutSpecialUpFunc(void( *callback )( int key, int x, int y ));

PARAMETERS callback Client hook for special key releases.

DESCRIPTION This function provides a way to detect the release of a keyboard key. The keys are reported exactly as with glutSpecialFunc(), save that the callback registered via this function is used to report the event.

This callback is bound to the current window .

CAVEATS Windows created via glutCreateMenuWindow() always cascade keyboard and mouse events to their parent.

SEE ALSO glutSpecialFunc(3) glutKeyboardUpFunc(3)
