NAME glutPostWindowRedisplay - Mark an indicated window as needing a redis- play.

LIBRARY OpenGLUT - window

SYNOPSIS #include <openglut.h>

void glutPostWindowRedisplay(int windowID);

PARAMETERS windowID The OpenGLUT window id to be affected.

DESCRIPTION Similar to glutPostRedisplay(), except that instead of affecting the current window , this function affects an arbitrary window, indicated by the windowID parameter.
