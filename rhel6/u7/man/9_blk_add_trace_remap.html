NAME blk_add_trace_remap - Add a trace for a remap operation

SYNOPSIS void blk_add_trace_remap(struct request_queue * q, struct bio * bio, dev_t dev, sector_t from);

ARGUMENTS q queue the io is for

bio the source bio

dev target device

from source sector

DESCRIPTION Device mapper or raid target sometimes need to split a bio because it spans a stripe (or similar). Add a trace for that action.
