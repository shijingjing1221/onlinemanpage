I2O_IOP_ADD(9) Message-based devices I2O_IOP_ADD(9)

NAME i2o_iop_add - Initialize the I2O controller and add him to the I2O core

SYNOPSIS int i2o_iop_add(struct i2o_controller * c);

ARGUMENTS c controller

DESCRIPTION Initialize the I2O controller and if no error occurs add him to the I2O core.

Returns 0 on success or negative error code on failure.
