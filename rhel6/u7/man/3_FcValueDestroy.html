NAME FcValueDestroy - Free a value

SYNOPSIS #include <fontconfig.h>

void FcValueDestroy(FcValue v); .fi

DESCRIPTION Frees any memory referenced by v. Values of type FcTypeString, FcType- Matrix and FcTypeCharSet reference memory, the other types do not.

VERSION Fontconfig version 2.8.0
