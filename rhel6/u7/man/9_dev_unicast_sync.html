DEV_UNICAST_SYNC(9) Network device support DEV_UNICAST_SYNC(9)

NAME dev_unicast_sync - Synchronize device´s unicast list to another device

SYNOPSIS int dev_unicast_sync(struct net_device * to, struct net_device * from);

ARGUMENTS to destination device

from source device

DESCRIPTION Add newly added addresses to the destination device and release addresses that have no users left. The source device must be locked by netif_tx_lock_bh.

This function is intended to be called from the dev->set_rx_mode function of layered software devices.
