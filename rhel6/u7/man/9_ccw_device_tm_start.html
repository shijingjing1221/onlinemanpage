NAME ccw_device_tm_start - perform start function

SYNOPSIS int ccw_device_tm_start(struct ccw_device * cdev, struct tcw * tcw, unsigned long intparm, u8 lpm);

ARGUMENTS cdev ccw device on which to perform the start function

tcw transport-command word to be started

intparm user defined parameter to be passed to the interrupt handler

lpm mask of paths to use

DESCRIPTION Start the tcw on the given ccw device. Return zero on success, non-zero otherwise.
