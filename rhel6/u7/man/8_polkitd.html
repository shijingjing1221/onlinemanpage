NAME polkitd - PolicyKit daemon

SYNOPSIS polkitd

DESCRIPTION polkitd provides the org.freedesktop.PolicyKit1 D-Bus service on the system message bus. Users or administrators should never need to start this daemon as it will be automatically started by dbus-daemon(1) whenever an application calls into the service.

AUTHOR Written by David Zeuthen davidz@redhat.com with a lot of help from many others.

BUGS Please send bug reports to either the distribution or the polkit-devel mailing list, see the link http://lists.freedesktop.org/mailman/listinfo/polkit-devel on how to subscribe.
