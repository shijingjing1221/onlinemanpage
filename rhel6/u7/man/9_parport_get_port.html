PARPORT_GET_PORT(9) Parallel Port Devices PARPORT_GET_PORT(9)

NAME parport_get_port - increment a port´s reference count

SYNOPSIS struct parport * parport_get_port(struct parport * port);

ARGUMENTS port the port

DESCRIPTION This ensures that a struct parport pointer remains valid until the matching parport_put_port call.
