NAME tracehook_consider_fatal_signal - suppress special handling of fatal signal

SYNOPSIS int tracehook_consider_fatal_signal(struct task_struct * task, int sig);

ARGUMENTS task task receiving the signal

sig signal number being sent

DESCRIPTION Return nonzero to prevent special handling of this termination signal. Normally handler for signal is SIG_DFL. It can be SIG_IGN if sig is ignored, in which case force_sig is about to reset it to SIG_DFL. When this returns zero, this signal might cause a quick termination that does not give the debugger a chance to intercept the signal.

Called with or without task->sighand->siglock held.
