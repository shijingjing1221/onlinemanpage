NAME unlockpt - unlock a pseudo-terminal master/slave pair

SYNOPSIS #define _XOPEN_SOURCE #include <stdlib.h>

int unlockpt(int fd);

DESCRIPTION The unlockpt() function unlocks the slave pseudo-terminal device (pty) corresponding to the master pty referred to by fd.

unlockpt() should be called before opening the slave side of a pty.

RETURN VALUE When successful, unlockpt() returns 0. Otherwise, it returns -1 and sets errno appropriately.

ERRORS EBADF The fd argument is not a file descriptor open for writing.

EINVAL The fd argument is not associated with a master pty.

VERSIONS unlockpt() is provided in glibc since version 2.1.

CONFORMING TO POSIX.1-2001.
