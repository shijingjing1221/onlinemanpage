I2O_IOP_SYSTAB_SET(9) Message-based devices I2O_IOP_SYSTAB_SET(9)

NAME i2o_iop_systab_set - Set the I2O System Table of the specified IOP

SYNOPSIS int i2o_iop_systab_set(struct i2o_controller * c);

ARGUMENTS c I2O controller to which the system table should be send

DESCRIPTION Before the systab could be set i2o_systab_build must be called.

Returns 0 on success or negative error code on failure.
