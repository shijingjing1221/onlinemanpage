NAME xcb_mapping_notify_event_t - keyboard mapping changed

SYNOPSIS #include <xcb/xproto.h>

EVENT FIELDS response_type The type of this event, in this case XCB_MAPPING_NOTIFY. This field is also present in the xcb_generic_event_t and can be used to tell events apart from each other.

sequence The sequence number of the last request processed by the X11 server.

request

first_keycode The first number in the range of the altered mapping.

count The number of keycodes altered.
