NAME ccs_update_schema - Tool to generate cluster relaxng schema for cluster config validation.

SYNOPSIS ccs_update_schema [OPTION]..

DESCRIPTION ccs_update_schema is part of the Cluster Configuration System (CCS). It is used to update the cluster relaxng schema that validates clus- ter.conf.

OPTIONS -h Help. Print out the usage.

-V Print the version information.

-v Be verbose. Mostly for debugging purposes.

-f Ignore local stored cache and regenerate a fresh schema.

SEE ALSO ccs_config_validate(8), cluster.conf(5)
