NAME gnutls_hex2bin - convert hex string into binary buffer.

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_hex2bin(const char * hex_data, size_t hex_size, char * bin_data, size_t * bin_size);

ARGUMENTS const char * hex_data string with data in hex format

size_t hex_size size of hex data

char * bin_data output array with binary data

size_t * bin_size when calling *bin_size should hold size of bin_data, on return will hold actual size of bin_data.

DESCRIPTION Convert a buffer with hex data to binary data.

RETURNS GNUTLS_E_SUCCESS on success, otherwise an error.

SINCE 2.4.0
