NAME gnutls_oprfi_enable_server - API function

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_oprfi_enable_server(gnutls_session_t session, gnutls_oprfi_callback_func cb, void * userdata);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

gnutls_oprfi_callback_func cb function pointer to Opaque PRF extension server callback.

void * userdata hook passed to callback function for passing application state.

DESCRIPTION Request that the server should attempt to accept the Opaque PRF Input TLS extension. If the client requests the extension, the provided callback cb will be invoked. The callback must have the following pro- totype:

int callback (gnutls_session_t session, void *userdata, size_t oprfi_len, const unsigned char *in_oprfi, unsigned char *out_oprfi);

The callback can inspect the client-provided data in the input parame- ters, and specify its own opaque prf input data in the output variable. The function must return 0 on success, otherwise the handshake will be aborted.
