SND_CTL_SYNC_VMASTER(9) Control/Mixer API SND_CTL_SYNC_VMASTER(9)

NAME snd_ctl_sync_vmaster_hook - Sync the vmaster hook

SYNOPSIS void snd_ctl_sync_vmaster_hook(struct snd_kcontrol * kcontrol);

ARGUMENTS kcontrol vmaster kctl element

DESCRIPTION Call the hook function to synchronize with the current value of the given vmaster element. NOP when NULL is passed to kcontrol or the hook doesn´t exist.
