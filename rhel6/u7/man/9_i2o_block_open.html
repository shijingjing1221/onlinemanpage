I2O_BLOCK_OPEN(9) Message-based devices I2O_BLOCK_OPEN(9)

NAME i2o_block_open - Open the block device

SYNOPSIS int i2o_block_open(struct block_device * bdev, fmode_t mode);

ARGUMENTS bdev block device being opened

mode file open mode

DESCRIPTION Power up the device, mount and lock the media. This function is called, if the block device is opened for access.

Returns 0 on success or negative error code on failure.
