DISABLE_CMF(9) The ccw bus DISABLE_CMF(9)

NAME disable_cmf - switch off the channel measurement for a specific device

SYNOPSIS int disable_cmf(struct ccw_device * cdev);

ARGUMENTS cdev The ccw device to be disabled

DESCRIPTION Returns 0 for success or a negative error value.

CONTEXT non-atomic
