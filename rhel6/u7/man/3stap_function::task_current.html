FUNCTION::TASK_CURRE(3stap) Context Functions FUNCTION::TASK_CURRE(3stap)

NAME function::task_current - The current task_struct of the current task

SYNOPSIS task_current:long()

ARGUMENTS None

DESCRIPTION This function returns the task_struct representing the current process. This address can be passed to the various task_*() functions to extract more task-specific data.
