NAME umad_dump - dump umad buffer to stderr

SYNOPSIS #include <infiniband/umad.h>

void umad_dump(void *umad);

DESCRIPTION umad_dump() dumps the given umad buffer to stderr.

RETURN VALUE umad_dump() returns no value.
