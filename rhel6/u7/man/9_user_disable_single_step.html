USER_DISABLE_SINGLE_(9) Kernel Internals USER_DISABLE_SINGLE_(9)

NAME user_disable_single_step - cancel user-mode single-step

SYNOPSIS void user_disable_single_step(struct task_struct * task);

ARGUMENTS task either current or a task stopped in TASK_TRACED

DESCRIPTION Clear task of the effects of user_enable_single_step and user_enable_block_step. This can be called whether or not either of those was ever called on task, and even if arch_has_single_step returned zero.
