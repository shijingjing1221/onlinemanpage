NAME i2o_block_sglist_free - Frees the SG list

SYNOPSIS void i2o_block_sglist_free(struct i2o_block_request * ireq);

ARGUMENTS ireq I2O block request from which the SG should be freed

DESCRIPTION Frees the SG list from the I2O block request.
