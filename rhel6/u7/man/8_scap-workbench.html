NAME scap-workbench - GUI tool for systems compliance evaluation

SYNOPSIS scap-workbench [XCCDF_FILE]

DESCRIPTION scap-workbench is GUI tool for security compliance checking. Compliance can be described with SCAP standards - XCCDF and OVAL.

This tool enables users to: * evaluate local and/or remote machines using SCAP content (scanning) * perform remediation on both local and remote machines (reme- diation) * adjust XCCDF scanning profiles (tailoring)

OPTIONS XCCDF_FILE If this parameter is provided the scanner will immediately open given XCCDF or source datastream (SDS) file after it starts.

SCAP CONTENT Sample content is provided by the OpenSCAP project (in the openscap- content package).

Other sources of SCAP content are:

National Vulnerability Database - http://web.nvd.nist.gov/view/ncp/repository

Red Hat content repository - http://www.redhat.com/security/data/oval/

scap-security-guide project - http://fedorahosted.org/scap-security- guide/

sce-community-content project - http://fedorahosted.org/sce-community- content/
