NAND_VERIFY_BUF16(9) Internal Functions Provided NAND_VERIFY_BUF16(9)

NAME nand_verify_buf16 - [DEFAULT] Verify chip data against buffer

SYNOPSIS int nand_verify_buf16(struct mtd_info * mtd, const uint8_t * buf, int len);

ARGUMENTS mtd MTD device structure

buf buffer containing the data to compare

len number of bytes to compare

DESCRIPTION Default verify function for 16bit buswith
