GET_REQUEST_WAIT(9) Block Devices GET_REQUEST_WAIT(9)

NAME get_request_wait - get a free request with retry

SYNOPSIS struct request * get_request_wait(struct request_queue * q, int rw_flags, struct bio * bio);

ARGUMENTS q request_queue to allocate request from

rw_flags RW and SYNC flags

bio bio to allocate request for (can be NULL)

DESCRIPTION Get a free request from q. This function keeps retrying under memory pressure and fails iff q is dead.

Must be called with q->queue_lock held and, Returns NULL on failure, with q->queue_lock held. Returns !NULL on success, with q->queue_lock *not held*.
