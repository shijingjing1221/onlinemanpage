NAME usb_anchor_empty - is an anchor empty

SYNOPSIS int usb_anchor_empty(struct usb_anchor * anchor);

ARGUMENTS anchor the anchor you want to query

DESCRIPTION returns 1 if the anchor has no urbs associated with it
