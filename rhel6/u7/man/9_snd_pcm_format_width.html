SND_PCM_FORMAT_WIDTH(9) PCM API SND_PCM_FORMAT_WIDTH(9)

NAME snd_pcm_format_width - return the bit-width of the format

SYNOPSIS int snd_pcm_format_width(snd_pcm_format_t format);

ARGUMENTS format the format to check

DESCRIPTION Returns the bit-width of the format, or a negative error code if unknown format.
