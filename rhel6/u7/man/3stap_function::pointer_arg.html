FUNCTION::POINTER_AR(3stap) Context Functions FUNCTION::POINTER_AR(3stap)

NAME function::pointer_arg - Return function argument as pointer value

SYNOPSIS pointer_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the unsigned value of argument n, same as ulong_arg. Can be used with any type of pointer.
