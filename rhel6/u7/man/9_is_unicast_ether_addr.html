NAME is_unicast_ether_addr - Determine if the Ethernet address is unicast

SYNOPSIS int is_unicast_ether_addr(const u8 * addr);

ARGUMENTS addr Pointer to a six-byte array containing the Ethernet address

DESCRIPTION Return true if the address is a unicast address.
