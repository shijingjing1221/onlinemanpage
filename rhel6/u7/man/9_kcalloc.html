KCALLOC(9) Memory Management in Linux KCALLOC(9)

NAME kcalloc - allocate memory for an array. The memory is set to zero.

SYNOPSIS void * kcalloc(size_t n, size_t size, gfp_t flags);

ARGUMENTS n number of elements.

size element size.

flags the type of memory to allocate (see kmalloc).
