NAME ata_link_init - Initialize an ata_link structure

SYNOPSIS void ata_link_init(struct ata_port * ap, struct ata_link * link, int pmp);

ARGUMENTS ap ATA port link is attached to

link Link structure to initialize

pmp Port multiplier port number

DESCRIPTION Initialize link.

LOCKING Kernel thread context (may sleep)
