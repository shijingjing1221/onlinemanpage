NAME XmGetXmDisplay A Display function that returns the XmDisplay object ID for a specified display

SYNOPSIS #include <Xm/Display.h> Widget XmGetXmDisplay( Display *display);

DESCRIPTION XmGetXmDisplay returns the XmDisplay object ID associated with a dis- play. The application can access Display resources with XtGetValues.

display Specifies the display for which the XmDisplay object ID is to be returned

For a complete definition of Display and its associated resources, see XmDisplay(3).

RETURN Returns the XmDisplay object ID for the specified display.

RELATED XmDisplay(3).
