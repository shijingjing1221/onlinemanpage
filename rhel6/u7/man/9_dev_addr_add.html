DEV_ADDR_ADD(9) Network device support DEV_ADDR_ADD(9)

NAME dev_addr_add - Add a device address

SYNOPSIS int dev_addr_add(struct net_device * dev, unsigned char * addr, unsigned char addr_type);

ARGUMENTS dev device

addr address to add

addr_type address type

DESCRIPTION Add a device address to the device or increase the reference count if it already exists.

The caller must hold the rtnl_mutex.
