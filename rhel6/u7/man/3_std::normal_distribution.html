Detailed Description template<typename _RealType = double> class std::normal_distribution< _RealType > A normal continuous distribution for random numbers.

The formula for the normal probability mass function is $ p(x) = ac{1}{ma t{2 i}} e^{- ac{{x - mean}^ {2}}{2 ma ^ {2}} } $.
