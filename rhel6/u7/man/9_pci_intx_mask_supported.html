NAME pci_intx_mask_supported - probe for INTx masking support

SYNOPSIS bool pci_intx_mask_supported(struct pci_dev * dev);

ARGUMENTS dev -- undescribed --

DESCRIPTION Check if the device dev support INTx masking via the config space command word.
