NAME blk_run_queue_async - run a single device queue in workqueue context

SYNOPSIS void blk_run_queue_async(struct request_queue * q);

ARGUMENTS q The queue to run

DESCRIPTION Tells kblockd to perform the equivalent of blk_run_queue on behalf of us.
