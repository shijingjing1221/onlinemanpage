NAME edd_dev_is_type - is this EDD device a ´type´ device?

SYNOPSIS int edd_dev_is_type(struct edd_device * edev, const char * type);

ARGUMENTS edev target edd_device

type a host bus or interface identifier string per the EDD spec

DESCRIPTION Returns 1 (TRUE) if it is a ´type´ device, 0 otherwise.
