FUNCTION::TASK_STIME(3stap) Task Time Tapset FUNCTION::TASK_STIME(3stap)

NAME function::task_stime - System time of the current task

SYNOPSIS task_stime:long()

ARGUMENTS None

DESCRIPTION Returns the system time of the current task in cputime. Does not include any time used by other tasks in this process, nor does it include any time of the children of this task.
