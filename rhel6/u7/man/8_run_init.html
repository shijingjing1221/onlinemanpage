NAME run_init - run an init script in the proper SELinux context

SYNOPSIS run_init SCRIPT [[ARGS]...]

DESCRIPTION Run a init script under the proper context, which is specified in /etc/selinux/POLICYTYPE/contexts/initrc_context.

FILES /etc/passwd - user account information /etc/shadow - encrypted passwords and age information /etc/selinux/POLICYTYPE/contexts/initrc_context - contains the context to run init scripts under

SEE ALSO newrole (1), runcon (1)
