SND_AC97_PCM_CLOSE(9) Control/Mixer API SND_AC97_PCM_CLOSE(9)

NAME snd_ac97_pcm_close - closes the given AC97 pcm

SYNOPSIS int snd_ac97_pcm_close(struct ac97_pcm * pcm);

ARGUMENTS pcm the ac97 pcm instance

DESCRIPTION It frees the locked AC97 slots.
