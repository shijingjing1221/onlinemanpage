NAME function::int_arg - Return function argument as signed int

SYNOPSIS int_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the value of argument n as a signed int (i.e., a 32-bit integer sign-extended to 64 bits).
