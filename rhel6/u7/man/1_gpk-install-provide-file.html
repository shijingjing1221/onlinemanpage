NAME gpk-install-provide-file - GNOME PackageKit Local File Provider

SYNOPSIS gpk-install-provide-file [ --verbose ] [ full_path ]

DESCRIPTION This manual page documents briefly the gpk-install-provide-file com- mand.

gpk-install-provide-file allows you to install a package that provides a filename on the system.

SEE ALSO gpk-install-local-file (1).
