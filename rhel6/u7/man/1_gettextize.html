NAME gettextize - install or upgrade gettext infrastructure

SYNOPSIS gettextize [OPTION]... [package-dir]

DESCRIPTION Prepares a source package to use gettext.

OPTIONS --help print this help and exit

--version print version information and exit

-f, --force force writing of new files even if old exist

--intl install libintl in a subdirectory

--po-dir=DIR specify directory with PO files

--no-changelog dont update or create ChangeLog files

--symlink make symbolic links instead of copying files

-n, --dry-run print modifications but dont perform them
