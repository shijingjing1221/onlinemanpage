PCI_SET_LTR(9) Hardware Interfaces PCI_SET_LTR(9)

NAME pci_set_ltr - set LTR latency values

SYNOPSIS int pci_set_ltr(struct pci_dev * dev, int snoop_lat_ns, int nosnoop_lat_ns);

ARGUMENTS dev PCI device

snoop_lat_ns snoop latency in nanoseconds

nosnoop_lat_ns nosnoop latency in nanoseconds

DESCRIPTION Figure out the scale and set the LTR values accordingly.
