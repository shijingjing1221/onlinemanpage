NAME hpipower - This sample openhpi application excercises HPI Power Management APIs.

SYNOPSIS hpipower [ -d -p -r -u -b -x ]

DESCRIPTION hpipower searches the Resource Presence Table (RPT) for resources with Power Capability. It sends the requested power action to the selected target.

OPTIONS -d power down target object -p power on target object -r reset target object -u unattended -b <n> Specify blade <n> (1...n) -x debug messages
