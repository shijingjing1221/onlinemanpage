NAME redhat-access-insights - Red Hat Access Insights Client tool

SYNOPSIS redhat-access-insights [options]

DESCRIPTION redhat-access-insights is designed to help customers proactively resolve issues affecting business operations in Red Hat Enterprise Linux and Cloud Infrastructure environments. View alerts or learn more at https://access.redhat.com/labs/insights/

OPTIONS -h, --help show this help message and exit

--version Display version

--register Register system to the Red Hat Access Insights service

--update-collection-rules Refresh collection rules from Red Hat

--daily Set Red Hat Access Insights to collect data once per day

--weekly Set Red Hat Access Insights to collect data once per week

--group=GROUP Group to add this system to during registration

--validate Validate remove.conf

DEBUG OPTIONS --test-connection Test connectivity to Red Hat

--force-reregister Forcefully reregister this machine to Red Hat. Use only as directed.

--verbose DEBUG output to stdout

--no-gpg Do not verify GPG signature

--no-upload Do not upload the archive

--no-tar-file Build the directory, but do not tar

--keep-archive Do not delete archive after upload

SEE ALSO redhat-access-insights.conf(8)
