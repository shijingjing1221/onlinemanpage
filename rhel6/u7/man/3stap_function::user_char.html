NAME function::user_char - Retrieves a char value stored in user space

SYNOPSIS user_char:long(addr:long)

ARGUMENTS addr the user space address to retrieve the char from

DESCRIPTION Returns the char value from a given user space address. Returns zero when user space data is not accessible.
