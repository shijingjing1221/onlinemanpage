NAME wmemset - set wide characters in memory

SYNOPSIS #include <wchar.h>

wchar_t *wmemset(wchar_t *ws, wchar_t wc, size_t n);

DESCRIPTION The wmemset() function shall copy the value of wc into each of the first n wide characters of the object pointed to by ws. This function shall not be affected by locale and all wchar_t values shall be treated identically. The null wide character and wchar_t values not corre- sponding to valid characters shall not be treated specially.

If n is zero, the application shall ensure that ws is a valid pointer, and the function shall copy zero wide characters.

RETURN VALUE The wmemset() functions shall return the value of ws.

ERRORS No errors are defined.
