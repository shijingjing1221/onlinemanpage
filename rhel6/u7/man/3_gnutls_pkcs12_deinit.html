NAME gnutls_pkcs12_deinit - This function deinitializes memory used by a gnutls_pkcs12_t structure

SYNOPSIS #include <gnutls/pkcs12.h>

void gnutls_pkcs12_deinit(gnutls_pkcs12_t pkcs12);

ARGUMENTS gnutls_pkcs12_t pkcs12 The structure to be initialized

DESCRIPTION This function will deinitialize a PKCS12 structure.
