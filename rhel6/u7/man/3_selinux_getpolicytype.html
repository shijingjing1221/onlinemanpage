NAME selinux_getpolicytype - get the type of SELinux policy running on the system

SYNOPSIS #include <selinux/selinux.h>

int selinux_getpolicytype();

DESCRIPTION selinux_getpolicytype Reads the contents of the /etc/selinux/config file to determine the SELinux policy used on the system.

RETURN VALUE On success, zero is returned. On failure, -1 is returned.
