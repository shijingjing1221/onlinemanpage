NAME skb_copy_datagram_from_iovec - Copy a datagram from an iovec.

SYNOPSIS int skb_copy_datagram_from_iovec(struct sk_buff * skb, int offset, const struct iovec * from, int from_offset, int len);

ARGUMENTS skb buffer to copy

offset offset in the buffer to start copying to

from io vector to copy to

from_offset offset in the io vector to start copying from

len amount of data to copy to buffer from iovec

DESCRIPTION Returns 0 or -EFAULT.

NOTE the iovec is not modified during the copy.
