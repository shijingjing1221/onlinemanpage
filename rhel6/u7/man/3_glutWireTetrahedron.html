NAME glutWireTetrahedron - Draw a wireframe tetrahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutWireTetrahedron(void);

DESCRIPTION This function draws a regular, wireframe 4-sided polyhedron centered at the origin. The distance from the origin to the vertices is 1.
