NAME probe::socket.create.return - Return from Creation of a socket

SYNOPSIS socket.create.return

VALUES success Was socket creation successful? (1 = yes, 0 = no)

protocol Protocol value

err Error code if success == 0

name Name of this probe

requester Requested by user process or the kernel (1 = kernel, 0 = user)

type Socket type value

family Protocol family value

CONTEXT The requester (user process or kernel)

DESCRIPTION Fires at the conclusion of creating a socket.
