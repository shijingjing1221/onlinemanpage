NAME gnutls_malloc - Allocates and returns data

SYNOPSIS #include <gnutls/gnutls.h>

void * gnutls_malloc(size_t s);

ARGUMENTS size_t s

DESCRIPTION This function will allocate s bytes data, and return a pointer to memory. This function is supposed to be used by callbacks.

The allocation function used is the one set by gnutls_global_set_mem_functions().
