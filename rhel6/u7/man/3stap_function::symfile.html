FUNCTION::SYMFILE(3stap) Context Functions FUNCTION::SYMFILE(3stap)

NAME function::symfile - Return the file name of a given address.

SYNOPSIS symfile:string(addr:long)

ARGUMENTS addr The address to translate.

DESCRIPTION Returns the file name of the given address, if known. If the file name cannot be found, the hex string representation of the address will be returned.
