NAME i2o_config_exit - Configuration OSM exit function

SYNOPSIS void i2o_config_exit(void);

ARGUMENTS void no arguments

DESCRIPTION If old ioctl´s are compiled in exit remove them and unregisters Configuration OSM from I2O core.
