CLK_GET_RATE(9) Clock Framework CLK_GET_RATE(9)

NAME clk_get_rate - obtain the current clock rate (in Hz) for a clock source. This is only valid once the clock source has been enabled.

SYNOPSIS unsigned long clk_get_rate(struct clk * clk);

ARGUMENTS clk clock source
