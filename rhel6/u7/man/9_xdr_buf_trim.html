NAME xdr_buf_trim - lop at most len bytes off the end of buf

SYNOPSIS void xdr_buf_trim(struct xdr_buf * buf, unsigned int len);

ARGUMENTS buf buf to be trimmed

len number of bytes to reduce buf by

DESCRIPTION Trim an xdr_buf by the given number of bytes by fixing up the lengths. Note that it´s possible that we´ll trim less than that amount if the xdr_buf is too small, or if (for instance) it´s all in the head and the parser has already read too far into it.
