NAME ata_gen_ata_sense - generate a SCSI fixed sense block

SYNOPSIS void ata_gen_ata_sense(struct ata_queued_cmd * qc);

ARGUMENTS qc Command that we are erroring out

DESCRIPTION Generate sense block for a failed ATA command qc. Descriptor format is used to accomodate LBA48 block address.

LOCKING None.
