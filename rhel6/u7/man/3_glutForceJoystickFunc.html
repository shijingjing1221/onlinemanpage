NAME glutForceJoystickFunc - Forces a joystick poll and callback.

LIBRARY OpenGLUT - input

SYNOPSIS #include <openglut.h>

void glutForceJoystickFunc(void);

DESCRIPTION Forces the OpenGLUT joystick code to poll your joystick(s) and to call your joystick callbacks with the result. The operation completes, including callbacks, before glutForceJoystickFunc() returns.

BUGS The original WINCE import used a #if to turn this function off. That is wrong. The proper way to handle the joystick code is to generate "null" events. The WINCE code needs to be fixed in og_joystick.c, not here.

SEE ALSO glutJoystickFunc(3)
