ETH_SKB_PAD(9) Network device support ETH_SKB_PAD(9)

NAME eth_skb_pad - Pad buffer to mininum number of octets for Ethernet frame

SYNOPSIS int eth_skb_pad(struct sk_buff * skb);

ARGUMENTS skb Buffer to pad

DESCRIPTION An Ethernet frame should have a minimum size of 60 bytes. This function takes short frames and pads them with zeros up to the 60 byte limit.
