NAME iscsi_destroy_session - destroy iscsi session

SYNOPSIS int iscsi_destroy_session(struct iscsi_cls_session * session);

ARGUMENTS session iscsi_session

DESCRIPTION Can be called by a LLD or iscsi_transport. There must not be any running connections.
