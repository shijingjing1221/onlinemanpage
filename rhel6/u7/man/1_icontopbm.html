NAME icontopbm - convert a Sun icon into a PBM image

SYNOPSIS icontopbm [iconfile]

DESCRIPTION This program is part of Netpbm(1).

icontopbm reads a Sun icon as input and produces a PBM image as output.

SEE ALSO pbmtoicon(1), pbm(1)
