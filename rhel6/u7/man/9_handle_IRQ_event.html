HANDLE_IRQ_EVENT(9) Internal Functions Provided HANDLE_IRQ_EVENT(9)

NAME handle_IRQ_event - irq action chain handler

SYNOPSIS irqreturn_t handle_IRQ_event(unsigned int irq, struct irqaction * action);

ARGUMENTS irq the interrupt number

action the interrupt action chain for this irq

DESCRIPTION Handles the action chain of an irq event
