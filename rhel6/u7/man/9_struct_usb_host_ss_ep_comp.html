NAME struct_usb_host_ss_ep_comp - Valid for SuperSpeed devices only

SYNOPSIS struct usb_host_ss_ep_comp { struct usb_ss_ep_comp_descriptor desc; unsigned char * extra; int extralen; };

MEMBERS desc endpoint companion descriptor, wMaxPacketSize in native byteorder

extra descriptors following this endpoint companion descriptor

extralen how many bytes of extra are valid
