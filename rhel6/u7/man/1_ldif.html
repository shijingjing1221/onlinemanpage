NAME ldif - manipulates an LDIF stream by adding a column with the defined attribute type

SYNOPSIS ldif [-b] attrtype

DESCRIPTION Manipulates an LDIF stream by adding a column with the defined attribute type

OPTIONS A summary of options is included below:

-b Output base64 binary format

USAGE ldif dn < /tmp/ldif
