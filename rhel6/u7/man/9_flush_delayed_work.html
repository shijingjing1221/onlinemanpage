NAME flush_delayed_work - block until a dwork_struct´s callback has terminated

SYNOPSIS void flush_delayed_work(struct delayed_work * dwork);

ARGUMENTS dwork the delayed work which is to be flushed

DESCRIPTION Any timeout is cancelled, and any pending work is run immediately.
