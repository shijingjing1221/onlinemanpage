NAME blk_rq_count_integrity_sg - Count number of integrity scatterlist elements

SYNOPSIS int blk_rq_count_integrity_sg(struct request * rq);

ARGUMENTS rq request with integrity metadata attached

DESCRIPTION Returns the number of elements required in a scatterlist corresponding to the integrity metadata in a request.
