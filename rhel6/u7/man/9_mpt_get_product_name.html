MPT_GET_PRODUCT_NAME(9) Message-based devices MPT_GET_PRODUCT_NAME(9)

NAME mpt_get_product_name - returns product string

SYNOPSIS void mpt_get_product_name(u16 vendor, u16 device, u8 revision, char * prod_name);

ARGUMENTS vendor pci vendor id

device pci device id

revision pci revision id

prod_name string returned

DESCRIPTION Returns product string displayed when driver loads, in /proc/mpt/summary and /sysfs/class/scsi_host/host<X>/version_product
