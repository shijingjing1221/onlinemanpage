DEV_ADDR_DEL(9) Network device support DEV_ADDR_DEL(9)

NAME dev_addr_del - Release a device address.

SYNOPSIS int dev_addr_del(struct net_device * dev, unsigned char * addr, unsigned char addr_type);

ARGUMENTS dev device

addr address to delete

addr_type address type

DESCRIPTION Release reference to a device address and remove it from the device if the reference count drops to zero.

The caller must hold the rtnl_mutex.
