DEV_UNICAST_DELETE(9) Network device support DEV_UNICAST_DELETE(9)

NAME dev_unicast_delete - Release secondary unicast address.

SYNOPSIS int dev_unicast_delete(struct net_device * dev, void * addr);

ARGUMENTS dev device

addr address to delete

DESCRIPTION Release reference to a secondary unicast address and remove it from the device if the reference count drops to zero.

The caller must hold the rtnl_mutex.
