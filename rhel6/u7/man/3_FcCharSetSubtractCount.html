NAME FcCharSetSubtractCount - Subtract and count charsets

SYNOPSIS #include <fontconfig.h>

FcChar32 FcCharSetSubtractCount(const FcCharSet *a); (const FcCharSet *b); .fi

DESCRIPTION Returns the number of chars that are in a but not in b.
