NAME hpisettime - This sample openhpi application excercises Event Log clock APIs.

SYNOPSIS hpisettime -d mm/dd/yyyy -t HH:MM:SS [-x]

DESCRIPTION hpisettime sets new date and time for the Event Log clock.

OPTIONS -d date in mm/dd/yyyy format -t time of day in 24-hr format -x displays eXtra debug messages
