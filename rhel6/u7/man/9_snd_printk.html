SND_PRINTK(9) Miscellaneous Functions SND_PRINTK(9)

NAME snd_printk - printk wrapper

SYNOPSIS snd_printk(fmt, args...);

ARGUMENTS fmt format string

args... variable arguments

DESCRIPTION Works like printk but prints the file and the line of the caller when configured with CONFIG_SND_VERBOSE_PRINTK.
