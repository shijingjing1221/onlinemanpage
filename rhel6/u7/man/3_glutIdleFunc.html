NAME glutIdleFunc - Sets the global idle callback.

LIBRARY OpenGLUT - idletimer

SYNOPSIS #include <openglut.h>

void glutIdleFunc(void( *callback )( void ));

PARAMETERS callback Client function for idle event.

DESCRIPTION When OpenGLUTs glutMainLoop() is doing nothing else, it checks to see if an idle callback set. If so, OpenGLUT invokes that callback.

This callback is not bound to any window.

CAVEATS There is at most one idle callback for your entire application.

SEE ALSO glutTimerFunc(3) glutMainLoop(3) glutMainLoopEvent(3)
