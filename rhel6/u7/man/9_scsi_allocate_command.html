NAME scsi_allocate_command - get a fully allocated SCSI command

SYNOPSIS struct scsi_cmnd * scsi_allocate_command(gfp_t gfp_mask);

ARGUMENTS gfp_mask allocation mask

DESCRIPTION This function is for use outside of the normal host based pools. It allocates the relevant command and takes an additional reference on the pool it used. This function *must* be paired with scsi_free_command which also has the identical mask, otherwise the free pool counts will eventually go wrong and you´ll trigger a bug.

This function should *only* be used by drivers that need a static command allocation at start of day for internal functions.
