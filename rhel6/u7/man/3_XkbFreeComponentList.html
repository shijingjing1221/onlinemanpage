XkbFreeComponentList(3) XKB FUNCTIONS XkbFreeComponentList(3)

NAME XkbFreeComponentList - Free the structure returned by XkbListComponents

SYNOPSIS void XkbFreeComponentList (XkbComponentListPtr list);

ARGUMENTS - list pointer to XkbComponentListRec to free

DESCRIPTION When you are done using the structure returned by XkbListComponents, free it using XkbFreeComponentList.
