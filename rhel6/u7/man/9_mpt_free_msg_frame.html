MPT_FREE_MSG_FRAME(9) Message-based devices MPT_FREE_MSG_FRAME(9)

NAME mpt_free_msg_frame - Place MPT request frame back on FreeQ.

SYNOPSIS void mpt_free_msg_frame(MPT_ADAPTER * ioc, MPT_FRAME_HDR * mf);

ARGUMENTS ioc Pointer to MPT adapter structure

mf Pointer to MPT request frame

DESCRIPTION This routine places a MPT request frame back on the MPT adapter´s FreeQ.
