PCI_ENABLE_SRIOV(9) Hardware Interfaces PCI_ENABLE_SRIOV(9)

NAME pci_enable_sriov - enable the SR-IOV capability

SYNOPSIS int pci_enable_sriov(struct pci_dev * dev, int nr_virtfn);

ARGUMENTS dev the PCI device

nr_virtfn number of virtual functions to enable

DESCRIPTION Returns 0 on success, or negative on failure.
