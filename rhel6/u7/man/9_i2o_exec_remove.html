I2O_EXEC_REMOVE(9) Message-based devices I2O_EXEC_REMOVE(9)

NAME i2o_exec_remove - Called on I2O device removal

SYNOPSIS int i2o_exec_remove(struct device * dev);

ARGUMENTS dev I2O device which was removed

DESCRIPTION Unregisters event notification from Executive I2O device.

Returns 0 on success.
