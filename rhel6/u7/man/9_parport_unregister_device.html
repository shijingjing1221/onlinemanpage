PARPORT_UNREGISTER_D(9) Parallel Port Devices PARPORT_UNREGISTER_D(9)

NAME parport_unregister_device - deregister a device on a parallel port

SYNOPSIS void parport_unregister_device(struct pardevice * dev);

ARGUMENTS dev pointer to structure representing device

DESCRIPTION This undoes the effect of parport_register_device.
