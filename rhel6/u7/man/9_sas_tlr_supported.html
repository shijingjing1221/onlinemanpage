NAME sas_tlr_supported - checking TLR bit in vpd 0x90

SYNOPSIS unsigned int sas_tlr_supported(struct scsi_device * sdev);

ARGUMENTS sdev scsi device struct

DESCRIPTION Check Transport Layer Retries are supported or not. If vpd page 0x90 is present, TRL is supported.
