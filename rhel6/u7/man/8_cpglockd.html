NAME cpglockd - CPG lock server for rgmanager

DESCRIPTION cpglockd utilizes the extended virtual synchrony features of the Corosync Cluster Engine to implement a simplistic, distributed lock server for rgmanager.

COMMAND LINE OPTIONS -F Dont wait for the current node to join the fencing domain at startup

-Q Dont wait for quorum formation at startup

-f Operate in the foreground mode; do not daemonize

-h Print command line usage.

SEE ALSO rgmanager(8), corosync(8)
