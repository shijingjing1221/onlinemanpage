NAME ccwgroup_probe_ccwdev - probe function for slave devices

SYNOPSIS int ccwgroup_probe_ccwdev(struct ccw_device * cdev);

ARGUMENTS cdev ccw device to be probed

DESCRIPTION This is a dummy probe function for ccw devices that are slave devices in a ccw group device.

RETURNS always 0
