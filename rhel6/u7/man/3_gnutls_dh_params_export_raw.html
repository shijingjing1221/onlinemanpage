NAME gnutls_dh_params_export_raw - export the raw DH parameters

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_dh_params_export_raw(gnutls_dh_params_t params, gnutls_datum_t * prime, gnutls_datum_t * generator, unsigned int * bits);

ARGUMENTS gnutls_dh_params_t params Holds the DH parameters

gnutls_datum_t * prime will hold the new prime

gnutls_datum_t * generator will hold the new generator

unsigned int * bits if non null will hold is the primes number of bits

DESCRIPTION This function will export the pair of prime and generator for use in the Diffie-Hellman key exchange. The new parameters will be allocated using gnutls_malloc() and will be stored in the appropriate datum.

RETURNS On success, GNUTLS_E_SUCCESS (zero) is returned, otherwise an error code is returned.
