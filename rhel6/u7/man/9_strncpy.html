STRNCPY(9) Basic C Library Functions STRNCPY(9)

NAME strncpy - Copy a length-limited, NUL-terminated string

SYNOPSIS char * strncpy(char * dest, const char * src, size_t count);

ARGUMENTS dest Where to copy the string to

src Where to copy the string from

count The maximum number of bytes to copy

DESCRIPTION The result is not NUL-terminated if the source exceeds count bytes.

In the case where the length of src is less than that of count, the remainder of dest will be padded with NUL.
