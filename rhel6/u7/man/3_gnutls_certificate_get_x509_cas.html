NAME gnutls_certificate_get_x509_cas - Used to export all the CAs from a gnutls_certificate_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_certificate_get_x509_cas(gnutls_certificate_credentials_t sc, gnutls_x509_crt_t ** x509_ca_list, unsigned int * ncas);

ARGUMENTS gnutls_certificate_credentials_t sc is a gnutls_certificate_credentials_t structure.

gnutls_x509_crt_t ** x509_ca_list will point to the CA list. Should be treated as constant

unsigned int * ncas the number of CAs

DESCRIPTION This function will export all the CAs associated with the given creden- tials.

SINCE 2.4.0
