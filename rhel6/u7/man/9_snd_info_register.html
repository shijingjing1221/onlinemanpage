SND_INFO_REGISTER(9) Proc Info API SND_INFO_REGISTER(9)

NAME snd_info_register - register the info entry

SYNOPSIS int snd_info_register(struct snd_info_entry * entry);

ARGUMENTS entry the info entry

DESCRIPTION Registers the proc info entry.

Returns zero if successful, or a negative error code on failure.
