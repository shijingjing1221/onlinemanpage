SND_FREE_PAGES(9) Management of Cards and Device SND_FREE_PAGES(9)

NAME snd_free_pages - release the pages

SYNOPSIS void snd_free_pages(void * ptr, size_t size);

ARGUMENTS ptr the buffer pointer to release

size the allocated buffer size

DESCRIPTION Releases the buffer allocated via snd_malloc_pages.
