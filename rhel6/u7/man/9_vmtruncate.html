VMTRUNCATE(9) Memory Management in Linux VMTRUNCATE(9)

NAME vmtruncate - unmap mappings freed by truncate syscall

SYNOPSIS int vmtruncate(struct inode * inode, loff_t newsize);

ARGUMENTS inode inode of the file used

newsize file offset to start truncating

DESCRIPTION This function is deprecated and truncate_setsize or truncate_pagecache should be used instead, together with filesystem specific block truncation.
