NAME mod_timer_pinned - modify a timer´s timeout

SYNOPSIS int mod_timer_pinned(struct timer_list * timer, unsigned long expires);

ARGUMENTS timer the timer to be modified

expires new timeout in jiffies

DESCRIPTION mod_timer_pinned is a way to update the expire field of an active timer (if the timer is inactive it will be activated) and not allow the timer to be migrated to a different CPU.

mod_timer_pinned(timer, expires) is equivalent to:

del_timer(timer); timer->expires = expires; add_timer(timer);
