NAME Tk_GetHWND, Tk_AttachHWND - manage interactions between the Windows handle and an X window

SYNOPSIS #include <tkPlatDecls.h>

HWND Tk_GetHWND(window)

Window Tk_AttachHWND(tkwin, hwnd)

ARGUMENTS Window window (in) X token for window.

Tk_Window tkwin (in) Tk window for window.

HWND hwnd (in) Windows HWND for window. _________________________________________________________________

DESCRIPTION Tk_GetHWND returns the Windows HWND identifier for X Windows window given by window.

Tk_AttachHWND binds the Windows HWND identifier to the specified Tk_Window given by tkwin. It returns an X Windows window that encapsu- lates the HWND.

KEYWORDS identifier, window
