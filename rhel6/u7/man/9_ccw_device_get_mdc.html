NAME ccw_device_get_mdc - accumulate max data count

SYNOPSIS int ccw_device_get_mdc(struct ccw_device * cdev, u8 mask);

ARGUMENTS cdev ccw device for which the max data count is accumulated

mask mask of paths to use

DESCRIPTION Return the number of 64K-bytes blocks all paths at least support for a transport command. Return values <= 0 indicate failures.
