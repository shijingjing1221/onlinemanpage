NAME gtnameserv - Naming service

SYNOPSIS tnameserv [OPTIONS]

DESCRIPTION To be written ...

OPTIONS -ORBInitialPort PORT Port on which naming service is to be started.

-ior FILE File in which to store naming services IOR reference.

Standard options:

-help Print help text, then exit.

-version Print version number, then exit.

-JOPTION Pass argument to the Java runtime.
