NAME probe::ipmib.FragFails - Count datagram fragmented unsuccessfully

SYNOPSIS ipmib.FragFails

VALUES skb pointer to the struct sk_buff being acted on

op Value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function ipmib_filter_key. If the packet passes the filter is is counted in the global FragFails (equivalent to SNMP´s MIB IPSTATS_MIB_FRAGFAILS)
