FUNCTION::ASMLINKAGE(3stap) Context Functions FUNCTION::ASMLINKAGE(3stap)

NAME function::asmlinkage - Mark function as declared asmlinkage

SYNOPSIS asmlinkage()

ARGUMENTS None

DESCRIPTION Call this function before accessing arguments using the *_arg functions if the probed kernel function was declared asmlinkage in the source.
