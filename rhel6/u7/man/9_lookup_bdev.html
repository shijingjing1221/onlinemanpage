NAME lookup_bdev - lookup a struct block_device by name

SYNOPSIS struct block_device * lookup_bdev(const char * pathname);

ARGUMENTS pathname special file representing the block device

DESCRIPTION Get a reference to the blockdevice at pathname in the current namespace if possible and return it. Return ERR_PTR(error) otherwise.
