NAME gnutls_handshake_set_max_packet_length - set the maximum size of the handshake

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_handshake_set_max_packet_length(gnutls_session_t session, size_t max);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

size_t max is the maximum number.

DESCRIPTION This function will set the maximum size of all handshake messages. Handshakes over this size are rejected with GNUTLS_E_HAND- SHAKE_TOO_LARGE error code. The default value is 48kb which is typi- cally large enough. Set this to 0 if you do not want to set an upper limit.

The reason for restricting the handshake message sizes are to limit Denial of Service attacks.
