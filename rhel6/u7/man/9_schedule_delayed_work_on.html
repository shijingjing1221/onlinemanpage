NAME schedule_delayed_work_on - queue work in global workqueue on CPU after delay

SYNOPSIS int schedule_delayed_work_on(int cpu, struct delayed_work * dwork, unsigned long delay);

ARGUMENTS cpu cpu to use

dwork job to be done

delay number of jiffies to wait

DESCRIPTION After waiting for a given time this puts a job in the kernel-global workqueue on the specified CPU.
