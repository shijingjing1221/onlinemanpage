NAME repl-monitor - Directory Server replication monitor

SYNOPSIS repl-monitor [-f configuration-file] [-h host] [-p port] [-r] [-c con- nection] [-a alias] [-k color] [-u refresh-url] [-s] [-t refresh-inter- val] [-v]

DESCRIPTION Outputs the status of all of the configured Directory Servers partici- pating in replication. The servers to query for status are specified in the configuration file.

OPTIONS A summary of options is included below:

-h, --host host Hostname of DS server

-p, --port port TCP port

-f, --configfile configuration-file Configuration file

-c, --conn connection Uses the same format as the configfile directive

-a, --alias alias Uses the same format as the configfile directive

-k, --color color Uses the same format as the configfile directive

-r, --skip-header Removes extra HTML tags

-u, --refreshurl refresh url Refresh url

-t, --interval refresh interval Refresh interval

-W, --prompt Prompt for passwords

-s, --text Print plain text report
