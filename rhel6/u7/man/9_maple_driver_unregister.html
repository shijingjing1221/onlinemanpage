NAME maple_driver_unregister - unregister a maple driver.

SYNOPSIS void maple_driver_unregister(struct maple_driver * drv);

ARGUMENTS drv maple driver to unregister.

DESCRIPTION Cleans up after maple_driver_register. To be invoked in the exit path of any module drivers.
