SDL_CDName(3) SDL API Reference SDL_CDName(3)

NAME SDL_CDName - Returns a human-readable, system-dependent identifier for the CD-ROM.

SYNOPSIS #include "SDL.h"

const char *SDL_CDName(int drive);

DESCRIPTION Returns a human-readable, system-dependent identifier for the CD-ROM. drive is the index of the drive. Drive indices start to 0 and end at SDL_CDNumDrives()-1.

EXAMPLES · "/dev/cdrom"

· "E:"

· "/dev/disk/ide/1/master"
