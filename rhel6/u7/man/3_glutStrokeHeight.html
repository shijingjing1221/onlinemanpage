NAME glutStrokeHeight - Returns the height of a given font.

LIBRARY OpenGLUT - fonts

SYNOPSIS #include <openglut.h>

GLfloat glutStrokeHeight(void *fontID);

PARAMETERS fontID A GLUT stroked font identifier.

DESCRIPTION This function reports the height of a font, given by fontID, as a global characteristic of that font.

Returns 0 if fontID is invalid.

CAVEATS Does not report the height used by individual characters. This may limit its usefulness; perhaps we should change it? (And/or add a new function.)
