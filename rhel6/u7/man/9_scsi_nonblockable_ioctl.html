NAME scsi_nonblockable_ioctl - Handle SG_SCSI_RESET

SYNOPSIS int scsi_nonblockable_ioctl(struct scsi_device * sdev, int cmd, void __user * arg, int ndelay);

ARGUMENTS sdev scsi device receiving ioctl

cmd Must be SC_SCSI_RESET

arg pointer to int containing SG_SCSI_RESET_{DEVICE,BUS,HOST}

ndelay file mode O_NDELAY flag
