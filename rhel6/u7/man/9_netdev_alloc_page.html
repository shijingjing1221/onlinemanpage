NETDEV_ALLOC_PAGE(9) Linux Networking NETDEV_ALLOC_PAGE(9)

NAME netdev_alloc_page - allocate a page for ps-rx on a specific device

SYNOPSIS struct page * netdev_alloc_page(struct net_device * dev);

ARGUMENTS dev network device to receive on

DESCRIPTION Allocate a new page node local to the specified device.

NULL is returned if there is no free memory.
