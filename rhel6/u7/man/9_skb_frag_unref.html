SKB_FRAG_UNREF(9) Linux Networking SKB_FRAG_UNREF(9)

NAME skb_frag_unref - release a reference on a paged fragment of an skb.

SYNOPSIS void skb_frag_unref(struct sk_buff * skb, int f);

ARGUMENTS skb the buffer

f the fragment offset

DESCRIPTION Releases a reference on the f´th paged fragment of skb.
