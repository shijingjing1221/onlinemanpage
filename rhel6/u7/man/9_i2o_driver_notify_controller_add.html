NAME i2o_driver_notify_controller_add - Send notification of added controller

SYNOPSIS void i2o_driver_notify_controller_add(struct i2o_driver * drv, struct i2o_controller * c);

ARGUMENTS drv I2O driver

c I2O controller

DESCRIPTION Send notification of added controller to a single registered driver.
