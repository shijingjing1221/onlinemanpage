SND_REQUEST_CARD(9) Management of Cards and Device SND_REQUEST_CARD(9)

NAME snd_request_card - try to load the card module

SYNOPSIS void snd_request_card(int card);

ARGUMENTS card the card number

DESCRIPTION Tries to load the module snd-card-X for the given card number via request_module. Returns immediately if already loaded.
