D_DELETE(9) The Linux VFS D_DELETE(9)

NAME d_delete - delete a dentry

SYNOPSIS void d_delete(struct dentry * dentry);

ARGUMENTS dentry The dentry to delete

DESCRIPTION Turn the dentry into a negative dentry if possible, otherwise remove it from the hash queues so it can be deleted later
