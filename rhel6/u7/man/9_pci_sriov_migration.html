PCI_SRIOV_MIGRATION(9) Hardware Interfaces PCI_SRIOV_MIGRATION(9)

NAME pci_sriov_migration - notify SR-IOV core of Virtual Function Migration

SYNOPSIS irqreturn_t pci_sriov_migration(struct pci_dev * dev);

ARGUMENTS dev the PCI device

DESCRIPTION Returns IRQ_HANDLED if the IRQ is handled, or IRQ_NONE if not.

Physical Function driver is responsible to register IRQ handler using VF Migration Interrupt Message Number, and call this function when the interrupt is generated by the hardware.
