NAME cpow, cpowf, cpowl - complex power functions

SYNOPSIS #include <complex.h>

double complex cpow(double complex x, double complex y); float complex cpowf(float complex x, float complex y); long double complex cpowl(long double complex x, long double complex y);

DESCRIPTION These functions shall compute the complex power function x**y, with a branch cut for the first parameter along the negative real axis.

RETURN VALUE These functions shall return the complex power function value.

ERRORS No errors are defined.
