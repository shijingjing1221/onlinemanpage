NAME print_tainted - return a string to represent the kernel taint state.

SYNOPSIS const char * print_tainted(void);

ARGUMENTS void no arguments

DESCRIPTION ´P´ - Proprietary module has been loaded. ´F´ - Module has been forcibly loaded. ´S´ - SMP with CPUs not designed for SMP. ´R´ - User forced a module unload. ´M´ - System experienced a machine check exception. ´B´ - System has hit bad_page. ´U´ - Userspace-defined naughtiness. ´D´ - Kernel has oopsed before ´A´ - ACPI table overridden. ´W´ - Taint on warning. ´C´ - modules from drivers/staging are loaded. ´I´ - Working around severe firmware bug. ´H´ - Hardware is unsupported.

The string is overwritten by the next call to print_tainted.
