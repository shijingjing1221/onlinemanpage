NAME iscsi_destroy_flashnode_sess - destory flashnode session entry

SYNOPSIS void iscsi_destroy_flashnode_sess(struct iscsi_bus_flash_session * fnode_sess);

ARGUMENTS fnode_sess pointer to flashnode session entry to be destroyed

DESCRIPTION Deletes the flashnode session entry and all children flashnode connection entries from sysfs
