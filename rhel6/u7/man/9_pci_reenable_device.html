PCI_REENABLE_DEVICE(9) Hardware Interfaces PCI_REENABLE_DEVICE(9)

NAME pci_reenable_device - Resume abandoned device

SYNOPSIS int pci_reenable_device(struct pci_dev * dev);

ARGUMENTS dev PCI device to be resumed

DESCRIPTION Note this function is a backend of pci_default_resume and is not supposed to be called by normal code, write proper resume handler and use it instead.
