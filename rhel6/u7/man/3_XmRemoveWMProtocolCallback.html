NAME XmRemoveWMProtocolCallback A VendorShell convenience interface that removes a callback from the internal list

SYNOPSIS #include <Xm/Xm.h> #include <Xm/Protocols.h> void XmRemoveWMProtocolCallback( Widget shell, Atom protocol, XtCallbackProc callback, XtPointer closure);

DESCRIPTION XmRemoveWMProtocolCallback is a convenience interface. It calls XmRe- moveProtocolCallback with the property value set to the atom returned by interning WM_PROTOCOLS.

shell Specifies the widget with which the protocol property is associated

protocol Specifies the protocol atom

callback Specifies the procedure to call when a protocol message is received

closure Specifies the client data to be passed to the callback when it is invoked

For a complete definition of VendorShell and its associated resources, see VendorShell(3).

RELATED VendorShell(3), XmAddWMProtocolCallback(3), XmInternAtom(3), and XmRe- moveProtocolCallback(3).
