NAME utrace_control_pid - control a thread being traced by a tracing engine

SYNOPSIS __must_check int utrace_control_pid(struct pid * pid, struct utrace_engine * engine, enum utrace_resume_action action);

ARGUMENTS pid thread to affect

engine attached engine to affect

action

enum utrace_resume_action for thread to do

DESCRIPTION This is the same as utrace_control, but takes a struct pid pointer rather than a struct task_struct pointer. The caller must hold a ref on pid, but does not need to worry about the task staying valid. If it´s been reaped so that pid points nowhere, then this call returns -ESRCH.
