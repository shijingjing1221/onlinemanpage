KFREE(9) Memory Management in Linux KFREE(9)

NAME kfree - free previously allocated memory

SYNOPSIS void kfree(const void * objp);

ARGUMENTS objp pointer returned by kmalloc.

DESCRIPTION If objp is NULL, no operation is performed.

Don´t free memory not originally allocated by kmalloc or you will run into trouble.
