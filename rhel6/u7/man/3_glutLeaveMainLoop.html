NAME glutLeaveMainLoop - Breaks out of OpenGLUTs glutMainLoop()

LIBRARY OpenGLUT - mainloop

SYNOPSIS #include <openglut.h>

void glutLeaveMainLoop(void);

DESCRIPTION This function allows you to unilaterally tell OpenGLUT that you are done and wish to exit. This is useful if you have also told OpenGLUT to return to you rather than to call exit() directly.
