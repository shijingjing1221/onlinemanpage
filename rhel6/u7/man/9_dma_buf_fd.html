NAME dma_buf_fd - returns a file descriptor for the given dma_buf

SYNOPSIS int dma_buf_fd(struct dma_buf * dmabuf, int flags);

ARGUMENTS dmabuf [in] pointer to dma_buf for which fd is required.

flags [in] flags to give to fd

DESCRIPTION On success, returns an associated ´fd´. Else, returns error.
