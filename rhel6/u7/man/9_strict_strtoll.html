STRICT_STRTOLL(9) Basic C Library Functions STRICT_STRTOLL(9)

NAME strict_strtoll - convert a string to a long long strictly

SYNOPSIS int strict_strtoll(const char * cp, unsigned int base, long long * res);

ARGUMENTS cp The string to be converted

base The number base to use

res The converted result value

DESCRIPTION strict_strtoll is similiar to strict_strtoull, but it allows the first character of a string is ´-´.

It returns 0 if conversion is successful and *res is set to the converted value, otherwise it returns -EINVAL and *res is set to 0.
