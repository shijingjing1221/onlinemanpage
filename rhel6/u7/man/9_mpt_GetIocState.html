MPT_GETIOCSTATE(9) Message-based devices MPT_GETIOCSTATE(9)

NAME mpt_GetIocState - Get the current state of a MPT adapter.

SYNOPSIS u32 mpt_GetIocState(MPT_ADAPTER * ioc, int cooked);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

cooked Request raw or cooked IOC state

DESCRIPTION Returns all IOC Doorbell register bits if cooked==0, else just the Doorbell bits in MPI_IOC_STATE_MASK.
