NAME rio_local_set_device_id - Set the base/extended device id for a port

SYNOPSIS void rio_local_set_device_id(struct rio_mport * port, u16 did);

ARGUMENTS port RIO master port

did Device ID value to be written

DESCRIPTION Writes the base/extended device id from a device.
