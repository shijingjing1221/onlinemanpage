KZALLOC(9) Memory Management in Linux KZALLOC(9)

NAME kzalloc - allocate memory. The memory is set to zero.

SYNOPSIS void * kzalloc(size_t size, gfp_t flags);

ARGUMENTS size how many bytes of memory are required.

flags the type of memory to allocate (see kmalloc).
