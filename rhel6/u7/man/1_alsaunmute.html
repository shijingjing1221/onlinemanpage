NAME alsaunmute - a simple script to initialize ALSA sound devices

SYNOPSIS alsaunmute

DESCRIPTION alsaunmute is a shell script which runs the alsactl(1) utility with init argument (see the script for details).

OPTIONS None.

AUTHORS alsaunmute is written by Jaroslav Kysela <perex@perex.cz>.

SEE ALSO alsactl(1), alsamixer(1), amixer(1)
