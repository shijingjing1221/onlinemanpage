NAME gnutls_pkcs7_deinit - deinitializes a #gnutls_pkcs7_t structure

SYNOPSIS #include <gnutls/x509.h>

void gnutls_pkcs7_deinit(gnutls_pkcs7_t pkcs7);

ARGUMENTS gnutls_pkcs7_t pkcs7 The structure to be initialized

DESCRIPTION This function will deinitialize a PKCS7 structure.
