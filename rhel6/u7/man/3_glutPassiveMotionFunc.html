NAME glutPassiveMotionFunc - Sets the non-dragging (gliding?) mouse-motion callback.

LIBRARY OpenGLUT - input

SYNOPSIS #include <openglut.h>

void glutPassiveMotionFunc(void( *callback )( int x, int y ));

PARAMETERS callback Client mouse-glide hook.

DESCRIPTION If you set this callback on a window, then every time that OpenGLUT detects a change in the mouse position inside that window, with no but- tons pressed on that mouse, OpenGLUT will invoke callback with the win- dow coordinates of the mouse.

This callback is bound to the current window .

CAVEATS Windows created via glutCreateMenuWindow() always cascade keyboard and mouse events to their parent.

SEE ALSO glutMotionFunc(3) glutMouseFunc(3) glutEntryFunc(3)
