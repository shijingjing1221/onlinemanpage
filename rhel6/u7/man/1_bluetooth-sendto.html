NAME bluetooth-sendto - GTK application for transfering files over Bluetooth

SYNOPSIS bluetooth-sendto [--device=XX:XX:XX:XX:XX:XX] [file] [file...]

DESCRIPTION bluetooth-sendto will display a dialog for transfering files over Blue- tooth. bluetooth-sendto is part of gnome-bluetooth, see also http://live.gnome.org/GnomeBluetooth

OPTIONS --device Define the device address to send the file to

file The file(s) to send to the device
