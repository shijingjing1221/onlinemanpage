NAME nand_read_page_raw_syndrome - [Intern] read raw page data without ecc

SYNOPSIS int nand_read_page_raw_syndrome(struct mtd_info * mtd, struct nand_chip * chip, uint8_t * buf, int page);

ARGUMENTS mtd mtd info structure

chip nand chip info structure

buf buffer to store read data

page page number to read

DESCRIPTION We need a special oob layout and handling even when OOB isn´t used.
