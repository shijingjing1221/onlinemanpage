ACCT_AUTO_CLOSE(9) Accounting Framework ACCT_AUTO_CLOSE(9)

NAME acct_auto_close - turn off a filesystem´s accounting if it is on

SYNOPSIS void acct_auto_close(struct super_block * sb);

ARGUMENTS sb super block for the filesystem

DESCRIPTION If the accounting is turned on for a file in the filesystem pointed to by sb, turn accounting off.
