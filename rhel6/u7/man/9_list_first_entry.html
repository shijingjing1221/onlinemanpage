LIST_FIRST_ENTRY(9) Data Types LIST_FIRST_ENTRY(9)

NAME list_first_entry - get the first element from a list

SYNOPSIS list_first_entry(ptr, type, member);

ARGUMENTS ptr the list head to take the element from.

type the type of the struct this is embedded in.

member the name of the list_struct within the struct.

DESCRIPTION Note, that list is expected to be not empty.
