NAME cman_notify - CMAN Notification Daemon run-part alike script

SYNOPSIS cman_notify

DESCRIPTION The cmannotifyd daemon talks to CMAN and provides a mechanism to notify external entities about cluster changes. cman_notify script is in charge to execute all notification scripts in a run-part alike way.

NOTES cman_notify should never be called standalone except for debugging pur- poses.
