NAME bio_uncopy_user - finish previously mapped bio

SYNOPSIS int bio_uncopy_user(struct bio * bio);

ARGUMENTS bio bio being terminated

DESCRIPTION Free pages allocated from bio_copy_user and write back data to user space in case of a read.
