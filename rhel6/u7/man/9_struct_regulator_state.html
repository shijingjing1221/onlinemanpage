NAME struct_regulator_state - regulator state during low power system states

SYNOPSIS struct regulator_state { int uV; unsigned int mode; int enabled; };

MEMBERS uV Operating voltage during suspend.

mode Operating mode during suspend.

enabled Enabled during suspend.

DESCRIPTION This describes a regulators state during a system wide low power state.
