SAS_PHY_FREE(9) SCSI mid layer SAS_PHY_FREE(9)

NAME sas_phy_free - free a SAS PHY

SYNOPSIS void sas_phy_free(struct sas_phy * phy);

ARGUMENTS phy SAS PHY to free

DESCRIPTION Frees the specified SAS PHY.

NOTE This function must only be called on a PHY that has not sucessfully been added using sas_phy_add.
