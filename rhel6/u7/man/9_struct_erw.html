STRUCT ERW(9) The ccw bus STRUCT ERW(9)

NAME struct_erw - extended report word

SYNOPSIS struct erw { __u32 res0:3; __u32 auth:1; __u32 pvrf:1; __u32 cpt:1; __u32 fsavf:1; __u32 cons:1; __u32 scavf:1; __u32 fsaf:1; __u32 scnt:6; __u32 res16:16; };

MEMBERS res0 reserved

auth authorization check

pvrf path-verification-required flag

cpt channel-path timeout

fsavf failing storage address validity flag

cons concurrent sense

scavf secondary ccw address validity flag

fsaf failing storage address format

scnt sense count, if cons == 1

res16 reserved
