Detailed Description template<typename _RealType = double> class std::gamma_distribution< _RealType > A gamma continuous distribution for random numbers.

The formula for the gamma probability mass function is $ p(x) = ac{1}{Gamma(lpha)} x^{lpha - 1} e^{-x} $.
