NAME ata_link_next - link iteration helper

SYNOPSIS struct ata_link * ata_link_next(struct ata_link * link, struct ata_port * ap, enum ata_link_iter_mode mode);

ARGUMENTS link the previous link, NULL to start

ap ATA port containing links to iterate

mode iteration mode, one of ATA_LITER_*

LOCKING Host lock or EH context.

RETURNS Pointer to the next link.
