NAME telldir - current location of a named directory stream

SYNOPSIS #include <dirent.h>

long telldir(DIR *dirp);

DESCRIPTION The telldir() function shall obtain the current location associated with the directory stream specified by dirp.

If the most recent operation on the directory stream was a seekdir(), the directory position returned from the telldir() shall be the same as that supplied as a loc argument for seekdir().

RETURN VALUE Upon successful completion, telldir() shall return the current location of the specified directory stream.

ERRORS No errors are defined.
