NAME ata_sg_setup - DMA-map the scatter-gather table associated with a command.

SYNOPSIS int ata_sg_setup(struct ata_queued_cmd * qc);

ARGUMENTS qc Command with scatter-gather table to be mapped.

DESCRIPTION DMA-map the scatter-gather table associated with queued_cmd qc.

LOCKING spin_lock_irqsave(host lock)

RETURNS Zero on success, negative on error.
