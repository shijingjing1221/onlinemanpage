NAME kmsg_dump_unregister - unregister a kmsg dumper.

SYNOPSIS int kmsg_dump_unregister(struct kmsg_dumper * dumper);

ARGUMENTS dumper pointer to the kmsg_dumper structure

DESCRIPTION Removes a dump device from the system. Returns zero on success and -EINVAL otherwise.
