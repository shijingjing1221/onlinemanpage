NAME arptables-save - Save ARP Tables

SYNOPSIS arptables-save [-c] [-t table]

DESCRIPTION arptables-save is used to dump the contents of an ARP Table in easily parseable format to STDOUT. Use I/O-redirection provided by your shell to write to a file.

-c, --counters include the current values of all packet and byte counters in the output

-t, --table tablename

restrict output to only one table. If not specified, output includes all available tables.

BUGS None known as of arptables-jf-0.0.0 release
