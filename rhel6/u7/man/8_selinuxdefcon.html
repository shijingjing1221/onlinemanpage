NAME selinuxdefcon - report default SELinux context for user

SYNOPSIS selinuxdefcon [-l level] user fromcon

DESCRIPTION selinuxdefcon reports the default context for the specified user from the specified context

-l level mcs/mls level

EXAMPLE # selinuxdefcon jsmith system_u:system_r:sshd_t:s0 unconfined_u:unconfined_r:unconfined_t:s0-s0:c0.c1023
