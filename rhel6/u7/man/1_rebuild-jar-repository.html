NAME rebuild-jar-repository - rebuild a JAR repository based on the current JVM

SYNOPSIS rebuild-jar-repository [OPTION]... DIRECTORY

DESCRIPTION Rebuild a jar repository (built by build-jar-repository ) based on the current JVM settings

OPTIONS If no option is specified the default action will be to rebuild with symbolic links

-c, --copy copy files

-h, --hard create hard links

-s, --soft, --symbolic create symbolic links (default)

EXAMPLES rebuild-jar-repository . Will update all links created by build-jar-repository in the current working directory. Updated links will be symbolic links.

rebuild-jar-repository -h /tmp Will update all links created by build-jar-repository in the /tmp. Updated links will be hard links.
