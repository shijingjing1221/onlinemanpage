PHY_GET_EEE_ERR(9) Network device support PHY_GET_EEE_ERR(9)

NAME phy_get_eee_err - report the EEE wake error count

SYNOPSIS int phy_get_eee_err(struct phy_device * phydev);

ARGUMENTS phydev target phy_device struct

DESCRIPTION it is to report the number of time where the PHY failed to complete its normal wake sequence.
