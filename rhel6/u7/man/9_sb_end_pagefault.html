NAME sb_end_pagefault - drop write access to a superblock from a page fault

SYNOPSIS void sb_end_pagefault(struct super_block * sb);

ARGUMENTS sb the super we wrote to

DESCRIPTION Decrement number of processes handling write page fault to the filesystem. Wake up possible waiters wanting to freeze the filesystem.
