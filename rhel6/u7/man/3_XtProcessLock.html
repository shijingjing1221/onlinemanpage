XtProcessLock(3) XT FUNCTIONS XtProcessLock(3)

NAME XtProcessLock, XtProcessUnlock - lock and unlock process

SYNTAX void XtProcessLock(void);

void XtProcessUnlock(void);

DESCRIPTION XtProcessLock is used to lock all process global data.

XtProcessUnlock unlocks the process.
