NAME pmdagpfs - gpfs filesystem statistics performance metrics domain agent (PMDA)

DESCRIPTION pmdagpfs is a Performance Metrics Domain Agent (PMDA) which exports metric values from the /usr/lpp/mmfs/bin/mmpmon program to provide information on mounted gpfs filesystems.

INSTALLATION Install the gpfs PMDA by using the Install script as root:

# cd $PCP_PMDAS_DIR/gpfs # ./Install

To uninstall, do the following as root:

# cd $PCP_PMDAS_DIR/gpfs # ./Remove

pmdagpfs is launched by pmcd(1) and should never be executed directly. The Install and Remove scripts notify pmcd(1) when the agent is installed or removed.

FILES $PCP_PMDAS_DIR/gpfs/Install installation script for the pmdagpfs agent

$PCP_PMDAS_DIR/gpfs/Remove undo installation script for the pmdagpfs agent

$PCP_LOG_DIR/pmcd/gpfs.log default log file for error messages from pmdagpfs

PCP ENVIRONMENT Environment variables with the prefix PCP_ are used to parameterize the file and directory names used by PCP. On each installation, the file /etc/pcp.conf contains the local values for these variables. The $PCP_CONF variable may be used to specify an alternative configuration file, as described in pcp.conf(5).

SEE ALSO PCPIntro(1), pmcd(1), and mmpmon(8).
