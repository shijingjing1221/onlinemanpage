NAME glutSolidRhombicDodecahedron - Draw a solid rhombic dodecahedron

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidRhombicDodecahedron(void);

DESCRIPTION This function draws a solid-shaded dodecahedron whose facets are rhombic and whose vertices are at unit radius. No facet lies normal to any coor- dinate axes. The polyhedron is centered at the origin.
