NAME nand_read_oob - [MTD Interface] NAND read data and/or out-of-band

SYNOPSIS int nand_read_oob(struct mtd_info * mtd, loff_t from, struct mtd_oob_ops * ops);

ARGUMENTS mtd MTD device structure

from offset to read from

ops oob operation description structure

DESCRIPTION NAND read data and/or out-of-band data
