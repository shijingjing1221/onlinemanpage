NAND_READ_PAGE_SYNDR(9) Internal Functions Provided NAND_READ_PAGE_SYNDR(9)

NAME nand_read_page_syndrome - [REPLACABLE] hardware ecc syndrom based page read

SYNOPSIS int nand_read_page_syndrome(struct mtd_info * mtd, struct nand_chip * chip, uint8_t * buf, int page);

ARGUMENTS mtd mtd info structure

chip nand chip info structure

buf buffer to store read data

page page number to read

DESCRIPTION The hw generator calculates the error syndrome automatically. Therefor we need a special oob layout and handling.
