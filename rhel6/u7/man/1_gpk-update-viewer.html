NAME gpk-update-viewer - GNOME PackageKit Update Viewer

SYNOPSIS gpk-update-viewer [ --verbose ]

DESCRIPTION This manual page documents briefly the gpk-update-viewer command.

gpk-update-viewer allows you to view and select updates to install. the system.
