NAME sputoppm - convert an Atari uncompressed Spectrum file to a PPM image

SYNOPSIS sputoppm

[spufile]

DESCRIPTION This program is part of Netpbm(1).

sputoppm reads an Atari uncompressed Spectrum file as input and pro- duces a PPM image as output.

SEE ALSO spctoppm(1), ppm(1)
