NAME kjs - KDE ECMAScript compatible interpreter

SYNOPSIS kjs [--help | -help | -h] [-e statement] [script]

DESCRIPTION kjs is KDE ECMAScript/JavaScript engine.

APPLICATION OPTIONS --help,-help,-h Show kjscmd usage.

-e statement Executes the statement and exits. For example:

kjs -e "print(hello world)"

script Executes the statements included in file script.

USAGE kjs file, as many other interpreters, when called without parameters uses a line editor to enter statements, showing the prompt

JS> To exit the line editor type quit() or press Ctrl+D.
