NAME probe::nfs.aop.readpage - NFS client synchronously reading a page

SYNOPSIS nfs.aop.readpage

VALUES i_size file length in bytes

dev device identifier

rsize read size (in bytes)

sb_flag super block flags

file file argument

page_index offset within mapping, can used a page identifier and position identifier in the page frame

__page the address of page

size number of pages to be read in this execution

i_flag file flags

ino inode number

DESCRIPTION Read the page over, only fires when a previous async read operation failed
