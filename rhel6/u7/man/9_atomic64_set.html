NAME atomic64_set - set atomic64 variable

SYNOPSIS void atomic64_set(atomic64_t * ptr, u64 new_val);

ARGUMENTS ptr pointer to type atomic64_t

new_val value to assign

DESCRIPTION Atomically sets the value of ptr to new_val.
