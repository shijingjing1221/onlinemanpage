ARCH_HAS_SINGLE_STEP(9) Kernel Internals ARCH_HAS_SINGLE_STEP(9)

NAME arch_has_single_step - does this CPU support user-mode single-step?

SYNOPSIS arch_has_single_step(void);

ARGUMENTS None

DESCRIPTION If this is defined, then there must be function declarations or inlines for user_enable_single_step and user_disable_single_step. arch_has_single_step should evaluate to nonzero iff the machine supports instruction single-step for user mode. It can be a constant or it can test a CPU feature bit.
