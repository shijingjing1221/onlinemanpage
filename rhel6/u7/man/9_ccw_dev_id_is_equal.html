NAME ccw_dev_id_is_equal - compare two ccw_dev_ids

SYNOPSIS int ccw_dev_id_is_equal(struct ccw_dev_id * dev_id1, struct ccw_dev_id * dev_id2);

ARGUMENTS dev_id1 a ccw_dev_id

dev_id2 another ccw_dev_id

RETURNS 1 if the two structures are equal field-by-field, 0 if not.

CONTEXT any
