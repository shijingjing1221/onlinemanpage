NAME probe::nfs.proc.create - NFS client creating file on server

SYNOPSIS nfs.proc.create

VALUES version NFS version (the function is used for all NFS version)

flag indicates create mode (only for NFSv3 and NFSv4)

prot transfer protocol

filelen length of file name

filename file name

fh file handle of parent dir

server_ip IP address of server
