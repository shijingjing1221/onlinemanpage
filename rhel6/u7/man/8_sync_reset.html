NAME sync_reset - Account reset utility. DANGER.

SYNOPSIS sync_reset [ -C config-file ] [ -v ] [ -f ]

DESCRIPTION Sync_reset is a small utility program to destroy user accounts on a system. The only safeguard which is in place is the obligary force option.

OPTIONS -C config-file Read configuration options from config-file.

-v Verbose mode.

-f Force operation. Without this flag sync_reset just bails out with an error. Principally here to try and prevent accidents with command autorepeat.

FILES /etc/imapd.conf

SEE ALSO sync_client(8)
