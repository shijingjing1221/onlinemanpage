PCI_SRIOV_GET_TOTALV(9) Hardware Interfaces PCI_SRIOV_GET_TOTALV(9)

NAME pci_sriov_get_totalvfs - - get total VFs supported on this devic3

SYNOPSIS int pci_sriov_get_totalvfs(struct pci_dev * dev);

ARGUMENTS dev the PCI PF device

DESCRIPTION For a PCIe device with SRIOV support, return the PCIe SRIOV capability value of TotalVFs or the value of drvttl if the driver reduced it. Otherwise, -EINVAL.
