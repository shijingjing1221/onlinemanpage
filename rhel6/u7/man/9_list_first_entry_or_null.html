NAME list_first_entry_or_null - get the first element from a list

SYNOPSIS list_first_entry_or_null(ptr, type, member);

ARGUMENTS ptr the list head to take the element from.

type the type of the struct this is embedded in.

member the name of the list_struct within the struct.

DESCRIPTION Note that if the list is empty, it returns NULL.
