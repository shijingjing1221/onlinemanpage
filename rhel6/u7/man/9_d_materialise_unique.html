NAME d_materialise_unique - introduce an inode into the tree

SYNOPSIS struct dentry * d_materialise_unique(struct dentry * dentry, struct inode * inode);

ARGUMENTS dentry candidate dentry

inode inode to bind to the dentry, to which aliases may be attached

DESCRIPTION Introduces an dentry into the tree, substituting an extant disconnected root directory alias in its place if there is one
