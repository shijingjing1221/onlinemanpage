NAME ktime_compare - Compares two ktime_t variables for less, greater or equal

SYNOPSIS int ktime_compare(const ktime_t cmp1, const ktime_t cmp2);

ARGUMENTS cmp1 comparable1

cmp2 comparable2

DESCRIPTION Returns ... cmp1 < cmp2: return <0 cmp1 == cmp2: return 0 cmp1 > cmp2: return >0
