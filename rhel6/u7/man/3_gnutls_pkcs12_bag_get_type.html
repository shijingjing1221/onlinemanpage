NAME gnutls_pkcs12_bag_get_type - This function returns the bag s type

SYNOPSIS #include <gnutls/pkcs12.h>

gnutls_pkcs12_bag_type_t gnutls_pkcs12_bag_get_type(gnutls_pkcs12_bag_t bag, int indx);

ARGUMENTS gnutls_pkcs12_bag_t bag The bag

int indx The element of the bag to get the type

DESCRIPTION This function will return the bags type.

RETURNS One of the gnutls_pkcs12_bag_type_t enumerations.
