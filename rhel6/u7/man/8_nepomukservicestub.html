NAME nepomukservicestub - KDE wrapper around Nepomuk service plugins.

SYNOPSIS nepomukservicestub [--help] [Generic options] [Qt&#8482;-options] [KDE-options] {servicename}

DESCRIPTION nepomukservicestub runs a Nepomuk service plugin in its own process.

It is used by nepomukserver to run all configured Nepomuk services.

GENERIC OPTIONS --author Show author information.

--help Show help about options.

--help-all Show all options.

--help-kde Show KDE specific options.

--help-qt Show Qt&#8482; specific options.

--license Show license information.

-v --version Show version information

APPLICATION OPTIONS servicename The name of the service plugin to run

SEE ALSO nepomukserver(8)
