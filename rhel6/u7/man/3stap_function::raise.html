NAME function::raise - raise a signal in the current thread

SYNOPSIS raise(signo:long)

ARGUMENTS signo signal number

DESCRIPTION This function calls the kernel send_sig routine on the current thread, with the given raw unchecked signal number. It may raise an error if send_sig failed. It requires guru mode.
