USB_VENDOR_AND_INTER(9) Host-Side Data Types and Macro USB_VENDOR_AND_INTER(9)

NAME USB_VENDOR_AND_INTERFACE_INFO - describe a specific usb vendor with a class of usb interfaces

SYNOPSIS USB_VENDOR_AND_INTERFACE_INFO(vend, cl, sc, pr);

ARGUMENTS vend the 16 bit USB Vendor ID

cl bInterfaceClass value

sc bInterfaceSubClass value

pr bInterfaceProtocol value

DESCRIPTION This macro is used to create a struct usb_device_id that matches a specific vendor with a specific class of interfaces.

This is especially useful when explicitly matching devices that have vendor specific bDeviceClass values, but standards-compliant interfaces.
