NAME gpk-backend-status - GNOME PackageKit Local File Provider

SYNOPSIS gpk-backend-status [ --verbose ]

DESCRIPTION This manual page documents briefly the gpk-backend-status command.

gpk-backend-status allows you to install a package that provides a filename on the system.
