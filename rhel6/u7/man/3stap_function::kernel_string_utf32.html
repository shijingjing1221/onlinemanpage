NAME function::kernel_string_utf32 - Retrieves UTF-32 string from kernel memory

SYNOPSIS kernel_string_utf32:string(addr:long)

ARGUMENTS addr The kernel address to retrieve the string from

DESCRIPTION This function returns a null terminated UTF-8 string converted from the UTF-32 string at a given kernel memory address. Reports an error on string copy fault or conversion error.
