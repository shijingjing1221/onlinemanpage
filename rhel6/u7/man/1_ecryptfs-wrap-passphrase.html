NAME ecryptfs-wrap-passphrase - wrap an eCryptfs mount passphrase.

SYNOPSIS ecryptfs-unwrap-passphrase [file]

printf "%s1s" "passphrase to wrap" "wrapping passphrase" | ecryptfs- wrap-passphrase [file] -

DESCRIPTION ecryptfs-wrap-passphrase is a utility to wrap an eCryptfs mount passphrase, using a specified wrapping passphrase, and write the encrypted output to file.

SEE ALSO ecryptfs(7), ecryptfs-rewrap-passphrase(1), ecryptfs-unwrap- passphrase(1)
