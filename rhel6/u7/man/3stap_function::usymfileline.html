NAME function::usymfileline - Return the file name and line number of an address.

SYNOPSIS usymfileline:string(addr:long)

ARGUMENTS addr The address to translate.

DESCRIPTION Returns the file name and the (approximate) line number of the given address, if known. If the file name or the line number cannot be found, the hex string representation of the address will be returned.
