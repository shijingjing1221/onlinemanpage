NAME virt-pki-validate - validate libvirt PKI files are configured correctly

SYNOPSIS virt-pki-validate

DESCRIPTION This tool validates that the necessary PKI files are configured for a secure libvirt server or client using the TLS encryption protocol. It will report any missing certificate or key files on the host. It should be run as root to ensure it can read all the necessary files

EXIT STATUS Upon successful validation, an exit status of 0 will be set. Upon failure a non-zero status will be set.
