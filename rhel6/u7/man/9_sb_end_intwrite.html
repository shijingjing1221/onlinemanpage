NAME sb_end_intwrite - drop write access to a superblock for internal fs purposes

SYNOPSIS void sb_end_intwrite(struct super_block * sb);

ARGUMENTS sb the super we wrote to

DESCRIPTION Decrement fs-internal number of writers to the filesystem. Wake up possible waiters wanting to freeze the filesystem.
