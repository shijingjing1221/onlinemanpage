NAME lreverse - Reverse the order of a list

SYNOPSIS lreverse list _________________________________________________________________

DESCRIPTION The lreverse command returns a list that has the same elements as its input list, list, except with the elements in the reverse order.

EXAMPLES lreverse {a a b c} _c b a a lreverse {a b {c d} e f} _f e {c d} b a

SEE ALSO list(n), lsearch(n), lsort(n)

KEYWORDS element, list, reverse
