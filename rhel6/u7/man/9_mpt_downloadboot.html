NAME mpt_downloadboot - DownloadBoot code

SYNOPSIS int mpt_downloadboot(MPT_ADAPTER * ioc, MpiFwHeader_t * pFwHeader, int sleepFlag);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

pFwHeader Pointer to firmware header info

sleepFlag Specifies whether the process can sleep

DESCRIPTION FwDownloadBoot requires Programmed IO access.

Returns 0 for success -1 FW Image size is 0 -2 No valid cached_fw Pointer <0 for fw upload failure.
