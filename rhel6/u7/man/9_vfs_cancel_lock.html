NAME vfs_cancel_lock - file byte range unblock lock

SYNOPSIS int vfs_cancel_lock(struct file * filp, struct file_lock * fl);

ARGUMENTS filp The file to apply the unblock to

fl The lock to be unblocked

DESCRIPTION Used by lock managers to cancel blocked requests
