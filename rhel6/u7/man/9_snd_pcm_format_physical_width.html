SND_PCM_FORMAT_PHYSI(9) PCM API SND_PCM_FORMAT_PHYSI(9)

NAME snd_pcm_format_physical_width - return the physical bit-width of the format

SYNOPSIS int snd_pcm_format_physical_width(snd_pcm_format_t format);

ARGUMENTS format the format to check

DESCRIPTION Returns the physical bit-width of the format, or a negative error code if unknown format.
