NAME ccw_device_get_path_mask - get currently available paths

SYNOPSIS __u8 ccw_device_get_path_mask(struct ccw_device * cdev);

ARGUMENTS cdev ccw device to be queried

RETURNS 0 if no subchannel for the device is available, else the mask of currently available paths for the ccw device´s subchannel.
