NAME function::errno_str - Symbolic string associated with error code

SYNOPSIS errno_str:string(err:long)

ARGUMENTS err The error number received

DESCRIPTION This function returns the symbolic string associated with the giver error code, such as ENOENT for the number 2, or E#3333 for an out-of-range value such as 3333.
