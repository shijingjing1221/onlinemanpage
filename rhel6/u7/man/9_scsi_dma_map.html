SCSI_DMA_MAP(9) SCSI mid layer SCSI_DMA_MAP(9)

NAME scsi_dma_map - perform DMA mapping against command´s sg lists

SYNOPSIS int scsi_dma_map(struct scsi_cmnd * cmd);

ARGUMENTS cmd scsi command

DESCRIPTION Returns the number of sg lists actually used, zero if the sg lists is NULL, or -ENOMEM if the mapping failed.
