NAME function::user_string2_utf16 - Retrieves UTF-16 string from user memory with alternative error string

SYNOPSIS user_string2_utf16:string(addr:long,err_msg:string)

ARGUMENTS addr The user address to retrieve the string from

err_msg The error message to return when data isn´t available

DESCRIPTION This function returns a null terminated UTF-8 string converted from the UTF-16 string at a given user memory address. Reports the given error message on string copy fault or conversion error.
