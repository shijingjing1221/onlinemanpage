EDD_RELEASE(9) Firmware Interfaces EDD_RELEASE(9)

NAME edd_release - free edd structure

SYNOPSIS void edd_release(struct kobject * kobj);

ARGUMENTS kobj kobject of edd structure

DESCRIPTION This is called when the refcount of the edd structure reaches 0. This should happen right after we unregister, but just in case, we use the release callback anyway.
