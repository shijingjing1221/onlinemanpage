XtDisplayToApplicationContext(3) XT FUNCTIONS XtDisplayToApplicationContext(3)

NAME XtDisplayToApplicationContext - obtain an application context

SYNTAX XtAppContext XtDisplayToApplicationContext(Display* dpy);

ARGUMENTS dpy Specifies the display that you want the application context for.

DESCRIPTION The XtDisplayToApplicationContext function returns the application con- text for the specified display.
