NAME corosync - The Corosync Cluster Engine.

SYNOPSIS corosync [-f] [-p] [-v]

DESCRIPTION corosync Corosync provides clustering infracture such as membership, messaging and quorum.

OPTIONS -f Start application in foreground.

-p Do not set process priority.

-v Display version and SVN revision of Corosync and exit.
