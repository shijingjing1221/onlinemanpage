NAME pi3topbm - convert an Atari Degas .pi3 file into a PBM image

SYNOPSIS pi3topbm

[pi3file]

DESCRIPTION This program is part of Netpbm(1).

pi3topbm reads an Atari Degas .pi3 file as input and produces a PBM image as output.

SEE ALSO pbmtopi3(1), pbm(1), pi1toppm(1), ppmtopi1(1)
