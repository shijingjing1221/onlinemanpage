SEQ_PATH(9) The Linux VFS SEQ_PATH(9)

NAME seq_path - seq_file interface to print a pathname

SYNOPSIS int seq_path(struct seq_file * m, struct path * path, char * esc);

ARGUMENTS m the seq_file handle

path the struct path to print

esc set of characters to escape in the output

DESCRIPTION return the absolute path of ´path´, as represented by the dentry / mnt pair in the path parameter.
