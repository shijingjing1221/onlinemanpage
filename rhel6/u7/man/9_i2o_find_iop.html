NAME i2o_find_iop - Find an I2O controller by id

SYNOPSIS struct i2o_controller * i2o_find_iop(int unit);

ARGUMENTS unit unit number of the I2O controller to search for

DESCRIPTION Lookup the I2O controller on the controller list.

Returns pointer to the I2O controller on success or NULL if not found.
