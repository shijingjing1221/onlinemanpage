PCI_ENABLE_LTR(9) Hardware Interfaces PCI_ENABLE_LTR(9)

NAME pci_enable_ltr - enable latency tolerance reporting

SYNOPSIS int pci_enable_ltr(struct pci_dev * dev);

ARGUMENTS dev PCI device

DESCRIPTION Enable LTR on dev if possible, which means enabling it first on upstream ports.

RETURNS Zero on success, errno on failure.
