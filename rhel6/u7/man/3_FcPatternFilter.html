NAME FcPatternFilter - Filter the objects of pattern

SYNOPSIS #include <fontconfig.h>

FcPattern * FcPatternFilter(FcPattern *p); (const FcObjectSet *); .fi

DESCRIPTION Returns a new pattern that only has those objects from p that are in os. If os is NULL, a duplicate of p is returned.

VERSION Fontconfig version 2.8.0
