NAME FcLangSetCompare - compare language sets

SYNOPSIS #include <fontconfig.h>

FcLangResult FcLangSetCompare(const FcLangSet *ls_a); (const FcLangSet *ls_b); .fi

DESCRIPTION FcLangSetCompare compares language coverage for ls_a and ls_b. If they share any language and territory pair, this function returns FcLangE- qual. If they share a language but differ in which territory that lan- guage is for, this function returns FcLangDiffentTerritory. If they share no languages in common, this function returns FcLangDifferent- Lang.
