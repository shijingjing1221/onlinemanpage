NAME pci_check_and_mask_intx - mask INTx on pending interrupt

SYNOPSIS bool pci_check_and_mask_intx(struct pci_dev * dev);

ARGUMENTS dev -- undescribed --

DESCRIPTION Check if the device dev has its INTx line asserted, mask it and return true in that case. False is returned if not interrupt was pending.
