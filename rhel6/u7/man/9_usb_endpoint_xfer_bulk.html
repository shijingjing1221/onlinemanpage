NAME usb_endpoint_xfer_bulk - check if the endpoint has bulk transfer type

SYNOPSIS int usb_endpoint_xfer_bulk(const struct usb_endpoint_descriptor * epd);

ARGUMENTS epd endpoint to be checked

DESCRIPTION Returns true if the endpoint is of type bulk, otherwise it returns false.
