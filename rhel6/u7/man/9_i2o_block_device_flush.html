I2O_BLOCK_DEVICE_FLU(9) Message-based devices I2O_BLOCK_DEVICE_FLU(9)

NAME i2o_block_device_flush - Flush all dirty data of I2O device dev

SYNOPSIS int i2o_block_device_flush(struct i2o_device * dev);

ARGUMENTS dev I2O device which should be flushed

DESCRIPTION Flushes all dirty data on device dev.

Returns 0 on success or negative error code on failure.
