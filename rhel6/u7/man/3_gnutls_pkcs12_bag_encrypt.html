NAME gnutls_pkcs12_bag_encrypt - This function will encrypt a bag

SYNOPSIS #include <gnutls/pkcs12.h>

int gnutls_pkcs12_bag_encrypt(gnutls_pkcs12_bag_t bag, const char * pass, unsigned int flags);

ARGUMENTS gnutls_pkcs12_bag_t bag The bag

const char * pass The password used for encryption, must be ASCII

unsigned int flags should be one of gnutls_pkcs_encrypt_flags_t elements bit- wise ord

DESCRIPTION This function will encrypt the given bag.

RETURNS On success, GNUTLS_E_SUCCESS (zero) is returned, otherwise an error code is returned.
