ACPI_BUS_HOT_REMOVE_(9) Device drivers infrastructure ACPI_BUS_HOT_REMOVE_(9)

NAME acpi_bus_hot_remove_device - remove a device and its children

SYNOPSIS void acpi_bus_hot_remove_device(void * context);

ARGUMENTS context struct acpi_eject_event pointer (freed in this func)

DESCRIPTION Hot-remove a device and its children. This function frees up the memory space passed by arg context, so that the caller may call this function asynchronously through acpi_os_hotplug_execute.
