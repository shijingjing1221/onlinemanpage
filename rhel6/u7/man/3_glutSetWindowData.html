NAME glutSetWindowData - Set the user data for the current window

LIBRARY OpenGLUT - window

SYNOPSIS #include <openglut.h>

void glutSetWindowData(void* data);

PARAMETERS data Arbitrary client-supplied pointer.

DESCRIPTION This associates an arbitrary void* value with the current window . This is especially useful in client-side callbacks that service many win- dows, if the client needs to know more about the window than OpenGLUT normally will provide.

SEE ALSO glutGetWindowData(3)
