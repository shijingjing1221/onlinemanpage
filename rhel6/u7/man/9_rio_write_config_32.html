RIO_WRITE_CONFIG_32(9) RapidIO driver interface RIO_WRITE_CONFIG_32(9)

NAME rio_write_config_32 - Write 32 bits to configuration space

SYNOPSIS int rio_write_config_32(struct rio_dev * rdev, u32 offset, u32 data);

ARGUMENTS rdev RIO device

offset Offset into device configuration space

data Data to be written

DESCRIPTION Writes 32 bits of data to the specified offset within the RIO device´s configuration space.
