I2O_BLOCK_RELEASE(9) Message-based devices I2O_BLOCK_RELEASE(9)

NAME i2o_block_release - Release the I2O block device

SYNOPSIS int i2o_block_release(struct gendisk * disk, fmode_t mode);

ARGUMENTS disk gendisk device being released

mode file open mode

DESCRIPTION Unlock and unmount the media, and power down the device. Gets called if the block device is closed.

Returns 0 on success or negative error code on failure.
