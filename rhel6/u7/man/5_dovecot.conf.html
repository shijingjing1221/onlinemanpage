NAME dovecot.conf - The configuration file for dovecot imap and pop3 server

FULL PATH /etc/dovecot.conf

DESCRIPTION The dovecot.conf file is a configuration file for the dovecot(1) imap and pop3 server. The dovecot.conf configuration file contains descrip- tion to all available options. Some of these options are described also in offline wiki documentation placed in /usr/share/doc/dovecot*/wiki/.

For backup purposes unmodified version of dovecot.conf can be found in /usr/share/doc/dovecot-<version>/example-config/dovecot.conf.default

SEE ALSO doveadm(1), dovecot(1), /usr/share/doc/dovecot*/wiki/ /usr/share/doc/dovecot*/dovecot.conf.default
