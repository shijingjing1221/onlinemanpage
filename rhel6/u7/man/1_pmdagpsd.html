NAME pmdagpsd - gpsd performance metrics domain agent (PMDA)

DESCRIPTION pmdagpsd is a Performance Metrics Domain Agent (PMDA) which exports values from the gpsd daemon.

INSTALLATION If you want access to the names and values for gpsd, do the following as root:

# cd $PCP_PMDAS_DIR/gpsd # ./Install

If you want to undo the installation, do the following as root:

# cd $PCP_PMDAS_DIR/gpsd # ./Remove

pmdagpsd is launched by pmcd(1) and should never be executed directly. The Install and Remove scripts notify pmcd(1) when the agent is installed or removed.

FILES $PCP_PMDAS_DIR/gpsd/Install installation script for the pmdagpsd agent

$PCP_PMDAS_DIR/gpsd/Remove undo installation script for the pmdagpsd agent

$PCP_LOG_DIR/pmcd/gpsd.log default log file for error messages from pmdagpsd

SEE ALSO pmcd(1).
