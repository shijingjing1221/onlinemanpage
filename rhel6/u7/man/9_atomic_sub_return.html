NAME atomic_sub_return - subtract integer and return

SYNOPSIS int atomic_sub_return(int i, atomic_t * v);

ARGUMENTS i integer value to subtract

v pointer of type atomic_t

DESCRIPTION Atomically subtracts i from v and returns v - i
