IPC_ALLOC(9) Kernel IPC facilities IPC_ALLOC(9)

NAME ipc_alloc - allocate ipc space

SYNOPSIS void * ipc_alloc(int size);

ARGUMENTS size size desired

DESCRIPTION Allocate memory from the appropriate pools and return a pointer to it. NULL is returned if the allocation fails
