NAME gnutls_x509_privkey_import - import a DER or PEM encoded key

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_privkey_import(gnutls_x509_privkey_t key, const gnutls_datum_t * data, gnutls_x509_crt_fmt_t format);

ARGUMENTS gnutls_x509_privkey_t key The structure to store the parsed key

const gnutls_datum_t * data The DER or PEM encoded certificate.

gnutls_x509_crt_fmt_t format One of DER or PEM

DESCRIPTION This function will convert the given DER or PEM encoded key to the native gnutls_x509_privkey_t format. The output will be stored in key .

If the key is PEM encoded it should have a header of "RSA PRIVATE KEY", or "DSA PRIVATE KEY".

RETURNS On success, GNUTLS_E_SUCCESS is returned, otherwise a negative error value.
