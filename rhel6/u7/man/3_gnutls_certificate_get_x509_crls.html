NAME gnutls_certificate_get_x509_crls - Used to export all the CRLs from a gnutls_certificate_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_certificate_get_x509_crls(gnutls_certificate_credentials_t sc, gnutls_x509_crl_t ** x509_crl_list, unsigned int * ncrls);

ARGUMENTS gnutls_certificate_credentials_t sc is a gnutls_certificate_credentials_t structure.

gnutls_x509_crl_t ** x509_crl_list the exported CRL list. Should be treated as constant

unsigned int * ncrls the number of exported CRLs

DESCRIPTION This function will export all the CRLs associated with the given cre- dentials.

SINCE 2.4.0
