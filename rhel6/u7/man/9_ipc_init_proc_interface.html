IPC_INIT_PROC_INTERF(9) Kernel IPC facilities IPC_INIT_PROC_INTERF(9)

NAME ipc_init_proc_interface - Create a proc interface for sysipc types using a seq_file interface.

SYNOPSIS void ipc_init_proc_interface(const char * path, const char * header, int ids, int (*show) (struct seq_file *, void *));

ARGUMENTS path Path in procfs

header Banner to be printed at the beginning of the file.

ids ipc id table to iterate.

show show routine.
