NAME i2o_bus_store_scan - Scan the I2O Bus Adapter

SYNOPSIS ssize_t i2o_bus_store_scan(struct device * d, struct device_attribute * attr, const char * buf, size_t count);

ARGUMENTS d device which should be scanned

attr device_attribute

buf output buffer

count buffer size

DESCRIPTION Returns count.
