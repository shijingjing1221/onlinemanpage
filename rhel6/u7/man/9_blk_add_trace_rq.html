NAME blk_add_trace_rq - Add a trace for a request oriented action

SYNOPSIS void blk_add_trace_rq(struct request_queue * q, struct request * rq, u32 what);

ARGUMENTS q queue the io is for

rq the source request

what the action

DESCRIPTION Records an action against a request. Will log the bio offset + size.
