MPT_DIAG_RESET(9) Message-based devices MPT_DIAG_RESET(9)

NAME mpt_diag_reset - Perform hard reset of the adapter.

SYNOPSIS int mpt_diag_reset(MPT_ADAPTER * ioc, int ignore, int sleepFlag);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

ignore Set if to honor and clear to ignore the reset history bit

sleepFlag CAN_SLEEP if called in a non-interrupt thread, else set to NO_SLEEP (use mdelay instead)

DESCRIPTION This routine places the adapter in diagnostic mode via the WriteSequence register and then performs a hard reset of adapter via the Diagnostic register. Adapter should be in ready state upon successful completion.

RETURNS 1 hard reset successful 0 no reset performed because reset history bit set -2 enabling diagnostic mode failed -3 diagnostic reset failed
