I2O_PROC_IOP_REMOVE(9) Message-based devices I2O_PROC_IOP_REMOVE(9)

NAME i2o_proc_iop_remove - Removes an I2O controller from the i2o proc tree

SYNOPSIS void i2o_proc_iop_remove(struct proc_dir_entry * dir, struct i2o_controller * c);

ARGUMENTS dir parent proc dir entry

c I2O controller which should be removed

DESCRIPTION Iterate over each i2o proc entry and search controller c. If it is found remove it from the tree.
