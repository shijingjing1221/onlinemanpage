NAME cp1251 - the CP 1251 character set encoded in octal, decimal, and hex- adecimal

DESCRIPTION The Windows Code Pages include several 8-bit extensions to the ASCII character set (also known as ISO 646-IRV). CP 1251 encodes the charac- ters used in Cyrillic scripts. The following table displays the char- acters in CP 1251, which are printable and unlisted in the ascii(7) manual page. The fourth column will only show the proper glyphs in an environment configured for CP 1251.
