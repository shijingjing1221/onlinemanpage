NAME vgck check volume group metadata

SYNOPSIS vgck [--commandprofile ProfileName] [-d|--debug] [-h|-?|--help] [-v|--verbose] [VolumeGroupName...]

DESCRIPTION vgck checks LVM metadata for each named volume group for consistency.

OPTIONS See lvm(8) for common options.
