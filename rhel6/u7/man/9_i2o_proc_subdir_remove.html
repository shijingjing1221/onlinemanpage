NAME i2o_proc_subdir_remove - Remove child entries from a proc entry

SYNOPSIS void i2o_proc_subdir_remove(struct proc_dir_entry * dir);

ARGUMENTS dir proc dir entry from which the childs should be removed

DESCRIPTION Iterate over each i2o proc entry under dir and remove it. If the child also has entries, remove them too.
