NAME dmi_get_system_info - return DMI data value

SYNOPSIS const char * dmi_get_system_info(int field);

ARGUMENTS field data index (see enum dmi_field)

DESCRIPTION Returns one DMI data value, can be used to perform complex DMI data checks.
