NAME sas_remove_children - tear down a devices SAS data structures

SYNOPSIS void sas_remove_children(struct device * dev);

ARGUMENTS dev device belonging to the sas object

DESCRIPTION Removes all SAS PHYs and remote PHYs for a given object
