NAME FcToLower - convert upper case ASCII to lower case

SYNOPSIS #include <fontconfig.h>

FcChar8 FcToLower(FcChar8c); .fi

DESCRIPTION This macro converts upper case ASCII c to the equivalent lower case letter.
