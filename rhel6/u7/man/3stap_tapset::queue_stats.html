DESCRIPTION qs_wait Function to record enqueue requests

See function::qs_run(3stap) for details.

See function::qsq_utilization(3stap) for details.

qsq_service_time Amount of time per request service

qsq_throughput Number of requests served per unit time

See function::qsq_throughput(3stap) for details.

qsq_print Prints a line of statistics for the given queue
