MPT_PRINT_IOC_SUMMAR(9) Message-based devices MPT_PRINT_IOC_SUMMAR(9)

NAME mpt_print_ioc_summary - Write ASCII summary of IOC to a buffer.

SYNOPSIS void mpt_print_ioc_summary(MPT_ADAPTER * ioc, char * buffer, int * size, int len, int showlan);

ARGUMENTS ioc Pointer to MPT_ADAPTER structure

buffer Pointer to buffer where IOC summary info should be written

size Pointer to number of bytes we wrote (set by this routine)

len Offset at which to start writing in buffer

showlan Display LAN stuff?

DESCRIPTION This routine writes (english readable) ASCII text, which represents a summary of IOC information, to a buffer.
