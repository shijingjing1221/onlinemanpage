NAME hrtimer_forward - forward the timer expiry

SYNOPSIS u64 hrtimer_forward(struct hrtimer * timer, ktime_t now, ktime_t interval);

ARGUMENTS timer hrtimer to forward

now forward past this time

interval the interval to forward

DESCRIPTION Forward the timer expiry so it will expire in the future. Returns the number of overruns.
