NAME kthread_should_stop - should this kthread return now?

SYNOPSIS int kthread_should_stop(void);

ARGUMENTS void no arguments

DESCRIPTION When someone calls kthread_stop on your kthread, it will be woken and this will return true. You should then return, and your return value will be passed through to kthread_stop.
