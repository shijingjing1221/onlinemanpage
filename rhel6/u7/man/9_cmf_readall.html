CMF_READALL(9) The ccw bus CMF_READALL(9)

NAME cmf_readall - read the current channel measurement block

SYNOPSIS int cmf_readall(struct ccw_device * cdev, struct cmbdata * data);

ARGUMENTS cdev the channel to be read

data a pointer to a data block that will be filled

DESCRIPTION Returns 0 on success, a negative error value otherwise.

CONTEXT any
