NAME blk_plug_device_unlocked - plug a device without queue lock held

SYNOPSIS void blk_plug_device_unlocked(struct request_queue * q);

ARGUMENTS q The struct request_queue to plug

DESCRIPTION Like blk_plug_device(), but grabs the queue lock and disables interrupts.
