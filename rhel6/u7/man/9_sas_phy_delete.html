SAS_PHY_DELETE(9) SCSI mid layer SAS_PHY_DELETE(9)

NAME sas_phy_delete - remove SAS PHY

SYNOPSIS void sas_phy_delete(struct sas_phy * phy);

ARGUMENTS phy SAS PHY to remove

DESCRIPTION Removes the specified SAS PHY. If the SAS PHY has an associated remote PHY it is removed before.
