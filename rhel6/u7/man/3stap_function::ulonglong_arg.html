NAME function::ulonglong_arg - Return function argument as 64-bit value

SYNOPSIS ulonglong_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the value of argument n as a 64-bit value. (Same as longlong_arg.)
