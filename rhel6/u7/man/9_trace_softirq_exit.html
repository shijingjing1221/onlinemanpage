NAME trace_softirq_exit - called immediately after the softirq handler returns

SYNOPSIS void trace_softirq_exit(struct softirq_action * h, struct softirq_action * vec);

ARGUMENTS h pointer to struct softirq_action

vec pointer to first struct softirq_action in softirq_vec array

DESCRIPTION The h parameter contains a pointer to the struct softirq_action that has handled the softirq. By subtracting the vec pointer from the h pointer, we can determine the softirq number. Also, when used in combination with the softirq_entry tracepoint we can determine the softirq latency.
