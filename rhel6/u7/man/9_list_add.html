NAME list_add - add a new entry

SYNOPSIS void list_add(struct list_head * new, struct list_head * head);

ARGUMENTS new new entry to be added

head list head to add it after

DESCRIPTION Insert a new entry after the specified head. This is good for implementing stacks.
