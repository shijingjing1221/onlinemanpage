NAME glutSolidTetrahedron - Draw a solid tetrahedron.

LIBRARY OpenGLUT - geometry

SYNOPSIS #include <openglut.h>

void glutSolidTetrahedron(void);

DESCRIPTION This function draws a regular, solid 4-sided polyhedron centered at the origin. The distance from the origin to the vertices is 1.
