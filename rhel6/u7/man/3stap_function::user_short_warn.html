NAME function::user_short_warn - Retrieves a short value stored in user space

SYNOPSIS user_short_warn:long(addr:long)

ARGUMENTS addr the user space address to retrieve the short from

DESCRIPTION Returns the short value from a given user space address. Returns zero when user space and warns (but does not abort) about the failure.
