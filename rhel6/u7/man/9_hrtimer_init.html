NAME hrtimer_init - initialize a timer to the given clock

SYNOPSIS void hrtimer_init(struct hrtimer * timer, clockid_t clock_id, enum hrtimer_mode mode);

ARGUMENTS timer the timer to be initialized

clock_id the clock to be used

mode timer mode abs/rel
