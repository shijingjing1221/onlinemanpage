NAME gnutls_pem_base64_encode_alloc - convert raw data to Base64 encoded

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_pem_base64_encode_alloc(const char * msg, const gnutls_datum_t * data, gnutls_datum_t * result);

ARGUMENTS const char * msg is a message to be put in the encoded header

const gnutls_datum_t * data contains the raw data

gnutls_datum_t * result will hold the newly allocated encoded data

DESCRIPTION This function will convert the given data to printable data, using the base64 encoding. This is the encoding used in PEM messages. This function will allocate the required memory to hold the encoded data.

You should use gnutls_free() to free the returned data.

RETURNS On success, GNUTLS_E_SUCCESS (0) is returned, otherwise an error code is returned.
