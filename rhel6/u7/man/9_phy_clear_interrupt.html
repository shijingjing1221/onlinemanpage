PHY_CLEAR_INTERRUPT(9) Network device support PHY_CLEAR_INTERRUPT(9)

NAME phy_clear_interrupt - Ack the phy device´s interrupt

SYNOPSIS int phy_clear_interrupt(struct phy_device * phydev);

ARGUMENTS phydev the phy_device struct

DESCRIPTION If the phydev driver has an ack_interrupt function, call it to ack and clear the phy device´s interrupt.

Returns 0 on success on < 0 on error.
