BLK_FREE_TAGS(9) Block Devices BLK_FREE_TAGS(9)

NAME blk_free_tags - release a given set of tag maintenance info

SYNOPSIS void blk_free_tags(struct blk_queue_tag * bqt);

ARGUMENTS bqt the tag map to free

DESCRIPTION For externally managed bqt frees the map. Callers of this function must guarantee to have released all the queues that might have been using this tag map.
