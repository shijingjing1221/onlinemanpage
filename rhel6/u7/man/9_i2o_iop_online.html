I2O_IOP_ONLINE(9) Message-based devices I2O_IOP_ONLINE(9)

NAME i2o_iop_online - Bring a controller online into OPERATIONAL state.

SYNOPSIS int i2o_iop_online(struct i2o_controller * c);

ARGUMENTS c I2O controller

DESCRIPTION Send the system table and enable the I2O controller.

Returns 0 on success or negative error code on failure.
