Tk_NameOfImage(3) Tk Library Procedures Tk_NameOfImage(3)

______________________________________________________________________________

NAME Tk_NameOfImage - Return name of image.

SYNOPSIS #include <tk.h>

const char * Tk_NameOfImage(typePtr)

ARGUMENTS Tk_ImageMaster *masterPtr (in) Token for image, which was passed to image managers cre- ateProc when the image was cre- ated. _________________________________________________________________

DESCRIPTION This procedure is invoked by image managers to find out the name of an image. Given the token for the image, it returns the string name for the image.

KEYWORDS image manager, image name
