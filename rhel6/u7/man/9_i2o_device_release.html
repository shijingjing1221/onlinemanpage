I2O_DEVICE_RELEASE(9) Message-based devices I2O_DEVICE_RELEASE(9)

NAME i2o_device_release - release the memory for a I2O device

SYNOPSIS void i2o_device_release(struct device * dev);

ARGUMENTS dev I2O device which should be released

DESCRIPTION Release the allocated memory. This function is called if refcount of device reaches 0 automatically.
