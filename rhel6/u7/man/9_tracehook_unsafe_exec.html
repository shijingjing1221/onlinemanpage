NAME tracehook_unsafe_exec - check for exec declared unsafe due to tracing

SYNOPSIS int tracehook_unsafe_exec(struct task_struct * task);

ARGUMENTS task current task doing exec

DESCRIPTION Return LSM_UNSAFE_* bits applied to an exec because of tracing.

task->cred_guard_mutex is held by the caller through the do_execve.
