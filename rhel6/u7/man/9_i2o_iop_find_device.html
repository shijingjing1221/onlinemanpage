I2O_IOP_FIND_DEVICE(9) Message-based devices I2O_IOP_FIND_DEVICE(9)

NAME i2o_iop_find_device - Find a I2O device on an I2O controller

SYNOPSIS struct i2o_device * i2o_iop_find_device(struct i2o_controller * c, u16 tid);

ARGUMENTS c I2O controller where the I2O device hangs on

tid TID of the I2O device to search for

DESCRIPTION Searches the devices of the I2O controller for a device with TID tid and returns it.

Returns a pointer to the I2O device if found, otherwise NULL.
