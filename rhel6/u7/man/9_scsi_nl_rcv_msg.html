NAME scsi_nl_rcv_msg - Receive message handler.

SYNOPSIS void scsi_nl_rcv_msg(struct sk_buff * skb);

ARGUMENTS skb socket receive buffer

DESCRIPTION Extracts message from a receive buffer. Validates message header and calls appropriate transport message handler
