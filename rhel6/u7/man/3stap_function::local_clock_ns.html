FUNCTION::LOCAL_CLOC(3stap) Timestamp Functions FUNCTION::LOCAL_CLOC(3stap)

NAME function::local_clock_ns - Number of nanoseconds on the local cpu´s clock

SYNOPSIS local_clock_ns:long()

ARGUMENTS None

DESCRIPTION This function returns the number of nanoseconds on the local cpu´s clock. This is always monotonic comparing on the same cpu, but may have some drift between cpus (within about a jiffy).
