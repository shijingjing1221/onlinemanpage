NAME dynamic_irq_init_keep_chip_data - initialize a dynamically allocated irq

SYNOPSIS void dynamic_irq_init_keep_chip_data(unsigned int irq);

ARGUMENTS irq irq number to initialize

DESCRIPTION does not set irq_to_desc(irq)->chip_data to NULL
