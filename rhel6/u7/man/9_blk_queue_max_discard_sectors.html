NAME blk_queue_max_discard_sectors - set max sectors for a single discard

SYNOPSIS void blk_queue_max_discard_sectors(struct request_queue * q, unsigned int max_discard_sectors);

ARGUMENTS q the request queue for the device

max_discard_sectors maximum number of sectors to discard
