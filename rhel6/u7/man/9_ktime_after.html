NAME ktime_after - Compare if a ktime_t value is bigger than another one.

SYNOPSIS bool ktime_after(const ktime_t cmp1, const ktime_t cmp2);

ARGUMENTS cmp1 comparable1

cmp2 comparable2

RETURN true if cmp1 happened after cmp2.
