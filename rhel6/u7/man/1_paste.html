NAME paste - merge lines of files

SYNOPSIS paste [OPTION]... [FILE]...

DESCRIPTION Write lines consisting of the sequentially corresponding lines from each FILE, separated by TABs, to standard output. With no FILE, or when FILE is -, read standard input.

Mandatory arguments to long options are mandatory for short options too.

-d, --delimiters=LIST reuse characters from LIST instead of TABs

-s, --serial paste one file at a time instead of in parallel

--help display this help and exit

--version output version information and exit
