NAME gnutls_openpgp_crt_init - initialize a #gnutls_openpgp_crt_t structure

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_crt_init(gnutls_openpgp_crt_t * key);

ARGUMENTS gnutls_openpgp_crt_t * key The structure to be initialized

DESCRIPTION This function will initialize an OpenPGP key structure.

RETURNS GNUTLS_E_SUCCESS on success, or an error code.
