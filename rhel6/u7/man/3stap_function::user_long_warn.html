NAME function::user_long_warn - Retrieves a long value stored in user space

SYNOPSIS user_long_warn:long(addr:long)

ARGUMENTS addr the user space address to retrieve the long from

DESCRIPTION Returns the long value from a given user space address. Returns zero when user space and warns (but does not abort) about the failure. Note that the size of the long depends on the architecture of the current user space task (for those architectures that support both 64/32 bit compat tasks).
