NAME tracehook_expect_breakpoints - guess if task memory might be touched

SYNOPSIS int tracehook_expect_breakpoints(struct task_struct * task);

ARGUMENTS task current task, making a new mapping

DESCRIPTION Return nonzero if task is expected to want breakpoint insertion in its memory at some point. A zero return is no guarantee it won´t be done, but this is a hint that it´s known to be likely.

May be called with task->mm->mmap_sem held for writing.
