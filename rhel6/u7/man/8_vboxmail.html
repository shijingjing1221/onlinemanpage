NAME vboxmail - script to send notification of new voice message

SYNOPSIS vboxmail messagename callername callerid mail-to-address

DESCRIPTION vboxmail is used by vboxgetty to send notification of a new voice mes- sage via email. It is not meant to be called directly by any user.

SEE ALSO vboxgetty(8), vboxgetty.conf(5)
