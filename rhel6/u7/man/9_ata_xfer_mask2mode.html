NAME ata_xfer_mask2mode - Find matching XFER_* for the given xfer_mask

SYNOPSIS u8 ata_xfer_mask2mode(unsigned long xfer_mask);

ARGUMENTS xfer_mask xfer_mask of interest

DESCRIPTION Return matching XFER_* value for xfer_mask. Only the highest bit of xfer_mask is considered.

LOCKING None.

RETURNS Matching XFER_* value, 0xff if no match found.
