I2O_DEVICE_CLAIM(9) Message-based devices I2O_DEVICE_CLAIM(9)

NAME i2o_device_claim - claim a device for use by an OSM

SYNOPSIS int i2o_device_claim(struct i2o_device * dev);

ARGUMENTS dev I2O device to claim

DESCRIPTION Do the leg work to assign a device to a given OSM. If the claim succeeds, the owner is the primary. If the attempt fails a negative errno code is returned. On success zero is returned.
