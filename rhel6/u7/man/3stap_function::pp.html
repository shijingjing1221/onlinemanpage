FUNCTION::PP(3stap) Context Functions FUNCTION::PP(3stap)

NAME function::pp - Returns the active probe point

SYNOPSIS pp:string()

ARGUMENTS None

DESCRIPTION This function returns the fully-resolved probe point associated with a currently running probe handler, including alias and wild-card expansion effects. Context: The current probe point.
