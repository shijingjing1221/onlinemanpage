NAME gnutls_session_get_master_secret - get the session s master secret value

SYNOPSIS #include <gnutls/gnutls.h>

const void * gnutls_session_get_master_secret(gnutls_session_t ses- sion);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

DESCRIPTION Return a pointer to the 48-byte master secret in the session. The pointer must not be modified or deallocated.

If a master secret value has not yet been established, the output will be garbage; in particular, a NULL return value should not be expected.

Consider using gnutls_prf() rather than extracting the master secret and use it to derive further data.

RETURNS pointer to master secret data.
