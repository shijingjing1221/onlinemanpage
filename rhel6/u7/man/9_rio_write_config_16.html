RIO_WRITE_CONFIG_16(9) RapidIO driver interface RIO_WRITE_CONFIG_16(9)

NAME rio_write_config_16 - Write 16 bits to configuration space

SYNOPSIS int rio_write_config_16(struct rio_dev * rdev, u32 offset, u16 data);

ARGUMENTS rdev RIO device

offset Offset into device configuration space

data Data to be written

DESCRIPTION Writes 16 bits of data to the specified offset within the RIO device´s configuration space.
