NAME gnutls_x509_rdn_get_oid - parse an RDN sequence and returns an OID.

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_rdn_get_oid(const gnutls_datum_t * idn, int indx, void * buf, size_t * sizeof_buf);

ARGUMENTS const gnutls_datum_t * idn should contain a DER encoded RDN sequence

int indx Indicates which OID to return. Use 0 for the first one.

void * buf a pointer to a structure to hold the peers name OID

size_t * sizeof_buf holds the size of buf

DESCRIPTION This function will return the specified Object identifier, of the RDN sequence.

RETURNS On success, GNUTLS_E_SUCCESS is returned, or GNUTLS_E_SHORT_MEM- ORY_BUFFER is returned and *sizeof_buf is updated if the provided buffer is not long enough, otherwise a negative error value.

SINCE 2.4.0
