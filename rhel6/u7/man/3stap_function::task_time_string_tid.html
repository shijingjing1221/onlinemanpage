NAME function::task_time_string_tid - Human readable string of task time usage

SYNOPSIS task_time_string_tid:string(tid:long)

ARGUMENTS tid Thread id of the given task

DESCRIPTION Returns a human readable string showing the user and system time the given task has used up to now. For example usr: 0m12.908s, sys: 1m6.851s.
