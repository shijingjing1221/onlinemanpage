PCI_ENABLE_IDO(9) Hardware Interfaces PCI_ENABLE_IDO(9)

NAME pci_enable_ido - enable ID-based Ordering on a device

SYNOPSIS void pci_enable_ido(struct pci_dev * dev, unsigned long type);

ARGUMENTS dev the PCI device

type which types of IDO to enable

DESCRIPTION Enable ID-based ordering on dev. type can contain the bits PCI_EXP_IDO_REQUEST and/or PCI_EXP_IDO_COMPLETION to indicate which types of transactions are allowed to be re-ordered.
