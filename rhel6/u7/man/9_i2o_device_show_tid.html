I2O_DEVICE_SHOW_TID(9) Message-based devices I2O_DEVICE_SHOW_TID(9)

NAME i2o_device_show_tid - Displays TID of I2O device

SYNOPSIS ssize_t i2o_device_show_tid(struct device * dev, struct device_attribute * attr, char * buf);

ARGUMENTS dev device of which the TID should be displayed

attr pointer to device attribute

buf buffer into which the TID should be printed

DESCRIPTION Returns the number of bytes which are printed into the buffer.
