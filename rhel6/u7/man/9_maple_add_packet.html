NAME maple_add_packet - add a single instruction to the maple bus queue

SYNOPSIS int maple_add_packet(struct maple_device * mdev, u32 function, u32 command, size_t length, void * data);

ARGUMENTS mdev maple device

function function on device being queried

command maple command to add

length length of command string (in 32 bit words)

data remainder of command string
