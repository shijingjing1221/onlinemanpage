NAME sas_rphy_unlink - unlink SAS remote PHY

SYNOPSIS void sas_rphy_unlink(struct sas_rphy * rphy);

ARGUMENTS rphy SAS remote phy to unlink from its parent port

DESCRIPTION Removes port reference to an rphy
