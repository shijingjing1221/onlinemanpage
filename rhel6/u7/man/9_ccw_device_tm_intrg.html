NAME ccw_device_tm_intrg - perform interrogate function

SYNOPSIS int ccw_device_tm_intrg(struct ccw_device * cdev);

ARGUMENTS cdev ccw device on which to perform the interrogate function

DESCRIPTION Perform an interrogate function on the given ccw device. Return zero on success, non-zero otherwise.
