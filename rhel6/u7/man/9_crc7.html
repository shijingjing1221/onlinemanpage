CRC7(9) Basic Kernel Library Functions CRC7(9)

NAME crc7 - update the CRC7 for the data buffer

SYNOPSIS u8 crc7(u8 crc, const u8 * buffer, size_t len);

ARGUMENTS crc previous CRC7 value

buffer data pointer

len number of bytes in the buffer

CONTEXT any

DESCRIPTION Returns the updated CRC7 value.
