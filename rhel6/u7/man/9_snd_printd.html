SND_PRINTD(9) Miscellaneous Functions SND_PRINTD(9)

NAME snd_printd - debug printk

SYNOPSIS snd_printd(fmt, args...);

ARGUMENTS fmt format string

args... variable arguments

DESCRIPTION Works like snd_printk for debugging purposes. Ignored when CONFIG_SND_DEBUG is not set.
