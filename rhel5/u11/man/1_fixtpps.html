NAME fixtpps - filter to fix Tpscript documents to work with PSUtils

SYNOPSIS fixtpps < Tpscript.ps > Fixed.ps

DESCRIPTION Fixtpps is a perl filter which "fixes" PostScript generated from the Troff Tpscript driver so that it works correctly with Angus Duggans psutils package.
