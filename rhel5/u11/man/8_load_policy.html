NAME load_policy - load a new SELinux policy into the kernel

SYNOPSIS load_policy [-bq]

DESCRIPTION load_policy is the tool used to load/replace the policy in the kernel. By default, load_policy will preserve the current policy boolean values when loading the policy.

OPTIONS -b reset the policy boolean values to the saved policy settings.

-q suppress warning messages.
