NAME gnutls_pkcs7_import - This function will import a DER or PEM encoded PKCS7

SYNOPSIS #include <gnutls/x509.h>

int gnutls_pkcs7_import(gnutls_pkcs7_t pkcs7, const gnutls_datum_t * data, gnutls_x509_crt_fmt_t format);

ARGUMENTS gnutls_pkcs7_t pkcs7 The structure to store the parsed PKCS7.

const gnutls_datum_t * data The DER or PEM encoded PKCS7.

gnutls_x509_crt_fmt_t format One of DER or PEM

DESCRIPTION This function will convert the given DER or PEM encoded PKCS7 to the native gnutls_pkcs7_t format. The output will be stored in pkcs7.

If the PKCS7 is PEM encoded it should have a header of "PKCS7".

Returns 0 on success.
