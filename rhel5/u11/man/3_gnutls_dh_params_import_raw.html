NAME gnutls_dh_params_import_raw - This function will import DH parameters

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_dh_params_import_raw(gnutls_dh_params_t dh_params, const gnutls_datum_t * prime, const gnutls_datum_t * generator);

ARGUMENTS gnutls_dh_params_t dh_params Is a structure that will hold the prime numbers

const gnutls_datum_t * prime holds the new prime

const gnutls_datum_t * generator holds the new generator

DESCRIPTION This function will replace the pair of prime and generator for use in the Diffie-Hellman key exchange. The new parameters should be stored in the appropriate gnutls_datum.
