NAME gnutls_certificate_free_keys - Used to free all the keys from a gnutls_certificate_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_certificate_free_keys(gnutls_certificate_credentials_t sc);

ARGUMENTS gnutls_certificate_credentials_t sc is an gnutls_certificate_credentials_t structure.

DESCRIPTION This function will delete all the keys and the certificates associated with the given credentials. This function must not be called when a TLS negotiation that uses the credentials is in progress.
