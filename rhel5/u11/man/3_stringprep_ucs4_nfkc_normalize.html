NAME stringprep_ucs4_nfkc_normalize - normalize Unicode string

SYNOPSIS #include <stringprep.h>

uint32_t * stringprep_ucs4_nfkc_normalize(uint32_t * str, ssize_t len);

ARGUMENTS uint32_t * str a Unicode string.

ssize_t len length of str array, or -1 if str is nul-terminated.

DESCRIPTION Converts UCS4 string into UTF-8 and runs stringprep_utf8_nfkc_normal- ize().

RETURN VALUE a newly allocated Unicode string, that is the NFKC normalized form of str.
