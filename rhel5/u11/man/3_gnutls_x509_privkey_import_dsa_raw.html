NAME gnutls_x509_privkey_import_dsa_raw - This function will import a raw DSA key

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_privkey_import_dsa_raw(gnutls_x509_privkey_t key, const gnutls_datum_t * p, const gnutls_datum_t * q, const gnutls_datum_t * g, const gnutls_datum_t * y, const gnutls_datum_t * x);

ARGUMENTS gnutls_x509_privkey_t key The structure to store the parsed key

const gnutls_datum_t * p holds the p

const gnutls_datum_t * q holds the q

const gnutls_datum_t * g holds the g

const gnutls_datum_t * y holds the y

const gnutls_datum_t * x holds the x

DESCRIPTION This function will convert the given DSA raw parameters to the native gnutls_x509_privkey_t format. The output will be stored in key.
