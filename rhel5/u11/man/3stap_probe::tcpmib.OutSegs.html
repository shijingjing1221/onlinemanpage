NAME probe::tcpmib.OutSegs - Count the sending of a TCP segment

SYNOPSIS tcpmib.OutSegs

VALUES sk pointer to the struct sock being acted on

op value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function tcpmib_filter_key. If the packet passes the filter is is counted in the global OutSegs (equivalent to SNMPs MIB TCP_MIB_OUTSEGS)
