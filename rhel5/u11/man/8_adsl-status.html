NAME adsl-status - Shell script to report on status of PPPoE link

SYNOPSIS adsl-status [config_file]

DESCRIPTION adsl-status is a shell script which checks the status of the PPPoE link established by the Roaring Penguin user-space PPPoE client. If you omit config_file, the default file /etc/sysconfig/network- scripts/ifcfg-ppp0 is used.

AUTHOR adsl-status was written by David F. Skoll <dfs@roaringpenguin.com>.

The pppoe home page is http://www.roaringpenguin.com/pppoe/.
