NAME fixscribeps - filter to fix Scribe documents so PSUtils work

SYNOPSIS fixscribeps < Scribe.ps > Fixed.ps

DESCRIPTION Fixscribeps is a perl filter which "fixes" PostScript from Scribe so that it works correctly with Angus Duggan s psutils package.
