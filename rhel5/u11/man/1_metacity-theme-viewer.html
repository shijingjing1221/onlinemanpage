NAME metacity-theme-viewer - view metacity themes

SYNOPSIS metacity-theme-viewer [ THEMENAME ]

DESCRIPTION metacity-theme-viewer allows you to preview any installed Metacity theme.

When designing a new Metacity theme, you can use metacity-theme-viewer to measure the performance of a window frame option, and to preview the option.

OPTIONS THEMENAME Name of the theme to be shown (Atlanta by default). It is case- sensitive.

FILES /usr/share/themes system themes directory

/usr/share/themes/*/metacity-1/metacity-theme-1.xml theme specification file
