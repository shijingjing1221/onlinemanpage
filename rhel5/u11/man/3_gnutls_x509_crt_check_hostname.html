NAME gnutls_x509_crt_check_hostname - This function compares the given host- name with the hostname in the certificate

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_check_hostname(gnutls_x509_crt_t cert, const char * hostname);

ARGUMENTS gnutls_x509_crt_t cert should contain an gnutls_x509_crt_t structure

const char * hostname A null terminated string that contains a DNS name

DESCRIPTION This function will check if the given certificates subject matches the given hostname. This is a basic implementation of the matching described in RFC2818 (HTTPS), which takes into account wildcards, and the subject alternative name PKIX extension.

Returns non zero on success, and zero on failure.
