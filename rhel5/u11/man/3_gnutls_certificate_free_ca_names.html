NAME gnutls_certificate_free_ca_names - Used to free all the CA names from a gnutls_certificate_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_certificate_free_ca_names(gnutls_certificate_credentials_t sc);

ARGUMENTS gnutls_certificate_credentials_t sc is an gnutls_certificate_credentials_t structure.

DESCRIPTION This function will delete all the CA name in the given credentials. Clients may call this to save some memory since in client side the CA names are not used.

CA names are used by servers to advertize the CAs they support to clients.
