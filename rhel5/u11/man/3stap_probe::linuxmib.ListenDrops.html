NAME probe::linuxmib.ListenDrops - Count of times conn request that were dropped

SYNOPSIS linuxmib.ListenDrops

VALUES sk Pointer to the struct sock being acted on

op Value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function linuxmib_filter_key. If the packet passes the filter is is counted in the global ListenDrops (equivalent to SNMP s MIB LINUX_MIB_LISTENDROPS)
