NAME function::proc_mem_data_pid - Program data size (data + stack) in pages

SYNOPSIS proc_mem_data_pid:long(pid:long)

ARGUMENTS pid The pid of process to examine

DESCRIPTION Returns the given process data size (data + stack) in pages, or zero when the process doesnt exist or the number of pages couldnt be retrieved.
