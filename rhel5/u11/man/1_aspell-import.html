NAME aspell-import - import old personal dictionaries into GNU Aspell

SYNOPSIS aspell-import

DESCRIPTION aspell-import is a command that will search for old personal dictionar- ies and will import them into GNU Aspell. It will look for both Ispell and Aspell dictionaries. To use it just run it from the command prompt.

OPTIONS This program does not accept any command-line options.
