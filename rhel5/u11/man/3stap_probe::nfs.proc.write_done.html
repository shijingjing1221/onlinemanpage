NAME probe::nfs.proc.write_done - NFS client response to a write RPC task

SYNOPSIS nfs.proc.write_done

VALUES count number of bytes written

status result of last operation

version NFS version

prot transfer protocol

valid fattr->valid ,indicates which fields are valid

timestamp V4 timestamp, which is used for lease renewal

server_ip IP address of server

DESCRIPTION Fires when a reply to a write RPC task is received or some write error occurs (timeout or socket shutdown).
