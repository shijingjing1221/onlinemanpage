NAME FcFontSetAdd - Add to a font set

SYNOPSIS #include <fontconfig.h>

FcBool FcFontSetAdd (FcFontSet *s, FcPattern *font);

DESCRIPTION Adds a pattern to a font set. Note that the pattern is not copied before being inserted into the set.

VERSION Fontconfig version 2.4.1
