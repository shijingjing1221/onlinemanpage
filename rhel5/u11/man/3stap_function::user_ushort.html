NAME function::user_ushort - Retrieves an unsigned short value stored in user space

SYNOPSIS user_ushort:long(addr:long)

ARGUMENTS addr the user space address to retrieve the unsigned short from

DESCRIPTION Returns the unsigned short value from a given user space address. Returns zero when user space data is not accessible.
