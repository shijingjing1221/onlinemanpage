NAME mkfifo - make FIFOs (named pipes)

SYNOPSIS mkfifo [OPTION] NAME...

DESCRIPTION Create named pipes (FIFOs) with the given NAMEs.

Mandatory arguments to long options are mandatory for short options too.

-m, --mode=MODE set permission mode (as in chmod), not a=rw - umask

-Z, --context=CONTEXT set security context (quoted string)

--help display this help and exit

--version output version information and exit
