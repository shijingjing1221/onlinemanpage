NAME probe::tcpmib.OutRsts - Count the sending of a reset packet

SYNOPSIS tcpmib.OutRsts

VALUES sk pointer to the struct sock being acted on

op value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function tcpmib_filter_key. If the packet passes the filter is is counted in the global OutRsts (equivalent to SNMPs MIB TCP_MIB_OUTRSTS)
