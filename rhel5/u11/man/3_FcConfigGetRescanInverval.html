NAME FcConfigGetRescanInverval - Get config rescan interval

SYNOPSIS #include <fontconfig.h>

int FcConfigGetRescanInverval (FcConfig *config);

DESCRIPTION Returns the interval between automatic checks of the configuration (in seconds) specified in config. The configuration is checked during a call to FcFontList when this interval has passed since the last check.

VERSION Fontconfig version 2.4.1
