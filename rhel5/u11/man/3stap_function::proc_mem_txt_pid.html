NAME function::proc_mem_txt_pid - Program text (code) size in pages

SYNOPSIS proc_mem_txt_pid:long(pid:long)

ARGUMENTS pid The pid of process to examine

DESCRIPTION Returns the given process text (code) size in pages, or zero when the process doesnt exist or the number of pages couldn t be retrieved.
