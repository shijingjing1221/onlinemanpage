NAME gnutls_ia_endphase_send - API function

SYNOPSIS #include <gnutls/extra.h>

int gnutls_ia_endphase_send(gnutls_session_t session, int final_p);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

int final_p Set iff this should signal the final phase.

DESCRIPTION Send a TLS/IA end phase message.

In the client, this should only be used to acknowledge an end phase message sent by the server.

In the server, this can be called instead of gnutls_ia_send() if the server wishes to end an application phase.

RETURN VALUE Return 0 on success, or an error code.
