NAME printenv - print all or part of environment

SYNOPSIS printenv [VARIABLE]... printenv OPTION

DESCRIPTION If no environment VARIABLE specified, print them all.

--help display this help and exit

--version output version information and exit

NOTE: your shell may have its own version of printenv, which usually supersedes the version described here. Please refer to your shells documentation for details about the options it supports.
