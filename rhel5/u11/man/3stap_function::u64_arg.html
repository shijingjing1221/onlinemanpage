NAME function::u64_arg - Return function argument as unsigned 64-bit value

SYNOPSIS u64_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the unsigned 64-bit value of argument n, same as ulonglong_arg.
