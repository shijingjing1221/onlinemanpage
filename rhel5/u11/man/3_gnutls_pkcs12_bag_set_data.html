NAME gnutls_pkcs12_bag_set_data - This function inserts data into the bag

SYNOPSIS #include <gnutls/pkcs12.h>

int gnutls_pkcs12_bag_set_data(gnutls_pkcs12_bag_t bag, gnutls_pkcs12_bag_type_t type, const gnutls_datum_t * data);

ARGUMENTS gnutls_pkcs12_bag_t bag The bag

gnutls_pkcs12_bag_type_t type The datas type

const gnutls_datum_t * data the data to be copied.

DESCRIPTION This function will insert the given data of the given type into the bag.

Returns the index of the added bag on success, or a negative value on error.
