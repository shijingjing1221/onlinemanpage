NAME gnutls_certificate_activation_time_peers - This function returns the peers certificate activation time

SYNOPSIS #include <gnutls/gnutls.h>

time_t gnutls_certificate_activation_time_peers(gnutls_session_t ses- sion);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION This function will return the peers certificate activation time. This is the creation time for openpgp keys.

Returns (time_t) -1 on error.
