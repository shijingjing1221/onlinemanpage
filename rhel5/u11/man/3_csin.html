NAME csin, csinf, csinl - complex sine function

SYNOPSIS #include <complex.h>

double complex csin(double complex z); float complex csinf(float complex z); long double complex csinl(long double complex z);

Link with -lm.

DESCRIPTION The complex sine function csin(z) is defined as (exp(i*z)-exp(-i*z))/(2*i).

CONFORMING TO C99

SEE ALSO cabs(3), csin(3), ctan(3), complex(7)
