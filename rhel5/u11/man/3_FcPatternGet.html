NAME FcPatternGet - Return a value from a pattern

SYNOPSIS #include <fontconfig.h>

FcResult FcPatternGet (FcPattern *p, const char *object, intid, FcValue *v);

DESCRIPTION Returns in v the idth value associated with the property object. The value returned is not a copy, but rather refers to the data stored within the pattern directly. Applications must not free this value.

VERSION Fontconfig version 2.4.1
