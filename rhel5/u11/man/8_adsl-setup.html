NAME adsl-setup - Shell script to configure Roaring Penguin PPPoE client

SYNOPSIS adsl-setup

DESCRIPTION adsl-setup is a shell script which prompts you for various pieces of information and sets up an /etc/sysconfig/network-scripts/ifcfg-ppp0 configuration script for the adsl-start, adsl-stop and adsl-connect scripts.

AUTHOR adsl-setup was written by David F. Skoll <dfs@roaringpenguin.com>.

The pppoe home page is http://www.roaringpenguin.com/pppoe/.
