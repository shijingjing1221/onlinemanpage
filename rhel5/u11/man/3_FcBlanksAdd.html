NAME FcBlanksAdd - Add a character to an FcBlanks

SYNOPSIS #include <fontconfig.h>

FcBool FcBlanksAdd (FcBlanks *b, FcChar32ucs4);

DESCRIPTION Adds a single character to an FcBlanks object, returning FcFalse if this process ran out of memory.

VERSION Fontconfig version 2.4.1
