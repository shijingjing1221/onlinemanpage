NAME is_selinux_enabled - check whether SELinux is enabled

SYNOPSIS #include <selinux/selinux.h>

int is_selinux_enabled();

DESCRIPTION is_selinux_enabled returns 1 if SELinux is running or 0 if it is not. May change soon.
