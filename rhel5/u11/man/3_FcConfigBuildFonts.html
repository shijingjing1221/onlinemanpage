NAME FcConfigBuildFonts - Build font database

SYNOPSIS #include <fontconfig.h>

FcBool FcConfigBuildFonts (FcConfig *config);

DESCRIPTION Builds the set of available fonts for the given configuration. Note that any changes to the configuration after this call have indetermi- nate effects. Returns FcFalse if this operation runs out of memory.

VERSION Fontconfig version 2.4.1
