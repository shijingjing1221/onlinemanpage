NAME probe::nfs.fop.llseek - NFS client llseek operation

SYNOPSIS nfs.fop.llseek

VALUES dev device identifier

origin the original position. The possible value could be: SEEK_SET (offset set to offset bytes), SEEK_CUR (offset set to its current location plus offset bytes), or SEEK_END (offset set to the size of the file plus offset bytes).

ino inode number

offset the offset of the file will be repositioned
