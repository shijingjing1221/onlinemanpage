NAME function::u32_arg - Return function argument as unsigned 32-bit value

SYNOPSIS u32_arg:long(n:long)

ARGUMENTS n index of argument to return

DESCRIPTION Return the unsigned 32-bit value of argument n, same as uint_arg.
