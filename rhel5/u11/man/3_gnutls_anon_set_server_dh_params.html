NAME gnutls_anon_set_server_dh_params - This function will set the DH param- eters for a server to use

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_anon_set_server_dh_params(gnutls_anon_server_credentials_t res, gnutls_dh_params_t dh_params);

ARGUMENTS gnutls_anon_server_credentials_t res is a gnutls_anon_server_credentials_t structure

gnutls_dh_params_t dh_params is a structure that holds diffie hellman parameters.

DESCRIPTION This function will set the diffie hellman parameters for an anonymous server to use. These parameters will be used in Anonymous Diffie Hell- man cipher suites.
