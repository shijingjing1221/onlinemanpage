NAME probe::nfs.proc.commit_setup - NFS client setting up a commit RPC task

SYNOPSIS nfs.proc.commit_setup

VALUES count bytes in this commit

bitmask1 V4 bitmask representing the set of attributes supported on this filesystem

version NFS version

bitmask0 V4 bitmask representing the set of attributes supported on this filesystem

prot transfer protocol

size bytes in this commit

offset the file offset

server_ip IP address of server

DESCRIPTION The commit_setup function is used to setup a commit RPC task. Is is not doing the actual commit operation. It is does not exist in NFSv2.
