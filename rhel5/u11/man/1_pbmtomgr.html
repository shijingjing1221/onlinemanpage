NAME pbmtomgr - convert a PBM image into a MGR bitmap

SYNOPSIS pbmtomgr

[pbmfile]

DESCRIPTION This program is part of Netpbm(1).

pbmtomgr reads a PBM image as input and produces a MGR bitmap as out- put.

MGR(1)is a window manager that is a smaller alternative to the X Win- dows System.

SEE ALSO mgrtopbm(1), pbm(1)
