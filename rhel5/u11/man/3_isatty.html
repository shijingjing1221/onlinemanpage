NAME isatty - does this descriptor refer to a terminal

SYNOPSIS #include <unistd.h>

int isatty(int desc);

DESCRIPTION returns 1 if desc is an open descriptor connected to a terminal and 0 else.

CONFORMING TO SVr4, 4.3BSD, POSIX.1-2001.

SEE ALSO fstat(2), ttyname(3)
