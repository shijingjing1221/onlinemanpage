NAME gnutls_init - This function initializes the session to null (null encryption etc...).

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_init(gnutls_session_t * session, gnutls_connection_end_t con_end);

ARGUMENTS gnutls_session_t * session is a pointer to a gnutls_session_t structure.

gnutls_connection_end_t con_end is used to indicate if this session is to be used for server or client. Can be one of GNUTLS_CLIENT and GNUTLS_SERVER.

DESCRIPTION This function initializes the current session to null. Every session must be initialized before use, so internal structures can be allo- cated. This function allocates structures which can only be freed by calling gnutls_deinit(). Returns zero on success.
