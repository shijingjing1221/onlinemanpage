FUNCTION::LOCAL_CLOC(3stap) Timestamp Functions FUNCTION::LOCAL_CLOC(3stap)

NAME function::local_clock_s - Number of seconds on the local cpus clock

SYNOPSIS local_clock_s:long()

ARGUMENTS None

DESCRIPTION This function returns the number of seconds on the local cpus clock. This is always monotonic comparing on the same cpu, but may have some drift between cpus (within about a jiffy).
