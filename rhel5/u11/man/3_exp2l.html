NAME exp2, exp2f, exp2l - base-2 exponential function

SYNOPSIS #include <math.h>

double exp2(double x); float exp2f(float x); long double exp2l(long double x);

Compile with -std=c99; link with -lm.

DESCRIPTION The exp2() function returns the value of 2 raised to the power of x.

CONFORMING TO SVr4, POSIX.1-2001, 4.3BSD, C89. The float and long double variants are C99 requirements.

SEE ALSO cbrt(3), cexp2(3), exp(3), exp10(3), sqrt(3)
