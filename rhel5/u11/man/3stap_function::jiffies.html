FUNCTION::JIFFIES(3stap) Timestamp Functions FUNCTION::JIFFIES(3stap)

NAME function::jiffies - Kernel jiffies count

SYNOPSIS jiffies:long()

ARGUMENTS None

DESCRIPTION This function returns the value of the kernel jiffies variable. This value is incremented periodically by timer interrupts, and may wrap around a 32-bit or 64-bit boundary. See HZ.
