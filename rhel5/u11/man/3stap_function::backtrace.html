FUNCTION::BACKTRACE(3stap) Context Functions FUNCTION::BACKTRACE(3stap)

NAME function::backtrace - Hex backtrace of current stack

SYNOPSIS backtrace:string()

ARGUMENTS None

DESCRIPTION This function returns a string of hex addresses that are a backtrace of the stack. Output may be truncated as as per maximum string length (MAXSTRINGLEN).
