NAME system-config-netboot - Network Booting Configuration Utility

SYNTAX system-config-netboot [OPTION]

DESCRIPTION This is a graphical tool for enabling, configuring and disabling net- work booting

OPTIONS Only standard GNOME/GTK+ command line options.

FILES /usr/bin/system-config-netboot /usr/share/system-config-netboot/*

EXAMPLES To run this program simply type:

system-config-netboot

LIMITATIONS This utility configures PXELinux network boot to allow network instal- lations and diskless clients.
