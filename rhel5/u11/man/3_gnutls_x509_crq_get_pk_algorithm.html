NAME gnutls_x509_crq_get_pk_algorithm - This function returns the certifi- cate requests PublicKey algorithm

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crq_get_pk_algorithm(gnutls_x509_crq_t crq, unsigned int * bits);

ARGUMENTS gnutls_x509_crq_t crq should contain a gnutls_x509_crq_t structure

unsigned int * bits if bits is non null it will hold the size of the parame- ters in bits

DESCRIPTION This function will return the public key algorithm of a PKCS 10 cer- tificate request.

If bits is non null, it should have enough size to hold the parameters size in bits. For RSA the bits returned is the modulus. For DSA the bits returned are of the public exponent.

Returns a member of the gnutls_pk_algorithm_t enumeration on success, or a negative value on error.
