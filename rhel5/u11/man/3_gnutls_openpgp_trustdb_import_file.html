NAME gnutls_openpgp_trustdb_import_file - This function will import a RAW or BASE64 encoded key

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_trustdb_import_file(gnutls_openpgp_trustdb_t trustdb, const char * file);

ARGUMENTS gnutls_openpgp_trustdb_t trustdb The structure to store the parsed key.

const char * file The file that holds the trustdb.

DESCRIPTION This function will convert the given RAW or Base64 encoded trustdb to the native gnutls_openpgp_trustdb_t format. The output will be stored in trustdb.

Returns 0 on success.
