NAME function::addr_to_node - Returns which node a given address belongs to within a NUMA system

SYNOPSIS addr_to_node:long(addr:long)

ARGUMENTS addr the address of the faulting memory access

DESCRIPTION This function accepts an address, and returns the node that the given address belongs to in a NUMA system.
