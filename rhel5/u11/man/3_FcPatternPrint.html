NAME FcPatternPrint - Print a pattern for debugging

SYNOPSIS #include <fontconfig.h>

void FcPatternPrint (const FcPattern *p);

DESCRIPTION Prints an easily readable version of the pattern to stdout. There is no provision for reparsing data in this format, its just for diagnos- tics and debugging.

VERSION Fontconfig version 2.4.1
