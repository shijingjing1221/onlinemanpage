NAME probe::nfsd.open - NFS server opening a file for client

SYNOPSIS nfsd.open

VALUES access indicates the type of open (read/write/commit/readdir...)

fh file handle (the first part is the length of the file handle)

client_ip the ip address of client

type type of file (regular file or dir)
