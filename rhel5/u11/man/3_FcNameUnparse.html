NAME FcNameUnparse - Convert a pattern back into a string that can be parsed

SYNOPSIS #include <fontconfig.h>

FcChar8 * FcNameUnparse (FcPattern *pat);

DESCRIPTION Converts the given pattern into the standard text format described above. The return value is not static, but instead refers to newly allocated memory which should be freed by the caller.

VERSION Fontconfig version 2.4.1
