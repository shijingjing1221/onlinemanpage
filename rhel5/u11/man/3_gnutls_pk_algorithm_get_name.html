NAME gnutls_pk_algorithm_get_name - Returns a string with the name of the specified public key algorithm

SYNOPSIS #include <gnutls/gnutls.h>

const char * gnutls_pk_algorithm_get_name(gnutls_pk_algorithm_t algo- rithm);

ARGUMENTS gnutls_pk_algorithm_t algorithm is a pk algorithm

DESCRIPTION Returns a string that contains the name of the specified public key algorithm or NULL.
