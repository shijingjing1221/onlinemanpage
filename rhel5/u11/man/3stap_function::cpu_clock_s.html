FUNCTION::CPU_CLOCK_(3stap) Timestamp Functions FUNCTION::CPU_CLOCK_(3stap)

NAME function::cpu_clock_s - Number of seconds on the given cpu s clock

SYNOPSIS cpu_clock_s:long(cpu:long)

ARGUMENTS cpu Which processors clock to read

DESCRIPTION This function returns the number of seconds on the given cpus clock. This is always monotonic comparing on the same cpu, but may have some drift between cpus (within about a jiffy).
