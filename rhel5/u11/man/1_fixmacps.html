NAME fixmacps - filter to fix Macintosh documents with saner version of md

SYNOPSIS fixmacps < Macintosh.ps > Fixed.ps

DESCRIPTION Fixmacps is a perl filter which "fixes" PostScript generated from a Macintosh PC that it works correctly with Angus Duggan s psutils pack- age.

FILES /usr/lib/psutils/md68_0.ps, /usr/lib/psutils/md71_0.ps - sanitised ver- sions of the md prologue.
