NAME probe::vm.write_shared_copy - Page copy for shared page write

SYNOPSIS vm.write_shared_copy

VALUES name Name of the probe point

zero boolean indicating whether it is a zero page (can do a clear instead of a copy)

address The address of the shared write

CONTEXT The process attempting the write.

DESCRIPTION Fires when a write to a shared page requires a page copy. This is always preceded by a vm.shared_write.
