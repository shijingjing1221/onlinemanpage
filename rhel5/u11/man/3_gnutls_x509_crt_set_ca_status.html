NAME gnutls_x509_crt_set_ca_status - This function will set the basic- Constraints extension

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_set_ca_status(gnutls_x509_crt_t crt, unsigned int ca);

ARGUMENTS gnutls_x509_crt_t crt should contain a gnutls_x509_crt_t structure

unsigned int ca true(1) or false(0). Depending on the Certificate authority status.

DESCRIPTION This function will set the basicConstraints certificate extension.

Returns 0 on success.
