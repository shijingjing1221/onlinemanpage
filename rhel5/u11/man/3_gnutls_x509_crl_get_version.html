NAME gnutls_x509_crl_get_version - This function returns the CRLs version number

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crl_get_version(gnutls_x509_crl_t crl);

ARGUMENTS gnutls_x509_crl_t crl should contain a gnutls_x509_crl_t structure

DESCRIPTION This function will return the version of the specified CRL.

Returns a negative value on error.
