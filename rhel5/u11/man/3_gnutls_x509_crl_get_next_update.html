NAME gnutls_x509_crl_get_next_update - This function returns the CRLs nex- tUpdate time

SYNOPSIS #include <gnutls/x509.h>

time_t gnutls_x509_crl_get_next_update(gnutls_x509_crl_t crl);

ARGUMENTS gnutls_x509_crl_t crl should contain a gnutls_x509_crl_t structure

DESCRIPTION This function will return the time the next CRL will be issued. This field is optional in a CRL so it might be normal to get an error instead.

Returns (time_t)-1 on error.
