NAME findchip - checks the FIR chipset.

SYNOPSIS findchip [ -d dongle ] [ -v ] [ -l ] [ <chip1 .. chipN> ] [ -h ]

OPTIONS -d enables debug output

-v be verbose

-l lists supported chipsets

-h prints help message

DESCRIPTION findchip checks the FIR chipset.

findchip must be run as root or installed setuid root, as it requires root privileges.
