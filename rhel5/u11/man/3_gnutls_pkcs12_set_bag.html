NAME gnutls_pkcs12_set_bag - This function inserts a Bag into a PKCS12 structure

SYNOPSIS #include <gnutls/pkcs12.h>

int gnutls_pkcs12_set_bag(gnutls_pkcs12_t pkcs12, gnutls_pkcs12_bag_t bag);

ARGUMENTS gnutls_pkcs12_t pkcs12 should contain a gnutls_pkcs12_t structure

gnutls_pkcs12_bag_t bag An initialized bag

DESCRIPTION This function will insert a Bag into the PKCS12 structure. Returns 0 on success.
