NAME gnutls_extra_check_version - This function checks the librarys version

SYNOPSIS #include <gnutls/extra.h>

const char * gnutls_extra_check_version(const char * req_version);

ARGUMENTS const char * req_version the version to check

DESCRIPTION Check that the version of the gnutls-extra library is at minimum the requested one and return the version string; return NULL if the condi- tion is not satisfied. If a NULL is passed to this function, no check is done, but the version string is simply returned.
