NAME gnutls_pkcs12_bag_set_friendly_name - This function sets a friendly name into the bag element

SYNOPSIS #include <gnutls/pkcs12.h>

int gnutls_pkcs12_bag_set_friendly_name(gnutls_pkcs12_bag_t bag, int indx, const char * name);

ARGUMENTS gnutls_pkcs12_bag_t bag The bag

int indx The bags element to add the id

const char * name the name

DESCRIPTION This function will add the given key friendly name, to the specified, by the index, bag element. The name will be encoded as a Friendly name bag attribute, which is usually used to set a user name to the local private key and the certificate pair.

Returns 0 on success, or a negative value on error.
