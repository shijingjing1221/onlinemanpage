NAME pdiff - produce a pretty comparison between files

SYNOPSIS pdiff FILE1 FILE2 [-- A2PS-OPTIONS...]

DESCRIPTION Pretty print the differences between FILE1 and FILE2.

OPTIONS -h, --help display this help and exit

-v, --version display version information and exit

-q, --quiet dont print informational messages

-l, --lines search for line differences (diff)

-w, --words search for word differences (wdiff)

-o, --output=FILE save the output in FILE

Options for a2ps are given after --, for instance

$ pdiff COPYING COPYING.LIB -- -Pdisplay
