NAME gnutls_x509_crt_set_crq - This function will associate the Certificate with a request

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_set_crq(gnutls_x509_crt_t crt, gnutls_x509_crq_t crq);

ARGUMENTS gnutls_x509_crt_t crt should contain a gnutls_x509_crt_t structure

gnutls_x509_crq_t crq holds a certificate request

DESCRIPTION This function will set the name and public parameters from the given certificate request to the certificate. Only RSA keys are currently supported.

Returns 0 on success.
