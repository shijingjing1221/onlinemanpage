NAME getent - get entries from administrative database

SYNOPSIS getent database [key ...]

DESCRIPTION The getent program gathers entries from the specified administrative database using the specified search keys. Where database is one of aliases, ethers, group, hosts, netgroup, networks, passwd, protocols, rpc, services or shadow.
