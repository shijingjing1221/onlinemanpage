NAME pam_getenv - get a PAM environment variable

SYNOPSIS #include <security/pam_appl.h>

const char *pam_getenv(pam_handle_t *pamh, const char *name);

DESCRIPTION The pam_getenv function searches the PAM environment list as associated with the handle pamh for a string that matches the string pointed to by name. The return values are of the form: "name=value".

RETURN VALUES The pam_getenv function returns NULL on failure.

SEE ALSO pam_start(3), pam_getenvlist(3), pam_putenv(3), pam(8)
