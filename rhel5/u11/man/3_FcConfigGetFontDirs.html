NAME FcConfigGetFontDirs - Get font directories

SYNOPSIS #include <fontconfig.h>

FcStrList * FcConfigGetFontDirs (FcConfig *config);

DESCRIPTION Returns the list of font directories in config. This includes the con- figured font directories along with any directories below those in the filesystem.

VERSION Fontconfig version 2.4.1
