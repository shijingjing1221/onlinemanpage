NAME probe::linuxmib.TCPMemoryPressures - Count of times memory pressure was used

SYNOPSIS linuxmib.TCPMemoryPressures

VALUES sk Pointer to the struct sock being acted on

op Value to be added to the counter (default value of 1)

DESCRIPTION The packet pointed to by skb is filtered by the function linuxmib_filter_key. If the packet passes the filter is is counted in the global TCPMemoryPressures (equivalent to SNMPs MIB LINUX_MIB_TCPMEMORYPRESSURES)
