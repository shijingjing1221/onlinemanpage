NAME gnutls_x509_crt_sign - This function will sign a certificate with a key

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_sign(gnutls_x509_crt_t crt, gnutls_x509_crt_t issuer, gnutls_x509_privkey_t issuer_key);

ARGUMENTS gnutls_x509_crt_t crt should contain a gnutls_x509_crt_t structure

gnutls_x509_crt_t issuer is the certificate of the certificate issuer

gnutls_x509_privkey_t issuer_key holds the issuers private key

DESCRIPTION This function is the same a gnutls_x509_crt_sign2() with no flags, and SHA1 as the hash algorithm.

Returns 0 on success.
