NAME probe::scheduler.cpu_off - Process is about to stop running on a cpu

SYNOPSIS scheduler.cpu_off

VALUES task_prev the process leaving the cpu (same as current)

name name of the probe point

idle boolean indicating whether current is the idle process

task_next the process replacing current

CONTEXT The process leaving the cpu.
