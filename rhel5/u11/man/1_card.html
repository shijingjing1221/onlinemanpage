NAME card - print reference card of program options

SYNOPSIS card [OPTION]... PROGRAM...

DESCRIPTION Print a reference card of the PROGRAMs thanks to their inline help.

OPTIONS -h, --help display this help and exit

-v, --version display version information and exit

-o, --output=FILE save the output in FILE

-l, --language=LL print the help in the language LL (default: English)

--command=CMD perform pretty-printing on the output of CMD (e.g. --com- mand=netscape -help)

Options for a2ps are given after --, for instance

$ card -lfr a2ps -- -Pdisplay -4
