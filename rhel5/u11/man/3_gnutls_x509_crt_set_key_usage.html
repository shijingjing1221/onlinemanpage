NAME gnutls_x509_crt_set_key_usage - This function will set the keyUsage extension

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_set_key_usage(gnutls_x509_crt_t crt, unsigned int usage);

ARGUMENTS gnutls_x509_crt_t crt should contain a gnutls_x509_crt_t structure

unsigned int usage an ORed sequence of the GNUTLS_KEY_* elements.

DESCRIPTION This function will set the keyUsage certificate extension.

Returns 0 on success.

REPORTING BUGS Report bugs to <bug-gnutls@gnu.org>.
