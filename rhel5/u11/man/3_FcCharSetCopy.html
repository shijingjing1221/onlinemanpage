NAME FcCharSetCopy - Copy a charset

SYNOPSIS #include <fontconfig.h>

FcCharSet * FcCharSetCopy (FcCharSet *src);

DESCRIPTION Makes a copy of src; note that this may not actually do anything more than increment the reference count on src.

VERSION Fontconfig version 2.4.1
