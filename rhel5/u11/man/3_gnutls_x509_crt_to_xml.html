NAME gnutls_x509_crt_to_xml - This function parses an RDN sequence

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_to_xml(gnutls_x509_crt_t cert, gnutls_datum_t * res, int detail);

ARGUMENTS gnutls_x509_crt_t cert should contain a gnutls_x509_crt_t structure

gnutls_datum_t * res The datum that will hold the result

int detail The detail level (must be GNUTLS_XML_SHOW_ALL or GNUTLS_XML_NORMAL)

DESCRIPTION This function will return the XML structures of the given X.509 cer- tificate. The XML structures are allocated internally (with malloc) and stored into res.

NOTE This function is currently not implemented. See the NEWS entry for version 1.3.5.

Returns a negative error code in case of an error.
