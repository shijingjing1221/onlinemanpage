NAME system-install-packages - Install or update a list of packages

SYNOPSIS system-install-packages PACKAGE...

DESCRIPTION system-install-packages will attempt to install the list of packages provided on the command line. Any dependencies will be resolved using the system s configured repositories. system-install-packages will prompt for confirmation at each step to ensure you are comfortable with the proposed operations.
