NAME stringprep_utf8_to_locale - encode UTF-8 string to locale encoding

SYNOPSIS #include <stringprep.h>

char * stringprep_utf8_to_locale(const char * str);

ARGUMENTS const char * str input zero terminated string.

DESCRIPTION Convert string encoded in UTF-8 into the locales character set by using stringprep_convert().

RETURN VALUE Returns newly allocated zero-terminated string which is str transcoded into the locales character set.
