NAME gnutls_certificate_set_verify_flags - This function will set the flags to be used at certificate verification

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_certificate_set_verify_flags(gnutls_certificate_creden- tials_t res, unsigned int flags);

ARGUMENTS gnutls_certificate_credentials_t res is a gnutls_certificate_credentials_t structure

unsigned int flags are the flags

DESCRIPTION This function will set the flags to be used at verification of the cer- tificates. Flags must be OR of the gnutls_certificate_verify_flags enumerations.
