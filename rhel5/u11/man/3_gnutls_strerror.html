NAME gnutls_strerror - Returns a string with a description of an error

SYNOPSIS #include <gnutls/gnutls.h>

const char * gnutls_strerror(int error);

ARGUMENTS int error is an error returned by a gnutls function. Error is always a negative value.

DESCRIPTION This function is similar to strerror(). Differences: it accepts an error number returned by a gnutls function; In case of an unknown error a descriptive string is sent instead of NULL.
