NAME strcat, strncat - concatenate two strings

SYNOPSIS #include <string.h>

char *strcat(char *dest, const char *src);

char *strncat(char *dest, const char *src, size_t n);

DESCRIPTION The strcat() function appends the src string to the dest string over- writing the character at the end of dest, and then adds a termi- nating character. The strings may not overlap, and the dest string must have enough space for the result.

The strncat() function is similar, except that it will use at most n characters from src. Since the result is always terminated with , at most n+1 characters are written.

RETURN VALUE The strcat() and strncat() functions return a pointer to the resulting string dest.

CONFORMING TO SVr4, 4.3BSD, C89, C99.
