NAME pirut - A graphical software installer and remover

SYNOPSIS pirut

DESCRIPTION pirut is a graphical frontend to yum(8). Pirut can install, remove, or update software packages. Pirut also allows searching, or viewing and installing software package groups.

SEE ALSO pup(8) puplet(8) yum(8) system-install-packages(8) system-cdinstall-helper(8)
