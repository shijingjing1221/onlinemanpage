NAME findfs - Find a filesystem by label or UUID

SYNOPSIS findfs LABEL=label

findfs UUID=uuid

DESCRIPTION findfs will search the disks in the system looking for a filesystem which has a label matching label or a UUID equal to uuid. If the filesystem is found, the device name for the filesystem will be printed on stdout.

AUTHOR findfs was written by Theodore Tso (tytso@mit.edu).

AVAILABILITY findfs is part of the e2fsprogs package and is available from http://e2fsprogs.sourceforge.net.

SEE ALSO fsck(8)
