NAME gnutls_x509_crq_set_challenge_password - This function will set a chal- lenge password

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crq_set_challenge_password(gnutls_x509_crq_t crq, const char * pass);

ARGUMENTS gnutls_x509_crq_t crq should contain a gnutls_x509_crq_t structure

const char * pass holds a null terminated password

DESCRIPTION This function will set a challenge password to be used when revoking the request.

Returns 0 on success.
