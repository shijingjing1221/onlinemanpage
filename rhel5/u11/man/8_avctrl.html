NAME avctrl - Bluetooth Audio/Video control utility

SYNOPSIS avctrl [ options ] <command>

DESCRIPTION avctrl is used to control the Audio/Video dongles.

OPTIONS -h Gives a list of possible options.

-q Dont display any messages.
