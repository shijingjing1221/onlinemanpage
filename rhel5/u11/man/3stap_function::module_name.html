FUNCTION::MODULE_NAM(3stap) Context Functions FUNCTION::MODULE_NAM(3stap)

NAME function::module_name - The module name of the current script

SYNOPSIS module_name:string()

ARGUMENTS None

DESCRIPTION This function returns the name of the stap module. Either generated randomly (stap_[0-9a-f]+_[0-9a-f]+) or set by stap -m <module_name>.
