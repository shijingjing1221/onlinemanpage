NAME gnutls_certificate_get_ours - This function returns the raw certificate sent in the last handshake

SYNOPSIS #include <gnutls/gnutls.h>

const gnutls_datum_t * gnutls_certificate_get_ours(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION This function will return the certificate as sent to the peer, in the last handshake. These certificates are in raw format. In X.509 this is a certificate list. In OpenPGP this is a single certificate. Returns NULL in case of an error, or if no certificate was used.
