NAME function::set_kernel_pointer - Writes a pointer value to kernel memory.

SYNOPSIS set_kernel_pointer(addr:long,val:long)

ARGUMENTS addr The kernel address to write the pointer to

val The pointer which is to be written

DESCRIPTION Writes the pointer value to a given kernel memory address. Reports an error when writing to the given address fails. Requires the use of guru mode (-g).
