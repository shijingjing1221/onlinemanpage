NAME fgconsole - print the number of the active VT.

SYNOPSIS fgconsole

DESCRIPTION If the active Virtual Terminal is /dev/ttyN, then prints N on standard output.

NOTES Under devfs, the consoles are in /dev/vc/N. devfsd may maintain sym- links for compatibility.

SEE ALSO chvt(1).
