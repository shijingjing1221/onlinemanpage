NAME gnutls_anon_allocate_client_credentials - Used to allocate a creden- tials structure

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_anon_allocate_client_credentials(gnutls_anon_client_creden- tials_t * sc);

ARGUMENTS gnutls_anon_client_credentials_t * sc is a pointer to an gnutls_anon_client_credentials_t struc- ture.

DESCRIPTION This structure is complex enough to manipulate directly thus this helper function is provided in order to allocate it.

Returns a negative value in case of an error.
