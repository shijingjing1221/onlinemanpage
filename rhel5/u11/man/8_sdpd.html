NAME sdpd Bluetooth SDP daemon

SYNOPSIS sdpd [options]

DESCRIPTION sdpd allows Bluetooth devices connected to the host to advertise via SDP the Bluetooth services available.

OPTIONS -n Dont detach from the controlling terminal.

BUGS None yet known.
