NAME users - print the user names of users currently logged in to the cur- rent host

SYNOPSIS users [OPTION]... [ FILE ]

DESCRIPTION Output who is currently logged in according to FILE. If FILE is not specified, use /var/run/utmp. /var/log/wtmp as FILE is common.

--help display this help and exit

--version output version information and exit
