NAME gnutls_safe_renegotiation_set - API function

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_safe_renegotiation_set(gnutls_session_t session, int value);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

int value 0 to disable and 1 to enable

DESCRIPTION Used to enable and disable safe renegotiation for the current session. Normally you shouldnt cope with this function since the default (enable) is sufficient, but there might be servers that cannot handle or correctly handle the extension.
