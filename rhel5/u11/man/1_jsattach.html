NAME jsattach - attach a serial line to a joystick device

SYNOPSIS jsattach <mode> <device>

DESCRIPTION jsattach is used for attaching a serial line to a joystick device via a line dicipline.

Exactly one of the following modes must be specified on the command line.

MODES --sball4 Use this option for the SpaceBall 4000 FLX.

--spaceorb Use this option for the SpaceOrb 360 or SpaceBall Avenger.

--magellan Use this option for the Magellan or SpaceMouse.

--warrior Use this option for the WingMan Warrior.

SEE ALSO jscal(1), jstest(1).
