NAME gnutls_rsa_export_get_modulus_bits - This function returns the bits used in RSA-export key exchange

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_rsa_export_get_modulus_bits(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION This function will return the bits used in the last RSA-EXPORT key exchange with the peer. Returns a negative value in case of an error.
