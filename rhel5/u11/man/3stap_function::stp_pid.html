FUNCTION::STP_PID(3stap) Context Functions FUNCTION::STP_PID(3stap)

NAME function::stp_pid - The process id of the stapio process

SYNOPSIS stp_pid:long()

ARGUMENTS None

DESCRIPTION This function returns the process id of the stapio process that launched this script. There could be other SystemTap scripts and stapio processes running on the system.
