NAME icc2ps - little cms PostScript converter.

SYNOPSIS icc2ps [options]

DESCRIPTION lcms is a standalone CMM engine, which deals with the color management. It implements a fast transformation between ICC profiles. icc2ps is little cms PostScript converter.

OPTIONS -b Black point compensation (CRD only).

-i profile Input profile: Generates Color Space Array (CSA).

-o profile Output profile: Generates Color Rendering Dictionary(CRD).

-t <0,1,2,3> Intent (0=Perceptual, 1=Colorimetric, 2=Saturation, 3=Absolute).

-u Do NOT generate resource name on CRD.
