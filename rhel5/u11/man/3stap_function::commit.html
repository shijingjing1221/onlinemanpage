FUNCTION::COMMIT(3stap) Speculation FUNCTION::COMMIT(3stap)

NAME function::commit - Write out all output related to a speculation buffer

SYNOPSIS commit(id:long)

ARGUMENTS id of the buffer to store the information in

DESCRIPTION Output all the output for id in the order that it was entered into the speculative buffer by speculative.
