NAME ovdb_server - overview helper server

SYNOPSYS Use "ovdb_init" to start ovdb_server

DESCRIPTION If the "readserver" parameter in ovdb.conf is true, "ovdb_init" will start "ovdb_server".

"ovdb_server" opens the overview database, and accesses it on behalf of the nnrpd reader processes.

To shut down ovdb_server, send a TERM signal to the process ID in pathrun/ovdb_server.pid . The parent process will shut down its chil- dren and wait for their exit before exiting itself.
