NAME FcCharSetAddChar - Add a character to a charset

SYNOPSIS #include <fontconfig.h>

FcBool FcCharSetAddChar (FcCharSet *fcs, FcChar32ucs4);

DESCRIPTION FcCharSetAddChar adds a single unicode char to the set, returning FcFalse on failure, either as a result of a constant set or from run- ning out of memory.

VERSION Fontconfig version 2.4.1
