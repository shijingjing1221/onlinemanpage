NAME gnutls_psk_allocate_server_credentials - Used to allocate an gnutls_psk_server_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_psk_allocate_server_credentials(gnutls_psk_server_creden- tials_t * sc);

ARGUMENTS gnutls_psk_server_credentials_t * sc is a pointer to an gnutls_psk_server_credentials_t struc- ture.

DESCRIPTION This structure is complex enough to manipulate directly thus this helper function is provided in order to allocate it.

Returns 0 on success.
