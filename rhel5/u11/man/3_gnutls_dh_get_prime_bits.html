NAME gnutls_dh_get_prime_bits - This function returns the bits used in DH authentication

SYNOPSIS #include <gnutls/gnutls.h>

int gnutls_dh_get_prime_bits(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls session

DESCRIPTION This function will return the bits of the prime used in the last Diffie Hellman authentication with the peer. Should be used for both anonymous and ephemeral diffie Hellman. Returns a negative value in case of an error.
