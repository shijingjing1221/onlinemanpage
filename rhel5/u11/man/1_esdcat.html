ESDCAT(1) Enlightened Sound Daemon ESDCAT(1)

NAME esdcat - The Enlightened Sound Daemon

DESCRIPTION esdcat pipes data from a pipe to the sound device

USAGE esdcat ["-s" server] ["-n" name] ["-b"] ["-m"] ["-r" freq] < file
