NAME vgmknodes - recreate volume group directory and logical volume special files

SYNOPSIS vgmknodes [-d|--debug] [-h|--help] [--refresh] [-v|--verbose] [[Vol- umeGroupName | LogicalVolumePath]...]

DESCRIPTION Checks the LVM2 special files in /dev that are needed for active logi- cal volumes and creates any missing ones and removes unused ones.

OPTIONS --refresh If any logical volume in the volume group is active, reload its metadata. This is not necessary in normal operation, but may be useful if something has gone wrong or if you re doing clustering manually without a clustered lock manager.

See lvm for common options.

SEE ALSO lvm(8), vgscan(8), dmsetup(8)
