NAME gnutls_rsa_params_deinit - This function will deinitialize the RSA parameters

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_rsa_params_deinit(gnutls_rsa_params_t rsa_params);

ARGUMENTS gnutls_rsa_params_t rsa_params Is a structure that holds the parameters

DESCRIPTION This function will deinitialize the RSA parameters structure.
