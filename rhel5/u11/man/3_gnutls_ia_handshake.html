NAME gnutls_ia_handshake - API function

SYNOPSIS #include <gnutls/extra.h>

int gnutls_ia_handshake(gnutls_session_t session);

ARGUMENTS gnutls_session_t session is a gnutls_session_t structure.

DESCRIPTION Perform a TLS/IA handshake. This should be called after gnutls_hand- shake() iff gnutls_ia_handshake_p().

Return 0 on success, or an error code.
