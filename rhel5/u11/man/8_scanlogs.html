NAME scanlogs - summarize INN log files.

SYNOPSIS scanlogs [ norotate ]

DESCRIPTION Scanlogs summarizes the information recorded in the INN log files (see newslog(5)). By default, it also rotates and cleans out the logs. It is normally invoked by the news.daily(8) script.

KEYWORDS The following keywords are accepted:

norotate Using this keyword disables the rotating and cleaning aspect of the log processing: the logs files are only scanned for informa- tion and no contents are altered.

If scanlogs is invoked more than once a day, the norotate keyword should be used to prevent premature log cleaning.
