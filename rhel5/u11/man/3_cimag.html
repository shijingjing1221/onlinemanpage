CIMAG(3) complex math routines CIMAG(3)

NAME cimag, cimagf, cimagl - get imaginary part of a complex number

SYNOPSIS #include <complex.h>

double cimag(double complex z); float cimagf(float complex z); long double cimagl(long double complex z);

Link with -lm.

DESCRIPTION The cimag() function returns the imaginary part of the complex number z.

One has z = creal(z) + I*cimag(z).

NOTE gcc also supports __imag__. That is a GNU extension.

CONFORMING TO C99

SEE ALSO cabs(3), creal(3), complex(7)
