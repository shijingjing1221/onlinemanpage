NAME foomatic-gswrapper - foomatic wrapper for ghostscript

SYNOPSIS foomatic-gswrapper [ gs option ] ...

This is a little Ghostscript regularization script. It massages argu- ments to make Ghostscript execute properly as a filter, with output on stdout and errors etc on stderr.

EXIT STATUS foomatic-gswrapper returns 0 on success.

AUTHOR Manfred Wassmann <manolo@NCC-1701.B.Shuttle.de> for the foomatic project using output from the associated binary.

BUGS None so far.

Please send bug reports to foomatic-devel@linuxprinting.org.
