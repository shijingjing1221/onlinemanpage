NAME issue - pre-login message and identification file

DESCRIPTION The file /etc/issue is a text file which contains a message or system identification to be printed before the login prompt. It may contain various @char and char sequences, if supported by mingetty(8).

FILES /etc/issue

SEE ALSO mingetty(8), motd(5)
