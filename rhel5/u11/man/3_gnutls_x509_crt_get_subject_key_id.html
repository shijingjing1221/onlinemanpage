NAME gnutls_x509_crt_get_subject_key_id - This function returns the certifi- cates key identifier

SYNOPSIS #include <gnutls/x509.h>

int gnutls_x509_crt_get_subject_key_id(gnutls_x509_crt_t cert, void * ret, size_t * ret_size, unsigned int * critical);

ARGUMENTS gnutls_x509_crt_t cert should contain a gnutls_x509_crt_t structure

void * ret

size_t * ret_size

unsigned int * critical will be non zero if the extension is marked as critical (may be null)

DESCRIPTION This function will return the X.509v3 certificates subject key identi- fier. This is obtained by the X.509 Subject Key identifier extension field (2.5.29.14).

Returns 0 on success and a negative value in case of an error.
