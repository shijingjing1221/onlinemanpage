NAME gnutls_anon_free_client_credentials - Used to free an allocated gnutls_anon_client_credentials_t structure

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_anon_free_client_credentials(gnutls_anon_client_creden- tials_t sc);

ARGUMENTS gnutls_anon_client_credentials_t sc is an gnutls_anon_client_credentials_t structure.

DESCRIPTION This structure is complex enough to manipulate directly thus this helper function is provided in order to free (deallocate) it.
