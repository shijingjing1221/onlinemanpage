NAME function::proc_mem_shr_pid - Program shared pages (from shared mappings)

SYNOPSIS proc_mem_shr_pid:long(pid:long)

ARGUMENTS pid The pid of process to examine

DESCRIPTION Returns the shared pages (from shared mappings) of the given process, or zero when the process doesnt exist or the number of pages couldnt be retrieved.
