NAME stringprep_locale_to_utf8 - convert locale encoded string to UTF-8

SYNOPSIS #include <stringprep.h>

char * stringprep_locale_to_utf8(const char * str);

ARGUMENTS const char * str input zero terminated string.

DESCRIPTION Convert string encoded in the locales character set into UTF-8 by using stringprep_convert().

RETURN VALUE Returns newly allocated zero-terminated string which is str transcoded into UTF-8.
