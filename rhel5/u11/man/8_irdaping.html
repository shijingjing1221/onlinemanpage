NAME irdaping - sends IrDA test frames

SYNOPSIS irdaping [ daddr ] [ -s framesize ]

OPTIONS daddr ...

-s framesize ...

DESCRIPTION This is a program similar to ping(8). It sends IrDA test frames (added some userdata which contains the frame number and the time the frame was sent). You can change the size of the frame by using the -s option. To get daddr you may use irdadump.

irdaping must be run as root or installed setuid root, as it requires root privileges.

EXAMPLES irdaping <daddr> [-s <framesize>]

You may use an alias, which does not use any parameter alias irp- ing="irdaping grep daddr /proc/net/irda/discovery|sed s/.*daddr:/" It works fine when there is only one discovered client.
