NAME gnutls_pkcs12_bag_decrypt - This function will decrypt an encrypted bag

SYNOPSIS #include <gnutls/pkcs12.h>

int gnutls_pkcs12_bag_decrypt(gnutls_pkcs12_bag_t bag, const char * pass);

ARGUMENTS gnutls_pkcs12_bag_t bag The bag

const char * pass The password used for encryption. This can only be ASCII.

DESCRIPTION This function will decrypt the given encrypted bag and return 0 on suc- cess.
