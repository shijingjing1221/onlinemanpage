NAME gnutls_openpgp_privkey_deinit - This function deinitializes memory used by a gnutls_openpgp_privkey_t structure

SYNOPSIS #include <gnutls/openpgp.h>

void gnutls_openpgp_privkey_deinit(gnutls_openpgp_privkey_t key);

ARGUMENTS gnutls_openpgp_privkey_t key The structure to be initialized

DESCRIPTION This function will deinitialize a key structure.
