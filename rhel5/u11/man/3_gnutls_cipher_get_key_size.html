NAME gnutls_cipher_get_key_size - Returns the length of the ciphers key size

SYNOPSIS #include <gnutls/gnutls.h>

size_t gnutls_cipher_get_key_size(gnutls_cipher_algorithm_t algorithm);

ARGUMENTS gnutls_cipher_algorithm_t algorithm is an encryption algorithm

DESCRIPTION Returns the length (in bytes) of the given cipher s key size. Returns 0 if the given cipher is invalid.
