NAME nohup - run a command immune to hangups, with output to a non-tty

SYNOPSIS nohup COMMAND [ARG]... nohup OPTION

DESCRIPTION Run COMMAND, ignoring hangup signals.

--help display this help and exit

--version output version information and exit

NOTE: your shell may have its own version of nohup, which usually supersedes the version described here. Please refer to your shells documentation for details about the options it supports.
