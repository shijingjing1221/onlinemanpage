NAME gnutls_certificate_set_openpgp_key - Used to set keys in a gnutls_cer- tificate_credentials_t structure

SYNOPSIS #include <gnutls/extra.h>

int gnutls_certificate_set_openpgp_key(gnutls_certificate_credentials_t res, gnutls_openpgp_key_t key, gnutls_openpgp_privkey_t pkey);

ARGUMENTS gnutls_certificate_credentials_t res is an gnutls_certificate_credentials_t structure.

gnutls_openpgp_key_t key contains an openpgp public key

gnutls_openpgp_privkey_t pkey is an openpgp private key

DESCRIPTION This function sets a certificate/private key pair in the gnutls_cer- tificate_credentials_t structure. This function may be called more than once (in case multiple keys/certificates exist for the server).
