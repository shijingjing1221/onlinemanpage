NAME gnutls_openpgp_key_check_hostname - This function compares the given hostname with the hostname in the key

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_key_check_hostname(gnutls_openpgp_key_t key, const char * hostname);

ARGUMENTS gnutls_openpgp_key_t key should contain an gnutls_openpgp_key_t structure

const char * hostname A null terminated string that contains a DNS name

DESCRIPTION This function will check if the given key s owner matches the given hostname. This is a basic implementation of the matching described in RFC2818 (HTTPS), which takes into account wildcards.

Returns non zero on success, and zero on failure.
