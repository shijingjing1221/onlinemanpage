NAME gnutls_openpgp_key_get_fingerprint - Gets the fingerprint

SYNOPSIS #include <gnutls/openpgp.h>

int gnutls_openpgp_key_get_fingerprint(gnutls_openpgp_key_t key, void * fpr, size_t * fprlen);

ARGUMENTS gnutls_openpgp_key_t key the raw data that contains the OpenPGP public key.

void * fpr the buffer to save the fingerprint.

size_t * fprlen the integer to save the length of the fingerprint.

DESCRIPTION Returns the fingerprint of the OpenPGP key. Depends on the algorithm, the fingerprint can be 16 or 20 bytes.
