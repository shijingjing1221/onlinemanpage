NAME FcPatternRemove - Remove one object of the specified type from the pat- tern

SYNOPSIS #include <fontconfig.h>

FcBool FcPatternRemove (FcPattern *p, const char *object, intid);

DESCRIPTION Removes the value associated with the property object at position id , returning whether the property existed and had a value at that position or not.
