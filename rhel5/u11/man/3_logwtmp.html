NAME updwtmp, logwtmp - append an entry to the wtmp file

SYNOPSIS #include <utmp.h>

void updwtmp(const char *wtmp_file, const struct utmp *ut); void logwtmp(const char *line, const char *name, const char *host);

DESCRIPTION updwtmp() appends the utmp structure ut to the wtmp file.

logwtmp() constructs a utmp structure using line, name, host, current time and current process ID. Then it calls updwtmp() to append the structure to the utmp file.

AVAILABILITY Both functions are available under glibc2, but not under libc5. How- ever, logwtmp() used to occur in the old libbsd. These days, the logwtmp() function is included in libutil. (Hence youll need to add -lutil to your compiler command line to get it.)

CONFORMING TO Not in POSIX.1-2001. Present on Solaris, NetBSD, and perhaps other systems.

FILES /var/log/wtmp database of past user logins
