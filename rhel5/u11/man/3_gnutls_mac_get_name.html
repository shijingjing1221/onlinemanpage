NAME gnutls_mac_get_name - Returns a string with the name of the specified mac algorithm

SYNOPSIS #include <gnutls/gnutls.h>

const char * gnutls_mac_get_name(gnutls_mac_algorithm_t algorithm);

ARGUMENTS gnutls_mac_algorithm_t algorithm is a MAC algorithm

DESCRIPTION Returns a string that contains the name of the specified MAC algorithm or NULL.
