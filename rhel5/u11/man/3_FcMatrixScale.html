NAME FcMatrixScale - Scale a matrix

SYNOPSIS #include <fontconfig.h>

void FcMatrixScale (FcMatrix *matrix, doublesx, doubledy);

DESCRIPTION FcMatrixScale multiplies matrix x values by sx and y values by sy. This is done by multiplying by the matrix:

sx 0 0 sy

VERSION Fontconfig version 2.4.1
