NAME probe::sunrpc.svc.register - Register an RPC service with the local portmapper

SYNOPSIS sunrpc.svc.register

VALUES prog the number of the program

progname the name of the program

sv_name the service name

port the port number

prot the IP protocol number

DESCRIPTION If proto and port are both 0, then unregister a service.
