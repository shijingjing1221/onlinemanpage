NAME gnutls_free - Returns a free() like function

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_free(void * ptr);

ARGUMENTS void * ptr

DESCRIPTION This function will free data pointed by ptr.

The deallocation function used is the one set by gnutls_global_set_mem_functions().
