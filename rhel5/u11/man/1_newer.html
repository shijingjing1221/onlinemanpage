NAME newer - compare file modification times

SYNOPSIS newer [option] file1 file2

DESCRIPTION Exit successfully if file1 exists and is not older than file2. Also exit successfully if file2 doesnt exist.

OPTIONS newer accepts the following options:

--help Print help message and exit.

--version Print version information and exit.
