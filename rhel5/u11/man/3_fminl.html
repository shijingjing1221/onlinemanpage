FMIN(3) math routines FMIN(3)

NAME fmin, fminf, fminl - find minimum value

SYNOPSIS #include <math.h>

double fmin(double x, double y); float fminf(float x, float y); long double fminl(long double x, long double y);

Compile with -std=c99; link with -lm.

DESCRIPTION Find the lesser value of x and y.

CONFORMING TO C99

SEE ALSO fmax(3)
