NAME texconfig - configures teTeX

texconfig-sys - configures teTeX system-wide

SYNOPSIS texconfig

texconfig-sys

DESCRIPTION texconfig allows to configure and maintain teTeX in an easy and conve- nient manner. It uses the dialog program to draw nice input boxes. After answering the dialoges, the appropriate files are changed auto- matically and all necessary steps are involved to apply the new set- tings.

SEE ALSO dialog(1), initex(1), mf(1), vi(1), EDITOR environment-variable.
