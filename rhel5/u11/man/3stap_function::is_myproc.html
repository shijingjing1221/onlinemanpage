FUNCTION::IS_MYPROC(3stap) Context Functions FUNCTION::IS_MYPROC(3stap)

NAME function::is_myproc - Determines if the current probe point has occurred in the users own process

SYNOPSIS is_myproc:long()

ARGUMENTS None

DESCRIPTION This function returns 1 if the current probe point has occurred in the users own process.
