NAME gnutls_global_deinit - This function deinitializes the global data

SYNOPSIS #include <gnutls/gnutls.h>

void gnutls_global_deinit( void);

ARGUMENTS void

DESCRIPTION This function deinitializes the global data, that were initialized using gnutls_global_init().

Note! This function is not thread safe. See the discussion for gnutls_global_init() for more information.
