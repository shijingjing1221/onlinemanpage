NAME

YAML::Marshall − YAML marshalling class you can mixin to your classes

SYNOPSIS 

    package Bar;
    use Foo −base;
    use YAML::Marshall −mixin;

DESCRIPTION

For classes that want to handle their own YAML serialization.
