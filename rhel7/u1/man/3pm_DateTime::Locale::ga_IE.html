NAME

DateTime::Locale::ga_IE

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'ga_IE' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Irish Ireland.
