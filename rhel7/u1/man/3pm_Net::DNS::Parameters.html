NAME 

    Net::DNS::Parameters − DNS parameter assignments

SYNOPSIS 

    use Net::DNS::Parameters;

DESCRIPTION

Net::DNS::Parameters is a Perl package representing the DNS parameter allocation (key,value) tables as recorded in the definitive registry file maintained and published by IANA.
