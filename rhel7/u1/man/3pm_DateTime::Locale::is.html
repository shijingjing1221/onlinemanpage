NAME

DateTime::Locale::is

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'is' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Icelandic.

DATA

This locale inherits from the DateTime::Locale::root locale.

It contains the following data.
