NAME

sfcbtrace − toggle the tracemask for SFCB trace output

SYNOPSIS

sfcbtrace <trace_mask> <shm_key>

OPTIONS

trace_mask − an unsigned long or hex value for component(s) to trace
(default=0)
shm_key − the shared memory ID being used by SFCB (default=deb001)
