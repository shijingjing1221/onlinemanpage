NAME

DateTime::Locale::kcg_NG

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'kcg_NG' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Tyap Nigeria.
