NAME

pear.conf − configuration file for PEAR applications.

DESCRIPTION

The pear.conf is used by PEAR applications to store their configuration. This file should not be edited.

The pear command provides the config-show, config-get and config-set sub-commands needed to change the configuration.

FILES

/etc/pear.conf

The system configuration file.

$HOME/.pearrc

The user configuration file.
