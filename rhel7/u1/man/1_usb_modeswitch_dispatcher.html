NAME

usb_modeswitch_dispatcher - dispatcher not intended for direct invocation.

SYNOPSIS

usb_modeswitch_dispatcher

DESCRIPTION

usb_modeswitch_dispatcher is doing additional device checking and using the usb_modeswitch binary to switch with the selected device config file. If no drivers are taking care of the device after the mode switch, the dispatcher will try to load and bind the "option" serial driver, in order to make the device usable.

This program is called by udev and is not supposed to be called directly by the user.
