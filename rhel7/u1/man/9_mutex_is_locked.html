NAME

mutex_is_locked − is the mutex locked

SYNOPSIS

int mutex_is_locked(struct mutex * lock);

ARGUMENTS

lock

the mutex to be queried

DESCRIPTION

Returns 1 if the mutex is locked, 0 if unlocked.
