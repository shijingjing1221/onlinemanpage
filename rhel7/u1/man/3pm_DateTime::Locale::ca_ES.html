NAME

DateTime::Locale::ca_ES

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'ca_ES' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Catalan Spain.
