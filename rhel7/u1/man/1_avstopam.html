NAME

avstopam - convert an AVS X image to a Netpbm image

SYNOPSIS

avstopam [avsfile]

DESCRIPTION

This program is part of Netpbm(1)

avstopam reads a Stardent <acronym xml:lang="en" title="Application Visualization System">AVS</acronym> X image as input and produces a Netpbm image as output.

avsfile is the input file, which defaults to Standard Input. Output is always on Standard Output.
