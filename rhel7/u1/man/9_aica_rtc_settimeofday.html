NAME

aica_rtc_settimeofday − Set the AICA RTC to the current time

SYNOPSIS

int aica_rtc_settimeofday(const time_t secs);

ARGUMENTS

secs

contains the time_t to set

DESCRIPTION

Adjusts the given tv to the AICA Epoch and sets the RTC seconds counter.
