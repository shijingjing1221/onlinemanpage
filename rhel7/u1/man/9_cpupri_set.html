NAME

cpupri_set − update the cpu priority setting

SYNOPSIS

void cpupri_set(struct cpupri * cp, int cpu, int newpri);

ARGUMENTS

cp

The cpupri context

cpu

The target cpu

newpri

The priority (INVALID−RT99) to assign to this CPU

NOTE

Assumes cpu_rq(cpu)−>lock is locked

RETURNS

(void)
