NAME

gnutls_x509_aki_get_id − API function

SYNOPSIS

#include <gnutls/x509-ext.h>

int gnutls_x509_aki_get_id(gnutls_x509_aki_t aki, gnutls_datum_t * id);

ARGUMENTS

gnutls_x509_aki_t aki

The authority key ID structure

gnutls_datum_t * id

Will hold the identifier

DESCRIPTION

This function will return the key identifier as stored in the aki structure. The identifier should be treated as constant.

RETURNS

On success, GNUTLS_E_SUCCESS (0) is returned, GNUTLS_E_REQUESTED_DATA_NOT_AVAILABLE if the index is out of bounds, otherwise a negative error value.

SINCE

3.3.0
