NAME

arpaname − translate IP addresses to the corresponding ARPA names

SYNOPSIS

arpaname {ipaddress ...}

DESCRIPTION

arpaname translates IP addresses (IPv4 and IPv6) to the corresponding IN−ADDR.ARPA or IP6.ARPA names.
