NAME

cfg80211_cqm_rssi_notify − connection quality monitoring rssi event

SYNOPSIS

ARGUMENTS

dev

network device

rssi_event

the triggered RSSI event

gfp

context flags

DESCRIPTION

This function is called when a configured connection quality monitoring rssi threshold reached event occurs.
