NAME

Net::LDAP::Control::SyncState − LDAPv3 Sync State control object

SYNOPSIS

See Net::LDAP::Control::SyncRequest

DESCRIPTION

"Net::LDAP::Control::SyncState" provides an interface for the creation and manipulation of objects that represent the "Sync State Control" as described by RFC 4533.

CONSTRUCTOR ARGUMENTS

In addition to the constructor arguments described in Net::LDAP::Control the following are provided.
state
entryUIID
cookie

METHODS

As with Net::LDAP::Control each constructor argument described above is also available as a method on the object which will return the current value for the attribute if called without an argument, and set a new value for the attribute if called with an argument.

SEE ALSO

Net::LDAP, Net::LDAP::Control, Net::LDAP::Control::SyncRequest, Net::LDAP::Control::SyncDone, http://www.ietf.org/rfc/rfc4533.txt
