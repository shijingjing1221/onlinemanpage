NAME

ptaskset (python−schedutils) − set or get the affinity of a process

SYNOPSIS

ptaskset [options] [mask | cpu−list] [pid | cmd [args...]]

OPTIONS

−p, −−pid

operate on existing given pid

−c, −−cpu−list

display and specify cpus in list format

−h, −−help

display help
