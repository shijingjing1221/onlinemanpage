NAME

DateTime::Locale::ro_MD

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'ro_MD' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Romanian Moldova.

DATA

This locale inherits from the DateTime::Locale::ro locale.
