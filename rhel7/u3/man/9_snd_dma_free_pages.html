NAME

snd_dma_free_pages − release the allocated buffer

SYNOPSIS

void snd_dma_free_pages(struct snd_dma_buffer * dmab);

ARGUMENTS

dmab

the buffer allocation record to release

DESCRIPTION

Releases the allocated buffer via snd_dma_alloc_pages.
