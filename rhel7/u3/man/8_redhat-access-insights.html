NAME

redhat−access−insights − Red Hat Access Insights Client tool

SYNOPSIS

redhat-access-insights [options]

DESCRIPTION

redhat−access−insights is designed to help customers proactively resolve issues affecting business operations in Red Hat Enterprise Linux and Cloud Infrastructure environments. View alerts or learn more at https://access.redhat.com/insights/

OPTIONS

-h, --help

show this help message and exit

--version

Display version

--register

Register system to the Red Hat Access Insights Service

--unregister

Unregister system from the Red Hat Access Insights Service

--update-collection-rules

Refresh collection rules from Red Hat Set Red Hat Access Insights to collect data once per week

--group=GROUP

Group to add this system to during registration

--retry=RETRIES

Number of times to retry uploading. 300 seconds between tries

--validate

Validate remove.conf

--quiet

Only display error messages to stdout

--silent

Display no messages to stdout

--no-schedule

Disable automatic scheduling

-c CONF, --conf=CONF

Pass a custom config file

--to-stdout

print archive to stdout; sets --silent and --no-upload

--compressor

Specifies the compression algorithm to use. Choices are gz, bz2, xz, and none. Defaults to gz.

--from-stdin

This accepts a JSON document from stdin rather than read rule config from a file or from Red Hat directly. This reads a JSON document from stdin with the keys: "uploader.json", with a value of a raw string representing an uploader config JSON document; and "sig", with a value of the raw signature content. Make sure the uploader JSON is not actually parsed into the resulting dict from stdin, i.e. it should be a raw string with JSON special characters escaped within.

DEBUG OPTIONS

--test-connection

Test connectivity to Red Hat

--force-reregister

Forcefully reregister this machine to Red Hat. Use only as directed.

--verbose

DEBUG output to stdout

--no-gpg

Do not verify GPG signature

--no-upload

Do not upload the archive

--no-tar-file

Build the directory, but do not tar

--keep-archive

Do not delete archive after upload
