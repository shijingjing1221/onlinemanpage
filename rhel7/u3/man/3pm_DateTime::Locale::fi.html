NAME

DateTime::Locale::fi

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'fi' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Finnish.

This locale inherits from the DateTime::Locale::root locale.
