NAME

mutex_init − initialize the mutex

SYNOPSIS

mutex_init(mutex);

ARGUMENTS

mutex

the mutex to be initialized

DESCRIPTION

Initialize the mutex to unlocked state.

It is not allowed to initialize an already locked mutex.
