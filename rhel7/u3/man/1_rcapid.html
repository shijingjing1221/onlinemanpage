NAME

rcapid - remote CAPI server

SYNOPSIS

rcapid [-l loglevel]

DESCRIPTION

rcapid is a server to communicate with CAPI clients. It is designed to run from inetd. add the following line to /etc/services:

capi 6000/tcp

Add the following line to /etc/inetd.conf:

capi stream tcp nowait.9999 /usr/sbin/rcapid rcapid

OPTIONS

-l loglevel

the level of verbosity in the logging. This is logged in /tmp/rcapid.log
