NAME

glutRemoveMenuItem — Remove a given menu item.

LIBRARY

OpenGLUT - menus

SYNOPSIS

#include <openglut.h>

void

glutRemoveMenuItem(int item);

PARAMETERS

item Integer position down the list

DESCRIPTION

Walks the list of the menu items and deletes the numbered item in the list.
