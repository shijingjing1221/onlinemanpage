NAME

struct_cfg80211_pmksa − PMK Security Association

SYNOPSIS

struct cfg80211_pmksa {
const u8 * bssid;
const u8 * pmkid;
};

MEMBERS

bssid

The AP's BSSID.

pmkid

The PMK material itself.

DESCRIPTION

This structure is passed to the set/del_pmksa method for PMKSA caching.
