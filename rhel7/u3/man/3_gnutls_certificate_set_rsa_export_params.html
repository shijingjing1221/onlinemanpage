NAME

gnutls_certificate_set_rsa_export_params − API function

SYNOPSIS

#include <gnutls/compat.h>

void gnutls_certificate_set_rsa_export_params(gnutls_certificate_credentials_t res, gnutls_rsa_params_t rsa_params);

ARGUMENTS

gnutls_certificate_credentials_t res

is a gnutls_certificate_credentials_t structure

gnutls_rsa_params_t rsa_params

is a structure that holds temporary RSA parameters.

DESCRIPTION

This function will set the temporary RSA parameters for a certificate server to use. These parameters will be used in RSA−EXPORT cipher suites.
