NAME

__skb_alloc_pages − allocate pages for ps−rx on a skb and preserve pfmemalloc data

SYNOPSIS

struct page * __skb_alloc_pages(gfp_t gfp_mask, struct sk_buff * skb, unsigned int order);

ARGUMENTS

gfp_mask

alloc_pages_node mask. Set __GFP_NOMEMALLOC if not for network packet RX

skb

skb to set pfmemalloc on if __GFP_MEMALLOC is used

order

size of the allocation

DESCRIPTION

Allocate a new page.

NULL is returned if there is no free memory.
