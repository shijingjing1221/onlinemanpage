NAME

gnutls_x509_crq_set_private_key_usage_period − API function

SYNOPSIS

#include <gnutls/x509.h>

int gnutls_x509_crq_set_private_key_usage_period(gnutls_x509_crq_t crq, time_t activation, time_t expiration);

ARGUMENTS

gnutls_x509_crq_t crq

a certificate of type gnutls_x509_crq_t

time_t activation

The activation time

time_t expiration

The expiration time

DESCRIPTION

This function will set the private key usage period extension (2.5.29.16).

RETURNS

On success, GNUTLS_E_SUCCESS (0) is returned, otherwise a negative error value.
