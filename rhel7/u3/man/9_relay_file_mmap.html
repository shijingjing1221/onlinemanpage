NAME

relay_file_mmap − mmap file op for relay files

SYNOPSIS

int relay_file_mmap(struct file * filp, struct vm_area_struct * vma);

ARGUMENTS

filp

the file

vma

the vma describing what to map

DESCRIPTION

Calls upon relay_mmap_buf to map the file into user space.
