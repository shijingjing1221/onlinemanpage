NAME

irq_get_next_irq − get next allocated irq number

SYNOPSIS

unsigned int irq_get_next_irq(unsigned int offset);

ARGUMENTS

offset

where to start the search

DESCRIPTION

Returns next irq number after offset or nr_irqs if none is found.
