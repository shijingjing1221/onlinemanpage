DESCRIPTION

ldns_pkt_edns() returns true if this packet needs and EDNS rr to be sent. At the moment the only reason is an expected packet size larger than 512 bytes, but for instance dnssec would be a good reason too.

packet: the packet to check
Returns true if packet needs edns rr

ldns_pkt_edns_udp_size() return the packet’s edns udp size

packet: the packet
Returns the size

ldns_pkt_edns_extended_rcode() return the packet’s edns extended rcode

packet: the packet
Returns the rcode

ldns_pkt_edns_version() return the packet’s edns version

packet: the packet
Returns the version

ldns_pkt_edns_z() return the packet’s edns z value

Returns the z value
