NAME

rio_inb_pwrite_handler − process inbound port−write message

SYNOPSIS

ARGUMENTS

pw_msg

pointer to inbound port−write message

DESCRIPTION

Processes an inbound port−write message. Returns 0 if the request has been satisfied.
