NAME

foomatic-printermap-to-gutenprint-xml − Updates Gutenprint driver XML file

SYNOPSIS

foomatic-printermap-to-gutenprint-xml

DESCRIPTION

This command updates the driver XML file for the Gutenprint driver to the state of the foomatic-printermap file in the Gutenprint source tree. Used to update the driver XML file for Gutenprint in the OpenPrinting database.

It is very unlikely you need to do this.
