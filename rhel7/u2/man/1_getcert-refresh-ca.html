NAME

getcert

SYNOPSIS

getcert refresh-ca [options]

DESCRIPTION

Forces certmonger to refresh information specific to a CA, such as locally-stored copies of its certificates.

OPTIONS

−c NAME

Refresh information about the CA which has the specified nickname.

−a

Refresh information about all known CAs.

BUGS

Please file tickets for any that you find at https://fedorahosted.org/certmonger/
