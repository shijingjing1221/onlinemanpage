NAME

DateTime::Locale::trv_TW

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'trv_TW' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Taroko Taiwan.
