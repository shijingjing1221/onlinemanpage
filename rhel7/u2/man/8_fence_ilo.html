NAME

fence_ilo - Fence agent for HP iLO

fence_ilo2 - Fence agent for HP iLO2 (symlink)

DESCRIPTION

fence_ilo is an I/O Fencing agent used for HP servers with the Integrated Light Out (iLO) PCI card.The agent opens an SSL connection to the iLO card. Once the SSL connection is established, the agent is able to communicate with the iLO card through an XML stream.

fence_ilo accepts options on the command line as well as from stdin. Fenced sends parameters through stdin when it execs the agent. fence_ilo can be run by itself with command line options. This is useful for testing and for turning outlets on or off from scripts.

Vendor URL: http://www.hp.com

PARAMETERS

-u, --ipport=[port]

TCP/UDP port to use for connection with device (Default Value: 443)

-t, --notls

Disable TLS negotiation, force SSL 3.0

-r, --ribcl-version=[version]

Force ribcl version to use

--ssl-secure

SSL connection with verifying fence device’s certificate

-o, --action=[action]

Fencing Action (Default Value: reboot)

-6, --inet6-only

Forces agent to use IPv6 addresses only

-a, --ip=[ip]

IP Address or Hostname

-n, --plug=[ip]

IP address or hostname of fencing device (together with --port-as-ip)

-4, --inet4-only

Forces agent to use IPv4 addresses only

-S, --password-script=[script]

Script to retrieve password

--tls1.0

Disable TLS negotiaton, force TLS 1.0

-p, --password=[password]

Login password or passphrase

-z, --ssl

SSL connection (Default Value: 1)

--ssl-insecure

SSL connection without verifying fence device’s certificate

-l, --username=[name]

Login Name This parameter is always required.

-v, --verbose

Verbose mode

-D, --debug-file=[debugfile]

Write debug information to given file

-V, --version

Display version information and exit

-h, --help

Display help and exit

--power-wait=[seconds]

Wait X seconds after issuing ON/OFF (Default Value: 0)

--login-timeout=[seconds]

Wait X seconds for cmd prompt after login (Default Value: 10)

--power-timeout=[seconds]

Test X seconds for status change after ON/OFF (Default Value: 20)

--delay=[seconds]

Wait X seconds before fencing is started (Default Value: 0)

--shell-timeout=[seconds]

Wait X seconds for cmd prompt after issuing command (Default Value: 3)

--port-as-ip

Make "port/plug" to be an alias to IP address

--retry-on=[attempts]

Count of attempts to retry power on (Default Value: 3)

ACTIONS

on

Power on machine.

off

Power off machine.

reboot

Reboot machine.

status

This returns the status of the plug/virtual machine.

list

List available plugs with aliases/virtual machines if there is support for more then one device. Returns N/A otherwise.

list-status

List available plugs with aliases/virtual machines and their power state if it can be obtained without additional commands.

monitor

Check the health of fence device

metadata

Display the XML metadata describing this resource.
