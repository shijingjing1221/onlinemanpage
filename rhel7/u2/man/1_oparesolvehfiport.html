NAME

oparesolvehfiport

(Host) Permits the Intel(R) Omni-Path Fabric Host Software style Intel(R) Omni-Path Host Fabric Interface ( HFI ) number and port number arguments to be converted to an Host Software style HFI name and physical port number. This can be useful when writing scripts that can accept FastFabric -style arguments, and interact directly with OFED commands.

Syntax

oparesolvehfiport hfi port

Options

hfi

HFI to send by , default is first HFI

port

Port to send by, default is first active port[START-DRAFT-COMMENT][gdr] Does there need to be an Examples section?[END-DRAFT-COMMENT]
