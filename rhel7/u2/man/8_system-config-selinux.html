NAME

system−config−selinux − SELinux Management tool

SYNOPSIS

system-config-selinux

DESCRIPTION

system-config-selinux provides a graphical interface for managing the SELinux configuration.

OPTIONS

None

FILES

/usr/bin/system-config-selinux

Examples

To run the program type:

system-config-selinux

SEE ALSO

selinux(1), semanage(8)
