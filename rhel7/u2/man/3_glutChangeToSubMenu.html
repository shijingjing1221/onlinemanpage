NAME

glutChangeToSubMenu — Replace a menu entry with a submenu.

LIBRARY

OpenGLUT - menus

SYNOPSIS

#include <openglut.h>

void

glutChangeToSubMenu(int item, const char *label, int subMenuID);

PARAMETERS

item Integer position down the list

label Submenu item text

subMenuID Submenu identifier

DESCRIPTION

Walks the list of the menu items and replaces the numbered item in the list with the given submenu.
