NAME

ata_xfer_mode2shift − Find matching xfer_shift for XFER_*

SYNOPSIS

int ata_xfer_mode2shift(unsigned long xfer_mode);

ARGUMENTS

xfer_mode

XFER_* of interest

DESCRIPTION

Return matching xfer_shift for xfer_mode.

LOCKING

None.

RETURNS

Matching xfer_shift, −1 if no match found.
