NAME

/etc/odbc.ini, $HOME/.odbc.ini - unixODBC data sources configuration

DESCRIPTION

/etc/odbc.ini is text configuration file for the system wide ODBC data sources (i. e. database connections). $HOME/.odbc.ini contains the configuration for user-specific data sources.

SEE ALSO

odbcinst(1), isql(1), odbcinst.ini(5)
