NAME

__skb_frag_set_page − sets the page contained in a paged fragment

SYNOPSIS

void __skb_frag_set_page(skb_frag_t * frag, struct page * page);

ARGUMENTS

frag

the paged fragment

page

the page to set

DESCRIPTION

Sets the fragment frag to contain page.
