NAME

system−config−keyboard − configure the keyboard layout of the system

SYNOPSIS

system−config−keyboard [−−help] [−−noui] [−−text] [<keyboardtype>]

DESCRIPTION

system−config−keyboard is a graphical user interface that allows the user to change the default keyboard of the system.

−−help

Display a short help message and list of keyboard types.

−−noui

Run in command line mode.

−−text

Run in text interface mode.

<keyboardtype> Run with −−help to get a list of supported

keyboard types.
