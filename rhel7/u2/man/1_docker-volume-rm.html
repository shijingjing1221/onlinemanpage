NAME

docker−volume−rm − Remove a volume

SYNOPSIS

docker volume rm [−−help] VOLUME [VOLUME...]

DESCRIPTION

Removes one or more volumes. You cannot remove a volume that is in use by a container.

$ docker volume rm hello
hello

OPTIONS

−−help
Print usage statement
