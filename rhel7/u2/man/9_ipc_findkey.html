NAME

ipc_findkey − find a key in an ipc identifier set

SYNOPSIS

struct kern_ipc_perm * ipc_findkey(struct ipc_ids * ids, key_t key);

ARGUMENTS

ids

Identifier set

key

The key to find

Requires ipc_ids.rwsem locked. Returns the LOCKED pointer to the ipc structure if found or NULL if not. If key is found ipc points to the owning ipc structure
