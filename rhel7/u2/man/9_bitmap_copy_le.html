NAME

bitmap_copy_le − copy a bitmap, putting the bits into little−endian order.

SYNOPSIS

void bitmap_copy_le(void * dst, const unsigned long * src, int nbits);

ARGUMENTS

dst

destination buffer

src

bitmap to copy

nbits

number of bits in the bitmap

DESCRIPTION

Require nbits % BITS_PER_LONG == 0.
