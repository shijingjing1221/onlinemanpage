NAME

cinnamon-screensaver − screen saver and locker

SYNOPSIS

cinnamon-screensaver [OPTIONS...]

DESCRIPTION

Cinnamon Screensaver is the default screen saver and locker in a GNOME desktop. It is designed with simplicity and security in mind.

OPTIONS

−−no−daemon

Don’t become a daemon

−−debug

Enable debugging code

This program also accepts the standard GTK options.

AUTHORS
