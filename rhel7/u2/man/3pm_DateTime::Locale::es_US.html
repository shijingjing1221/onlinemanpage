NAME

DateTime::Locale::es_US

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'es_US' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Spanish United States.
