NAME

DateTime::Locale::ha_NE

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'ha_NE' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Hausa Niger.
