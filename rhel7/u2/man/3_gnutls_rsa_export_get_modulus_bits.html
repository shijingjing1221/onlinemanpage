NAME

gnutls_rsa_export_get_modulus_bits − API function

SYNOPSIS

#include <gnutls/compat.h>

int gnutls_rsa_export_get_modulus_bits(gnutls_session_t session);

ARGUMENTS

gnutls_session_t session

is a gnutls session

DESCRIPTION

Get the export RSA parameter’s modulus size.

RETURNS

The bits used in the last RSA−EXPORT key exchange with the peer, or a negative error code in case of error.
