NAME

oddjob-mkhomedir.conf - system message bus configuration for oddjob-mkhomedir

DESCRIPTION

The /etc/dbus-1/system.d/oddjob-mkhomedir.conf configuration file specifies D-Bus-level access controls for the com.redhat.oddjob_mkhomedir service, in particular which users can invoke methods provided by the service. For more information, please see the D-Bus documentation.
