NAME

getcert

SYNOPSIS

getcert modify-ca [options]

DESCRIPTION

Modifies the helper command in a certmonger CA configuration.

OPTIONS

−c NAME

The nickname of the CA configuration to modify.

−e COMMAND

The new helper command to run for communicating with the CA.

−v

Be verbose about errors. Normally, the details of an error received from the daemon will be suppressed if the client can make a diagnostic suggestion.

BUGS

Please file tickets for any that you find at https://fedorahosted.org/certmonger/
