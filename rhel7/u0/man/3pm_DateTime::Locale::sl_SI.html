NAME

DateTime::Locale::sl_SI

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'sl_SI' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Slovenian Slovenia.

This locale inherits from the DateTime::Locale::sl locale.
