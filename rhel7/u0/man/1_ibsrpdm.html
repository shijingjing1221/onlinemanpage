NAME

ibsrpdm − Discover SRP targets on an InfiniBand Fabric

SYNOPSIS

ibsrpdm [OPTION]

DESCRIPTION

List InfiniBand SCSI RDMA Protocol (SRP) targets on an IB fabric.

OPTIONS

−c

Generate output suitable for piping directly to a /sys/class/infiniband_srp/srp−<device>−<port>/add_target file

−v

Print more verbose output

−d DEVICE

Use device file DEVICE (default /dev/infiniband/umad0)

SEE ALSO

srp_daemon(1)
