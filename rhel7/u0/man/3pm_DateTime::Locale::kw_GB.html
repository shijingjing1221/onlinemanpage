NAME

DateTime::Locale::kw_GB

SYNOPSIS 

  use DateTime;
  my $dt = DateTime−>now( locale => 'kw_GB' );
  print $dt−>month_name();

DESCRIPTION

This is the DateTime locale package for Cornish United Kingdom.
