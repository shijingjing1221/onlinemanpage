NAME

libquvi−scripts − Overview of query media library scripts

DESCRIPTION

libquvi-scripts contains the embedded lua scripts that libquvi uses for parsing the media details. Some additional utility scripts are also included.

<http://quvi.sourceforge.net/doc/>
