NAME

Net::DNS::RR::MB − DNS MB resource record

SYNOPSIS

"use Net::DNS::RR";

DESCRIPTION

Class for DNS Mailbox ( MB ) resource records.

METHODS

madname

    print "madname = ", $rr−>madname, "\n";

Returns the domain name of the host which has the specified mailbox.
