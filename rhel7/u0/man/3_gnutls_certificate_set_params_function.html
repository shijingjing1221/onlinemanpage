NAME

gnutls_certificate_set_params_function − API function

SYNOPSIS

#include <gnutls/gnutls.h>

void gnutls_certificate_set_params_function(gnutls_certificate_credentials_t res, gnutls_params_function * func);

ARGUMENTS

gnutls_certificate_credentials_t res

is a gnutls_certificate_credentials_t structure

gnutls_params_function * func

is the function to be called

DESCRIPTION

This function will set a callback in order for the server to get the Diffie−Hellman or RSA parameters for certificate authentication. The callback should return GNUTLS_E_SUCCESS (0) on success.
