NAME

sputoppm - convert an Atari uncompressed Spectrum file to a PPM image

SYNOPSIS

sputoppm

[spufile]

DESCRIPTION

This program is part of Netpbm(1)

sputoppm reads an Atari uncompressed Spectrum file as input and produces a PPM image as output.
