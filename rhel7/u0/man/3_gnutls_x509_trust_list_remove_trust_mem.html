NAME

gnutls_x509_trust_list_remove_trust_mem − API function

SYNOPSIS

#include <gnutls/x509.h>

int gnutls_x509_trust_list_remove_trust_mem(gnutls_x509_trust_list_t list, const gnutls_datum_t * cas, gnutls_x509_crt_fmt_t type);

ARGUMENTS

gnutls_x509_trust_list_t list

The structure of the list

const gnutls_datum_t * cas

A buffer containing a list of CAs (optional)

gnutls_x509_crt_fmt_t type

The format of the certificates

DESCRIPTION

This function will add the given certificate authorities to the trusted list.

RETURNS

The number of added elements is returned.

SINCE

3.1.10
