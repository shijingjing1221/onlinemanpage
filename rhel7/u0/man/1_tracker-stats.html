NAME

tracker-stats − Provides statistics on the data available for querying

SYNOPSIS

tracker-stats [OPTION...]

DESCRIPTION

tracker-stats takes no arguments and returns statistics based on the ontology classes outlined in the Nepomuk ontology specification. These classes can be used to further query more information using SparQL with tracker-sparql.

OPTIONS

−?, −−help

Show a brief help message.

−V, −−version

Print version.
