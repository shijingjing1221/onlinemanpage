NAME

grub-macbless — Mac-style bless on HFS or HFS+

SYNOPSIS

grub-macbless [-v | --verbose] [-p | --ppc] FILE | [-x | --x86] FILE

OPTIONS

--x86

Bless for x86 based Macs.

--ppc

Bless for PPC based Macs.

--verbose

Print verbose messages.
